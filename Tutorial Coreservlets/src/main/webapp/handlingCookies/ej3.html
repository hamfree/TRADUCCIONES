Soluciones Ejercicios 7

Solución Enunciado 3 - Página Inicial

3. Haga una pequeña página que muestre alguna información simple de su elección. Haga otra página que permita a los usuarios elegir el color de primer plano y fondo que ellos quieran usar en la primera página. Por ejemplo, si un usuario nunca visita la página de elección de colores, la página principal informante usaría colores por defecto de tu elección. Pero si un usuario visita la página de elección de color y elije un amarillo para el fondo y un rojo para el primer plano, todas las visitas subsiguientes a la página principal por el usuario resultarán en rojo sobre amarillo. No hay necesidad de revisar los colores; puede aceptar cualquier valor de color que el usuario le de.
Si está un poco oxidado con el HTML, puede fijar los colores como sigue: 

<BODY BGCOLOR="colorName" TEXT="colorName">
o 
<BODY BGCOLOR="#RRGGBB" TEXT="#RRGGBB">

(Donde R,G y B son valores hexadecimales para los componentes rojo, verde y azul. P.e. #FF00FF es magenta -- 255 para el rojo, 0 para el verde y 255 para el azul).
