Ejercicios

  1. 1. Haga una página JSP que aleatoriamente seleccione un color de fondo para cada petición. Sólo tienes que elegir al azar entre un pequeño conjunto de predefinido colores. Asegúrese de no utilizar la hoja de estilos CSS, ya que sobreescribirá los colores dados por <body bgcolor="...">. Nota: En Eclipse, todo el código JAVA va en src/subdirectorio-como-nombrePaquete, independientemente de que el código Java sea un servlet o algún otro tipo de clase Java. Así, si usted quiere un clase llamada UtilesColor con un método estático llamado colorAleatorio y la tiene en un paquete llamado utiles, usted podrá escribir algo como esto: 
    		package utiles;
		
		public class UtilesColor {
		    public static String colorAleatorio() {
		        if (Math.random() < 0.5) {
		            return("BLUE");
		        } else {
		            return("RED");
		        }
		    }
		}

    En el JSP, usted puede llamar al método con

     
		<%= utiles.UtilesColor.colorAleatorio() %>

    También, si usted quiere probar algún código Java separadamente desde una página JSP, no que en Eclipse, si su clase tiene un método "main", usted puede hacer click derecho en cualquier lugar del fichero, elegir "Run As - Java Application", y Eclipse ejecutará el main y pondrá la salida en la consola abajo.

  2. Haga una página JSP que permita al usuario suminitrar un parámetro de petición indicando el color de fondo. si no se suministra ningún parámetro, se seleccionará un color de fondo aleatorio.
  3. Haga una página JSP que permita al usuario suministrar un parámetro de petición indicando el color de fondo. Si no se suministró ningún parámetro, el color de fondo más reciemente usado (de una petición previa de cualquier usuario) debería ser usado.
