10 - Ejercicios-Invoking-Java-Code-with-JSP-Scripting

Solución Enunciado 1

1. Haga una página JSP que aleatoriamente seleccione un color de fondo para cada petición. Sólo tienes que elegir al azar entre un pequeño conjunto de colores predefinidos. Asegúrese de no utilizar la hoja de estilos CSS, ya que sobreescribirá los colores dados por <body bgcolor="...">. Nota: En Eclipse, todo el código JAVA va en src/subdirectorio-como-nombrePaquete, independientemente de que el código Java sea un servlet o algún otro tipo de clase Java. Así, si usted quiere un clase llamada UtilesColor con un método estático llamado colorAleatorio y la tiene en un paquete llamado utiles, usted podrá escribir algo como esto:

        package utiles;
        
        public class UtilesColor {
            public static String colorAleatorio() {
                if (Math.random() < 0.5) {
                    return("BLUE");
                } else {
                    return("RED");
                }
            }
        }

En el JSP, usted puede llamar al método con

 
        <%= utiles.UtilesColor.colorAleatorio() %>

También, si usted quiere probar algún código Java separadamente desde una página JSP, observe que en Eclipse, si su clase tiene un método "main", usted puede hacer click derecho en cualquier lugar del fichero, elegir "Run As - Java Application", y Eclipse ejecutará el main y pondrá la salida en la consola abajo.
