Formulario de registro

5. Haga un formulario "registro" que recoja nombre, apellidos y dirección de correo electrónico. Envíe los datos a un servlet que los muestre. Siéntase libre de modificar el formulario HTML ThreeParams y el servlet para llevar a cabo esto. A continuación, modifique el servlet para que utilize un valor por defecto (o de un mensaje de error) si el usuario omite cualquiera de los tres parámetros requeridos.
