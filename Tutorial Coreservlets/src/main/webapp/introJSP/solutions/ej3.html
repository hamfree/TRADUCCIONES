09 - Ejercicios-JSP-Intro

Solución Enunciado 3

3. Si usted quisiera un servlet para construir una página web que muestre una imagen, ¿dónde pondría usted la imagen y cómo se referiría a ella desde el servlet? ¿Como difiere esto de un JSP? (Esta cuestión es mayormente fácil, pero aun así ligeramente más complicada de lo que suena al principio).

Respuesta

La imagen se colocaría bajo el directorio "WebContent", el mismo directorio donde se colocan las páginas HTML, los JSP's y otros recursos como las hojas de estilo. Desde el servlet utilizaría la etiqueta IMG de HTML dentro de un out.println() y como ruta indicaría la ruta a la imagen tomando como raíz el directorio "WebContent" de forma que si la imagen se encontrara, por ejemplo, en:
../09-Ejercicios-Jsp-Intro/WebContent/imagenes/imagen.jpg
la etiqueta en el servlet quedaría como :
<img src="imagenes/imagen.jpg" alt="">

Difiere de un JSP en que el servlet se encuentra en un fichero java, colocado en una ruta del CLASSPATH y el JSP en un fichero de extensión JSP que se encontrará en alguna ruta bajo "WebContent" junto a las páginas HTML estáticas, las imágenes y otros recursos web.
