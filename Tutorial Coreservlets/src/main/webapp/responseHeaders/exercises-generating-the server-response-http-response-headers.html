Ejercicios

  1. Repite el primer problema del conjunto previo de ejercicios (el que redirige a los usuarios aleatoriamente). En vez de usar response.sendRedirect, fija el apropiado código de estado y cabecera de respuesta explícitamente. ¿Obtiene el mismo resultado?
  2. Escriba un servlet que instruya al navegador a reconectar cada cinco segundos. Muestre el tiempo ( importe java.util.* y entonces imprima new Date()) en cada conexión.
  3. Escriba un servlet que genera una hoja Excel con 10 filas y 5 columnas. Muestre un número aleatorio en cada celda. Inténtelo desde Firefox e Internet Explorer. ¿Qué ocurre cuando pulsa el botón "Recargar"?
  4. Escriba un servlet que devuelva una página para los usuarios de Internet Explorer indicando que serán enviados a http://www.microsoft.com después de 10 segundos. Enviélos allí después de ese período. Envíe a los usuarios de Firefox a http://www.mozilla.org después del mismo retardo. Tendrá que leer acerca de la cabecera Refresh en el libro (página 203) para ver la opción de añadir una URL específica a dónde conectarse.
