08-Ejercicios Rastreo Sesión

Solución Enunciado 3

3. Haga un servlet que imprima una lista de las URLs de las páginas que el usuario actual ha utilizado para enlazar a el (dentro de la sesión de navegación actual). Esto es, si un usuario ha seguido enlaces de hipertexto desde tres diferentes páginas al servlet, el servlet mostrará al usuario las URLs de esas tres páginas. Comprueba su servlet realizando un par de páginas web estáticas que enlacen a él. Si se siente inspirado, modifique la aproximación básica desde las notas de forma que almacene entradas repetidas; si el mismo usuario sigue un enlace desde la página 1 al servlet dos veces, el servlet mostrará la URL de la página sólo una vez en la lista. Empiece con mi clase ShowItems, y haga unos pequeños cambios para obtener los datos de una cabecera de la petición (¿cuál?) y cambie de doPost a doGet. Si usted no está familiarizado con las esstructuras de datos relacionadas con listas en Java, vea las notas en la siguiente página referente a la clase ArrayList.
