Modificando la Aplicación Java EE

El servidor GlassFish admite el desarrollo iterativo. Siempre que realice un cambio en una aplicación Java EE, debe volver a implementar la aplicación.

Para Modificar un Archivo Class

Para modificar un archivo de clase en un enterprise bean, cambie el código fuente, vuelva a compilarlo y vuelva a implementar la aplicación. Por ejemplo, para actualizar el tipo de cambio en el método de negocio dollarToYen de la clase ConverterBean, debe seguir estos pasos.

Para modificar ConverterServlet, el procedimiento es el mismo.

  1. Edite ConverterBean.java y guarde el archivo.

  2. Vuelva a compilar el archivo fuente.

    • Para volver a compilar ConverterBean.java en el EID NetBeans, haga clic con el botón derecho en el proyecto converter y seleccione Ejecutar.

      Esto vuelve a compilar el archivo ConverterBean.java, reemplaza el archivo de clase anterior en el directorio de compilación y vuelve a implementar la aplicación en el servidor GlassFish.

    • Vuelva a compilar ConverterBean.java usando Maven.

  3. En una ventana de terminal, vaya al directorio tut-install`/examples/ejb/converter/`.

  4. Introduzca el siguiente comando:

                          
mvn install

    Este comando vuelve a empaquetar e implementar la aplicación.
