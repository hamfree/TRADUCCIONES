Usando La Funcionalidad Ajax con la Tecnología JavaServer Faces

La funcionalidad Ajax se puede agregar a una aplicación JavaServer Faces de una de las siguientes maneras:

  • Agregando el código JavaScript requerido a una aplicación

  • Usando la biblioteca de recursos Ajax incorporada

En versiones anteriores de la plataforma Java EE, las aplicaciones JavaServer Faces proporcionaban la funcionalidad Ajax al agregar el JavaScript necesario a la página web. En la plataforma Java EE, el soporte estándar de Ajax es proporcionado por una biblioteca de recursos JavaScript incorporada.

Con el soporte de esta biblioteca de recursos de JavaScript, los componentes estándar de la interfaz de usuario de JavaServer Faces, como botones, etiquetas o campos de texto, pueden habilitarse para la funcionalidad Ajax. También puede cargar esta biblioteca de recursos y usar sus métodos directamente desde el código de bean administrado. Las siguientes secciones del tutorial describen el uso de la biblioteca de recursos Ajax incorporada.

Además, debido a que el modelo de componentes de la tecnología JavaServer Faces se puede ampliar, se pueden crear componentes personalizados con la funcionalidad Ajax.

Los ejemplos del tutorial incluyen una versión Ajax de la aplicación guessnumber, ajaxguessnumber. Consulte La Aplicación de Ejemplo ajaxguessnumber para obtener más información.
