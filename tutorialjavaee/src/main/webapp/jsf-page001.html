Configurando una Página

Una página web típica de JavaServer Faces incluye los siguientes elementos:

  • Un conjunto de declaraciones de espacio de nombres que declaran las bibliotecas de etiquetas JavaServer Faces

  • Opcionalmente, las etiquetas HTML head (h:head) y body (h:body)

  • Una etiqueta de formulario (h:form) que representa los componentes de entrada del usuario

Para agregar los componentes de JavaServer Faces a su página web, debe proporcionar acceso a la página a las dos bibliotecas de etiquetas estándar: la biblioteca de etiquetas del kit de representación HTML de JavaServer Faces y la biblioteca de etiquetas principal de JavaServer Faces. La biblioteca de etiquetas HTML estándar de JavaServer Faces define etiquetas que representan componentes comunes de la interfaz de usuario HTML. La biblioteca de etiquetas principal de JavaServer Faces define etiquetas que realizan acciones principales y son independientes de un kit de representación en particular.

Para obtener una lista completa de las etiquetas JavaServer Faces Facelets y sus atributos, consulte la documentación de la biblioteca de etiquetas JavaServer Faces Facelets.

Para utilizar cualquiera de las etiquetas de JavaServer Faces, debe incluir las directivas adecuadas en la parte superior de cada página que especifiquen las bibliotecas de etiquetas.

Para las aplicaciones Facelets, las directivas del espacio de nombres XML identifican de forma única el URI de la biblioteca de etiquetas y el prefijo de la etiqueta.

Por ejemplo, cuando crea una página XHTML de Facelets, incluya directivas de espacio de nombres de la siguiente manera: 

              
<html xmlns="http://www.w3.org/1999/xhtml"
      xmlns:h="http://xmlns.jcp.org/jsf/html"
      xmlns:f="http://xmlns.jcp.org/jsf/core">

El URI del espacio de nombres XML identifica la ubicación de la biblioteca de etiquetas y el valor del prefijo se utiliza para distinguir las etiquetas que pertenecen a esa biblioteca de etiquetas específica. También puede utilizar otros prefijos en lugar del h o f estándar. Sin embargo, al incluir la etiqueta en la página, debe utilizar el prefijo que haya elegido para la biblioteca de etiquetas. Por ejemplo, en la siguiente página web se debe hacer referencia a la etiqueta form usando el prefijo h porque la directiva de biblioteca de etiquetas anterior usa el prefijo h para distinguir las etiquetas definidas en la biblioteca de etiquetas HTML: 

              
<h:form ...>

Las secciones Adición de componentes a una página mediante etiquetas de biblioteca de etiquetas HTML y Utilización de etiquetas principales describe cómo utilizar las etiquetas de componentes de la biblioteca de etiquetas HTML estándar de JavaServer Faces y las etiquetas principales de la biblioteca de etiquetas principal de JavaServer Faces.
