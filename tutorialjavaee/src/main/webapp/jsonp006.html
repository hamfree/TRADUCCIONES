La Aplicación de Ejemplo jsonpmodel

Esta sección describe cómo compilar y ejecutar la aplicación de ejemplo jsonpmodel. Este ejemplo es una aplicación web que demuestra cómo crear un modelo de objetos a partir de datos de formulario, cómo analizar datos JSON y cómo escribir datos JSON mediante la API del modelo de objetos.

La aplicación de ejemplo jsonpmodel está en el directorio tut-install`/examples/web/jsonp/jsonpmodel`.

Aquí se tratan los siguientes temas:

  • Componentes de la Aplicación de Ejemplo jsonpmodel

  • Ejecutando la Aplicación de Ejemplo jsonpmodel

Componentes de la Aplicación de Ejemplo jsonpmodel

La aplicación de ejemplo jsonpmodel contiene los siguientes archivos.

  • Tres páginas de JavaServer Faces.

    • La página index.xhtml contiene un formulario para recopilar información.

    • La página modelcreated.xhtml contiene un área de texto que muestra datos JSON.

    • La página parsejson.xhtml contiene una tabla que muestra los elementos del modelo de objetos.

  • El bean administrado ObjectModelBean.java, que es un bean administrado con ámbito de sesión que almacena los datos del formulario y dirige la navegación entre las páginas de Facelets. Este archivo también contiene código que utiliza la IPA del modelo de objetos JSON.

El código utilizado en ObjectModelBean.java para crear un modelo de objetos a partir de los datos del formulario es similar al ejemplo de Creando an Modelo de Objeto desde Código de la Aplicación. El código para escribir la salida JSON del modelo es similar al ejemplo en Escribiendo un Modelo de Objeto a un Flujo. El código para navegar por el árbol del modelo de objetos es similar al ejemplo de Navegando un Modelo de Objeto.

Ejecutando la Aplicación de Ejemplo jsonpmodel

Esta sección describe cómo ejecutar la aplicación de ejemplo jsonpmodel utilizando el EID de NetBeans y desde la línea de comandos.

Aquí se tratan los siguientes temas:

  • Para Ejecutar la Aplicación de Ejemplo jsonpmodel Usando el EID de NetBeans

  • Para Ejecutar la Aplicación de Ejemplo jsonpmodel Usando Maven

Para Ejecutar la Aplicación de Ejemplo jsonpmodel Usando el EID de NetBeans

  1. Asegúrese de que GlassFish Server se haya iniciado (consulte Arrancando y Parando el Servidor GlassFish).

  2. En el menú Archivo, elija Abrir proyecto.

  3. En el cuadro de diálogo Abrir proyecto, vaya a:

                            
tut-install/examples/web/jsonp

  4. Seleccione la carpeta jsonpmodel.

  5. Haga clic en Abrir proyecto.

  6. En la pestaña Proyectos, haga clic con el botón derecho en el proyecto jsonpmodel y seleccione Ejecutar.

    Este comando compila y empaqueta la aplicación en un archivo WAR (jsonpmodel.war) ubicado en el directorio target, lo implementa en el servidor y abre una ventana del navegador web con el siguiente URL: 

                            
http://localhost:8080/jsonpmodel/

  7. Edite los datos en la página y haga clic en Crear un objeto JSON para enviar el formulario. La siguiente página muestra un objeto JSON que contiene los datos del formulario.

  8. Haga clic en Analizar JSON. La página siguiente contiene una tabla que enumera los nodos del árbol del modelo de objetos.

Para Ejecutar la Aplicación de Ejemplo jsonpmodel Usando Maven

  1. Asegúrese de que GlassFish Server se haya iniciado (consulte Arrancando y Parando el Servidor GlassFish).

  2. En una ventana de terminal, vaya a:

                            
tut-install/examples/web/jsonp/jsonpmodel

  3. Ingrese el siguiente comando para implementar la aplicación:

                            
mvn install

  4. Abra una ventana del navegador web e ingrese la siguiente dirección:

                            
http://localhost:8080/jsonpmodel/

  5. Edite los datos en la página y haga clic en Crear un objeto JSON para enviar el formulario. La siguiente página muestra un objeto JSON que contiene los datos del formulario.

  6. Haga clic en Analizar JSON. La página siguiente contiene una tabla que enumera los nodos del árbol del modelo de objetos.
