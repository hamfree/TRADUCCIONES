La Aplicación de Ejemplo mood

La aplicación de ejemplo mood, ubicada en el directorio tut-install`/examples/web/servlet/mood/`, es un ejemplo simple que muestra los estados de ánimo de Duke en diferentes momentos del día. El ejemplo muestra cómo desarrollar una aplicación simple usando las anotaciones @WebServlet, @WebFilter y @WebListener para crear un servlet, un listener y un filtro.

Aquí se tratan los siguientes temas:

  • Componentes de la Aplicación de Ejemplo mood

  • Ejecutando el Ejemplo mood

Componentes de la Aplicación de Ejemplo mood

La aplicación de ejemplo mood consta de tres componentes: mood.web.MoodServlet, mood.web.TimeOfDayFilter y mood.web .SimpleServletListener.

MoodServlet, la capa de presentación de la aplicación, muestra el estado de ánimo de Duke en un gráfico, según la hora del día. La anotación @WebServlet especifica el patrón de URL: 

                
@WebServlet("/report")
public class MoodServlet extends HttpServlet {
    ...

TimeOfDayFilter establece un parámetro de inicialización que indica que Duke está despierto: 

                
@WebFilter(filterName = "TimeOfDayFilter",
urlPatterns = {"/*"},
initParams = {
    @WebInitParam(name = "mood", value = "awake")})
public class TimeOfDayFilter implements Filter {
    ...

El filtro llama al método doFilter, que contiene una instrucción switch que establece el estado de ánimo de Duke en función de la hora actual.

SimpleServletListener registra los cambios en el ciclo de vida del servlet. Las entradas de registro aparecen en el registro del servidor.

Ejecutando el Ejemplo mood

Puede usar el EID de NetBeans o Maven para compilar, empaquetar, implementar y ejecutar el ejemplo de mood.

Aquí se tratan los siguientes temas:

  • Para Ejecutar el Ejemplo mood Usando el EID NetBeans

  • Para Ejecutar el Ejemplo mood Usando Maven

Para Ejecutar el Ejemplo mood Usando el EID NetBeans

  1. Asegúrese de que GlassFish Server se haya iniciado (consulte Arrancando y Parando el Servidor GlassFish).

  2. Desde el menú Archivo, elija Abrir Proyecto.

  3. En el cuadro de diálog Abrir Proyecto, navegue a :

                            
tut-install/examples/web/servlet

  4. Seleccione la carpeta mood.

  5. Pulse Abrir Proyecto.

  6. En la pestaña Proyectos, haga click derecho en el proyecto mood y seleccione Construir.

  7. En un navegador web, introduzca la URL siguiente:

                            
http://localhost:8080/mood/report

    La URL especifica el contexto raíz, seguido por el patrón de URL.

    Una página web aparece con el título "Servlet MoodServelte at /mood", una cadena de texto describiendo el estado de ánimo de Duke, y un gráfico ilustrativo.

Para Ejecutar el Ejemplo mood Usando Maven

  1. Asegúrese de que GlassFish Server se haya iniciado (consulte Arrancando y Parando el Servidor GlassFish).

  2. En una ventana de terminal, vaya a:

                            
tut-install/examples/web/servlet/mood/

  3. Introduzca el comando siguiente para desplegar la aplicación:

                            
mvn install

  4. En un navegador web, introduzca la URL siguiente:

                            
http://localhost:8080/mood/report

    La URL especifica el contexto raíz, seguido por el patrón de URL.

    Una página web aparece con el título "Servlet MoodServelte at /mood", una cadena de texto describiendo el estado de ánimo de Duke, y un gráfico ilustrativo.
