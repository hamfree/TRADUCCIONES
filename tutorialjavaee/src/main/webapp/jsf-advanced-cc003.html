Validando Valores de un Componente Compuesto

JavaServer Faces proporciona las siguientes etiquetas para validar los valores de los componentes de entrada. Estas etiquetas se pueden utilizar con la etiqueta composite:valueHolder o composite:editableValueHolder.

La tabla 14-2 enumera las etiquetas de validación de uso común. Ver Usando los Validadores Estándar para obtener detalles y una lista completa.

Tabla 14-2 Etiquetas de validación

Nombre de la Etiqueta

Descripción

f:validateBean

Delega la validación del valor local a la IPA de Validación de Beans.

f:validateRegex

Utiliza el atributo pattern para validar el componente de envoltura. El patrón completo se compara con el valor String del componente. Si coincide, es válido.

f:validateRequired

Refuerza la presencia de un valor. Tiene el mismo efecto que establecer el elemento required del atributo de un componente compuesto en true.
