Arquetipos Maven de Java EE en el Tutorial

Algunos de los capítulos tienen instrucciones sobre cómo crear una aplicación de ejemplo utilizando arquetipos de Maven. Los arquetipos son plantillas para generar un proyecto Maven en particular. El Tutorial incluye varios arquetipos de Maven para generar proyectos Java EE.

Instalando los Arquetipos del Tutorial

Debe instalar los arquetipos de Maven incluidos en su repositorio local de Maven antes de poder crear nuevos proyectos basados en los arquetipos. Puede instalar los arquetipos utilizando EID de NetBeans o Maven.

Instalando los Arquetipos del Tutorial Usando el EID NetBeans

  1. Desde el menú Archivo, seleccione Abrir Proyecto.

  2. En el cuadro de diálogo Abrir proyecto, navegue hasta:

    
tut-install/examples

  3. Seleccione la carpeta archetypes.

  4. Pulse Abrir Proyecto.

  5. En la pestaña Proyectos, haga clic con el botón derecho en el proyecto arquetipos y seleccione Construir.

Instalando los Arquetipos del Tutorial Usando Maven

  1. En una ventana de terminal, vaya a:

    
tut-install/examples/archetypes/

  2. Ingrese el siguiente comando:

    
mvn install
