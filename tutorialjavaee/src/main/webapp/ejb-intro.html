35 Enterprise Beans

Los Enterprise beans son componentes Java EE que implementan la tecnología Enterprise JavaBeans (EJB). Los Enterprise beans se ejecutan en el contenedor EJB, un entorno de tiempo de ejecución dentro del Servidor GlassFish (ver Tipos de Contenedor). Aunque transparente para el desarrollador de aplicaciones, el contenedor EJB proporciona servicios a nivel de sistema, como transacciones y seguridad, a sus beans empresariales (Enterprise Beans, en inglés). Estos servicios le permiten construir e implementar rápidamente beans empresariales, que forman el núcleo de las aplicaciones Java EE transaccionales.
