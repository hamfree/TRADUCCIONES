Implementación y Ensamblaje de Aplicaciones Java EE

Una aplicación Java EE está empaquetada en una o más unidades estándar para su implementación en cualquier sistema compatible con la plataforma Java EE. Cada unidad contiene

  • Un componente o componentes funcionales, como un bean empresarial, una página web, un servlet o un subprograma.

  • Un descriptor de implementación opcional que describe su contenido.

Una vez que se ha producido una unidad Java EE, está lista para implementarse. La implementación generalmente implica el uso de la herramienta de implementación de una plataforma para especificar información específica de la ubicación, como una lista de usuarios locales que pueden acceder a ella y el nombre de la base de datos local. Una vez implementada en una plataforma local, la aplicación está lista para ejecutarse.
