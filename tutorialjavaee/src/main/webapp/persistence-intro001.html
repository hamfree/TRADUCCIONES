Introducción a la IPA de Persistencia Java

The IPA de Java Persistence provides Java developers with an object/relational mapping facility for managing relational data in Java applications. Java Persistence consists of four areas:

  • The IPA de Java Persistence

  • The query language

  • The Java Persistence Criteria API

  • Object/relational mapping metadata
