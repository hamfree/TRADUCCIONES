Creación y Ejecución de Ejemplos del Servicio de Mensajes de Java

Los ejemplos están en el directorio tut-install/examples/jms/.

Para compilar y ejecutar cada ejemplo:

  1. Utilice el EID de NetBeans o Maven para compilar, empaquetar y, en algunos casos, implementar el ejemplo.

  2. Use el EID de NetBeans, Maven o el comando appclient para ejecutar el cliente de la aplicación, o use el navegador para ejecutar los ejemplos de la aplicación web.

Antes de implementar o ejecutar los ejemplos, debe crear recursos para ellos. Algunos ejemplos tienen un archivo glassfish-resources.xml que se usa para crear recursos para ese ejemplo y otros. Puede usar el comando asadmin para crear los recursos.

Para usar los comandos asadmin y appclient, debe colocar los directorios bin del servidor GlassFish en su ruta de comando, como se describe en Consejos de instalación del SDK.
