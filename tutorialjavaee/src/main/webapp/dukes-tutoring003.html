Interfaz de Administración

La interfaz de administración de la Tutoría de Duke es utilizada por el personal del centro de tutoría para administrar los datos empleados por la interfaz principal: los estudiantes, los tutores de los estudiantes y las direcciones. La interfaz de administración utiliza muchos de los mismos componentes que la interfaz principal. Aquí se describen los componentes adicionales que solo se utilizan en la interfaz de administración.

Aquí se tratan los siguientes temas:

  • Enterprise Beans Utilizados en la Interfaz de Administración

  • Archivos Facelets Utilizados en la Interfaz de Administración

  • Beans CDI Gestionados Utilizados en la Interfaz de Administración

  • Clases Auxiliares Utilizadas en la Interfaz de Administración

Enterprise Beans Utilizados en la Interfaz de Administración

El siguiente enterprise bean, en el paquete dukestutoring.ejb del proyecto dukes-tutoring-war, se utiliza en la interfaz de administración.

  • AdminBean: Un bean de sesión sin estado para toda la lógica de negocio utilizada en la interfaz de administración. Llama a los métodos de seguridad para permitir la invocación de los métodos de negocio solo por parte de usuarios autorizados.

Archivos Facelets Utilizados en la Interfaz de Administración

Los siguientes archivos Facelets, bajo src/main/webapp/, se utilizan en la interfaz de administración:

  • admin/adminTemplate.xhtml: Plantilla para la interfaz de administración

  • admin/index.xhtml: página de inicio para la interfaz de administración

  • login.xhtml: página de inicio de sesión para la interfaz de administración con restricciones de seguridad

  • loginError.xhtml: Página que se muestra si hay errores al autenticar al usuario administrador

  • Directorio admin/address: páginas que le permiten crear, editar y eliminar entidades de Address

  • Directorio admin/guardian: páginas que le permiten crear, editar y eliminar entidades Guardian

  • Directorio admin/student: páginas que le permiten crear, editar y eliminar entidades Student

  • resources/components/formLogin.xhtml: componente compuesto para un formulario de inicio de sesión utilizando la seguridad Java EE

  • WEB-INF/includes/adminNav.xhtml: fragmento XHTML para la barra de navegación de la interfaz de administración

Beans CDI Gestionados Utilizados en la Interfaz de Administración

Los beans gestionados por CDI que se utilizan en la interfaz de administración se encuentran en el paquete dukestutoring.web del proyecto dukes-tutoring-war.

  • StudentBean.java: un bean administrado para las páginas de Facelets utilizadas para crear y editar estudiantes. Los nombres y apellidos tienen anotaciones de Validación de Beans que requieren que se completen los campos. Los números de teléfono tienen anotaciones de Validación de Beans para garantizar que los datos enviados estén bien formados.

  • GuardianBean.java: un bean administrado para las páginas de Facelets que se utiliza para crear tutores y asignar tutores a los estudiantes. Los nombres y apellidos tienen anotaciones de Validación de Beans que requieren que se completen los campos. Los números de teléfono tienen anotaciones de Validación de Beans para garantizar que los datos enviados estén bien formados.

  • AddressBean.java: un bean administrado para las páginas de Facelets que se utiliza para crear direcciones para los estudiantes. Los atributos de calle, ciudad, provincia y código postal tienen anotaciones de Validación de Beans que requieren que se completen los campos, y el atributo de código postal tiene una anotación adicional para garantizar que los datos estén correctamente formados.

Clases Auxiliares Utilizadas en la Interfaz de Administración

Las siguientes clases auxiliares, que se encuentran en el paquete dukestutoring.web.util del proyecto dukes-tutoring-war, se utilizan en la interfaz de administración.

  • EntityConverter: una clase principal para StudentConverter y GuardianConverter que define un caché para almacenar las clases de entidad al convertir las entidades para su uso en la interfaz de usuario de componentes JavaServer Faces el caché ayuda a aumentar el rendimiento. El caché se almacena en el contexto JavaServer Faces.

  • StudentConverter: un conversor JavaServer Faces para la clase de entidad Student. Esta clase contiene métodos para convertir instancias de Student en cadenas y viceversa, para que puedan usarse en los componentes de la interfaz de usuario de la aplicación.

  • GuardianConverter: similar a StudentConverter, esta clase es un convertidor para la clase de entidad Guardian.
