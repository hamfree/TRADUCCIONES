Herramientas del Servidor GlassFish

GlassFish Server es una implementación compatible de la plataforma Java EE. Además de admitir todas las API descritas en las secciones anteriores, GlassFish Server incluye una serie de herramientas Java EE que no forman parte de la plataforma Java EE, pero que se proporcionan para comodidad del desarrollador.

Esta sección resume brevemente las herramientas que componen GlassFish Server. Las instrucciones para iniciar y detener GlassFish Server, iniciar la Consola de administración e iniciar y detener Apache Derby se encuentran en Capítulo 2, "Uso de los ejemplos de tutoriales" .

GlassFish Server contiene las herramientas enumeradas en Tabla 1-1. La información de uso básica de muchas de las herramientas aparece a lo largo del tutorial. Para obtener información detallada, consulte la ayuda en línea de las herramientas de la GUI.

Tabla 1-1 Herramientas de servidor GlassFish

Herramienta

Descripción

Consola de Administración

Una utilidad de administración del servidor GlassFish de GUI basada en la web. Se utiliza para detener GlassFish Server y administrar usuarios, recursos y aplicaciones.

asadmin

Una utilidad de administración del servidor GlassFish de línea de comandos. Se utiliza para iniciar y detener GlassFish Server y para administrar usuarios, recursos y aplicaciones.

appclient

Una herramienta de línea de comandos que inicia el contenedor del cliente de la aplicación e invoca la aplicación del cliente empaquetada en el archivo JAR del cliente de la aplicación.

capture-schema

Una herramienta de línea de comandos para extraer información de esquema de una base de datos, produciendo un archivo de esquema que GlassFish Server puede usar para la persistencia administrada por contenedor.

package-appclient

Una herramienta de línea de comandos para empaquetar las bibliotecas de contenedores del cliente de la aplicación y los archivos JAR.

Apache Derby

Una copia de la base de datos de Apache Derby.

xjc

Una herramienta de línea de comandos para transformar o vincular un esquema XML de origen a un conjunto de clases de contenido JAXB en el lenguaje de programación Java.

schemagen

Una herramienta de línea de comandos para crear un archivo de esquema para cada espacio de nombres al que se hace referencia en sus clases de Java.

wsimport

Una herramienta de línea de comandos para generar artefactos portátiles JAX-WS para un archivo WSDL determinado. Después de la generación, estos artefactos pueden empaquetarse en un archivo WAR con el WSDL y los documentos de esquema, junto con la implementación del punto final, y luego implementarse.

wsgen

Una herramienta de línea de comandos para leer una clase de punto final de servicio web y generar todos los artefactos portátiles JAX-WS necesarios para la implementación e invocación del servicio web.
