11 Usando Convertidores, Oyentes, y Validadores

El capítulo anterior describió los componentes y explicó cómo agregarlos a una página web. Este capítulo proporciona información sobre cómo agregar más funcionalidad a los componentes a través de convertidores, escuchas y validadores.

  • Los convertidores se utilizan para convertir los datos que se reciben de los componentes de entrada. Los convertidores permiten que una aplicación lleve las características fuertemente tipadas del lenguaje de programación Java al mundo basado en cadenas de la programación de servlets HTTP.

  • Los oyentes se utilizan para escuchar los eventos que suceden en la página y realizar las acciones definidas.

  • Los validadores se utilizan para validar los datos que se reciben de los componentes de entrada. Los validadores permiten que una aplicación exprese restricciones en los datos de entrada del formulario para garantizar que se cumplan los requisitos necesarios antes de que se procesen los datos de entrada.
