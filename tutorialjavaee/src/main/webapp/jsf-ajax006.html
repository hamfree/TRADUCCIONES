Gestionando Errores

JavaServer Faces maneja los errores Ajax mediante el uso del atributo onerror de la etiqueta f:ajax. El valor de este atributo es el nombre de una función de JavaScript.

Cuando hay un error al procesar una solicitud Ajax, JavaServer Faces llama a la función JavaScript onerror definida y le pasa un objeto de datos. El objeto de datos contiene todas las propiedades disponibles para el atributo onevent y, además, las siguientes propiedades:

  • description

  • errorName

  • errorMessage

El type es error. La propiedad status del objeto de datos contiene uno de los valores de error válidos enumerados en la tabla 13-4.

Tabla 13-4 Valores de Error Válidos para la Propiedad status del Objeto de Datos

Valores

Descripción

emptyResponse

No hay respuesta Ajax del servidor.

httpError

Uno de los errores HTTP válidos: request.status==null o request.status==undefined o request.status<200 o request.status>=300.

malformedXML

La respuesta del Ajax no está bien formada.

serverError

La respuesta Ajax contiene un elemento error.

En el siguiente ejemplo, los errores que ocurrieron al procesar la solicitud Ajax son manejados por la función JavaScript handlemyajaxerror: 


<f:ajax event="click" render="errormessage" onerror="handlemyajaxerror"/>
