48 Conceptos del Servicio de Mensajería de Java

Este capítulo proporciona una introducción a la IPA de Java Message Service (JMS), una IPA de Java que permite que las aplicaciones creen, envíen, reciban y lean mensajes mediante una comunicación fiable, asíncrona y débilmente acoplada.

  • Uso de Funciones Avanzadas de JMS

  • Uso de la IPA de JMS en Aplicaciones de Java EE
