Introducción a la Configuración de Aplicaciones JavaServer Faces

El proceso de construcción e implementación de aplicaciones JavaServer Faces simples se describe en capítulos anteriores de este tutorial, incluyendo Capítulo 6, "Introducción a las Aplicaciones Web", Capítulo 8, "Introducción a los Facelets", Capítulo 13, "Usando Ajax con la Tecnología JavaServer Faces" y Capítulo 14, "Componentes Compuestos: Temas Avanzados y un Ejemplo" . Sin embargo, cuando crea aplicaciones grandes y complejas, se requieren varias tareas de configuración adicionales. Estas tareas incluyen lo siguiente:

  • Registrar beans administrados con la aplicación para que todas las partes de la aplicación tengan acceso a ellos

  • Configurar beans gestionados y beans de modelo para que se creen instancias con los valores adecuados cuando una página hace referencia a ellos

  • Definir reglas de navegación para cada una de las páginas de la aplicación para que la aplicación tenga un flujo de páginas fluido, si se necesita una navegación no predeterminada.

  • Empaquetar la aplicación para incluir todas las páginas, recursos y otros archivos para que la aplicación se pueda implementar en cualquier contenedor compatible
