Monitorizando Eventos en el Cliente

Para monitorear las solicitudes de Ajax en curso, use el atributo onevent de la etiqueta f:ajax. El valor de este atributo es el nombre de una función de JavaScript. JavaServer Faces llama a la función onevent en cada etapa del procesamiento de una solicitud Ajax: inicio, finalización y éxito.

Al llamar a la función de JavaScript asignada a la propiedad onevent, JavaServer Faces le pasa un objeto de datos. El objeto de datos contiene las propiedades enumeradas en la tabla 13-3.

Tabla 13-3 Propiedades del Objeto de Datos onevent

Propiedad

Descripción

responseXML

La respuesta a la llamada Ajax en formato XML

responseText

La respuesta a la llamada de Ajax en formato de texto.

responseCode

La respuesta a la llamada Ajax en código numérico

source

La fuente del evento Ajax actual: el elemento DOM

status

El estado de la llamada actual de Ajax: begin, complete, o success

type

El tipo de llamada de Ajax: event

Al usar la propiedad status del objeto de datos, puede identificar el estado actual de la solicitud Ajax y monitorear su progreso. En el siguiente ejemplo, monitormyajaxevent es una función de JavaScript que monitorea la solicitud Ajax enviada por el evento: 


<f:ajax event="click" render="statusmessage" onevent="monitormyajaxevent"/>
