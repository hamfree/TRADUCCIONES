10 Usando la Tecnología JavaServer Faces en Páginas Web

Las páginas web (páginas de Facelets, en la mayoría de los casos) representan la capa de presentación para las aplicaciones web. El proceso de crear páginas web para una aplicación de JavaServer Faces incluye el uso de etiquetas de componente para agregar componentes a la página y cablearlos a los beans de respaldo, validadores, oyentes, convertidores, y otros objetos del lado del servidor que están asociados con la página.

Este capítulo explica cómo crear páginas web usando varios tipos de componentes y etiquetas principales. En el siguiente capítulo, aprenderá cómo agregar convertidores, validadores, y oyentes a las etiquetas de componente para proporcionar funcionalidad adicional a los componentes.

Muchos de los ejemplos en este capítulo fueron tomados del Capítulo 60, "Ejemplo de Estudio de Caso de la Librería de Duke."
