Empaquetando Archivos de Adaptadores de Recursos

Un archivo Adaptador de recursos (RAR) almacena archivos XML, clases de Java y otros objetos para aplicaciones de Arquitectura de los Conectores de JavaEE (JCA). Se puede implementar un adaptador de recursos en cualquier servidor JavaEE, al igual que una aplicación JavaEE. Un archivo RAR se puede contener en un archivo de tipo Enterprise (EAR), o puede existir como un archivo separado.

El archivo RAR contiene

  • Un archivo JAR con las clases de implementación del adaptador de recursos.

  • Un directorio META-INF/ opcional que puede almacenar un archivo ra.xml y/o un descriptor de despliegue específico del servidor utilizado para propósitos de configuración

Se puede implementar un archivo RAR en el servidor de aplicaciones como un componente independiente o como parte de una aplicación más grande. En ambos casos, el adaptador está disponible para todas las aplicaciones utilizando un procedimiento de búsqueda.
