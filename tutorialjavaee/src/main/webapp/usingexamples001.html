Software Requerido

Se requiere el siguiente software para ejecutar los ejemplos:

  • Plataforma Java, Edición Estándar

  • Kit del Software de Desarrollo Java EE 8

  • Tutorial de Componentes de Java EE 8

  • EID NetBeans

  • Apache Maven

Plataforma Java, Edición Estándar

Para compilar, implementar y ejecutar los ejemplos, necesita una copia de Java Platform, Standard Edition Development Kit (JDK). Debe utilizar JDK 7 Update 65 o superior o JDK 8 Update 20 o superior. Puede descargar el software JDK desde http://www.oracle.com/technetwork/java/javase/downloads/index.html .

Kit del Software de Desarrollo Java EE 8

GlassFish Server Open Source Edition 5.0 está diseñado como el entorno de compilación y tiempo de ejecución para los ejemplos de tutoriales. Para compilar, implementar y ejecutar los ejemplos, necesita una copia de GlassFish Server y, opcionalmente, NetBeans IDE. Para obtener GlassFish Server, debe instalar el kit de desarrollo de software (SDK) Java EE 8, que puede descargar desde http://www.oracle.com/technetwork/java/javaee/downloads/index.html .

Consejos de Instalación del KDJ

El SDK de Java EE 8 se instala desde un archivo ZIP. Establece el nombre de usuario de administración predeterminado como admin sin contraseña requerida. El puerto de administración se establece en 4848 y el puerto HTTP se establece en 8080.

Este tutorial hace referencia a as-install-parent, el directorio donde instala GlassFish Server. Por ejemplo, el directorio de instalación predeterminado en Microsoft Windows es C:\glassfish5, por lo que as-install-parent es C:\glassfish5. El propio GlassFish Server está instalado en as-install , el directorio glassfish en as-install-parent. Entonces, en Microsoft Windows, as-install es C:\glassfish5\glassfish.

Después de instalar GlassFish Server, agregue los siguientes directorios a su PATH para evitar tener que especificar la ruta completa cuando use comandos: 

                  
as-install-parent/bin
as-install/bin

Tutorial de Componentes de Java EE 8

El componente del tutorial, incluida la documentación y la fuente de ejemplo, se encuentra en Java EE 8 SDK.

EID NetBeans

El entorno de desarrollo integrado (IDE) de NetBeans es un IDE gratuito de código abierto para desarrollar aplicaciones Java, incluidas aplicaciones empresariales. NetBeans IDE es compatible con la plataforma Java EE. Puede crear, empaquetar, implementar y ejecutar los ejemplos de tutoriales desde NetBeans IDE.

Para ejecutar los ejemplos del tutorial, necesita la última versión de NetBeans IDE. Puede descargar NetBeans IDE desde https://netbeans.org/downloads/index.html . Asegúrese de descargar el paquete Java EE.

Para Instalar el EID NetBeans sin el servidor GlassFish

Cuando instale NetBeans IDE, no instale la versión de GlassFish Server que viene con NetBeans IDE. Para omitir la instalación de GlassFish Server, siga estos pasos.

  1. En la primera página del asistente de instalación de NetBeans IDE, anule la selección de la casilla de verificación de GlassFish Server y haga clic en Aceptar.

  2. Acepte tanto el Acuerdo de licencia como el Acuerdo de licencia de Junit.

    Algunos de los ejemplos de tutoriales utilizan la biblioteca Junit, por lo que debería instalarla.

  3. Continúe con la instalación de NetBeans IDE.

Para agregar el Servidor GlassFish como un Servidor Usando el EID NetBeans

Para ejecutar los ejemplos del tutorial en NetBeans IDE, debe agregar su GlassFish Server como servidor en NetBeans IDE. Siga estas instrucciones para agregar GlassFish Server a NetBeans IDE.

  1. En el menú Herramientas, elija Servidores.

  2. En el asistente de servidores, haga clic en Agregar servidor.

  3. En Elegir servidor, seleccione GlassFish Server y haga clic en Siguiente.

  4. En Ubicación del servidor, busque la ubicación del SDK de Java EE 8 y haga clic en Siguiente.

  5. En Ubicación del dominio, seleccione Registrar dominio local.

  6. Haga clic en Finalizar.

Apache Maven

Maven es una herramienta de compilación basada en tecnología Java desarrollada por Apache Software Foundation y se utiliza para compilar, empaquetar e implementar los ejemplos de tutoriales. Para ejecutar los ejemplos del tutorial desde la línea de comandos, necesita Maven 3.0 o superior. Si aún no tiene Maven, puede instalarlo desde:

http://maven.apache.org

Asegúrese de agregar el directorio maven-install`/bin` a su ruta.

Si está utilizando NetBeans IDE para compilar y ejecutar los ejemplos, incluye una copia de Maven.
