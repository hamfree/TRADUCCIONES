Registrando un Componente Personalizado

Además de registrar los representadores personalizados (como se explica en la sección anterior), también debe registrar los componentes personalizados que suelen estar asociados con los representadores personalizados. Utiliza una anotación @FacesComponent, como se describe en Creando Clases de Componentes Personalizados, o el elemento component del archivo de configuración de recursos de la aplicación.

A continuación, se muestra un elemento component hipotético del archivo de recursos de configuración de la aplicación que registra AreaComponent: 


<component>
    <component-type>DemoArea</component-type>
    <component-class>
        dukesbookstore.components.AreaComponent
    </component-class>
    <property>
        <property-name>alt</property-name>
        <property-class>java.lang.String</property-class>
    </property>
    <property>
        <property-name>coords</property-name>
        <property-class>java.lang.String</property-class>
    </property>
    <property>
        <property-name>shape</property-name>
        <property-class>java.lang.String</property-class> 
    </property>
</component>

Los atributos especificados en una etiqueta component anulan cualquier configuración en la anotación @FacesComponent.

El elemento component-type indica el nombre con el que debe registrarse el componente. Otros objetos que se refieren a este componente utilizan este nombre. Por ejemplo, el elemento component-type en la configuración de AreaComponent define un valor de DemoArea, que coincide con el valor devuelto por el método getComponentType de la clase AreaTag.

El elemento component-class indica el nombre de clase completo del componente. Los elementos property especifican las propiedades del componente y sus tipos.

Si el componente personalizado puede incluir facetas, puede configurar las facetas en la configuración del componente utilizando elementos facet, que están permitidos después de los elementos component-class. Ver Registrando un Renderizador Personalizado con un Kit de Renderizado para obtener más detalles sobre la configuración de facetas.
