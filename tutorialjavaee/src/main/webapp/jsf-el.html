9 Lenguaje de Expresión (EL)

Este capítulo presenta el Lenguaje de Expresión (también conocido como EL), que proporciona un mecanismo importante para permitir que la capa de presentación (páginas web) se comuniquen con la lógica de la aplicación (beans gestionados). El EL es utilizado por varias tecnologías JavaEE, como la tecnología JavaServer Faces, la tecnología JavaServer Pages (JSP) y Contextos e Inyección de Dependencias para Java EE (CDI). El EL también se puede utilizar en entornos independientes. Este capítulo solo cubre el uso de EL en contenedores Java EE.
