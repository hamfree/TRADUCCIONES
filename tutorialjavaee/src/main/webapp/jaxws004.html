Interoperabilidad de los Servicios Web y JAX-WS

JAX-WS admite la versión 1.1 del Perfil Básico de Interoperabilidad de Servicios Web (WS-I). El Perfil Básico de WS-I es un documento que aclara las especificaciones de SOAP 1.1 y WSDL 1.1 para promover la interoperabilidad de SOAP. Para obtener enlaces relacionados con WS-I, consulte Más información sobre JAX-WS.

Para admitir la versión 1.1 del Perfil Básico de WS-I, el tiempo de ejecución de JAX-WS admite las codificaciones doc/literal y rpc/literal para servicios, puertos estáticos, proxies dinámicos y la Interfaz de Invocación Dinámica (DII).
