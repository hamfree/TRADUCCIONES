27 Contextos e Inyección de Dependencias para Java EE: Temas Avanzados

This chapter describes more advanced features of Contexts and Dependency Injection for Java EE (CDI). Specifically, it covers additional features CDI provides to enable loose coupling of components with strong typing, in addition to those described in Descripción General de CDI.

  • Usando Beans Predefinidos en Aplicaciones CDI

  • Usando Eventos en Aplicaciones CDI

  • Usando Interceptores en Aplicaciones CDI

  • Usando Decoradores en Aplicaciones CDI

  • Usando Estereotipos en Aplicaciones CDI

  • Uso de los Literales de Anotaciones Incorporados

  • Usando las Interfaces de los Configuradores
