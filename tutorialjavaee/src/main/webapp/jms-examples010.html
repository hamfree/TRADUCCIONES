Usando el EID NetBeans para Crear Recursos JMS

Cuando escriba sus propias aplicaciones JMS, deberá crear recursos para ellas. Esta sección explica cómo utilizar el EID de NetBeans para crear archivos src/main/setup/glassfish-resources.xml similares a los que se utilizan en los ejemplos de este capítulo. También explica cómo usar el EID de NetBeans para eliminar los recursos.

También puede crear, enumerar y eliminar recursos JMS mediante la Consola de administración o con los comandos asadmin create-jms-resource, asadmin list-jms-resources y asadmin delete-jms-resources. Para obtener más información, consulte la documentación del servidor GlassFish o ingrese asadmin help nombre-comando.

Aquí se tratan los siguientes temas:

  • Para Crear Recursos JMS Usando el EID NetBeans

  • Para Eliminar Recursos JMS Usando el EID NetBeans

Para Crear Recursos JMS Usando el EID NetBeans

Siga estos pasos para crear un recurso JMS en el servidor GlassFish usando el EID de NetBeans. Repita estos pasos para cada recurso que necesite.

  1. Haga clic con el botón derecho en el proyecto para el que desea crear recursos y seleccione Nuevo, luego seleccione Otro.

  2. En el asistente Nuevo archivo, en Categorías, seleccione GlassFish.

  3. En Tipos de archivo, seleccione Recurso JMS.

  4. En la página Atributos generales - Recurso JMS, en el campo Nombre JNDI, introduzca el nombre del recurso.

    Por convención, los nombres de recursos JMS comienzan con jms/.

  5. Seleccione la opción para el tipo de recurso.

    Normalmente, es javax.jms.Queue, javax.jms.Topic o javax.jms.ConnectionFactory.

  6. Haga clic en Siguiente.

  7. En la página Propiedades de JMS, para una cola o un tema, introduzca un nombre para una cola física en el campo Valor de la propiedad Nombre.

    Puede introducir cualquier valor para este campo obligatorio.

    Las fábricas de conexiones no tienen propiedades requeridas. En algunas situaciones, es posible que deba especificar una propiedad.

  8. Haga clic en Finalizar.

    Se crea un archivo llamado glassfish-resources.xml en su proyecto Maven, en un directorio llamado src/main/setup/. En la pestaña Proyectos, puede encontrarlo en el nodo Otras fuentes. Deberá ejecutar el comando asadmin add-resources para crear los recursos en el servidor GlassFish.

Para Eliminar Recursos JMS Usando el EID NetBeans

  1. En la pestaña Servicios, expanda el nodo Servidores, luego expanda el nodo Servidor GlassFish.

  2. Expanda el nodo Recursos, luego expanda el nodo Recursos del conector.

  3. Expanda el nodo Recursos de objetos de administración.

  4. Haga clic con el botón derecho en cualquier destino que desee eliminar y seleccione Anular registro.

  5. Expanda el nodo Grupos de conexiones de conectores.

  6. Haga clic con el botón derecho en el grupo de conexiones que corresponde a la fábrica de conexiones que eliminó y seleccione Anular registro.

    Cuando elimina un grupo de conexiones de conector, el recurso de conector asociado también se elimina. Esta acción elimina la fábrica de conexiones.
