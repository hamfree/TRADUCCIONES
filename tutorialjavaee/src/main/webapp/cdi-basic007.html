Inyectando Beans

Para usar los beans que crea, los inyecta en otro bean que luego puede usar una aplicación, como una aplicación JavaServer Faces. Por ejemplo, podría crear un bean llamado Printer en el que inyectaría uno de los beans Greeting: 

              
import javax.inject.Inject;
public class Printer {
    @Inject Greeting greeting;
    ...
}

Este código inyecta la implementación @Default Greeting en el bean. El siguiente código inyecta la implementación @Informal: 

              
import javax.inject.Inject;
public class Printer {
    @Inject @Informal Greeting greeting;
    ...
}

Se necesita más para la imagen completa de este bean. Su uso del alcance necesita ser entendido. Además, para una aplicación JavaServer Faces, el bean debe ser accesible a través de EL.

Ahora que puede identificar el objetivo de la inyección, es importante comprender qué se puede inyectar y en qué contexto. JSF 2.3 proporciona productores que permiten inyectar los artefactos JSF más importantes. Para obtener información detallada, consulte el paquete javadoc para javax.faces.annotation.
