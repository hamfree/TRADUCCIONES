36 Introducción a los Enterprise Beans

Este capítulo muestra cómo desarrollar, implementar y ejecutar una aplicación Java EE simple llamada converter que utiliza un EJB para su lógica empresarial. El propósito de converter es calcular las conversiones de moneda entre el yen japonés, el euro y el dólar estadounidense. La aplicación converter consta de un bean empresarial, que realiza los cálculos, y un cliente web.

Aquí se tratan los siguientes temas:

  • Empezando Con los Enterprise Beans

  • Creando el Enterprise Bean

  • Modificando la Aplicación Java EE
