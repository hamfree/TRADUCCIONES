Agrupación de Componentes

Las secciones anteriores describen cómo asociar un solo componente de la interfaz de usuario con la funcionalidad Ajax. También puede asociar Ajax con más de un componente a la vez agrupándolos en una página. El siguiente ejemplo muestra cómo se pueden agrupar varios componentes utilizando la etiqueta f:ajax: 


<f:ajax>
    <h:form>
        <h:inputText id="input1" value="#{user.name}"/>
        <h:commandButton id="Submit"/>
    </h:form>
</f:ajax>

En el ejemplo, ninguno de los componentes está asociado con ningún atributo Ajax event o render todavía. Por lo tanto, no se llevará a cabo ninguna acción en caso de entrada del usuario. Puede asociar los componentes anteriores con un event y un atributo render de la siguiente manera: 


<f:ajax event="click" render="@all">
    <h:form>
        <h:inputText id="input1" value="#{user.name}"/>
        <h:commandButton id="Submit"/>
    </h:form>
</f:ajax>

En el ejemplo actualizado, cuando el usuario hace clic en cualquiera de los componentes, los resultados actualizados se mostrarán para todos los componentes. Puede ajustar aún más la acción de Ajax agregando eventos específicos a cada uno de los componentes, en cuyo caso la funcionalidad de Ajax se vuelve acumulativa. Considere el siguiente ejemplo: 


<f:ajax event="click" render="@all">
    ...
    <h:commandButton id="Submit">
        <f:ajax event="mouseover"/>
    </h:commandButton>
    ...
</f:ajax>

Ahora el componente de botón disparará una acción Ajax en caso de un evento mouseover así como un evento de clic del mouse.
