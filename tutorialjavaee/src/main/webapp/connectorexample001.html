Descripción General de los Ejemplos de Adaptadores de Recursos

El ejemplo de trading muestra cómo usar una interfaz de cliente personalizada simple para conectarse a un EIS desde una aplicación web. El adaptador de recursos de este ejemplo implementa el contrato de salida y la interfaz de cliente personalizada. El ejemplo de traffic muestra cómo usar un bean controlado por mensajes (MDB) para procesar actualizaciones de información de tráfico desde un EIS. El adaptador de recursos de este ejemplo implementa los contratos de gestión de trabajo y de entrada.
