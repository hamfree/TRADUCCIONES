Uso de Etiquetas Principales

Las etiquetas incluidas en la biblioteca de etiquetas principal de JavaServer Faces se utilizan para realizar acciones principales que no son realizadas por etiquetas HTML.

La tabla 10-8 enumera las etiquetas principales de manejo de eventos.

Tabla 10-8 Etiquetas principales de manejo de eventos

Etiqueta

Función

f:actionListener

Agrega un detector de acciones a un componente principal

f:phaseListener

Agrega un PhaseListener a una página

f:setPropertyActionListener

Registra un oyente de acción especial cuyo único propósito es insertar un valor en un bean gestionado cuando se envía un formulario

f:valueChangeListener

Agrega un oyente de cambio de valor a un componente principal

Tabla 10-9 Etiquetas principales de conversión de datos

Etiqueta

Función

f:converter

Agrega un convertidor arbitrario al componente principal

f:convertDateTime

Agrega una instancia de DateTimeConverter al componente principal

f:convertNumber

Agrega una instancia de NumberConverter al componente principal

Tabla 10-10 Etiquetas principales de facetas

Función

f:metadata

La table 10-13 enumera las etiquetas principales que pertenecen a otras categorías.

Tabla 10-13 Etiquetas de núcleo diversas

Categoría de Etiqueta

Etiqueta

Función

Configuración de atributos

f:attribute

Agrega atributos configurables a un componente principal

Localización

f:loadBundle

Especifica un ResourceBundle que se expone como un Map

Sustitución de parámetros

f:param

Sustituye parámetros en una instancia de MessageFormat y agrega pares de nombre-valor de cadena de consulta a una URL

Ajax

f:ajax

Asocia una acción Ajax con un solo componente o un grupo de componentes según la ubicación

Evento

f:event

Permite instalar un ComponentSystemEventListener en un componente

WebSocket

f:websocket

Permite que las comunicaciones del lado del servidor se envíen a todas las instancias de un socket que contenga el mismo nombre de canal.
