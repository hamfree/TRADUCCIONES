Descripción General de los Ejemplos de JMS

Las siguientes tablas enumeran los ejemplos utilizados en este capítulo, describen lo que hacen y vinculan a la sección que los describe completamente. El directorio de ejemplo para cada ejemplo es relativo al directorio tut-install/examples/jms/.

Tabla 49-1 Ejemplos de JMS que Muestran el Uso de Aplicaciones Clientes de Java EE

Directorio de Ejemplo

Descripción

simple/producer

Usar una aplicación cliente para enviar mensajes; ver Enviando mensajes

simple/synchconsumer

Usar una aplicación cliente para recibir mensajes sincrónicamente; ver Recepción de Mensajes Sincrónicamente

simple/asynchconsumer

Usar una aplicación cliente para recibir mensajes de forma asíncrona; ver Uso de un Detector de Mensajes para la Entrega Asincrónica de Mensajes

simple/messagebrowser

Usar una aplicación cliente para usar un QueueBrowser para navegar por una cola; ver Exploración de Mensajes en una Cola

simple/clientackconsumer

Uso de un cliente de aplicación para confirmar los mensajes recibidos de forma síncrona; ver Confirmación de Mensajes

durablesubscriptionexample

Usar una aplicación cliente para crear una suscripción duradera sobre un tema; ver Uso de Suscripciones Duraderas

transactedexample

Usar una aplicación cliente para enviar y recibir mensajes en transacciones locales (también usa mensajes de solicitud y respuesta); ver Uso de Transacciones Locales

shared/sharedconsumer

Uso de un cliente de aplicación para crear suscripciones de temas no duraderos compartidos; ver Uso de Suscripciones no Duraderas Compartidas

shared/shareddurableconsumer

Uso de un cliente de aplicación para crear suscripciones de temas duraderos compartidos; ver Uso de Suscripciones Duraderas Compartidas

Tabla 49-2 Ejemplos de JMS que Muestran el Uso de Componentes Java EE Web y EJB

Directorio de Ejemplo

Descripción

websimplemessage

Uso de beans administrados para enviar mensajes y recibir mensajes sincrónicamente; ver Enviar y Recibir Mensajes Usando una Aplicación Web Simple

simplemessage

Usar un cliente de aplicación para enviar mensajes y usar un bean controlado por mensajes para recibir mensajes de forma asíncrona; ver Recepción de Mensajes de Forma Asíncrona Mediante un Bean Controlado por Mensajes

clientsessionmdb

Usar un bean de sesión para enviar mensajes y usar un bean controlado por mensajes para recibir mensajes; ver Envío de Mensajes Desde un Bean de Sesión a un MDB

clientmdbentity

Uso de un cliente de aplicación, dos beans controlados por mensajes y persistencia de JPA para crear una aplicación de recursos humanos simple; ver Uso de una Entidad para Unir Mensajes de Dos MDB
