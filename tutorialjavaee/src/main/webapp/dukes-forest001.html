Descripción General del Caso de Estudio del Ejemplo del Bosque de Duke

El Bosque de Duke es una aplicación de comercio electrónico simple que contiene varias aplicaciones web e ilustra el uso de las siguientes IPA de Java EE:

  • Tecnología JavaServer Faces, incluido Ajax

  • Contextos e Inyección de Dependencias para Java EE (CDI)

  • IPA de Java para Servicios Web RESTful (JAX-RS)

  • IPA de Java Persistence (JPA)

  • IPA de Java para JavaBeans Validation (Validación de Beans)

  • Tecnología de Enterprise JavaBeans (EJB)

  • Servicio de Mensajes de Java (JMS)

La aplicación consta de los siguientes proyectos.

  • Almacén de Duke: Una aplicación web que cuenta con catálogo de productos, autorregistro de clientes y carrito de compras. También tiene una interfaz de administración para la gestión de productos, categorías y usuarios. El nombre del proyecto es dukes-store.

  • Envíos de Duke: una aplicación web que proporciona una interfaz para la gestión de envíos de pedidos. El nombre del proyecto es dukes-shipment.

  • Pagos de Duke: una aplicación de servicio web que tiene un servicio web RESTful para el pago de pedidos. El nombre del proyecto es dukes-payment.

  • Recursos de Duke: un proyecto de archivo Java simple que contiene todos los recursos utilizados por los proyectos web. Incluye mensajes, hojas de estilo CSS, imágenes, archivos JavaScript y componentes compuestos de JavaServer Faces. El nombre del proyecto es dukes-resources.

  • Entidades: un proyecto de archivo Java simple que contiene todas las entidades JPA. Este proyecto se comparte entre otros proyectos que utilizan las entidades. El nombre del proyecto es entities.

  • Eventos: un proyecto de archivo Java simple que contiene una clase POJO que se usa como un evento CDI. El nombre del proyecto es events.
