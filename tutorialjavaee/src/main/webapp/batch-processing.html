58 Procesamiento por Lotes

Este capítulo describe las aplicaciones por lotes para la Plataforma Java (JSR 352), que brinda soporte para definir, implementar y ejecutar trabajos por lotes. Los trabajos por lotes son tareas que se pueden ejecutar sin la interacción del usuario. El marco de trabajo por lotes se compone de un lenguaje de especificación de trabajos basado en XML, una API de Java y un tiempo de ejecución por lotes.
