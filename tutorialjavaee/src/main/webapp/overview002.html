Aspectos destacados de la plataforma Java EE 8

The key goals of the Java EE 8 platform are to modernize the infrastructure for enterprise Java for the cloud and microservices environments, emphasize HTML5 and HTTP/2 support, enhance ease of development through new Contexts and Dependency Injection features, and further enhance security and reliability of the platform.

The Java EE 8 platform includes the following new features:

  • New technologies, including the following:

    • IPA de Java para Enlace JSON

    • IPA de Seguridad de Java EE

  • New object model improvements for JSON Processing (see IPA de Java para procesamiento JSON for details), including support for the following:

    • JSON Pointer

    • JSON Patch

    • JSON Merge Patch

  • New features for RESTful web services (see IPA de Java para Servicios Web RESTful for details)

  • New features for servlets (see Tecnología de Servlets de Java for details)

  • New features for JavaServer Faces components (see Tecnología JavaServer Faces for details)

  • New Contexts and Dependency Injection features (see Contextos e Inyección de Dependencias para Java EE for details)

  • New JavaBean validation features (see Validación de Beans for details)
