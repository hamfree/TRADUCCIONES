Aspectos destacados de la plataforma Java EE 8

Los objetivos clave de la plataforma Java EE 8 son modernizar la infraestructura de Java empresarial para los entornos de nube y microservicios, enfatizar el soporte HTML5 y HTTP/2, mejorar la facilidad de desarrollo a través de nuevas características de Contextos e Inyección de Dependencias, y mejorar aún más la seguridad y confiabilidad de la plataforma.

La plataforma Java EE 8 incluye las siguientes características nuevas:

  • Nuevas tecnologías, incluyendo las siguientes:

    • IPA de Java para Enlace JSON

    • IPA de Seguridad de Java EE

  • Nuevas mejoras en el modelo de objetos para el procesamiento JSON (consulte IPA de Java para procesamiento JSON para obtener más detalles), incluida la compatibilidad con lo siguiente:

    • JSON Pointer

    • JSON Patch

    • JSON Merge Patch

  • Nuevas funciones para los servicios web RESTful (consulte IPA de Java para Servicios Web RESTful para obtener más detalles)

  • Nuevas funciones para los servlets (consulte Tecnología de Servlets de Java para obtener más detalles)

  • Nuevas características para los componentes de JavaServer Faces (consulte Tecnología JavaServer Faces para obtener más detalles)

  • Nuevas características para los Contextos y la Inyección de Dependencias (consulte Contextos e Inyección de Dependencias para Java EE para obtener más detalles)

  • Nuevas características de validación de JavaBeans (consulte Validación de Beans para obtener más detalles)
