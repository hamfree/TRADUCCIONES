Descripción General de la IPA de Enlazado JSON

Esta sección proporciona instrucciones básicas para usar la API del cliente JSON-B. Las instrucciones proporcionan una base para entender Ejecutando la Aplicación de Ejemplo jsonbbasics. Consulte la página del proyecto JSON-B para obtener documentación de la IPA y una guía del usuario más detallada.

Aquí se tratan los siguientes temas:

  • Creando una Instancia jasonb

  • Usando el Mapeado Predeterminado

  • Usando Personalizaciones

  • Usando Anotaciones

Creando una Instancia jasonb

Una instancia de jsonb proporciona acceso a métodos para vincular objetos a JSON. Se requiere una sola instancia de jsonb para la mayoría de las aplicaciones. Se crea una instancia de jsonb utilizando la interfaz JsonbBuilder, que es el punto de entrada de un cliente a la API de enlace JSON. Por ejemplo: 

                
Jsonb jsonb = JsonbBuilder.create();

Usando el Mapeado Predeterminado

JSON-B proporciona asignaciones predeterminadas para serializar y deserializar tipos básicos de Java y Java SE, así como clases de fecha y hora de Java. Para usar las asignaciones y el comportamiento de asignación predeterminados, cree una instancia de jsonb y use el método toJson para serializar a JSON y el método fromJson para deserializar nuevamente a un objeto. El siguiente ejemplo vincula un objeto Person simple que contiene un solo campo name. 

                
Jsonb jsonb = JsonbBuilder.create();
Person person = new Person();
person.name = "Fred";
Jsonb jsonb = JsonbBuilder.create();
// serializa a JSON
String result = jsonb.toJson(person);
// deserializa de JSON
person = jsonb.fromJson("{name:\"joe"\"}", Person.class);

Usando Personalizaciones

JSON-B admite muchas formas de personalizar el comportamiento de asignación predeterminado. Para las personalizaciones en tiempo de ejecución, se utiliza un objeto de configuración JsonbConfig al crear jsonbinstance. La clase JsonbConfig admite muchas opciones de configuración y también incluye opciones avanzadas para vincular tipos personalizados. Para obtener opciones avanzadas, consulte la interfaz JsonbAdapter y las interfaces JsonbSerializer y JsonbDeserializer.

El siguiente ejemplo crea un objeto de configuración que establece la propiedad FORMATTING para especificar si los datos JSON serializados se formatean o no con saltos de línea y sangría. 

                
JsonbConfig config = new JsonbConfig()
    .withFormatting(true);
Jsonb jsonb = JsonbBuilder.create(config);

Usando Anotaciones

JSON-B incluye muchas anotaciones que se pueden usar en tiempo de compilación para personalizar el comportamiento de asignación predeterminado. El siguiente ejemplo usa la anotación @JsonbProperty para cambiar el campo name a person-name cuando el objeto se serializa en JSON. 

                
public class Person {
    @JsonbProperty("person-name")
    private String name;
}

El documento JSON resultante se escribe como:

                
{
    "person-name": "Fred",
}
