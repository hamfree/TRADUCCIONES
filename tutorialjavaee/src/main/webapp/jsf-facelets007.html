Recursos Reubicables

Puede colocar una etiqueta de recurso en una parte de una página y especificar que se represente en otra parte de la página. Para hacer esto, usa el atributo target de una etiqueta que especifica un recurso. Los valores aceptables para este atributo son los siguientes.

  • "head" renderiza el recurso en el elemento head.

  • "body" renderiza el recurso en el elemento body.

  • "form" renderiza el recurso en el elemento form.

Por ejemplo, la siguiente etiqueta h:outputScript se coloca dentro de un elemento h:form, pero representa el JavaScript en el elemento head: 

              
<h:form>
    <h:outputScript name="myscript.js" library="mylibrary" target="head"/>
</h:form>

La etiqueta h:outputStylesheet también admite la reubicación de recursos, de manera similar.

Los recursos reubicables son esenciales para los componentes compuestos que usan hojas de estilo y también pueden ser útiles para los componentes compuestos que usan JavaScript. Consulte Aplicación de ejemplo de ejemplo de componente compuesto para ver un ejemplo.
