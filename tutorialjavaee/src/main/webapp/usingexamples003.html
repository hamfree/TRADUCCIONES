Arrancando la Consola de Administración

Para administrar GlassFish Server y gestionar usuarios, recursos y aplicaciones Java EE, utilice la herramienta Consola de administración. GlassFish Server debe estar ejecutándose antes de invocar la Consola de administración. Para iniciar la Consola de administración, abra un navegador en http://localhost:4848/.

Para arrancar la Consola de Administración Usando el EID NetBeans

  1. Haga clic en la pestaña Servicios.

  2. Expanda Servidores.

  3. Haga clic con el botón derecho en la instancia de GlassFish Server y seleccione Ver consola de administración del dominio.

    Nota:

    NetBeans IDE usa su navegador web predeterminado para abrir la Consola de administración.
