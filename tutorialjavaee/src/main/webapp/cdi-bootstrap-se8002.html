Configurando el Contenedor CDI

La configuración de la instancia de SeContainerInitializer permite la adición explícita de elementos en un archivo interno de beans sintéticos. El archivo de beans sintéticos representa el conjunto de beans que se cargaron al inicializar el contenedor. El contenido del archivo de beans sintéticos depende de si el descubrimiento está habilitado:

  • Si el descubrimiento está habilitado, el archivo de beans sintéticos se crea usando reglas de descubrimiento de beans estándar y contiene un superconjunto de todos los archivos JAR en el classpath. Se excluyen los archivos que no incluyen un archivo beans.xml.

  • Si el descubrimiento está deshabilitado y los beans se agregan mediante programación, el archivo de beans sintéticos contiene solo los beans que se agregaron mediante programación.
