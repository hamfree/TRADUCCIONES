Descripción General de EL

EL permite a los autores de páginas utilizar expresiones simples para acceder dinámicamente a datos de componentes JavaBeans. Por ejemplo, el atributo test de la siguiente etiqueta condicional se proporciona con una expresión EL que compara 0 con el número de elementos en el bean de ámbito de sesión llamado cart. 

              
<c:if test="${sessionScope.cart.numberOfItems > 0}">
  ...
</c:if>

Vea Uso de EL para hacer referencia a los beans gestionados para obtener más información sobre cómo utilizar EL en aplicaciones JavaServer Faces.

En resumen, EL proporciona una forma de utilizar expresiones simples para realizar las siguientes tareas:

  • Leer dinámicamente datos de aplicaciones almacenados en componentes JavaBeans, varias estructuras de datos y objetos implícitos

  • Escribir datos dinámicamente, como la entrada del usuario en formularios, en componentes JavaBeans

  • Invocar métodos públicos y estáticos arbitrarios

  • Realizar operaciones aritméticas, booleanas y de cadena de forma dinámica

  • Construir dinámicamente objetos de colección y realizar operaciones en colecciones

En una página de JavaServer Faces, una expresión EL se puede utilizar en texto estático o en el atributo de una etiqueta personalizada o acción estándar.

Finalmente, EL proporciona una IPA conectable para resolver expresiones, de modo que se puedan implementar resolutores personalizados que pueden manejar expresiones que aún no son compatibles con EL.
