Enlazado JSON en la Plataforma Java EE

Java EE incluye soporte para JSR 367, que proporciona una API que puede serializar objetos Java en documentos JSON y deserializar documentos JSON en objetos Java. El IPA de Java para Enlace JSON contiene los siguientes paquetes:

  • El paquete javax.json.bind contiene la interfaz de enlace, la interfaz del constructor y una clase de configuración. La Tabla 21-1 enumera las principales clases e interfaces en este paquete.

  • El paquete javax.json.bind.adapter contiene la interfaz JsonbAdapter, que proporciona métodos para enlazar tipos personalizados de Java convirtiéndolos en tipos conocidos.

  • El paquete javax.json.bind.annotation define anotaciones que se pueden usar para personalizar el comportamiento de enlace predeterminado. Las anotaciones se pueden utilizar para elementos de campo, propiedad JavaBean, tipo o paquete.

  • Las interfaces y clases del paquete javax.json.bind.config para personalizar el comportamiento de enlace predeterminado. La Tabla 21-2 enumera las principales clases e interfaces de este paquete.

  • El paquete javax.json.bind.serializer contiene interfaces que se utilizan para crear rutinas de serialización y deserialización para tipos personalizados que no se pueden asignar fácilmente mediante los métodos JsonbAdapter. La Tabla 21-3 enumera las principales interfaces de este paquete.

  • El paquete javax.json.bind.spi contiene una interfaz de proveedor de servicios (SPI) para crear implementaciones de enlace JSON. Este paquete contiene la clase JsonbProvider, que contiene los métodos que implementa un proveedor de servicios.

Tabla 21-1 Clases e interfaces principales en javax.json.bind

Clase o Interfaz

Descripción

Jsonb

Contiene los métodos JSON-B para seriar objetos Java a JSON y componer JSON a objetos Java.

JsonBuilder

Utilizado por los clientes para crear instancias de Jsonb.

JsonbConfig

Se utiliza para establecer propiedades de configuración en instancias de Jsonb. Las propiedades incluyen estrategias de enlace y propiedades para configurar serializadores y deserializadores personalizados.

JsonbException

Indica que ocurrió un problema durante el enlace JSON.

Tabla 21-2 Clases e interfaces principales en javax.json.bind.config

Clase o Interfaz

Descripción

PropertyNamingStrategy

Se utiliza para establecer cómo se traducen los nombres de las propiedades.

PropertyVisibilityStrategy

Se utiliza para establecer si los campos y los métodos deben considerarse propiedades que anulan el alcance predeterminado y el comportamiento de acceso al campo.

BinaryDataStrategy

Se utiliza para establecer la codificación binaria.

PropertyOrderStrategy

Se utiliza para establecer cómo se ordenan las propiedades durante la serialización.

Tabla 21-3 Clases e interfaces principales en javax.json.bind.serializer

Clase o Interfaz

Descripción

JsonbDeserializer

Se utiliza para crear una rutina de deserialización para un tipo personalizado.

JsonbSerializer

Se utiliza para crear una rutina de serialización para un tipo personalizado.
