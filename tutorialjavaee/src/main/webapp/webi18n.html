22 Internacionalización y Localización de Aplicaciones Web

El proceso de preparación de una aplicación para admitir más de un idioma y formato de datos se denomina internacionalización. La localización es el proceso de adaptar una aplicación internacionalizada para admitir una región o localidad específica. Los ejemplos de información dependiente de la configuración regional incluyen mensajes y etiquetas de interfaz de usuario, conjuntos de caracteres y codificación, y formatos de fecha y moneda. Aunque todas las interfaces de usuario del cliente deben estar internacionalizadas y localizadas, esto es particularmente importante para las aplicaciones web debido a la naturaleza global de la web.
