Usando las Interfaces de los Configuradores

La especificación CDI 2.0 define las siguientes interfaces de Configuradores (Configurators, en inglés), que se utilizan para definir y modificar dinámicamente objetos CDI:

Interfaz

Descripción

AnnotatedTypeConfigurator SPI

Ayuda a crear y configurar los siguientes tipos de metadatos:

AnnotatedType

AnnotatedField

AnnotatedConstructor

AnnotatedMethod

AnnotatedParameter

Interfaz InjectionPointConfigurator

Ayuda a configurar una instancia existente de InjectionPoint

Interfaz BeanAttributesConfigurator

Ayuda a configurar una nueva instancia de BeanAttributes

Interfaz BeanConfigurator

Ayuda a configurar una nueva instancia de Bean

Interfaz ObserverMethodConfigurator

Ayuda a configurar una instancia de ObserverMethod

Interfaz ProducerConfigurator

Ayuda a configurar una instancia de Producer
