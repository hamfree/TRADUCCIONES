Arrancando y Parando Apache Derby

GlassFish Server incluye Apache Derby.

Para iniciar Derby desde la línea de comando, abra una ventana de terminal o símbolo del sistema, cambie al directorio as-install/bin y ejecute: 


asadmin start-database

Para detener Derby desde la línea de comando, abra una ventana de terminal o símbolo del sistema, cambie al directorio as-install/bin y ejecute: 


asadmin stop-database

Para obtener información sobre Apache Derby incluido con GlassFish Server, consulte las Notas de la versión que se encuentran en el directorio as-install/javadb /.

Para Arrancar la Base de Datos del Servidor Usando el EID NetBeans

Cuando inicia GlassFish Server usando NetBeans IDE, el servidor de la base de datos se inicia automáticamente. Sin embargo, si alguna vez necesita iniciar el servidor manualmente, siga estos pasos.

  1. Haga clic en la pestaña Servicios.

  2. Expanda Bases de datos.

  3. Haga clic con el botón derecho en Java DB y seleccione Iniciar servidor.

Siguiente Pasos

Para detener la base de datos utilizando NetBeans IDE, haga clic con el botón derecho en Java DB y seleccione Detener servidor.
