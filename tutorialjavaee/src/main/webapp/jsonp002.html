Procesado JSON en la Plataforma Java EE

Java EE incluye soporte para JSR 374, que proporciona una API para analizar, transformar y consultar datos JSON utilizando el modelo de objetos o el modelo de transmisión descrito en Generando y Analizando Datos JSON. La IPA de Java para procesamiento JSON contiene los siguientes paquetes:

  • El paquete javax.json contiene una interfaz de lectura, una interfaz de escritura, una interfaz de constructor de modelos para el modelo de objetos y clases de utilidad y tipos de Java para elementos JSON. Este paquete también incluye varias clases que implementan otros estándares relacionados con JSON: JSON Pointer, Parche JSON y Parche JSON Merge. Estos estándares se utilizan para recuperar, transformar o manipular valores en un modelo de objetos. La Tabla 20-1 enumera las principales clases e interfaces de este paquete.

  • El paquete javax.json.stream contiene una interfaz de analizador y una interfaz de generador para el modelo de transmisión. La Tabla 20-2 enumera las principales clases e interfaces de este paquete.

  • El paquete javax.json.spi contiene una interfaz de proveedor de servicios (SPI) para conectar implementaciones para objetos de procesamiento JSON. Este paquete contiene la clase JsonProvider, que contiene los métodos que implementa un proveedor de servicios.

Tabla 20-1 Clases Principales e Interfaces en javax.json

Clase o Interfaz

Descripción

Json

Contiene métodos estáticos para crear instancias de analizadores, compiladores y generadores JSON. Esta clase también contiene métodos para crear objetos de generador, generador y analizador.

JsonReader

Lee datos JSON de una secuencia y crea un modelo de objeto en la memoria.

JsonObjectBuilder, JsonArrayBuilder

Cree un modelo de objeto o un modelo de matriz en la memoria agregando elementos del código de la aplicación.

JsonWriter

Escribe un modelo de objetos de la memoria en una secuencia.

JsonValue

Representa un elemento (como un objeto, una matriz o un valor) en datos JSON.

JsonStructure

Representa un objeto o una matriz en datos JSON. Esta interfaz es un subtipo de JsonValue.

JsonObject, JsonArray

Representa un objeto o una matriz en datos JSON. Estas dos interfaces son subtipos de JsonStructure.

JsonPointer

Contiene métodos para operar en objetivos específicos dentro de documentos JSON. Los objetivos pueden ser objetos JsonValue, JsonObject, o JsonArray.

JsonPatch

Una interfaz para admitir una secuencia de operaciones que se aplicará a un recurso JSON de destino. Las operaciones se definen dentro de un documento de parche JSON.

JsonMergePatch

Una interfaz para admitir actualizaciones para los recursos JSON de destino. Un documento de parche JSON se compara con el recurso de destino para determinar el conjunto específico de operaciones de cambio que se aplicarán.

JsonString, JsonNumber

Representa tipos de datos para elementos en datos JSON. Estas dos interfaces son subtipos de JsonValue.

JsonException

Indica que se produjo un problema durante el procesamiento de JSON.

Tabla 20-2 Clases Principales e Interfaces en javax.json.stream

Clase o Interfaz

Descripción

JsonParser

Representa un analizador basado en eventos que puede leer datos JSON de una secuencia o de un modelo de objetos.

JsonGenerator

Escribe datos JSON en un flujo, un elemento a la vez.
