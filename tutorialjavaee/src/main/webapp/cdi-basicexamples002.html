El Ejemplo de CDI simplegreeting

El ejemplo de simplegreeting ilustra algunas de las características más básicas de CDI: alcances, calificadores, inyección de beans y acceso a un bean administrado en una aplicación JavaServer Faces. Cuando ejecuta el ejemplo, hace clic en un botón que presenta un saludo formal o informal, dependiendo de cómo editó una de las clases. El ejemplo incluye cuatro archivos de origen, una página y una plantilla de Facelets y archivos de configuración.

Aquí se tratan los siguientes temas:

  • Los Archivos Fuente de simplegreeting

  • La Página y Plantilla de Facelets

  • Ejecutando el Ejemplo simplegreeting

Los Archivos Fuente de simplegreeting

Los cuatro archivos fuente para el ejemplo simplegreeting son:

  • La clase Saludo predeterminada, que se muestra en Beans como Objetos Inyectables

  • La definición de la interfaz del calificador @Informal y la clase InformalGreeting que implementa la interfaz, ambas mostradas en Usando Calificadores

  • La clase de bean administrado Printer, que inyecta una de las dos interfaces, que se muestra en su totalidad en Agregando Métodos Setter y Getter

Los archivos fuente se encuentran en el directorio tut-install/examples/cdi/simplegreeting/src/main/java/javaeetutorial/simplegreeting .

La Página y Plantilla de Facelets

Para usar el bean administrado en una aplicación sencilla de Facelets:

  1. Use un archivo de plantilla muy simple y una página index.xhtml.

    La página de la plantilla, template.xhtml, tiene este aspecto:

                          
<?xml version='1.0' encoding='UTF-8' ?>
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
          "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html lang="es"
      xmlns="http://www.w3.org/1999/xhtml"
      xmlns:h="http://xmlns.jcp.org/jsf/html"
      xmlns:ui="http://xmlns.jcp.org/jsf/facelets">
    <h:head>
        <meta http-equiv="Content-Type" content="text/html; charset=UTF-8"/>
        <h:outputStylesheet library="css" name="default.css"/>
        <title><ui:insert name="title">Default Title</ui:insert></title>
    </h:head>
    <body>
        <div id="container">
            <div id="header">
                <h2><ui:insert name="head">Head</ui:insert></h2>
            </div>
            <div id="space">
                <p></p>
            </div>
            <div id="content">
                <ui:insert name="content"/>
            </div>
        </div>
    </body>
</html>

  2. Para crear la página Facelets, redefina el título y el encabezado, luego agregue un pequeño formulario al contenido: 

                          
<?xml version='1.0' encoding='UTF-8' ?>
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
          "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html lang="es"
      xmlns="http://www.w3.org/1999/xhtml"
      xmlns:ui="http://xmlns.jcp.org/jsf/facelets"
      xmlns:h="http://xmlns.jcp.org/jsf/html">
    <ui:composition template="/template.xhtml">
        <ui:define name="title">Simple Greeting</ui:define>
        <ui:define name="head">Simple Greeting</ui:define>
        <ui:define name="content">
            <h:form id="greetme">
               <p><h:outputLabel value="Introduzca su nombre: " for="name"/>
                  <h:inputText id="name" value="#{printer.name}"/></p>
               <p><h:commandButton value="Diga Hola"
                                   action="#{printer.createSalutation}"/></p>
               <p><h:outputText value="#{printer.salutation}"/> </p>
            </h:form>
        </ui:define>
    </ui:composition>
</html>

    El formulario le pide al usuario que ingrese un nombre. El botón tiene la etiqueta Diga Hola y la acción definida para él es llamar al método createSalutation del bean administrado Printer. Este método, a su vez, llama al método greet de la clase Greeting definida.

    El texto de salida del formulario es el valor del saludo devuelto por el método setter. Dependiendo de si se inyecta la versión predeterminada o @Informal del saludo, esta es una de las siguientes, donde name es el nombre ingresado por el usuario: 

                          
Hola, name.
¡Qué hay, name!

    La página y la plantilla de Facelets se encuentran en el directorio tut-install/examples/cdi/simplegreeting/src/main/webapp/.

    El archivo CSS simple que utiliza la página Facelets se encuentra en la siguiente ubicación: 

                          
tut-install/examples/cdi/simplegreeting/src/main/webapp/resources/css/default.css

Ejecutando el Ejemplo simplegreeting

Puede usar el EID NetBeans o Maven para compilar, empaquetar, implementar y ejecutar la aplicación simplegreeting.

Aquí se tratan los siguientes temas:

  • Para Construir, Empaquetar, y Ejecutar el Ejemplo simplegreeting Usando el EID NetBeans

  • Para Construir, Empaquetar, y Desplegar el Ejemplo simplegreeting Usando Maven

  • Para Ejecutar el Ejemplo simplegreeting

Para Construir, Empaquetar, y Ejecutar el Ejemplo simplegreeting Usando el EID NetBeans

  1. Asegúrese de que GlassFish Server se haya iniciado (consulte Arrancando y Parando el Servidor GlassFish).

  2. En el menú Archivo, seleccione Abrir proyecto.

  3. En el cuadro de diálogo Abrir proyecto, vaya a:

                            
tut-install/examples/cdi

  4. Selecciona la carpeta simplegreeting.

  5. Haz clic en Abrir proyecto.

  6. Para modificar el archivo Printer.java, realice estos pasos:

    1. Expanda el nodo Paquetes de origen.

    2. Expanda el nodo greetings.

    3. Haga doble clic en el archivo Printer.java.

    4. En el editor, comente la anotación @Informal:

                                    
@Inject
//@Informal
Greeting greeting;

    5. Guarde el archivo.

  7. En la pestaña Proyectos, haga clic con el botón derecho en el proyecto simplegreeting y seleccione Crear.

    Este comando construye y empaqueta la aplicación en un archivo WAR, simplegreeting.war, ubicado en el directorio target, y luego lo implementa en el Servidor GlassFish.

Para Construir, Empaquetar, y Desplegar el Ejemplo simplegreeting Usando Maven

  1. Asegúrese de que GlassFish Server se haya iniciado (consulte Arrancando y Parando el Servidor GlassFish).

  2. En una ventana de terminal, vaya a:

                            
tut-install/examples/cdi/simplegreeting/

  3. Ingrese el siguiente comando para implementar la aplicación:

                            
mvn install

    Este comando construye y empaqueta la aplicación en un archivo WAR, simplegreeting.war, ubicado en el directorio target, y luego lo implementa en el Servidor GlassFish.

Para Ejecutar el Ejemplo simplegreeting

  1. En un navegador web, ingrese la siguiente URL:

                            
http://localhost:8080/simplegreeting

    Se abre la página Simple Greeting.

  2. Introduzca un nombre en el campo.

    Por ejemplo, suponga que ingresa Duke.

  3. Pulse Diga Hola.

    Si no modificó el archivo Printer.java, la siguiente cadena de texto aparece debajo del botón: 

                            
¡Qué hay, Duke!

    Si comentó la anotación @Informal en el archivo Printer.java, la siguiente cadena de texto aparece debajo del botón: 

                            
Hola, Duke.
