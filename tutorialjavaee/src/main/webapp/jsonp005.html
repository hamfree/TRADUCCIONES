JSON en los Servicios Web RESTful de Java EE

Esta sección explica cómo se relaciona el IPA de Java para procesar JSON con otros paquetes Java EE que brindan soporte JSON para servicios web RESTful. Ver Capítulo 32, "Construyendo Servicios Web RESTful con JAX-RS" para obtener más información sobre los servicios web RESTful.

Jersey, la implementación de referencia para JAX-RS (JSR 311) incluida en el Servidor GlassFish, brinda soporte para vincular datos JSON de métodos de recursos RESTful a objetos Java usando JAXB, como se describe en Usando JAX-RS con JAXB en Capítulo 34, "JAX-RS: Temas Avanzados y un Ejemplo". Sin embargo, la compatibilidad con JSON no forma parte de JAX-RS (JSR 311) o JAXB (JSR 222), por lo que es posible que ese procedimiento no funcione para implementaciones de Java EE que no sean el Servidor GlassFish.

El IPA de Java para procesar JSON (JSR 374) no admite explícitamente el enlace JSON en Java. Se está considerando un futuro JSR (JSON Binding) similar a JAXB para XML para una versión futura de Java EE.

Todavía puede usar IPA de Java para procesar JSON con métodos de recursos JAX-RS. Para obtener más información, consulte el código de muestra para el procesamiento JSON incluido con el KDJ de Java EE 8.
