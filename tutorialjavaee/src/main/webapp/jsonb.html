21 Enlazado JSON

Este capítulo describe el IPA de Java para Enlace JSON (JSON-B). JSON es un formato de intercambio de datos ampliamente utilizado en servicios web y otras aplicaciones conectadas. Para obtener una breve descripción general de JSON, consulte Introducción a JSON.

La especificación JSR 367 proporciona una capa de enlace estándar (metadatos y tiempo de ejecución) entre las clases Java y los documentos JSON. La implementación de referencia de JSON-B es Yasson, que se desarrolla a través de Eclipse.org y se incluye como parte de el Servidor GlassFish. Puede obtener más información sobre Yasson en https://projects.eclipse.org/projects/rt.yasson .
