Accediendo al Contexto de Web

El contexto en el que se ejecutan los componentes web es un objeto que implementa la interfaz ServletContext. Recupera el contexto web usando el método getServletContext. El contexto web proporciona métodos para acceder

  • Parámetros de inicialización

  • Recursos asociados al contexto web

  • Atributos con valor de objeto

  • Capacidades de registro

Los métodos de acceso del contador están sincronizados para evitar operaciones incompatibles por parte de servlets que se ejecutan simultáneamente. Un filtro recupera el objeto contador mediante el método getAttribute del contexto. El valor incrementado del contador se registra en el registro.
