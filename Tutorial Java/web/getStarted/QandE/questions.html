Preguntas y Ejercicios: Comenzando

Preguntas

Pregunta 1: Cuando usted compila un programa escrito en el lenguaje de programación Java, el compilador convierte el fichero fuente leíble por humanos en un código independiente de la plataforma que una Máquina Virtual de Java puede entender. ¿Cómo se llama este código independiente de la plataforma?

Pregunta 2: ¿Cuál de los siguientes no es un comentario valido?:

a. /** comment */
b. /* comment */
c. /* comment
d. // comment

Pregunta 3: ¿Cuál es la primera cosa que debería comprobar si ve el siguiente error en tiempo de ejecución?: 

Exception in thread "main" java.lang.NoClassDefFoundError:
HolaMundoApl.java.

Pregunta 4: ¿Cuál es la firma correcta del método main?

Pregunta 5: Cuando declara el método main, ¿qué modificador debe ir primero, public o static?

Pregunta 6: ¿Qué parámetros define el método main?

Ejercicios
