Respuestas a las Preguntas y Ejercicios: Comenzando

Preguntas

Pregunta 1: Cuando usted compila un programa escrito en el lenguaje de programación Java, el compilador convierte el fichero fuente leíble por humanos en un código independiente de la plataforma que una Máquina Virtual de Java puede entender. ¿Cómo se llama este código independiente de la plataforma?

Respuesta 1: Bytecode.

Pregunta 2: ¿Cuál de los siguientes no es un comentario valido?:

a. /** comment */
b. /* comment */
c. /* comment
d. // comment

Respuesta 2: c es un comentario no válido.

Pregunta 3: ¿Cuál es la primera cosa que debería comprobar si ve el siguiente error en tiempo de ejecución?: 

Exception in thread "main" java.lang.NoClassDefFoundError:
HolaMundoApl.java.

Respuesta 3: Compruebe su classpath. Su clase no pudo ser encontrada.

Pregunta 4: ¿Cuál es la firma correcta del método main?

Respuesta 4: La firma correcta es public static void main(String[] args) o public static void main(String... args)

Pregunta 5: Cuando declara el método main, ¿qué modificador debe ir primero, public o static?

Respuesta 5: Pueden estar en cualquier orden, pero la convención es public static.

Pregunta 6: ¿Qué parámetros define el método main?

Respuesta 6: El método main define un simple parámetro, nombrado usualmente args, cuyo tipo es un array de objetos String.

Ejercicios

Ejercicio 1: Cambiar el programa HolaMundoApl.java de forma que visualize ¡Hola Mundo! en vez de Hello World!.

Respuesta 1: Esta es la única línea de código que debe cambiar: 

System.out.println("¡Hola Mundo!"); //Muestra la cadena.

Ejercicio 2: Puede encontar un versión ligeramente modificada de HolaMundoApl aquí: HolaMundoApl2.java

El programa tiene un error. Arregle el error de forma que el programa compile exitosamente y se ejecute. ¿Cuál era el error?

Respuesta 2: Aquí está el error cuando intenta compilar el programa: 

HolaMundoApl2.java:41: error: unclosed string literal
        System.out.println("Hola Mundo!); // Muestra la cadena.
                           ^
HolaMundoApl2.java:41: error: ';' expected
        System.out.println("Hola Mundo!); // Muestra la cadena.
                                                               ^
HolaMundoApl2.java:43: error: reached end of file while parsing
}
 ^
3 errors

Para arreglar este error, necesita cerrar las comillas alrededor de la cadena. Aquí está la línea correta de código: 

 System.out.println("Hello World!"); //Muestra la cadena.
