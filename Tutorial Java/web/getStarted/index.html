Recorrido: Comenzando

Este recorrido ofrece todo lo que usted necesitará saber sobre cómo empezar con el lenguaje de programación Java.

El Fenómeno de la Tecnología Java Ofrece una introducción a la tecnología Java como un todo. Se analiza tanto el lenguaje de programación Java como la plataforma, ofreciendo una amplia visión de lo que esta tecnoligía puede hacer y cómo hará su vida más fácil.

La Aplicación The "¡Hola Mundo!" Éste enfoque práctico describe qué descargar, qué instalar, y qué escribir, para crear una aplicación simple "¡Hola Mundo!". Se ofrecen instrucciones separadas para el entorno de de desarrollo integrado (EDI Netbeans). Microsoft Windows, Solaris™ el Sistema Operativo Solaris (SO Solaris), Linux, y usuarios de Mac.

Una mirada más cercana a "¡Hola Mundo!" Analiza la aplicación "¡Hola Mundo!", describiendo cada sección de código en detalle. Cubre los comentarios del código fuente, el bloque de definición de clase HolaMundoApl, y el método main.

Problemas Comunes (y Su Solución) Este es el lugar donde ir si usted tiene problemas compilando o ejecutando los programas en este recorrido.
