¿Cómo Cambiará Mi Vida la Tecnología Java?

No podemos prometerle fama, fortuna, o incluso un trabajo si usted aprende el lenguaje de programación Java. Aun así, facilitará que haga sus programa mejor y requerirá menos esfuerzo que en otros lenguajes. Creemos que la tecnología Java le ayudará a realizar lo siguiente:

  • Comenzar rápidamente: Aunque el lenguaje de programación Java es un poderoso lenguaje orientado a objetos, es fácil de aprender, especialmente para programadores ya familiarizados con C o C++.
  • Escribir menos código: Comparaciones de las métricas de programación (número de clases, número de metodos, y demás) sugieren que un programa escrito en el lenguaje de programación Java puede ser hasta cuatro veces más pequeño que el mismo programa escrito en C++.
  • Escribir mejor código: El lenguaje de programación Java fomenta las buenas prácticas de codificación, y la recolección de basura automática le ayuda a evitar las pérdidas de memoria. Su orientación a objetos, su arquitectura de componentes JavaBeans™, y su amplia, y fácilmente extensible IPA le permite reutilizar código existente, comprobado e introducir menos errores.
  • Desarrollar programas más rápidamente: El lenguaje de programación Java es más simple que C++, y como tal, su tiempo de desarrollo podría ser hasta dos veces más rápido cuando escriba con él. Sus programas también requerirán menos líneas de código.
  • Evitar dependencias de la plataforma: Usted puede mantener su programa portable evitando el uso de librerías escritas en otros lenguajes.
  • Escriba una vez, ejecute en cualquier parte: Ya que las aplicaciones escritas con el lenguaje de programación Java son compiladas en bytecodes independientes-de-la-máquina, se ejecutarán consistentemente en cualquier plataforma Java.
  • Distribuir software más fácilmente: Con el software Java Web Start, los usuarios serán capaces de lanzar sus aplicaciones con un click simple del ratón. Una comprobación de versión automática al inicio se asegura de que los usuarios siempre estén al dia con la última versión de su software. Si una actualización está disponible, el software Java Web Start automáticamente actualizará sus instalaciones.
