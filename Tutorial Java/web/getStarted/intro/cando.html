¿Qué Puede Hacer la Tecnología Java?

El lenguaje de programación de alto nivel y de propósito general Java es una plataforma software poderosa. Cada implementación completa de la plataforma Java le ofrece las siguientes características:

  • Herramientas de Desarrollo : Las herramientas de desarrollo ofrecen todo lo que usted necesitará para compilar, ejecutar, monitorizar, depurar y documentar sus aplicaciones. Como un desarrollador nuevo, las herramientas principales que estár usando serán el compilador javac , el lanzador java , y la herramienta de documentación javadoc.

  • Interfaz de Programación de Aplicaciones (IPA) : La IPA ofrece la funcionalidad principal del lenguaje de programación Java. Ofrece un amplio catálogo de clases útiles listas para usar en sus propias aplicaciones. Se extiende por todo, desde objetos básicos, a la creación de redes y la seguridad, a la generación de XML y el acceso de base de datos, y más. La IPA de base es muy grande; para obtener una visión general de lo que contiene, consultar la Documentación de la Edición Estandar de la Plataforma Java 8 .

  • Tecnologías de Implementación : El software del KDJ ofrece mecanismos estándar tales como el software Java Web Start y el software Java Plug-In para implantar sus aplicaciones a usuarios finales.

  • Herramientas de Interfaz de Usuario : Las herramientas JavaFX, Swing, y Java 2D hacen posible el crear sofisticados Interfaces Gráficos de Usuario (GUIs en inglés, IGUs en español).

  • Librerías de Integración : Librerías de integración tales como Java IDL IPA, JDBC IPA, Java Naming and Directory Interface (JNDI) IPA, Java RMI, y la Invocación de Métodos Remotos de Java en Internet con la Tecnología del Protocolo Inter-ORB (Tecnología Java RMI-IIOP) habilita el acceso a bases de datos y la manipulación de objetos remotos.
