Sobre la Tecnología Java

La tecnología Java es a la vez un lenguaje de programación y una plataforma.

El Lenguaje de Programación Java

El lenguaje de programación Java es un lenguaje de alto nivel que puede ser caracterizado por las siguientes palabras de moda:

  • Simple
  • Orientado a objetos
  • Distribuido
  • Multihilo
  • Dinámico
  • Neutral a la Arquitectura
  • Portable
  • Alto rendimiento
  • Robusto
  • Seguro

Cada una de las palabras de moda precedentes están explicadas en El Entorno del Lenguaje Java , un libro blanco escrito por James Gosling y Henry McGilton.

En el lenguaje de programación Java, todo el código fuente es escrito primero en ficheros de texto plano cuyo nombre finaliza con la extensión .java. Estos ficheros fuente son entonces compilados en ficheros .class por el compilador javac . Un fichero .class no contiene código que sea nativo a su procesador; en vez de eso contiene bytecodes — el lenguaje máquina de la Máquina Virtual de Java 1 (Java VM). La herramienta de lanzamiento java entonces ejecuta su aplicación con una instancia de la Máquina Virtual de Java.

Una visión general del proceso de desarrollo de software.

Debido a que la MV de Java está disponible en muchos sistemas operativos diferentes, los mismos ficheros .class son capaces de ejecutarse en Microsoft Windows, el Sistema Operativo Solaris (Solaris OS), Linux, o Mac OS. Algunas máquinas virtuales, tales como Java SE HotSpot de un Vistazo , ejecutan pasos adicionales en tiempo de ejecución para darle a su aplicación un aumento de rendimiento. Esto incluye varias tareas como encontrar cuellos de botella en el rendimiento y recompilar (a código nativo) secciones de código frecuentemente usadas.

A través de la VM de Java, la misma aplicación es capaz de ejecutarse en múltiples plataformas.

La Plataforma Java

Una platforma es el entorno de hardware y software en el cual un programa se ejecuta. Hemos mencionado ya algunas de las plataformas más populares como Microsoft Windows, Linux, Solaris OS, y Mac OS. La mayoría de las plataformas pueden ser descritas como una combinación del sistema operativo y el hardware subyacente. La plataforma Java difiere de la mayoría de otras plataformas en que es una plataforma sólo-de-software que se ejecuta encima de otras plataformas basadas-en-hardware.

La plataforma Java tiene dos componentes:

  • La Máquina Virtual Java
  • El Interfaz de Programación de Aplicaciones Java (IPA)

Ya se le ha presentado la Máquina Virtual de Java; es la base de la plataforma Java y es portada en varias plataformas basadas-en-hardware.

La IPA es una gran colección de componentes de software ya hechos que ofrecen muchas útiles capacidades. Están agrupados en librerías de clases relacionadas e interfaces; estas librerías son conocidas como paquetes. La siguiente sección , ¿Qué Puede Hacer la Tecnología Java? pone de relieve algunas de la funcionalidades proporcionadas por la IPA.

La IPA y la Máquina Virtual de Java aislan el programa del hardware subyacente.

Como un entorno independiente-de-la-plataforma, la plataforma Java puede ser un poquito más lenta que el código nativo. Sin embargo, los avances en el compilador y las tecnologías de la máquina virtual están acercando el rendimiento al del código nativo sin amenazar la portabilidad.

Los términos "Máquina Virtual de Java" y "JVM" significan una Máquina Virtual para la plataforma de Java.
