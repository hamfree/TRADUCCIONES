Lección: La Aplicación "¡Hola Mundo!"

Las secciones listadas abajo ofrecen instrucciones detalladas para compilar y ejecutar una aplicación "¡Hola Mundo!". La primera sección ofrece información sobre cómo empezar con el EID de NetBeans, un entorno integrado de desarrollo que simplifica enormemente el proceso de desarrollo del software. El EID NetBeans se ejecuta en todas las plataformas listadas abajo. Las secciones restantes ofrecen instrucciones que son específicas a la plataforma para comenzar sin un entorno integrado de programación. Si usted encuentra problemas, asegúrese de consultar la sección de problemas comunes; en ella se ofrecen soluciones para muchos errores encontrados por los nuevos usuarios.

"¡Hola Mundo!" para el EID Netbeans Estas instrucciones son para los usuarios del EID NetBeans. El EID de NetBeans se ejecuta sobre la plataforma Java, lo que significa que puede utilizarlo con cualquier sistema operativo para el cual haya un KDJ 7 disponible. Estos sistemas operativos incluyen Microsoft Windows, Solaris OS, Linux y Mac OS X. Recomendamos utilizar el EID de NetBeans en vez de la línea de comandos siempre que sea posible.

"¡Hola Mundo!" para Microsoft Windows Estas instrucciones de línea de comandos son para usuarios de Windows XP Professional, Windows XP Home, Windows Server 2003, Windows 2000 Professional y Windows Vista.

"¡Hola Mundo!" para SO Solaris y Linux Estas instrucciones de línea de comandos son para usuarios de Solaris OS y Linux. Problemas Comunes (y Sus Soluciones) Consulte ésta página si tiene problemas compilando o ejecutando su aplicación.
