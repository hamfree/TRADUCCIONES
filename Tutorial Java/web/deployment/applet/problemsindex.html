Resolviendo Problemas Comunes de los Applets

Esta sección cubre algunos problemas comunes que usted podría encontrar cuando escribe applets de Java. Después de cada problema hay una lista de razones posibles y soluciones.

Problema: Mi applet no se visualiza.

  • Compruebe el registro de la Consola de Java por errores.
  • Comprueba la sintaxis del fichero (JNLP) Protocolo de Lanzamiento en Red de Java del applet. Los ficheros JNLP incorrectos son la razón más común por fallos sin errores evidentes.
  • Compruebe la sintaxis de JavaScript si el despliegue usa la función runApplet del Kit de Herramientas de Despliegue. Vea Desplegando un Applet por los detalles.

Problema: El registro de la Consola de Java visualiza java.lang.ClassNotFoundException.

  • Asegúrese de que sus ficheros fuente de Java se compilaron correctamente.
  • Si despliega usando la etiqueta <applet>, compruebe que el path al fichero JAR del applet se especifica con precisión en el atributo archive.
  • Si se inicia utilizando un archivo JNLP, verifique la ruta en la etiqueta jar en el archivo JNLP.
  • Asegúrese de que el archivo JAR del applet, el archivo JNLP, y la página web están ubicados en el directorio correcto y se referencian entre sí con precisión.

Problema: Soy capaz de construir el código una vez, pero ahora la construcción falla aunque no haya errores de compilación.

  • Cierre su navegador y ejecute la construcción de nuevo. El navegador probablemente tiene un bloqueo en el archivo JAR, debido al cual el proceso de construcción es incapaz de regenerar el archivo JAR.

Problema: Cuando intento cargar una página web que tiene un applet, mi navegador me redirige a www.java.com sin ninguna advertencia.

  • El applet en la página web probablemente fue desplegado usando el script del Kit de Herramienteas de Despliegue. El applet puede requerir una versión posterior del software del Entorno de Ejecución de Java que la versión que actualmente existe en el cliente. Compruebe el parámetro minimumVersion de la función runApplet en la página web del applet. Vea Desplegando un Applet por detalles.

Problema: He arreglado algunos errores y reconstruido el código fuente de mi applet. Cuando recargo la página web del applet, mis arreglos no están apareciendo.

  • Puede estar viendo una versión cacheada previamente del applet. Cierre el navegador. Abra el Panel de Control de Java y borre los ficheros temporales de internet. Esto eliminará su applet de la caché. Intente ver su applet otra vez.
