Visualizando Documentos en el Navegador

Un applet de Java puede cargar una página web en una ventana del navegador usando los métodos showDocument en la clase java.applet.AppletContext.

Aquí tiene las dos formas de showDocument: 

public void showDocument(java.net.URL url)
public void showDocument(java.net.URL url, String targetWindow)

La forma de un argumento de showDocument simplemente instruye al navegador a visualizar el documento en la URL especificada, sin especificar la ventana en la cual visualizar el documento.

La forma de dos argumentos de showDocument le permite especificar la ventana o el marco HTML en el cual visualizar el documento. El segundo argumento puede ser uno de los valores siguientes:

  • "_blank" – Muestra el documento en una ventana nueva, sin nombre.
  • "windowName" – Muestra el documento en una ventana llamada windowName. Esta ventana se crea si es necesario.
  • "_self" – Visualiza el documento en la ventana y marco que contiene el applet.
  • "_parent" – Visualiza el documento en el marco padre del marco del applet. Si el marco del applet no tiene marco padre, se actúa de la misma forma que con "_self".
  • "_top" – Muestra el documento en el marco de nivel superior. Si el marco del applet es el marco del nivel superior, se actúa de la misma forma que con "_self".
Nota: En esta descripción, los marcos no se refieren al JFrame de Swing, si no al marco HTML dentro de la ventana del navegador.

El applet siguiente le habilita a que intente cada argumento de ambas formas de showDocument. El applet abre una ventana que le permite escribir una URL y elegir una opción para el argumento targetWindow. Cuando usted presione Intro o pulse el botón Mostrar documento, el applet llama a showDocument.

Nota:  Si usted no ve el ejemplo ejecutándose, podría necesitar habilitar el intérprete de JavaScript en su navegador de forma que el script de las Herramientas de Desplieuge pueda funcionar adecuadamente.
