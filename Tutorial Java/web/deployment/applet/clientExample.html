Ejemplo de Applet de Cliente de Red

La clase QuoteClientApplet le permite obtener citas de una aplicación del lado del servidor que se ejecuta en el mismo host que este applet. Esta clase también visualiza la cita recibida desde el servidor.

Las clases QuoteServer.java y QuoteServerThread.java construyen la aplicación del lado del servidor que devuelven citas. Aquí tiene un fichero de texto ( one-liners.txt) que contiene un número de citas.

Realize los siguientes pasos para comprobar QuoteClientApplet.

  1. Descargue y salve los siguientes ficheros a su máquina local.
    • QuoteClientApplet
    • QuoteServer.java
    • QuoteServerThread.java
    • one-liners.txt
    • quoteApplet.html
  2. Incluya el código HTML siguiente en una página web para desplegar QuoteClientApplet. 
    <script src=
  "https://www.java.com/js/deployJava.js"></script>
<script> 
    var attributes =
      { code:'QuoteClientApplet.class',  width:500, height:100} ; 
    var parameters =
      { codebase_lookup:'true', permissions:'sandbox' };
    deployJava.runApplet(attributes, parameters, '1.6'); 
</script>
    Alternativamente, puede usar la página quoteApplet.html que contiene ya este código HTML.
  3. Compile la clase QuoteClientApplet.java. Copie los ficheros class generados en el mismo directorio donde usted salvó la página web.
  4. Compile las clases para la aplicación del lado del servidor, QuoteServer.java y QuoteServerThread.java.
  5. Copie el fichero one-liners.txt al directorio que tiene los ficheros class para la aplicación del lado del servidor (generados en el paso anterior).
  6. Inicie la aplicación del lado del servidor. 
    java QuoteServer

    Usted debería ver un mensaje con el número de puerto, como es mostrado en el ejemplo sigiuente. Anote el número de puerto. 

    QuoteServer listening on port:3862
  7. Abra la página web conteniendo su applet en un navegador introduciendo la URL de la página web. El nombre del anfitrión en la URL debería ser el mismo que el nombre del anfitrión en el cual se está ejecutando la aplicación del lado del servidor.

    Por ejemplo, si la aplicación del lado del servidor está ejecutándose en una máquina llamada JohnDoeMachine, usted introduciría una URL similar. El número de puerto exacto y la ruta variarán dependiendo de la configuración de su servidor web. 

    http://JohnDoeMachine:8080/quoteApplet/quoteApplet.html
    El QuoteClientApplet será visualizado en la página web.
  8. Introduzca el número de puerto de su aplicación del lado del servidor en el campo de texto del applet y pulse OK. Una cita es visualizada.
