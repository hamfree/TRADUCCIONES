Empezando Con los Applets

El applet HelloWorld mostrado a continuación es una clase Java que muestra la cadena "Hello World".

Nota:  Si usted no ve el ejemplo ejecutándose, podría necesitar habilitar el interpretador de JavaScript en su navegador de forma que el script del Kit de Herramientas de Implementación funcione apropiadamente.

Lo siguiente es el código fuente del applet HelloWorld: 


import javax.swing.JApplet;
import javax.swing.SwingUtilities;
import javax.swing.JLabel;

public class HelloWorld extends JApplet {
    //Llamado cuando el applets es cargado en el navegador.
    public void init() {
        //Ejecuta un trabajo en el hilo de envío de eventos; creando el IGU del applet.
        try {
            SwingUtilities.invokeAndWait(new Runnable() {
                public void run() {
                    JLabel lbl = new JLabel("Hello World");
                    add(lbl);
                }
            });
        } catch (Exception e) {
            System.err.println("createGUI didn't complete successfully");
        }
    }
}

Un applet tal como este es gestionado tipicamente y ejecutado por el software del Plug-in Java en el navegador.

Descargue el código fuente para el ejemplo Hello World para experimentar adicionalmente.
