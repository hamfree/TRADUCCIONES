Comunicando Con Otros Applets

Un applet de Java puede comunicarse con otros applets de Java usando funciones de JavaScript en la págian web padre. Las funciones de JavaScript habiitan la comunicación entre applets recibiendo mensajes desde un applet e invocando métodos de otros applets. Vea los siguientes temas para más información sobre la interacción entre el código de Java y el código de JavaScript:

  • Invocando Código JavaScript Desde un Applet
  • Invocando Métodos del Applet Desde Código JavaScript

Debería evitar usar los mecanismos siguientes para encontrar otros applets y compartir datos entre applets:

  • Evite usar variables estáticas para compartir datos entre applets.
  • No use los métodos getApplet y getApplets de la clase AppletContext para buscar otros appplets. Estos métodos sólo encuentran applets que están ejecutándose en la misma instancia del software del Entorno de Ejecución de Java.

Los applets deben originarse desde el mismo directorio en el servidor para poder comunicarse entre sí.

Los applets Sender y Receiver son mostrados a continuación. Cuando un usuario pulsa el botón para incrementar el contador, el applet Sender invoca una función de JavaScript para enviar una petición al applet Receiver. Al recibir la solicitud, el applet Receiver incrementa una variable de contador y muestra el valor de la variable.

Nota:  Si no ve el applet ejecutarse, necesita instalar al menos la versión Java SE Development Kit (KDJ) 6 update 10.
Nota:  Si no ve el ejemplo ejecutándose, podría necesitar habilitar el intérprete de JavaScript en su navegador de forma que el script de las Herramientas de Despliegue pueda funcionar adecuadamente.

Para habilitar la comunicación con otro applet, obtenga una referencia a una instancia de la clase netscape.javascript.JSObject. Use esta instancia para invocar funciones de JavaScript. El applet Sender usa una instancia de la clase netscape.javascript.JSObject para invocar una función JavaScript llamada sendMsgToIncrementCounter. 

try {
    JSObject window = JSObject.getWindow(this);
    window.eval("sendMsgToIncrementCounter()");
} catch (JSException jse) {
    // ...
}
Nota:  Para compilar código Java que tiene una referencia a clases en el paquete netscape.javascript, incluya <su path del KDJ>/jre/lib/plugin.jar en su classpath. En tiempo de ejecución el software del Complemento de Java hace automáticamente disponibles estas clases a los applets.

Escriba la función de JavaScript que recibirá las peticiones desde un applet e invoca métodos de otro applet en la página web. La función de JavaScript sendMsgToIncrementCounter invoca el método incrementCounter del applet Receiver. 

<script>
    function sendMsgToIncrementCounter() {
        var myReceiver = document.getElementById("receiver");
        myReceiver.incrementCounter();
    } 
<script>

Note que el código JavaScript usa el nombre receiver para obtener una referencia al applet Receiver en la página web. Este nombre deberiá ser el mismo que el valor del atributo id que es especificado cuando usted despliega el applet Receiver.

El método incrementCounter del applet Receiver es mostrado a continuación. 

public void incrementCounter() {
    ctr++;
    String text = " Current Value Of Counter: "
        + (new Integer(ctr)).toString();
    ctrLbl.setText(text);
}

Despliegue los applets en la página web tal como es mostrado en el trozo de código siguiente. Puede ver los applets Sender y Receiver y el código JavaScript asociado en AppletPage.html. 

<!-- Sender Applet -->
<script src="https://www.java.com/js/deployJava.js"></script>
<script> 
    var attributes = { code:'Sender.class',
        archive:'examples/dist/applet_SenderReceiver/applet_SenderReceiver.jar',
        width:300, height:50} ;
    var parameters = { permissions:'sandbox' };
    deployJava.runApplet(attributes, parameters, '1.6');
</script>

<!-- Receiver Applet -->
<script> 
    var attributes = { id:'receiver', code:'Receiver.class',
        archive:'examples/dist/applet_SenderReceiver/applet_SenderReceiver.jar',
        width:300, height:50} ;
    var parameters = { permissions:'sandbox' };
    deployJava.runApplet(attributes, parameters, '1.6');
</script>
