Desarrollando Applets Arrastrables

Un applet de Java que es desplegado especificando el parámetro draggable puede ser arrastrado fuera del navegador y ser transformado en una aplicación Java Web Start. El applet de Java puede ser arrastrado al presionar la tecla Alt y el botón izquierdo del ratón y arrastrando el ratón. Cuando la operación de arrastre empieza, el applet es eliminado de su contenedor padre (Applet o JApplet) y colocado en una nueva ventana de nivel superior no decorada (Frame o JFrame). Un pequeño botón de Cerrar flotante se muestra próximo al applet arrastrado. Cuando se hace click sobre el botón Cerrar flotante, el applet es devuelto al navegador. Los applets de Java que pueden ser arrastrados fuera del navegador se denominarán en lo sucesivo como applets arrastrables.

Puede personalizar el comportamiento de un applet arrastrable de las siguientes formas:

  • Puede cambiar la secuencia de teclas y el botón del ratón que se usan para arrastrar el applet fuera del navegador.
  • Puede agregar un acceso directo de escritorio que pueda ser usado para lanzar su aplicación fuera del navegador.
  • Puede definir cómo debería el applet ser cerrado después de que ha sido arrastrado fuera del navegador

Las secciones siguientes describen cómo implementar y personalizar un applet arrastrable. La clase MenuChooserApplet es usada para demostrar el desarrollo y despliegue de los applets arrastables. Abra AppletPage.html en un navegador para ver el applet Menu Chooser en una nueva página.

Nota:  Si no ve el applet ejecutarse, necesita instalar al menos la versión Java SE Development Kit (KDJ) 6 update 10.
Nota:  Si no ve el ejemplo ejecutándose, podría necesitar habilitar el intérprete de JavaScript en su navegador de forma que el script de las Herramientas de Despliegue pueda funcionar adecuadamente.

Habiliar la Capacidad para Arrastrar un Applet

Puede habilitar la capacidad de arrastrar un applet estableciendo el parámetro draggable a true cuando despliega el applet, como se muestra en el trozo de código: 

<script src="https://www.java.com/js/deployJava.js"></script>
<script>
    var attributes = { code:'MenuChooserApplet', width:900, height:300 };
    var parameters = { jnlp_href: 'draggableapplet.jnlp', draggable: 'true' };
    deployJava.runApplet(attributes, parameters, '1.6');
</script>

Cambiar la Secuencia de Teclas y el Botón del Ratón que se Utiliza para Arrastrar un Applet

Puede cambiar la secuencia de teclas y el botón del ratón que se usa para arrastrar un applet implementando el método isAppletDragStart- En el siguiente trozo de código, el applet puede ser arrastrado presionando el botón izquierdo del ratón y arrastrando el ratón-: 

 public boolean isAppletDragStart(MouseEvent e) {
        if(e.getID() == MouseEvent.MOUSE_DRAGGED) {
            return true;
        } else {
            return false;
        }
    }

Habilitar la Adición de un acceso directo de escritorio cuando el applet se desconecta del navegador

Si el usuario cierra la ventana del navegador o navega fuera de la página después de arrastrar un applet fuera de la página, se dice que el applet está desconectado del navegador. Puede crear un acceso directo de escritorio para applet cuando el applet está desconectado del navegador. El acceso directo de escritorio puede ser utilizado para lanzar la aplicación fuera del navegador. Para habilitar la creación de un acceso directo de escritorio, agregue las etiquetas offline-allowed y shortcut al fichero JNLP del applet. 

<information>
    <!-- ... -->
    <offline-allowed />
    <shortcut online="false">
        <desktop />
    </shortcut>
</information>
Nota: Dependiendo del ajuste para la Creación de Accesos Directos en el Panel de Control de Java, es posible que se solicite confirmación al usuario antes de crear el acceso directo.

Definir Cómo Debe Ser Cerrado el Applet

Puede definir cómo debe ser cerrado un applet. Por ejemplo, su applet Swing podría tener un JButton para cerrar el applet en vez de confiar en el botón de cierre flotante predeterminado.

El software del Complemento de Java le da al applet una instancia de la clase ActionListener. Esta instancia de la clase ActionListener, también referida como el oyente de cierre, puede ser usada para modificar el comportamiento por defecto del applet al cerrar.

Para definir cómo el applet debería ser cerrado, implemente los métodos setAppletCloseListener y appletRestored en su applet.

En el trozo de código siguiente, la clase MenuChooserApplet recibe el oyente de cierre y lo pasa a la instancia de la clase MenuItemChooser: 

MenuItemChooser display = null;
// ...
display = new MenuItemChooser();
// ...
public void setAppletCloseListener(ActionListener cl) {
    display.setCloseListener(cl);
}

public void appletRestored() {
    display.setCloseListener(null);
}

La clase MenuItemChooser es responsable de controlar el interfaz de usuario del applet. La clase MenuItemChooser define un JButton etiquetado "Close.". El código siguiente es ejecutado cuando el usuario pulsa este botón Cerrar: 

private void close() {
    // invoca actionPerformed del closeListener recibido
    // del software del Complemento de Java
    if (closeListener != null) {
        closeListener.actionPerformed(null);
    }
}

Solicitud y Personalización de la Decoración de Applets.

Comenzando en la versión de Java SE 7, cuando despliega un applet, puede especificar que la ventana del applet arrastrable esté decorada con un título de ventana por defecto o personalizado.

Para habilitar la decoración de la ventana de un applet arrastrable, especifique el parámetro java_decorated_frame con un valor de "true". Para habiliar un título de ventana personalizado, especifique el parámetro java_applet_title también. El valor de este parámetro será el texto del título de la ventana. 

<script src="https://www.java.com/js/deployJava.js"></script>
<script>
    var attributes =
      { code:'SomeDraggableApplet', width:100, height:100 };
    var parameters =
      { jnlp_href: 'somedraggableapplet.jnlp', 
          java_decorated_frame: 'true',
          java_applet_title: 'Un Título Personalizado'
      };
    deployJava.runApplet(attributes, parameters, '1.7');
</script>

Los parámetros java_decorated_frame y java_applet_title pueden también ser especificados en el fichero JNLP del applet como se muestra en el siguiente trozo de código: 

<applet-desc main-class="SayHello" name="main test" height="150" width="300">
    <param name="java_decorated_frame" value="true" />
    <param name="java_applet_title" value="" />
</applet-desc>
