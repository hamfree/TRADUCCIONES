Invocando Métodos del Applet Desde Código JavaScript

El código de JavaScript en una página web puede interactuar con los applets de Java embebidos en la página. El código JavaScript puede realizar operaciones tales como siguen:

  • Invocar métodos de los objetos Java
  • Obtener y establecer campos en objetos de Java
  • Obtener y establecer elementos de matrices de Java

La Especificación LiveConnect describe los detalles de cómo el código de JavaScript se comunica con el código de Java.

Se muestran advertencias de seguridad cuando el código JavaScript realiza llamadas a un applet de Java. Para suprimir estos avisos, agregue el atributo Caller-Allowable-Codebase al manifiesto del fichero JAR. Especifique la ubicación del código JavaScript que tiene permitido hacer llamadas al applet. Vea Atributos de manifiesto de archivos JAR para seguridad para obtener información sobre el atributo Caller-Allowable-Codebase.

Este tópico explora la comunicación del código de JavaScript con el applet de Java usando el ejemplo del applet Math. La clase MathApplet y la clase de soporte Calculator suministran un conjunto de métodos públicos y variables. El código de JavaScript en la página web invoca y evalúa estos miembros públicos para pasar datos y recuperar los resultados calculados.

Applet Math y Clases Relacionadas

Aquí está el código fuente para la clase MathApplet. El método getCalculator devuelve una referencia a la clase de ayuda Calculator. 

 
package jstojava;
import java.applet.Applet;

public class MathApplet extends Applet{

    public String userName = null;
         
    public String getGreeting() {
        return "Hello " + userName;
    }
    
    public Calculator getCalculator() {
        return new Calculator();
    } 
    
    public DateHelper getDateHelper() {
        return new DateHelper();
    }
    
    public void printOut(String text) {
        System.out.println(text);
    }
}

Los métodos en la clase Calculator permiten al usuario establecer dos valores, agregar números, y recuperar los números en un rango. 


package jstojava;

public class Calculator {
    private int a = 0;
    private int b = 0; // se asume que b > a
    
    public void setNums(int numA, int numB) {
        a = numA;
        b = numB;
    }
    
    public int add() {
        return a + b;
    }
    
    public int [] getNumInRange() {    
        int x = a;
        int len = (b - a) + 1;
        int [] range = new int [len];
        for (int i = 0; i < len; i++) {
            range[i]= x++;
            System.out.println("i: " + i + " ; range[i]: " + range[i]);
        }
        return range;
    }
}

El método getDate de la clase DateHelper devuelve la fecha actual. 


package jstojava;
import java.util.Date;
import java.text.SimpleDateFormat;

public class DateHelper {
    
    public static String label = null;
        
    public String getDate() {
        return label + " " + new SimpleDateFormat().format(new Date());
    }

}

Desplegar el Applet

Desplegar el applet en una página web, AppletPage.html Cuando despliegue el applet, asegúrese de que especifica un id para el applet. El id del applet es usado más tarde para obtener una referencia al objeto applet. 

<script src=
  "https://www.java.com/js/deployJava.js"></script>
<script>
    <!-- el id de un applet puede ser usado para obtener una referencia al
    objeto applet -->
    var attributes = { id:'mathApplet',
        code:'jstojava.MathApplet',  width:1, height:1} ;
    var parameters = { jnlp_href: 'math_applet.jnlp'} ;
    deployJava.runApplet(attributes, parameters, '1.6');
</script>

A continuación, agregue algún código de JavaScript a la página web AppletPage.html. El código JavaScript puede usar el id del applet como una referencia al objeto applet e invocar los métodos del applet. En el ejemplo mostrado a continuación, el código de JavaScript establece las variables de los miembros públicos del applet, invoca los métodos públicos, y recupera una referencia a otro objeto referenciado por el applet (Calculator). El código JavaScript es capaz de manejar tipos primitivos, de matriz y de devolución de objetos. 

<script language="javascript">
    function enterNums(){
        var numA = prompt('Introduzca número  \'a\'?','0');
        var numB = prompt(
            'Introduzca número \'b\' (debería ser mayor que el número \'a\' ?','1');
        // establece la variable pública del applet
        mathApplet.userName = "John Doe";

        // invoca métodos públicos del applet
        var greeting = mathApplet.getGreeting();

        // obtiene otra clase referenciada por el applet e
        // invoca sus métodos
        var calculator = mathApplet.getCalculator();
        calculator.setNums(numA, numB);

        // tipo de datos primitivo devuelto por el applet
        var sum = calculator.add();

        // matriz devuelta por el applet
        var numRange = calculator.getNumInRange();

        // compruebe el log de la consola de Java para este mensaje
        mathApplet.printOut("Comprobando la impresión a System.out");

        // obtiene otra clase, establece un campo estático e invoca sus métodos
        var dateHelper = mathApplet.getDateHelper();
        dateHelper.label = "La fecha de hoy es: ";
        var dateStr = dateHelper.getDate();
        <!-- ... -->
</script>

Abra AppletPage.html en un navegador para ver el applet Math.

Nota:  Si no ve el applet ejecutándose, necesita instalar al menos la versión Java SE Development Kit (KDJ) 6 update 10.
Nota:  Si no ve el ejemplo ejecutándose, podría necesitar habilitar el intérprete de JavaScript en su navegador de forma que el script de Herramientas de Despliegue pueda funcionar adecuamente.

Compruebe las restricciones de seguridad colocadas en los applets invocados por el código de JavaScript.
