Respuestas a Preguntas y Ejercicios: Applets

Preguntas

  1. Pregunta: ¿Qué clases puede un applet extender?

    Respuesta: Un applet puede extender la clase java.applet.Applet o la clase java.swing.JApplet.

    La clase java.applet.Applet extiende la clase java.awt.Panel y le habilita a usar las herramientas del IGU en el paquete AWT.

    La clase java.swing.JApplet es una subclae de java.applet.Applet que también le habilita a usar las herramientas del IGU de Swing.

  2. Pregunta: ¿Para qué usa el método start()?

    Respuesta: Usted usa el método start() cuando el applet debe realizar una tarea después de que es inicializado, pero antes de recibir la entrada del usuario. El método start() realiza el trabajo del applet o (lo más probable) inicia uno o más subprocesos para realizar el trabajo.

  3. Pregunta: Verdadero o falso: Un applet puede hacer conexiones de red a cualquier máqina en internet.

    Respuesta: Falso: Un applet puede sólo conectar al anfitrión de donde vino.

  4. Pregunta: ¿Cómo obtiene el valor de un parámetro especificado en el fichero JNLP desde dentro del código de un applet?

    Respuesta: Usted usa el método getParameter("Nombre de parámetro"), el cual devuelve el valor como cadena del parámetro.

  5. Pregunta: ¿Qué clases habilitan a los applets a interactuar con el código de JavaScript en la página web del applet?

    Respuesta: La clase netscape.javascript.JSObject habilita a los applets a interactura con el código JavaScript en la página web del applet.

  6. Pregunta: Verdadero o falso: Los applets pueden modificar los contenidos de la página web padre.

    Respuesta: Verdadero: Los applets pueden modificar los contenidos de la página web padre usando el método getDocument de la clase com.sun.java.browser.plugin2.DOM y la IPA Común del MOD.

Ejercicios

  1. Ejercicio: La página web padre del applet Exercise tiene una variable JavaScript llamada memberId. Escriba el código para establecer el valor de memberId en "123489" en el método start del applet.

    Respuesta: 

    import java.applet.Applet;
import netscape.javascript.*; // agregue plugin.jar al
                              // classpath durante la
                              // compilación

public class Exercise extends Applet {
    public void start() {
        try {
            JSObject window =
                JSObject.getWindow(this);
            window.setMember("memberId", "123489");
        } catch (JSException jse) {
            jse.printStackTrace();
        }
    }
}
