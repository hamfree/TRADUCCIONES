Lección : Applets de Java

Esta lección muesra lo báisco de los applets de Java, cómo desarrollar applets que interatúan extensivamente con su entorno, y cómo implantar applets.

Un applet de Java es una clase especial de progrmaa Java que un navegador habilitado con tecnología Java puede descargar de internet y ejecutar. Un applet está tipicamente embebido dentro de una página web y se ejecuta en el contexto de un navegador. Un applet debe ser una subclase de la clase java.applet.Applet. La clase Applet ofrece el interfaz estandar entre el applet y el entorno del navegador.

Swing ofrece una subclase especial de la clase Applet llamada javax.swing.JApplet. La clase JApplet debería ser utilizada por todos los applets que usan componentes de Swing para construir sus interfaces gráficos de usuario (GUIs).

El software del Plug-in Java del navegador gestiona el ciclo de vida de un applet.

Utilice un servidor web para probar los ejemplos en esta lección. El uso de applets locales no es recomendado, y los applets locales son bloqueados cuando la configuración del nivel de seguridad en el Panel de Control de Java está fijada en Alta o Muy Alta.

Nota: Algunas funcionalidades de los applets Java pueden no trabajar como está descrito en el SO de Mac. Esto es por la forma en que el software del Plug-in de Java se comunica con los navegadores en el SO de Mac.
Nota:  Por favor, asegúres de tener al menos la versión Java SE Development Kit (KDJ) 6 update 10 en su máquina cliente antes de continuar. Necesitará esto para ver los ejemplos de las aplicaciones ricas de internet y leer las secciones posteriores sin interrupciones.
