Invocando Código JavaScript Desde un Applet

Los applets de Java pueden invocar funciones de JavaScript presentes en la misma página web que el applet. La Especificación LiveConnect describe los detalles sobre cómo el código de JavaScript comunica con el código Java.

La clase netscape.javascript.JSObject habilita a los applets de Java el recuperar una referencia a los objetos de JavaScript e interactuar con la página web. El applet Data Summary (Resumen de Datos) descrito a continuación invoca código de JavaScript para recuperar información de la página web y escribe un resumen de datos de vuelta en la página web.

Supongamos que tiene una página web con algunas funciones JavaScript. El ejemplo AppletPage.html tiene funciones JavaScript para recuperar la edad, la dirección, y números de teléfono. Hay también una variable llamada userName que no tiene valor al principio. 

<head>
<title>Página del Applet Resumen de Datos - LiveConnect Java a JavaScript </title>
<meta http-equiv="Content-Type" content="text/html; charset=windows-1252"/>
<script language="javascript">
    var userName = "";
    
    // devuelve un número
    function getAge() { 
        return 25;
    }
    // devuelve un objeto
    function address() { 
        this.street = "1 Example Lane";
        this.city = "Santa Clara";
        this.state = "CA";
    }
    // devuelve un número
    function getPhoneNums() { 
        return ["408-555-0100", "408-555-0102"];
    } 
    function writeSummary(summary) {
        summaryElem =
            document.getElementById("summary");
        summaryElem.innerHTML = summary;
    }
    </script>

    <!-- ... -->      
</head>
<body>
    <script src =
      "https://www.java.com/js/deployJava.js"></script>
    <script> 
        <!-- ... -->
        deployJava.runApplet(attributes, parameters, '1.6'); 
    </script>          
    <!-- ... -->
    <p id="summary"/>  // este elemento HTML contiene
                             // el resumen
    <!-- ... -->
</body>

A continuación, considere una clase de applet llamado DataSummaryApplet. La clase DataSummaryApplet realiza las siguientes operaciones.

  • Invoca el método setMember de JSObject para establecer la variable userName a "John Doe".
  • Recupera la edad, la dirección, y los números de teléfono y construye una cadena conteniendo un resumen de estos datos.
  • Invoca la función de JavaScript writeSummary para escribir el resumen de vuelta en la página web.

Este applet necesita primero recuperar una referencia a JSObject como sigue: 

...
JSObject window = JSObject.getWindow(this);
...

Coloca la sentencia precedente en un bloque try ...catch.. para manejar netscape.javascript.JSException.

Ahora que el applet tiene una referencia a JSObject, puede invocar las funciones JavaScript relevantes usando los métodos eval y call de JSObject. 


package javatojs;

import java.applet.Applet;
import netscape.javascript.*; // agregue plugin.jar al classpath durante la compilación

public class DataSummaryApplet extends Applet {
    public void start() {
        try {
            JSObject window = JSObject.getWindow(this);

            String userName = "John Doe";

            // establece la variable de JavaScript
            window.setMember("userName", userName);

            // invoca la función de JavaScript
            Number age = (Number) window.eval("getAge()");

            // obtiene un objeto JavaScript y recupera su contenido
            JSObject address = (JSObject) window.eval("new address();");
            String addressStr = (String) address.getMember("street") + ", " +
                    (String) address.getMember("city") + ", " +
                    (String) address.getMember("state");

            // obtiene una matriz de JavaScript y recupera sus contenidos
            JSObject phoneNums = (JSObject) window.eval("getPhoneNums()");
            String phoneNumStr = (String) phoneNums.getSlot(0) + ", " +
                    (String) phoneNums.getSlot(1);

            // cambia dinámicamente el HTML de la página; escribe el resumen de datos
            String summary = userName + " : " + age + " : " +
                    addressStr + " : " + phoneNumStr;
            window.call("writeSummary", new Object[] {summary})   ;
        } catch (JSException jse) {
            jse.printStackTrace();
        }
    }
}

Para compilar código de Java que hace una referencia a clases en el paquete netscape.javascript, incluya <su camino al KDJ>/jre/lib/plugin.jar a su classpath. En tiempo de ejecución, el software del Complemento de Java hace que automáticamente estén disponibles estas clases a los applets.

El applet Data Summary (Resumen de Datos) visualiza el resultado siguiente en la página web: 

Resultado de las llamadas del applet de Java a JavaScript en esta página
                
John Doe : 25 : 1 Example Lane, Santa Clara, CA : 408-555-0100, 408-555-0102

Abra AppletPage.html en un navegador para ver el applet Data Summary.

Nota:  Si no ve el applet ejecutándose, necesita instalar al menos la versión Java SE Development Kit (KDJ) 6 update 10.
Nota:  Si no ve el ejemplo ejecutándose, podría necesitar habilitar el intérprete de JavaScript en su navegador de forma que el script de Herramientas de Despliegue pueda funcionar adecuamente.
