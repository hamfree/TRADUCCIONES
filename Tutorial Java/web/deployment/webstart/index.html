Lección: Java Web Start

El software de Java Web Start ofrece el poder de lanzar aplicaciones con todas las funciones con un simple click. Los usuarios pueden descargar y lanzar aplicaciones, tales como un programa completo de hoja de cálculo o un cliente de chat de Internet, sin pasar por largos procedimientos de instalación.

Con el software de Java Web Start, los usuarios pueden lanzar una aplicación de Java pulsando un enlace en una página web. El enlace apunta a un fichero de Protocolo de Lanzamiento en Red de Java (JNLP), el cual instruye al software de Java Web Start a descargar, guardar en caché, y ejecutar la aplicación.

El software de Java Web Start brinda a los desarrolladores y usuarios de Java muchas ventajas de implementación:

  • Con el software de Java Web Start, puede colocar una única aplicación Java en un servidor web para su implementación en una amplia variedad de plataformas, incluyendo Windows, Linux, y Solaris.
  • El software de Java Web Start admite múltiples versiones simultáneas de la plataforma Java. Una aplicación puede requerir una versión específica del software del Entorno de Ejecución de Java sin entrar en conflicto con las necesidades de otras aplicaciones.
  • Los usuarios pueden crear un acceso directo en el escritorio para iniciar una aplicación de Java Web Start fuera del navegador.
  • El software de Java Web Start aprovecha la seguridad inherente de la plataforma Java. Por defecto, las aplicaciones tienen acceso restringido al disco local y a los recursos de red.
  • Las aplicaciones iniciadas con el software Java Web Start se almacenan en caché localmente para mejorar el rendimiento.
  • Las actualizaciones a las aplicaciones de Java Web Start son automáticamente descargadas cuando la aplicación es ejecutada de forma independiente desde el escritorio del usuario.

El software de Java Web Start es instalado como parte del software del EEJ. Los usuarios no tienen que instalar el software de Java Web Start separadamente o realizar tareas adicionales para usar las aplicaciones de Java Web Start.
