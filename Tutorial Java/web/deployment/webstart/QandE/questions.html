Preguntas y Ejercicios: Java Web Start

Preguntas

  1. En un enlace que está para ejecutar una aplicación de Java Web Start, ¿Qué fichero es especificado como atributo href de la etiqueta a?

  2. ¿Qué tipo MIME debe reconocer un servidor Web para que pueda alojar aplicaciones Java Web Start?

  3. En el fichero JNLP de una aplicación, ¿qué dos elementos deben ser especificados dentro del elemento resources?

  4. ¿Qué interfaz ofrece la habilidad de controlar cómo son cacheados los recursos de la aplicación Java Web Start?

    1. BasicService
    2. DownloadService
    3. PersistenceService
    4. ExtendedService

  5. Verdadero o Falso: las aplicaciones Java Web Start se ejecutan en cajas de arena seguras por defecto.

  6. Verdadero o Falso: Si una aplicación Java Web Start se está ejecutando en una caja de arena segura, los ficheros JAR para la aplicación pueden residir en servidores diferentes.

  7. Para que una aplicación Java Web Start soporte operaciones fuera de la caja de arena segura, ¿qué tenemos que hacer?

Ejercicios

  1. Escriba el código XML que agregaría a un fichero JNLP para solicitar que la aplicación tenga acceso completo al sistema cliente.

  2. Para una aplicación Java Web Start, tiene usted dos iconos, one.gif y two.gif , en el directorio images en un fichero JAR. Escriba el código de la aplicación que usaría para acceder a estas imágenes.

Compruebe sus respuestas.
