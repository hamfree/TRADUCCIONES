Configurando un Servidor Web

Es posible que deba configurar su servidor web para manejar archivos de Protocolo de Inicio de Red Java (JNLP). Si el servidor web no está configurado apropiadamente, la aplicación Java Web Start no se lanzará cuando pulse en el enlace al fichero JNLP.

Configure el servidor web de forma que los ficheros con la extensión .jnlp estén configurados al tipo MIME application/x-java-jnlp-file

Los pasos específicos para configurarel tipo MIME JNLP variarán dependiendo del servidor web. Como un ejemplo, para configurar un servidor web Apache, debería agregar la siguiente línea al fichero mime.types. 

application/x-java-jnlp-file JNLP

Para otros servidores web, compruebe la documentación por instrucciones para el establecimiento de tipos MIME.
