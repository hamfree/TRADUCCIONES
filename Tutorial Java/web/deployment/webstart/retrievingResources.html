Recuperando Recursos

Use el método getResource para leer recursos de un fichero JAR. Por ejemplo, el código siguiente recupera imágenes de un fichero JAR. 

// Obtenemos el classloader actual
ClassLoader cl = this.getClass().getClassLoader();
// Creamos los iconos
Icon saveIcon  = new ImageIcon(cl.getResource("images/save.gif"));
Icon cutIcon   = new ImageIcon(cl.getResource("images/cut.gif"));

El ejemplo asume que las siguientes entradas existen en el fichero JAR de la aplicación:

  • images/save.gif
  • images/cut.gif
