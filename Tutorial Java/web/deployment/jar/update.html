Actualizando un Fichero JAR

La herramienta Jar ofrece una opción u la cual puede usar para actualizar los contenidos de un fichero JAR existente modificando su manifiesto o agregando ficheros.

El comando básico para agregar ficheros tiene este formato: 

jar uf fichero-jar fichero-entrada(s)

En este comando:

  • La opción u indica que quiere actualizar un fichero JAR existente.
  • La opción f indica que el fichero JAR a actualizar se especifica en la línea de comandos.
  • fichero-jar es el fichero JAR existente que va a ser actualizado.
  • fichero-entrada(s) es una lista delimitada por espacios de uno o más ficheros que quiere agregar al fichero JAR.

Se sobrescribirán los archivos que ya estén en el archivo que tengan el mismo nombre de ruta que el archivo que se agrega.

Cuando crea un fichero JAR nuevo, puede opcionalmente usar la opción -C para indicar un cambio de directorio. Para más información, vea la sección Creando un Fichero JAR.

Ejemplos

Recuerde que TicTacToe.jar tiene estos contenidos: 

META-INF/MANIFEST.MF
TicTacToe.class
TicTacToe.class
TicTacToe.java
audio/
audio/beep.au
audio/ding.au
audio/return.au
audio/yahoo1.au
audio/yahoo2.au
example1.html
images/
images/cross.gif
images/not.gif

Suponga que quiere agregar el fichero images/new.gif al fichero JAR. Puede lograrlo emitiendo este comando desde el directorio principal del directorio images: 

jar uf TicTacToe.jar images/new.gif

El archivo JAR revisado tendría esta tabla de contenido: 

META-INF/MANIFEST.MF
TicTacToe.class
TicTacToe.class
TicTacToe.java
audio/
audio/beep.au
audio/ding.au
audio/return.au
audio/yahoo1.au
audio/yahoo2.au
example1.html
images/
images/cross.gif
images/not.gif
images/new.gif

Puede usar la opción -C para "cambiar directorios" durante la ejecución de comando. Por ejemplo: 

jar uf TicTacToe.jar -C images new.gif

Este comando cambiaría al directorio images antes de agregar new.gif al archivo JAR. El directorio images no se incluiría en la ruta de acceso de new.gif cuando se agrega al archivo, lo que da como resultado una tabla de contenido similar a esta: 

META-INF/MANIFEST.MF
META-INF/MANIFEST.MF
TicTacToe.class
TicTacToe.class
TicTacToe.java
audio/
audio/beep.au
audio/ding.au
audio/return.au
audio/yahoo1.au
audio/yahoo2.au
example1.html
images/
images/cross.gif
images/not.gif
new.gif
