Usando Ficheros JAR: Lo Básico

Los ficheros JAR son empaquetados con el formato de ficheros ZIP, de forma que puede usarlos para tareas tales como compresión de datos sin pérdida, archivado, descompresión, y desempaquetado de archivos. Estas tareas se encuentran entre los usos más comunes de los archivos JAR, y puede obtener muchos beneficios de los archivos JAR utilizando solo estas características básicas.

Incluso si usted quiere aprovecharse de la funcionalidad avanzada ofrecias por el formato de ficheros Jar tal como el firmado electrónico, primero necesitará familiarizarse con las operaciones fundamentales.

Para realizar las tareas básicas con los ficheros JAR, use la Herramienta de Archivado de Java ofrecida como parte del Kit de Desarrollo de Java (KDJ). Debido a que la herramienta de Archivado de Java es invocada usando el comando jar, este tutorial se refiere a ella como 'la herramienta Jar'.

Como sinopsis y vista previa de algunos de los temas que se tratarán en esta sección, la siguiente tabla resume las operaciones comunes de archivos JAR:

Operaciones Comunes con ficheros JAR
Operación Comando
Para crear un fichero JAR jar cf jar-file fichero(s)-de-entrada
Para ver los contenidos de un fichero JAR jar tf fichero-jar
Para extraer los contenidos de un fichero JAR jar xf fichero-jar
Para extraer ficheros específicos de un fichero JAR jar xf fichero-jar fichero(s)-archivado(s)
Para ejecutar una aplicación empaquetada como un fichero JAR (requiere una cabecera de manifiesto Main-class) java -jar app.jar
Para invocar un applet empaquetado como un fichero JAR 
<applet code=AppletClassName.class
        archive="JarFileName.jar"
        width=width height=height>
</applet>
