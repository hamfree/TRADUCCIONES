Configurando la Información de la Versión del Paquete

Es posible que necesite incluir información sobre la versión del paquete en el manifiesto de un fichero JAR. Proporciona esta información con los siguientes encabezados en el manifiesto:

Encabezados en un manifiesto
Encabezado Definición
Name El nombre de la especificación.
Specification-Title El título de la especificación.
Specification-Version La versión de la especificación.
Specification-Vendor El vendedor de la especificación.
Implementation-Title El título de la implementación.
Implementation-Version El número de compilación de la implementación.
Implementation-Vendor El vendedor de la implementación.

Se puede asignar un conjunto de tales encabezados a cada paquete. Los encabezados de versiones deben aparecer directamente debajo del encabezado Name para el paquete. Este ejemplo muestra todos los encabezados de versión: 

Name: java/util/
Specification-Title: Java Utility Classes
Specification-Version: 1.2
Specification-Vendor: Example Tech, Inc.
Implementation-Title: java.util
Implementation-Version: build57
Implementation-Vendor: Example Tech, Inc.

Para más información sobre los encabezados de versionado de paquetes, vea la Especificación de Versionado de Paquetes.

Un ejemplo

Queremos incluir los encabezados en el ejemplo de encima en el manifiesto de MyJar.jar.

Primero creamos un fichero de texto llamado Manifest.txt con los contenidos siguientes: 

Name: java/util/
Specification-Title: Java Utility Classes
Specification-Version: 1.2
Specification-Vendor: Example Tech, Inc.
Implementation-Title: java.util 
Implementation-Version: build57
Implementation-Vendor: Example Tech, Inc.
Advertencia: El fichero de texto debe terminar con una nueva línea o retorno de carro. La última línea no será analizada apropiadamente si no termina con una nueva línea o retorno de carro.

Entonces creamos un fichero JAR llamado MyJar.jar introduciendo el comando siguiente: 

jar cfm MyJar.jar Manifest.txt MyPackage/*.class

Esto crea el fichero JAR con un manifiesto con los contenidos siguientes: 

Manifest-Version: 1.0
Created-By: 1.7.0_06 (Oracle Corporation)
Name: java/util/
Specification-Title: Java Utility Classes
Specification-Version: 1.2
Specification-Vendor: Example Tech, Inc.
Implementation-Title: java.util 
Implementation-Version: build57
Implementation-Vendor: Example Tech, Inc.
