Configurando el Punto de Entrada de la Aplicación

Si tiene una aplicación incluída en un fichero JAR, necesita alguna forma de indicar qué clase dentro del fichero JAR es el punto de entrada de su aplicación. Proporcione esta información con la cabecera Main-Class en el manifiesto, que tiene la forma general: 

Main-Class: nombreclase

El valor nombreclase es el nombre de la clase que es el punto de entrada de su aplicación.

Recuerde que el punto de entrada es la clase que tiene un método con la firma public static void main(String[] args).

Después de que ha fijado la cabecera Main-Class en el manifiesto, ejecuta el archivo JAR utilizando la forma siguiente del comando java: 

java -jar nombre-JAR

El método main de la clase especificada en la cabecera Main-Class es ejecutado.

Un Ejemplo

Queremos ejecutar el método main en la clase MyClass en el paquete MyPackage cuando ejecutamos el fichero JAR.

Primero creamos un fichero de texto llamado Manifest.txt con los contenidos siguientes: 

Main-Class: MyPackage.MyClass
Advertencia: El fichero de texto debe terminar con una línea nueva o retorno de carro. La última línea no será analizada apropiadamente si no termina con una línea nueva o retorno de carro.

Entonces creamos un fichero JAR llamado MyJar.jar introduciendo el comando siguiente: 

jar cfm MyJar.jar Manifest.txt MyPackage/*.class

Esto crea el fichero JAR con un manifiesto con los contenidos siguientes: 

Manifest-Version: 1.0
Created-By: 1.7.0_06 (Oracle Corporation)
Main-Class: MyPackage.MyClass

Cuando ejecuta el fichero JAR con el comando siguiente, el método main de MyClass se ejecuta: 

java -jar MyJar.jar

Establecer un Punto de Entrada con la Herramienta JAR

La bandera 'e' (por 'punto de entrada') crea o sobreescribe el atributo del manifiesto Main-Class. Pued ser usada mientras se crea o actualiza un fichero JAR. Úsela para especificar el punto de entrada de la aplicación sin editar o crear el fichero de manifiesto.
Por ejemplo, este comando crea app.jar donde el valor del atributo Main-Class en el manifiesto es establecido a MyApp: 

jar cfe app.jar MyApp MyApp.class

Puede invocar directamente esta aplicación ejecutando el comando siguiente: 

java -jar app.jar

Si el nombre de la clase de punto de entrada está en un paquete se puede usar un '.' (punto) carácter como delimitador. Por ejemplo, si Main.class está en un paquete llamado foo el punto de entrada puede ser especificado de las siguientes formas: 

jar cfe Main.jar foo.Main foo/Main.class
