Extrayendo el Contenido de un Fichero JAR

El comando básico a usar para extraer los contenidos de un fichero JAR es: 

jar xf jar-file [fichero-archivado(s)]

Veamos las opciones y argumentos en este comando:

  • La opción x indica que quiere extraer ficheros del archivo JAR.
  • Las opciones f indican que el fichero JAR del cual serán extraídos los ficheros está especificado en la línea de comandos, en lugar de a través de la entrada estándar.
  • El argumento fichero-archivado(s) es el nombre de un fichero (o una ruta y nombre de fichero) de un fichero JAR del cual vamos a extraer ficheros.
  • ficheros-archivados(s) es un argumento opcional que consiste de una lista de ficheros separados por espacios que van a ser extraídos del archivo. Si este argumento no está presente, la herramienta Jar extraerá todos los ficheros en el archivo.

Como siempre, el orden en el cual las opciones x y f aparezcan en el comando no importa, pero no deber espacios entre ellos.

Cuando extraiga ficheros, la herramienta Jar hace copias de los ficheros deseados y los escriben en el directorio actual, reproduciendo la estructura de directorios que los ficheros tienen en el archivo. El fichero JAR original permanece sin cambios.

Precaución: Cuando extrae archivos, la herramienta Jar sobrescribirá los archivos existentes que tengan el mismo nombre de ruta que los archivos extraídos.

Un Ejemplo

Extraigamos algunos ficheros del fichero JAR TicTacToe que hemos estado usando en las secciones previas. Recuerde que los contenidos de TicTacToe.jar son: 

META-INF/MANIFEST.MF
TicTacToe.class
TicTacToe.class
TicTacToe.java
audio/
audio/beep.au
audio/ding.au
audio/return.au
audio/yahoo1.au
audio/yahoo2.au
example1.html
images/
images/cross.gif
images/not.gif

Suponga que quiere extraer el fichero de clases TicTacToe y el fichero de imagen cross.gif. Para hacerlo puede usar este comando: 

jar xf TicTacToe.jar TicTacToe.class images/cross.gif

Este comando hace dos cosas:

  • Coloca una copia de TicTacToe.class en el directorio actual.
  • Crea el directorio images, si aún no existe, y coloca una copia de cross.gif dentro.

El fichero JAR TicTacToe original permanece sin cambios.

Se pueden extraer tantos archivos como desee del archivo JAR de la misma manera. Cuando el comando no especifica qué ficheros extraer, la herramienta Jar extrae todos los ficheros en el archivo. Por ejemplo, puede extraer todos los ficheros en el archvio TitTacToe usando este comando: 

jar xf TicTacToe.jar
