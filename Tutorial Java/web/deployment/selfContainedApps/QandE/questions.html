Preguntas y Ejercicios: Aplicaciones Autocontenidas

Preguntas

  1. ¿Cual es de los items siguients no es una ventaja de aplicaciones autocontenidas?

    1. Los usuarios instalan la aplicación vía un instalador que es familiar para ellos.
    2. La aplicación se ejecuta como una aplicación nativa.
    3. La aplicación requiere menos espacio en la máquina del usuario.
    4. Uste contrala versión del EEJ qu es utlizada por la aplicación.
    5. La aplicación no requiere de un navegador para ejecutarse.

  2. Verdadero o Falso: El tipo MIME debe ser siempre usado par configurar una asociación de fichero.

  3. ¿Qué elementos son usados para identificar los puntos de entrda para aplicaciones autocontenidas en la tarea de Ant <fx:deploy>?

Ejercicios

  1. Escriba la tarea Ant <fx:deploy> para generar un paquete Windows MSI para una aplicación simple nombrada Mi App de Ejemplo. El fichero JAR para la aplicación es en el directorio dist, la clase main es samples.MyApp, y ficheros de salida son escritos al directorio actual.

  2. Mejore el código desde el ejercicio anterior para crear paquetes para todos los instaladores de Windows y defina una asociación de ficheor para ficheros de texto.

Compruebe sus respuestas.
