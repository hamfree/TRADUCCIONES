Añadiendo una Librería Externa

Las Aplicaciones Autocontenidas incluyen todo lo que una aplicación necesita para ejecutar. Si tu aplicación requiere una librería externa, esa librería puede ser añadida al empaquetado para la aplicación. Añadir la libería puede ser hecho de diferentes formas.

La Demostración de Asociación de Fichero descrita en Usando Asociaciones de Fichero descarga la librería Groovy como parte del proceso de construcición. La librería se coloca en el directorio /lib en el proyecto de la aplicaicón. Este directorio es entonces copiado al directorio /dist desde el cual el empaquetado de la aplicación autocontenida es generado.

El siguiente código en la tarea -pre-init en el ficheor build.xml file muestra cómo la librería es descargada: 

<!-- descara y copia la librería groovy -->
<copy toFile="lib/groovy-all-2.3.8.jar">
    <resources>
      <url url="http://central.maven.org/maven2/org/codehaus/groovy/groovy-all/2.3.8/groovy-all-2.3.8.jar"/>
    </resources>
</copy>

Ver build.xml el código completo de construcción.

Usted puede descargar los ficheros fuentes para la Demostración Asociación de Fichero desde los Ejemplos de Aplicación Autocontenida.
