Lección: Haciendo Más con Aplicaciones Java Ricas de Internet

Los applets lanzados usando el Protocolo de Lanzamiento en Red de Java (JNLP) tienen capacidades similiares a aquellas aplicaciones de Java Web Start. Esta lección contiene temas comunes para el desarrollo y desplieuge de applets y aplicaciones Java Web Start (conocidas juntas como aplicaciones ricas de Internet). Si usted no está familiarizado con los applets o las aplicaciones Java Web Start, puede aprender más en las lecciones siguientes de los tutoriales de Java:

  • Para aprender más sobre el desarrollo del applet y su despliegue, visite la lección Applets de Java.
  • Para aprender más sobre el desarrollo y desplieuge de aplicaciones Java Web Start, visite la lección Java Web Start.
Nota:  Por favor asegúrese de que tiene al menos la versión Java SE Development Kit (KDJ) 6 update 10 en su máquina cliente antes de seguir adelante. Necesitará esto para ver aplicaciones ricas de internet de ejemplo y leer las secciones subsiguientes sin interrupción.
