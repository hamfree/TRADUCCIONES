Propiedades del Sistema

Este tema lista las propiedades del sistema que pueden ser acccedidas por las aplicaciones ricas de Internet (RIAs) que están restringidas a la caja de arena de seguridad y son lanzadas con o sin el Protocolo de Lanzamiento en Red de Java (JNLP). Algunas propiedades del sistema no pueden ser accedidas por RIAs en cajas de arena.

Propiedades Seguras del Sistema Accesibles por Todas las RIAs

Todas las RIAs pueden recuperar las propiedades del sistema seguras siguientes:

  • java.class.version
  • java.vendor
  • java.vendor.url
  • java.version
  • os.name
  • os.arch
  • os.version
  • file.separator
  • path.separator
  • line.separator

Propiedades Seguras del Sistema Accesibles por RIAs Lanzadas usando JNLP

Las RIAs lanzadas usando JNLP pueden establecer y recuperar las propiedades seguras siguientes:

  • awt.useSystemAAFontSettings
  • http.agent
  • http.keepAlive
  • java.awt.syncLWRequests
  • java.awt.Window.locationByPlatform
  • javaws.cfg.jauthenticator
  • javax.swing.defaultlf
  • sun.awt.noerasebackground
  • sun.awt.erasebackgroundonresize
  • sun.java2d.d3d
  • sun.java2d.dpiaware
  • sun.java2d.noddraw
  • sun.java2d.opengl
  • swing.boldMetal
  • swing.metalTheme
  • swing.noxp
  • swing.useSystemFontSettings

Propiedades del Sistema Prohibidas

Las RIAs en cajas de arena no pueden acceder a las propiedades del sistema siguientes:

  • java.class.path
  • java.home
  • user.dir
  • user.home
  • user.name
