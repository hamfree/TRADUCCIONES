Personalizando la Experiencia de Carga

Las Aplicaciones Ricas de Internet (RIAs, en inglés) pueden tomar unos pocos segundos en cargar dependiendo de factores tales como la velocidad de la red y los recursos requeridos por la ARI. Personalizar la experiencia de la carga de la ARI suministrando una pantalla de inicio o un indicador de progreso de carga personalizado para involucrar al usuario final durante el proceso de carga y para comunicar información de progreso medible.

Mientras que los mecanismos generales para personalizar la experiencia de carga para los applets y las aplicaciones Java Web Start son similares, hay diferencias sutiles en la sintaxis y en la implementación. Vea los temas siguientes con instrucciones paso a paso e información conceptual sobre la personalización de la experiencia de carga de la ARI:

  • Personalizando la Pantalla de Carga
  • Visualizando un Indicador de Progreso de Carga Personalizado para un applet
  • Visualizando un Indicador de Progreso de Carga Personalizado para una aplicación Java Web Start
  • Personalizando la Experiencia de Carga (información conceptual, capacidades del software cliente del Entorno de Ejecución de Java (EEJ))
