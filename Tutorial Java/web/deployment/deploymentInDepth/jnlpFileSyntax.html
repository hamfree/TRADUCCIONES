Estructura del Fichero JNLP

Este tema describe la sintaxis del fichero del Protocolo de Lanzamiento en Red de Java (JNLP) para las aplicaciones ricas de Internet (ARIs).

El trozo de código siguiente muestra un fichero JNLP de ejemplo para una aplicación Java Web Start: 

<?xml version="1.0" encoding="UTF-8"?>
<jnlp spec="1.0+" codebase="" href="">
    <information>
        <title>Demo Árbolo Dinámico</title>
        <vendor>Equipo Dinámico</vendor>
        <icon href="sometree-icon.jpg"/>
        <offline-allowed/>
    </information>
    <resources>
        <!-- Recursos de la Aplicación -->
        <j2se version="1.6+" href=
           "http://java.sun.com/products/autodl/j2se"/>
        <jar href="DynamicTreeDemo.jar"
            main="true" />

    </resources>
    <application-desc
         name="Aplicación Demo Árbol Dinámico"
         main-class="webstartComponentArch.DynamicTreeApplication"
         width="300"
         height="300">
     </application-desc>
     <update check="background"/>
</jnlp>

La tabla siguiente describe los elementos y atributos comunmente usados en los ficheros JNLP. Pulse el enlace padre para el ver el padre de un elemento.

Nota: Esta tabla no incluye todos los contenidos posibles del fichero JNLP. Para más información, vea Java Network Launching Protocol and IPA Specification Change Log .

Codificación de archivos JNLP

El software Java Web Start soporta la codificación de los ficheros JNLP en cualquiera de las codificaciones de caracteres soportadas por la plataforma Java. Para más información sobre la codificación de caracteres en la plataforma Java, vea la Guía de Codificaciones Soportadas. Para codificar un fichero JNLP, especifique una codificación en el prólogo XML de ese fichero. Por ejemplo, la línea siguiente indica que el fichero JNLP está codificado en UTF-16. 

<?xml version="1.0" encoding="utf-16"?>
Nota: El prólogo XML en sí mismo debe estar codificado en UTF-8.
