Cambiando el Botón de Lanzamiento

Usted puede cambiar la imagen del botón 'Launch' de su aplicación Java Web Start, si no le gusta el botón por defecto o si usted tiene otra imagen que ha estandarizado.

Use la variable deployJava.launchButtonPNG para que apunte a la ubicación de su imagen de botón 'Launch'.

Variable: deployJava.launchButtonPNG

Uso: Ofrecer una URL alternativa de imagen

En este ejemplo, el botón 'Launch' de la aplicación Bloc de Notas es ahora una imagen de Duke saludando. 

<script src="https://www.java.com/js/deployJava.js"></script>
<script>
    deployJava.launchButtonPNG='https://docs.oracle.com/javase/tutorial/images/DukeWave.gif';
    var url = "https://docs.oracle.com/javase/tutorialJWS/deployment/webstart/examples/Notepad.jnlp";
    deployJava.createWebStartLaunchButton(url, '1.6.0');
</script>

El nuevo botón 'Launch' (Duke saludando) de la aplicación Bloc de Notas a continuación. Pulse en la imagen de Duke para lanzar la aplicación Bloc de Notas.

Nota: Si no ve el ejemplo ejecutándos, es posible que deba habilitar el intérprete de JavaScript en su navegador de forma que el script de Herramientas de Despliegue pueda funcionar adecuadamente.
