Respuestas a Preguntas y Ejercicios: Despliegue en Profundidad

Preguntas

  1. Pregunta: ¿Qué script contiene funciones para desplegar applets y aplicaciones Java Web Start?

    Respuesta: El script de Herramientas de Despliegue contiene las funciones JavaScript que pueden ser usadas para desplegar applets y aplicaciones Java Web Start.

  2. Pregunta: Verdadero o Falso: Siempre debe firmar su ARI solo para asegurarse de que siempre funcione.

    Respuesta: Verdadero: Las ARIs no firmadas son bloqueadas por ranzones de seguridad. Siempre firme su ARI con un certificado válido de una autoridad de certificación de confianza.

Ejercicios

  1. Ejercicio: Escriba el código JavaScript para desplegar applet Ejercicio usando el fichero ex.jnlp.

    Respuesta: 

    <script src="https://www.java.com/js/deployJava.js"></script>
<script> 
    var attributes = {code:'Exercise', width:300, height:300} ; 
    var parameters = {jnlp_href: 'ex.jnlp'} ; 
    deployJava.runApplet(attributes, parameters, '1.6'); 
</script>
