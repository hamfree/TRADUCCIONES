Herramientas de Despliegue

El script de las Herramientas de Despliegue es un conjunto de funciones JavaScript que pueden ayudar a los desarrolladores a desplegar aplicaciones ricas de Internet (ARIs) de forma consistente entre varias configuraciones de navegadores y sistemas operativos. El script de las Herramientas de Despliegue evalúa el navegador subyacente y el sistema operativo, y despliega la ARI con el HTML correcto. Este script puede también asegurarse de que la versión requerida del software del Entorno de Ejecución de Java (EEJ) esté presente en la máquina cliente. El script de las Herramientas de Despliegue fue introducido en la versión de la Plataforma Java, Edición Estándar 6 actualización 10.

Ubicación del Script de las Herramientas de Despliegue

El script de las Herramientas de Despliegue se ubica en las siguientes direcciones de web:

  • http://www.java.com/js/deployJava.js (Vea nota.)
  • https://www.java.com/js/deployJava.js – Cuando despliegue su applet en una página segura, use el script de Herramientas de Despliegue desde esta ubicación segura para evitar los avisos de contenido mezclado cuando se carga la página.
Nota: La dirección web http://www.java.com/js/deployJava.js está siendo eliminada. Use la dirección web https://www.java.com/js/deployJava.js para lanzar todas las aplicaciones.

El código de JavaScript en esta ubicación ha sido minimizado de forma que pueda cargarse rápidamente. Puede ver la versión legible para humanos del código JavaScript con bloques de comentarios asociados en https://www.java.com/js/deployJava.txt .

Nota: El intérprete de JavaScript deberá estar habilitado en el navegador del cliente de forma que el script de las Herramientas de Despliegue puede ejecutarse y desplegar su ARI apropiadamente.
