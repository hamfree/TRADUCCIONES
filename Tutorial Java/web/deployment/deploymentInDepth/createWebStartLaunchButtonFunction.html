Desplegando una Aplicación Java Web Start

Usted puede desplegar aplicaciones Java Web Start usando la función createWebStartLaunchButton del script Herramientas de Despliegue. Las aplicaciones Java Web Start son lanzadas usando el Protocolo de Lanzamiento En Red de Java (JNLP). La función createWebStartLaunchButton genera un enlace (Etiqueta Ancla de HTML - <a>) al fichero JNLP de la aplicación Java Web Start.

Esta etiqueta ancla generada es el botón de las aplicaciones Java Web Start . Cuando el usuario final pulsa el botón Launch, el script de Herramientas de Despliegue se asegura que el softwara apropiado del Entorno de Ejecución de Java (EEJ) esté instalado y entonces lanza la aplicación Java Web Start.

Nota: Dependiendo del tipo de navegador, podría ser incapaz de ver el HTML generado por el script de Herramientas de Despliegue cuando intenta ver el fuente de la página web. Para ver el HTML generado, intente salvar la página HTML después de que haya sido cargada, o use una herramienta como Firebug (una extensión de Mozilla Firefox).
Nota: Si el ciente no tiene la mínima versión requerida del software del EEJ, el script de Herramientas de Despliegue redirige el navegador a http://www.java.com para permitir a los usuarios descargar el último software del EEJ.

Firma de la función: createWebStartLaunchButton: function(jnlp, minimumVersion) o createWebStartLaunchButton: function(jnlp)

Parámetros:

  • jnlp – La URL del fichero JNLP que contiene la información de despliegue para la aplicación Java Web Start. Esta URL debe ser una ruta absoluta.
  • minimumVersion – La versión mínima del software del EEJ requerida para ejecutar esta aplicación.

Uso:

  • Especificar una versión mínima del software del EEJ que se requierea para ejecutar la aplicación 
    <script src="https://www.java.com/js/deployJava.js"></script>
<script>
    var url = "http://java.sun.com/javase/technologies/desktop/javawebstart/apps/notepad.jnlp";
    deployJava.createWebStartLaunchButton(url, '1.6.0');
</script>
  • Habilitar que la aplicación Java Web Start se ejecute en cualquier versión del software del EEJ

    Use la función createWebStartLaunchButton: function(jnlp) si su aplicación no tiene un requisito de versión mínima del software del EEJ.

Nota: Cuando despliegua usando cualquiera de las funciones createWebStartLaunchButton descritas anteriormente, debe especificar un codebase absoluto en el fichero JNLP de la aplicación Java Web Start. Esto permite que la aplicación Java Web Start sea lanzada desde la línea de comandos con el comando javaws <path/to/local JNLP file>.
