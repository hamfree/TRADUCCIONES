Lección: Despliegue En Profundidad

El despliegue de las aplicaciones Ricas de Internet involucra diversas tecnologías y herramientas tales como el Protocolo de Lanzamiento en Red de Java (JNLP), Herramientas de Despliegue, pack200, jarsigner y otras. Esta lección explora cómo estas tecnologías y herramientas ayudan a los desarrolladores a desplegar ARIs (applets y aplicaciones Java Web Start).

Los componentes principales que están involucrados en el despliegue de ARIs son los siguientes:

  • La página HTML en la cual la ARI será desplegada
  • El fichero JNLP para la ARI
  • El fichero JAR contiene los ficheros de clases y recursos de la ARI

Estos componentes se mencionan en temas posteriores.
