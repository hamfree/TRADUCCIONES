Personalizando la Pantalla de Carga

Una pantalla de carga por defecto se visualiza cunado un applet está siendo cargado en la página web. Usted puede visualizar una pantalla de carga personalizada especficando los parámetros siguientes cuando despliegue el applet:

  • image - la imagen a ser mostrada en la pantalla de carga
  • boxbgcolor - el color de fondo del área en el cual el applet será visualizado
  • boxborder - si el applet tendrá un borde; por defecto es true
  • centerimage - la posición de la imagen; por defecto es false

La pantalla de carga puede visualizar una imagen estática o un gir animado.

El trozo de código de AppletPage.html muestra como personalizar la pantalla de carga para visualizar una animación de Duke, la mascota de Java. 

<script src="https://www.java.com/js/deployJava.js"></script>
<script> 
  var attributes = {code:'SwingSet2Applet.class',
      archive:'SwingSet2.jar', width:695, height:525} ; 
    <!-- customize splash screen display options -->
  var parameters = {jnlp_href: 'SwingSet2.jnlp', 
                    image: 'dukeanimated.gif', 
                    boxbgcolor: 'cyan', 
                    boxborder: 'true', 
                    centerimage: 'true'   
                    }; 
  deployJava.runApplet(attributes, parameters, '1.6'); 
</script>

Abra AppletPage.html en un navegador para ver la pantalla de carga cuando se cargue el applet Demo SwingSet2. Si ha pulsado este enlace antes y visto el applet Demo SwingSet2, asegúrese de limpiar la cache usando el Panel de Control de Java. Puede no ver la pantalla de carga si el applet se carga rápidamente.

Nota:  Si no ve el applet ejecutarse, necesita instalar al menos la versión 6 actualización 10 del Kit de Desarrollo de Java SE (KDJ).
Nota:  Si no ve el ejemplo ejecutarse, podría necesitar habilitar el interpretador de JavaScript en su navegador de forma que el script de Herramientas de Despliegue pueda funcionar adecuadamente.

Vea el tema Visualizando un Indicador de Progreso de Carga Personalizado para información sobre visualizar un indicador de progreso de carga personalizado cuando los recursos de applet están siendo cargados.
