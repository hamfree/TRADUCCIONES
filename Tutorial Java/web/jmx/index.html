Recorrido: Extensiones de Gestión de Java (JMX)

El recorrido de las Extensiones de Gestión de Java (JMX) proporciona una introdución a la tecnología JMX, la cual se incluye en la Plataforma Java, Edición Estándar (plataforma Java SE). Este recorrido presenta ejemplos de cómo usar las características más importantes de la tecnología JMX.

 Descripción de la Tecnología JMX proporciona una descripción breve de la tecnología JMX, incluyendo su propósito y características principales.

 Presentación de MBeans introduce el concepto fundamental de la tecnología JMX, los beans gestionados, también conocidos como MBeans. Esta lección también introduce a los MXBeans.

 Notificaciones introduce el mecanismo de notificaciones de la tecnología JMX.

 Gestión Remota muestra cómo implementar las capacidades de gestión remoto de la IPA de JMX y cómo crear una aplicación cliente JMX.

 Dónde Ir Desde Aquí proporciona referencias a documentación más avanzada que describe la tecnología JMX.
