Extensiones de Gestión de Java (JMX): Final de la Lección

Ha llegado al final del recorrido de "Extensiones de Gestión de Java (JMX)".

Si tiene comentarios o sugerencias sobre este recorrido, use nuestra página de comentarios para contárnoslo.

El propósito de este recorrido ha sido proveerle con una breve introducción a los elementos básicos de la tecnología JMX. Se pueden encontrar materiales más avanzados en otros lugares.

Los recursos siguientes serán útiles si desde estudiar más la tecnología JMX:

  • Documentación de la tecnología JMX para la plataforma Java SE. La documentación de la Tecnología JMX se incluye en la Documentación Técnica de Java SE .
  • Descripción general de la tecnología JMX. La documentación de la plataforma Java SE para la tecnología JMX contiene la Descripción General de la Tecnología JMX, la cual proporciona una descripción más detallada de la Tecnología JMX que que la provista en el recorrido presente.
  • Tutorial de la Tecnología JMX. Las lecciones en este recorrido se basan en los ejemplos usados en el Tutorial de la Tecnología JMX. Además de los elementos básicos presentados en el recorrido presente, el Tutorial de Tecnología JMX presenta aspectos más avanzados de la tecnología JMX.
  • Además de los ejemplos presentados en el tutorial de arriba, una vez ha instalado el Kit de Desarrollo Java (KDJ) 6, un aplicación de ejemplo que demuestra una implementación de la vida real de la IPA JMX pueden encontrarse en el directorio siguiente: 
    jdk_home/sample/jmx/jmx-scandir
    Donde jdk_home es el directorio en el cual el software KDJ se instala. El ejemplo jmx-scandir es un ejemplo avanzado, lo cual presenta conceptos avanzados de la IPA de JMX en un escenario del mundo real.
  • Supervisión y Gestión para la Plataforma Java. Las tecnologías e IPAs proporcionadas para la supervisión y gestión de la plataforma Java SE dependen en gran medida de la tecnología JMX. La Guía de Supervisión y Gestión de Java SE contienen un capítulo sobre Supervisión y Gestión Usando Tecnología JMX.
  • Tecnología Java Management Extension (JMX). La página de inicio de la tecnología JMX es el lugar para ir para todas las últimas noticias sobre la tecnología JMX.
