Lección: Presentación de MBeans

Esta lección introduce el concepto fundamenteal de la IPA de JMX, a saber, beans gestionados, o MBeans.

Un MBean es un objeto de Java gestionado, similar a un componente JavaBeans, que sigue los patrones de diseño establecidos en la especificación JMX. Un MBean puede representar un dispositivo, una aplicación, o cualquier recurso que necesite ser gestionado. Los MBeans exponen una interfaz de gestión que consiste en lo siguiente:

  • Un conjunto de atributos que se pueden leer o escribir, o ambos.
  • Un conjunto de operaciones invocables.
  • Una autodescripción.

La interfaz de gestión no cambia a lo largo de la vida de una instancia MBean. Los MBeans también pueden emitir notificaciones cuando ocurran ciertos eventos predefinidos.

La especificación JMX define cinco tipos de MBean:

  • MBeans estándar
  • MBeans dinámicos
  • MBeans abiertos
  • MBeans de modelo
  • MXBeans

Los ejemplos en este recorrido demuestran sólo los tipos más simples de MBean, a saber, MBeans estándar y MXBeans.
