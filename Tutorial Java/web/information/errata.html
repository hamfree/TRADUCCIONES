  • Problemas con el CD en Linux:

    Algunos sistemas de Linux automontan el CD con un sistema de ficheros que fuerza a todos los nombres de fichero a ser minúsculas. Esto rompen los enlaces en los ficheros HTML con nombres con mayúsculas y minúsculas mezcladas, y también evitan que los ejemplos se compilen.

    Si tiene este problema, trate de montar manualmente el CD con el sistema de ficheros UDF. Para hacer esto:

    • Conviértase en superusuario (root).
    • Desmonte el cd: 
      umount directorio_del_dispositivo_cd
    • Vuelva a montar el cd usando el sistema de ficheros UDF: 
      mount -t udf directorio_del_dispositivo_cd punto_de_montaje_cd
