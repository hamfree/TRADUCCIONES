Nuestro objetivo es hacer que los productos de Oracle, servicios y documentación de apoyo accesibles a todos los usuarios, incluidos los usuarios que son discapacitados. A tal fin, la documentación incluye características que hacen que la información esté disponible para los usuarios de tecnología de asistencia. Esta documentación está disponible en formato HTML y contiene marcadores para facilitar el acceso de la comunidad de discapacitados. Las normas de accesibilidad seguirán evolucionando con el tiempo, y Oracle participa activamente con otros líderes en el mercado y proveedores de tecnología para hacer frente a los obstáculos técnicos y que nuestra documentación puede ser accesible a todos nuestros clientes. Para obtener más información, visite el sitio web del Programa de Accesibilidad en Oracle http://www.oracle.com/us/corporate/accessibility/index.html .

Accesibilidad de los Códigos de Ejemplo en la Documentación

Los lectores de pantalla no siempre pueden leer correctamente los ejemplos de código de este documento. Las convenciones para escribir código necesitan que las llaves cerradas aparezcan en una línea por lo demás vacía; Sin embargo, algunos lectores de pantalla puede que no siempre lean una línea de texto que consta únicamente de un paréntesis o llave.

Accesibilidad de los enlaces a páginas externas en Documentación

Esta documentación puede contener enlaces a sitios web de otras empresas u organizaciones que Oracle no posee ni controla. Oracle no evalúa ni hace ninguna comunicación con respecto a la accesibilidad de estos sitios Web.

Acceso al Soporte de Oracle

Los clientes de Oracle tienen acceso al soporte electrónico a través de My Oracle Support. Para obtener más información, visite http://www.oracle.com/support/contact.html o visite http://www.oracle.com/accessibility/support.html si tiene problemas de audición.
