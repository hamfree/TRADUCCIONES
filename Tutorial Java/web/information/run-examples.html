¿Cómo compilo y ejecuto los ejemplos? Para las instrucciones, diríjase a la lección La Aplicación "¡Hola Mundo!" en el recorrido Comenzando. Esta lección incluye enlaces para descargar el último KDJ (Kit de Desarrollo Java o JDK en inglés) para Windows, Linux y Solaris. También hay instrucciones y enlaces para el EID Netbeans™, el cual funciona en todas las plataformas mencionadas.

¿Porqué obtengo un mensaje de error cuando intento compilar o ejecutar un ejemplo? Compruebe Problemas Comunes (y Sus Soluciones).

Yo ya tengo el EEJ (Entorno de Ejecución de Java o JRE en inglés). ¿Porqué necesito el KDJ (Entorno de Desarrollo de Java o JDK en inglés)? El Entorno de Ejecución de Java SE, o EEJ (JRE en inglés), le permite ejecutar programas que hayan sido compilados e instalados. Esperamos que usted usará los Tutoriales para expandir su entendimiento de la tecnología Java modificando y extendiendo los ejemplos. Para hacer esto, usted necesita el KDJ (Kit de Desarrollo Java, JDK en inglés).
