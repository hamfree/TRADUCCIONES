Introducir los Bytes de la Firma

A continuación, introduza los bytes de la firma desde el archivo especificado como segundo argumento de la línea de comandos. 

FileInputStream sigfis = new FileInputStream(args[1]);
byte[] sigToVerify = new byte[sigfis.available()]; 
sigfis.read(sigToVerify);
sigfis.close();

Ahora la matriz de bytes sigToVerify contiene los bytes de la firma.
