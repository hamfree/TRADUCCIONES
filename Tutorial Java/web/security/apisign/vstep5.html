Compilar y Ejecutar el Programa

Aquí está el código fuente completo para el programa VerSig.java, con algunos comentarios añadidos.

Compile y ejecute el programa. Recuerde, necesita especificar tres argumentos en la línea de comandos:

  • el nombre del archivo que contiene los bytes codificados de la clave pública
  • el nombre del archivo que contiene los bytes de la firma digital
  • el nombre del archivo de datos (para el que se generó la firma digital)

Ya que comprobará la salida del programa GenSig, los nombres de los archivos que debe usar son

  • suepk
  • sig
  • data

Aquí hay una ejecución de ejemplo, la negrita indica lo que usted teclea. 

%java VerSig suepk sig data
 la firma verifica: true
