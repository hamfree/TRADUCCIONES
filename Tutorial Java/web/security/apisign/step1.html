Preparar la Estructura Inicial del Programa

Aquí está la estructura básica del programa GenSig. Colóquelo en un archivo llamado GenSig.java. 

import java.io.*;
import java.security.*;

class GenSig {

    public static void main(String[] args) {

        /* Genera una firma DSA */

        if (args.length != 1) {
            System.out.println("Uso: GenSig nombreDelFicheroAFirmar");
        }
        else try {

        // el resto del código va aquí

        } catch (Exception e) {
            System.err.println("Captura excepción " + e.toString());
        }
    }
}


Notas:

  • Los métodos para firmar datos están en el paquete java.security, así que el programa importa todo de ese paquete. El programa también importa el paquete java.io, que contiene los métodos necesitados para ingresar los datos del archivo a firmar.

  • Se espera un único argumento, que especifica el archivo de datos a ser firmado.

  • El código escrito en los pasos siguiente irá entre los bloques try y catch.
