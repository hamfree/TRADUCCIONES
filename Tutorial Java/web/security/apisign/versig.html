Verificando una Signatura Digital

Si tiene datos de los cuales se generaron una firma digital, puede verificar la autenticidad de la firma digital. Para hacer eso, necesita

  • los datos
  • la firma
  • la clave pública correspondiente a la clave privada usada para firmar los datos

En este ejemplo escribe un programa VerSig para verificar la firma generada por el programa GenSig. Esto demuestra los pasos requeridos para verificar la autenticidad de una supuesta firma.

VerSig importa una clave pública y una firma digital que se supone es la firma de un archivo de datos especificado y después verifica la autenticidad de la firma. La clave pública, la firma, y los nombres del archivo de datos se especifican en la línea de comandos.

Los pasos para crear el programa de ejemplo VerSig para importar los archivos y verificar la firma son los siguientes.

  1. Preparar la Estructura Inicial del Programa

    Crear un archivo de texto llamado VerSig.java. Teclee en el programa inicial la estructura (sentencias import, nombre de clas, método main, etcetera).

  2. Introducir y Convertir los Bytes Codificados de la Clave Pública

    Importar los bytes codificados de la clave pública desde el archivo especificado con el primer argumento de la línea de comandos y los convierte en una PublicKey.

  3. Introducir los Bytes de la Firma

    Introducir los bytes de firma desde el archivo especificado como segundo argumento de línea de comandos.

  4. Verificar la Firma

    Obtener un objeto Signature e inicialízelo con la clave pública para verificar la firma. Suministre los datos cuya firma se va a verificar (del archivo especificado como tercer argumento de línea de comando) y verifique la firma.

  5. Compilar y Ejecutar el Programa
