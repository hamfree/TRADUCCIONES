Compilar y Ejecutar el Programa

Aquí está el código fuente completo para el programa GenSig.java, con algunos comentarios añadidos. Compílelo y ejecútelo. Recuerde, necesita especificar el nombre de un archivo que va a ser firmado, como en 

java GenSig data

Puede descargar y usar este archivo de ejemplo llamado data o cualquier otro archivo que quiera. El archivo no será modificado. Será leído de forma que se pueda generar una firma para él.

Después de ejecutar el programa, debería ver los archivos salvados suepk (clave pública) y sig (firma).
