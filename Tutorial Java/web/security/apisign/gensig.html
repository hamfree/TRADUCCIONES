Generando una Firma Digital

El programa GenSig que va a crear usará la IPA de Seguridad del KDJ para generar claves y una firma digital para datos usando la clave pricada y para exportar la clave pública y la firma a archivos. La aplicación toma el nombre del archivo de datos de la línea de comandos.

Los pasos siguientes crean el programa de ejemplo GenSig.

  1. Preparar la Estructura Inicial del Programa

    Cree un archivo de texto llamado GenSig.java. Teclee la estructura inicial del programa (sentencias import, nombre de clase, método main, y así).

  2. Generar las Claves Públicas y Privadas

    Genere un par de claves (clave pública y clave privada). La clave privada se necesita para firmar los datos. La clave pública será utilizada por el programa VerSig para verificar la firma.

  3. Firmando los Datos

    Obtenga un objeto Signature e inicialícelo para firmar. Proporciónele los datos que serán firmados, y genera la firma.

  4. Salvar la Firma y la Clave Pública en los Ficheros

    Salve los bytes de la firma en un archivo y los bytes de la clave pública en otro.

  5. Compilar y Ejecutar el Programa
