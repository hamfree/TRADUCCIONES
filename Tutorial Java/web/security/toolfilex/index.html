Lección: Intercambiando Ficheros

Si quiere enviar electrónicamente un documeto importante, (como un contrato) a algún otro, es una buena idea "firmar" digitalmente el documento, de forma que su destinatario pueda comprobar que el documento procede de usted y que no se modificó durante el transporte.

Esta lección muestra cómo usar las herramientas de Seguridad para el intercambio de un documento importante, en este caso un contrato.

Primero finja que es el remitente del contrato, Stan Smith. Esta lección le muestra los pasos que Stan daría para colocar el contrato en un archivo JAR, firmarlo, y exportar el certificado de clave pública para la clave pública correspondiente a la clave privada usada para firmar el archivo JAR.

Después finje que usted es Ruth, que ha recibo el archivo JAR firmado y el certificado. Usará keytool para importar el certificado en el almacén de certificados de Ruth en una entrada con el alias de stan, y usa la herramienta jarsigner para verificar la firma.

Para información posterior sobre firmas digitales, certificados, almacenes de claves, y las herramientas, vea la lección IPA y Herramientas Usadas para Código Seguro e Intercambio de Ficheros.

Nota: Esta lección asume que usted ejecuta todos los comandos dentro del mismo directorio.
