Crear un Fichero JAR Conteniendo el Contrato

La primera cosa que necesita es un archivo de contrato. Puede descargar y usar este archivo de ejemplo muy básico llamada contract. O puede usar cualquier otro archivo que quiera. Simplemente asegúrese de que el nombre del fichero es contract de forma que funcionará con los comandos especificados en esta lección.

Una vez obtuvo el archivo de contrato, colóquelo en un archivo JAR. En su ventana de comandos teclee lo siguiente: 

jar cvf Contract.jar contract

Este comando crea un archivo JAR llamado Contract.jar y colca el archivo contract dentro de él.
