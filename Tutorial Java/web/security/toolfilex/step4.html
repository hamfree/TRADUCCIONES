Exportar el Certificado de Clave Pública

Ahora tiene un archivo JAR firmado sContract.jar. Los destinatarios que quieren usar este archivo también quieren autenticar su firma. Para hacer esto, necesitan la clave pública que corresponde a la clave privada que usó para generar su firma. Usted proporciona su clave pública enviándoles una copia del certificado que contiene su clave pública. Copie ese certificado del almacén de claves examplestanstore a un archivo llamado StanSmith.cer con lo siguiente: 

keytool -export -keystore examplestanstore
-alias signLegal -file StanSmith.cer

Será preguntado por la contraseña del almacén.

Una vez tengan ese certificado y el archivo JAR firmado, sus destinatarios pueden usar la herramienta jarsigner para autenticar su firma. Vea Pasos para el Receptor del Contrato.
