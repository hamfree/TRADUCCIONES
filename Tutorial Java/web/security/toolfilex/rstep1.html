Importar el Certificado como Certificado de Confianza

Suponga que usted es Ruth y ha recibido de Stan Smith

  • El archivo JAR firmado sContract.jar conteniendo un contrato
  • El archivo StanSmith.cer que contiene el certificado de clave pública para la clave pública correspondiente a la clave privada usada para firmar el archivo JAR

Antes de que pueda usar la herramienta jarsigner para comprobar la autenticidad de la firma del archivo JAR, necesita importar el certificado de Stan en su almacén de claves.

Incluso aunque usted (actuando como Stan) creó estos archivos y en realidad no se han transportado a ninguna parte, puede simular ser otra persona que no sea el creador y remitente, Stan. Actuando como Ruth, teclee el comando siguiente para crear un almacén de claves llamado exampleruthstore e importe el certificado dentro de una entrada con el alias stan. 

keytool -import -alias stan -file StanSmith.cer -keystore exampleruthstore

Ya que el almacén de claves no existe aún, keytool lo creará para usted. Le preguntará por una contraseña para el almacén de claves.

keytool imprime la información del certificado y le pide que la verifique; Por ejemplo, comparando las huellas digitales del certificado visualizado con aquellas obtenidas de otra (confiable) fuente de información. (Cada huella digital es un número relativamente corto que identifica el certificado de manera única y confiable). Por ejemplo, en el mundo real puede telefonear a Stan y preguntarle cuáles deberían ser las huellas digitales. El puede obtener las huellas digitales del archivo StanSmith.cer que creó ejecutando el comando 

keytool -printcert -file StanSmith.cer

Si las huellas digitales que el ve son las mismas que las que le informa keytool a usted, entonces ambos pueden asumir que el certificado no ha sido modificado en el transporte. Puede de forma segura permitir que keytool proceda a colocar una entrada de "certificado de confianza" en su almacén de claves. Esta entrada contiene los datos del certificado de clave pública del archivo StanSmith.cer. keytool asigna el alias stan a esta nueva entrada.
