Lección: Características de Seguridad en Java SE

Las funciones de seguridad le ayudan a proteger sus programas y datos del daño, para mantener los datos protegidos y privados, y desplegar nuevas aplicaciones en entornos de ejecución conscientes de la seguridad.

Java también proporciona varias herramientas que le ayudan a manejar el acceso a los recursos del sistema; para crear, almacenar y mantener claves cifradas públicas y privadas (pares de claves) y certificados; y para crear y firmar ficheros jar durante el proceso de despliegue.

Note: Para un resumen breve de las características de seguridad admitidas en Java SE 7, vea el Informe Introducción a la Seguridad en el directorio de Guías de Seguridad.
