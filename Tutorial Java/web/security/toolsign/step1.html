Descarga y Uso de la Aplicación de Ejemplo

Esta lección usa una aplicación simple que le proporcionamos.

  1. Cree un archivo llamado Count.java en su computadora lcoal copiando o descargando el código fuente de Count.java. Los ejemplos en esta lección asumen que coloca count en el directorio C:\Test.
  2. La aplicación count necesita acceder a un archivo de texto que contiene los datos que procesará. Descargue un fichero de datos de ejemplo, o uso cualquier otro fichero de texto conveniente como datos.

    Importante: Coloque su fichero de datos en un directorio distinto del directorio que contiene su fichero de clase count descargado. Sugerimos C:\TestData\data.

    Más tarde en esta lección verá cómo una aplicación ejecutándose bajo un gestor de seguridad no puede leer un fichero a menos que tenga el permiso explícito para hacerlo. Sin embargo, una aplicación puede siempre leer un fichero desde el mismo directorio (o un subdirectorio). No necesita permiso explícito.
  3. Compile y después ejecute la aplicación Count para ver que hace.

    Cuando ejecute la aplicación count, necesita especificar (como un argumento) la ruta de un fichero a leer.

    java Count C:\TestData\data

Aquí tiene una ejecución de ejemplo: 

    C:\Test>java Count C:\TestData\data
    Counted 65 chars.
