Pasos para el Firmador de Código

El firmador de código realiza los pasos siguientes:

  1. Descarga y Uso de la Aplicación de Ejemplo.
  2. Crear un Fichero JAR Conteniendo el Fichero Class, usando la herramienta jar.
  3. Generar Claves (si estas aún no existen), usando el comando -genkey de keytool.
    Paso Opcional Generar una petición de firma de certificado (CSR) para el certificado de clave pública, e importar la respuesta de la autoridad de certificación (AC). Por simplicidad (y ya que usted solo pretende ser Susan Jones), este paso se omite. Vea Generar una Petición de Formado de Certificado (CSR) para un Certificado de Clave Pública para más información.
  4. Firmar el Fichero JAR, usando la herramienta jarsigner y la clave privada.
  5. Exportar el Certificado de Clave Pública, usando el comando -export de keytool. Luego proporcione el archivo JAR firmado y el certificado al remitente Ray.
