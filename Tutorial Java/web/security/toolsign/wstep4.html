Salvar el Fichero de Políticas

Para salvar el nuevo archivo de políticas que ha estado creando, elija el comando Save As del menú File. Esto mostrará el cuadro de diálogo Save As.

Navegue en la estructura de directorios para obtener el directorio en el cual salvar el archivo de políticas: el directorio Test en la unidad C:. Teclee el nombre de archivo 

exampleraypolicy

Después elija el botón Save. El archivo de políticas se salvó ahora, y su nombre y ruta se muestra en la caja de texto etiqueta Policy File.

Después salga de la Herramienta de Políticas selección el comando Exit del menú File.
