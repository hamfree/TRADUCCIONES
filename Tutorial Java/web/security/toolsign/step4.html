Firmar el Fichero JAR

Ahora está listo para firmar el archivo JAR. Teclee lo siguiente en su ventana de comandos para firmar el archivo JAR Count.jar, usando la clave privada en la entrada del almacén de clave con el alias signFiles, y nombre el archivo JAR firmado resultante sCount.jar: 

jarsigner -keystore examplestore -signedjar sCount.jar Count.jar signFiles

Será preguntado por la contraseña del almacén y la contraseña de la clave privada.

Nota: La herramienta jarsigner extrae el certificado de la entrada del almacén de claves cuyo alias es signFiles y lo adjunta a la firma generada del archivo JAR firmado.

Se recomienda el sellado de tiempo de la firma, y se muestra un aviso si la firma no tiene sello de tiempo. El sello de tiempo se usa para verificar que el certificado usado para firmar el archivo JAR era válido en el momento de la firma. Vea See Firmando Archivos JAR para información sobre las opciones de jarsigner que incluyen el sellado de tiempo con la firma.
