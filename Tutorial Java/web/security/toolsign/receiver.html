Pasos para el Receptor de Códigos

En esta lección, actuará como el receptor del archivo jar firmado que contiene el fichero count.class. Este solicita acceso a los recursos de su sistema en su sistema a los que normalmente no tendría permiso para acceder.

Este procedimiento requiere que realize los pasos siguientes listados abajo.

  1. Observar la Aplicación Restringida. Esta aplicación no tendrá la capacidad de acceder a sus recursos de sistema hasta que importe el certificado de Susan y cree un archivo de políticas.
  2. Importar el certificado de Susan como un certificado de confianza usando el comando keytool -import, y dele el alias susan.
  3. Configurar un archivo de políticas para otorgar permiso para la aplicación count firmada por susan para leer el fichero especificado en su sistema.
  4. Comprobar su aplicación count reconfigurada para verificar que con un certificado de confianza y acceso a su nuevo archivo de políticas que otorga su permiso para leer fichero en su sistema, count puede ahora leer su fichero de datos.
