Crear un Fichero JAR Conteniendo el Fichero Class

A continuación, cree un archivo JAR que contenga el archivo Count.class. Teclee lo siguiente en su ventana de comandos: 

jar cvf Count.jar Count.class

Esto crea un archivo JAR, Count.jar, y coloca el archivo Count.class dentro de este.
