Especificar el Almacén de Claves

Para esta lección usted otorgará a todo el código en los archivos JAR firmados por el alias susan acceso de lectura a todos los ficheros en el directorio C:\TestData\. Usted necesita

  1. Especificar el almacén de claves que contiene la información del certificado con alias susan
  2. Crear la entrada de política otorgando el permiso

El almacén de claves es el llamado exampleraystore creando en el paso Importar el Certificado como Certificado de Confianza.

Para especificar el almacén de claves, elija el comando Cambiar Almacén de Claves en el menú Editar de la ventana principal de la Herramienta de Políticas. Esto muestra un cuadro de diálogo en el que especifica la URL del almacén de claves y el tipo de almacén de claves.

Para especificar el almacén de claves llamado exampleraystore en el directorio Test de la unidad C:, teclee la siguiente URL file en la caja de texto etiquetada New KeyStore URL 

file:/C:/Test/exampleraystore

Puede dejar la caja de texto etiquetada New KeyStore Type en blanco si el tipo del almacén de claves es el predeterminado como se especifica en el archivo de propiedades de seguridad. Su almacén de claves será del tipo predeterminado, así que deje la caja de texto en blanco.

Nota: El valor de New KeyStore URL es una URL y por lo tanto siempre debe usar barras inclinadas (nunca barras invertidas) como separador de directorios.

Cuando haya terminado de especificar la URL del almacén de claves, elija OK. La caja de texto etiquetada Keystore está ahora rellenada con la URL.

A continuación, necesita especificar la nueva entrada de política.
