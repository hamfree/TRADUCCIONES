ExampleGame

Debajo está el código fuente para ExampleGame. Por simplicidad, ExampleGame no contiene en realidad el código para jugar un juego. Simplemente recupera o actualiza el marcador de puntuaciones más altas del usuario.

Para ver cual es el valor actual del marcador de puntuaciones más altas del usuario, podría ejecutar: 

java ExampleGame get

Para establecer un nuevo valor de puntuación alta para el usuario, podría ejecutar: 

java ExampleGame set score

Para recuperar la puntuación más alta actual del usuario, ExampleGame simplemente instancia un objeto HighScore y hace una llamada a su método getHighScore. Para establecer una nueva puntuación más alta para el usuario, ExampleGame instancia un objeto HighScore y llama a setHighScore, pasándole la nueva puntuación más alta del usuario.

Aquí está el código fuente para ExampleGame, ExampleGame.java: 


package com.gamedev.games;

import java.io.*;
import java.security.*;
import java.util.Hashtable;
import com.scoredev.scores.*;

public class ExampleGame
{
    public static void main(String args[])
	throws Exception 
    {
	HighScore hs = new HighScore("ExampleGame");

	if (args.length == 0)
	    usage();

	if (args[0].equals("set")) {
	    hs.setHighScore(Integer.parseInt(args[1]));
	} else if (args[0].equals("get")) {
	    System.out.println("score = "+ hs.getHighScore());
	} else {
	    usage();
	}
    }

    public static void usage()
    {
	System.out.println("ExampleGame get");
	System.out.println("ExampleGame set <score>");
	System.exit(1);
    }
}
