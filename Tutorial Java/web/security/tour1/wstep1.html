Iniciando la Herramienta de Políticas

Para iniciar la Herramienta de Políticas, teclee simplemente lo siguiente en la línea de comandos: 

policytool

Esto abre la ventana Herramienta de Políticas.

Siempre que la Herramienta de Políticas se inicia, esta intenta rellenar en esta ventana con la información de políticas del archivo de políticas de usuario. El archivo de políticas de usuario se llama .java.policy de forma predeterminada y está en su directorio de inicio. Si la Herramienta de Políticas no puede encontrar el archivo de políticas de usuario, emite una advertencia y visualiza una ventana en blanco de la Herramienta de Políticas (una ventana con encabezados y botones pero sin datos en ella), como se muestra en la siguiente figura.

Puede entonces proceder a abrir un archivo de políticas existente o crear un nuevo archivo de políticas.

La primera vez que ejecuta la Herramienta de Políticas, verá la ventana en blanco de la Herramienta de Políticas, ya que el archivo de políticas de usuario aún no existe. Puede inmediatamente proceder a crear un nuevo archivo de políticas, como se describe en el paso siguiente.
