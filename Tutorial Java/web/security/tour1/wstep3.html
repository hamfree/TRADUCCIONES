Salvando el fichero de Politícas

Para salvar el nuevo archivo de políticas que ha estado creando, elija el comando Salvar Como del menú Archivo. Esto visualiza el cuadro de diálogo Salvar Como.

Los ejemplos en la lección Visita Rápida para Controlar Aplicaciones asumen que ha almacenado el archivo de políticas en el directorio Test en la unidad C:.

Navegue al directorio Test. Teclee el nombre del archivo examplepolicy y pulse Salvar.

El archivo de políticas es ahora salvado, y su nombre y ruta se muestra en la caja de texto etiquetada Archivo de Políticas.

Salga de la Herramienta de Políticas eligiendo Salir del menú Archivo.
