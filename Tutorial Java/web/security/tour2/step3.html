Establecer el Fichero de Políticas para Conceder los Permisos Requeridos

Este paso usa la utilidad de Herramienta de Políticas para abrir el archivo de políticas llamado examplepolicy creado en la lección Crear un Archivo de Políticas . Agregará una nueva entrada de política otorgando permiso al código del directorio donde se almacena GetProps.class para leer los valores de las propiedades "user.home" y "java.home", como se muestra en la figura siguiente.

Los pasos son como siguen.

  1. Abrir el Fichero de Políticas
  2. Conceder los Permisos Requeridos
  3. Salvar el Fichero de Políticas
Nota para Usuarios de UNIX: Las instrucciones ilustran la creación del archivo de políticas para un sistema Windows. Los pasos son exactamente los mismos si está trabajando en un sistema UNIX, con las siguientes diferencias.
  • Usted recupera el archivo examplepolicy desde el directorio test en su directorio de inicio.
  • Para la URL CodeBase en el paso para otorgar los permisos requeridos, puede substituir file:${user.home}/test/ para file:/C:/Test/. Alternativamente podría especificar directamente su directorio de inicio en vez de referirse a la propiedad "user.home", como en file:/home/jones/test/.
