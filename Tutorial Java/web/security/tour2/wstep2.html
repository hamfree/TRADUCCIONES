Conceder los Permisos Requeridos

Para permitir a la aplicación GetProps permiso para leer los valores de las propiedades "user.home" y "java.home", debe crear una entrada de política otorgando esos permisos. Elija el botón Agregar Entrada de Política en la ventana principal Herramienta de Políticas. Esto muestra el cuadro de diálogo Entrada de Políticas, como se muestra en la siguiente figura.

Teclee la siguiente URL de fichero dentro de la caja de texto CodeBase para indicar que va a otorgar un permiso al código de un directorio especificado, que es el directorio en el cual se almacenó GetProps.class . 

file:/C:/Test/

(Tenga en cuenta que esta es una URL y, por lo tanto, siempre debe tener barras inclinadas, no barras invertidas)

Deje la caja de texto SignedBy en blanco, ya que no se le requiere que el código sea firmado.

Para agregar el permiso a leer el valor de la propiedad "user.home", elija el botón Agregar Permiso. Esto muestra el cuadro de diálogo Permisos.

Haga lo siguiente.

  1. Elija Permiso de Propiedad de la lista desplegable de Permisos. El nombre completo del tipo de permiso (java.util.PropertyPermission) aparece ahora en la caja de texto a la drecha de la lista desplegable.
  2. Teclee lo siguiente en la caja de texto a la derecha de la lista etiquetada Nombre Objetivo para especificar la propiedad "user.home": 
    user.home
  3. Especifique el permiso para leer esta propiedad seleccionando la opción read desde la lista desplegable de Acciones.

Ahora el cuadro de diálogo Permisos se parece a lo siguiente.

Elija el botón OK. El nuevo permiso aparece en una línea en la ventana de entrada de política.

Para agregar permiso de lectura al valor de la propiedad "java.home", elija el botón Agregar Permiso de nuevo. En el cuadro de diálogo Permisos, haga lo siguiente:

  1. Elija Permisod de Propiedad de la lista desplegable de Permisos.
  2. Teclee lo siguiente en la caja de texto a la derecha de la lista etiqueta Nombre Objetivo para especificar la propiedad "java.home": 
    java.home
  3. Especifique el permiso para leer esta propiedad eligiendo la opción read desde la lista desplegable de Acciones.

Ahora el cuadro de diálogo de Permisos se parece a lo siguiente.

Elija el botón OK. El nuevo permiso y el permiso agregado anteriormente aparecen en líneas en la ventana de entrada de política, como se muestra en la figura siguiente.

Ya ha terminado de especificar esta entrada de política, así que elija el boton Hecho en el diálogo Entrada de Política. La ventana Herramienta de Política ahora incluye una línea representando la nueva entrada de política, mostrando el valor CodeBase.
