Lección: Visita Rápida para Controlar Aplicaciones

Lección de requisito previo:  Creando un Fichero de Políticas

Esta lección muestra cómo usar un gestor de seguridad para otorgar o denegar acceso a los recursos del sistema a las aplicaciones. La lección también muestra como los accesos a los recursos, tales como leer o escribir un fichero, no se permiten para aplicaciones que se ejecuten con un gestor de seguridad a menot que esté explícitamente permitido por una entrada de permiso en un archivo de políticas.

Los pasos para esta lección son:

  1. Observar la Libertad de una Aplicación
  2. Ver como Restringir Aplicaciones
  3. Configurar el Fichero de Políticas para Conceder los Permisos Requeridos
  4. Ver los Efectos del Fichero de Politica
