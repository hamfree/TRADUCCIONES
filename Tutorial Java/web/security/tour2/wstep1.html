Abrir el Fichero de Políticas

Iniciar Herramienta de Políticas tecleando lo siguiente en la línea de comandos: 

policytool

Esto muestra la ventana de Herramienta de Políticas. Para abrir el archivo de políticas examplepolicy, use el comando Abrir en el menú Archivo. Esto le presentará un diálogo Abrir, que puede usar para navegar por la estructura de directorios hasta que obtenga el directorio que contiene el archivo de políticas (es decir, el directorio C:\Test\).

Elija el archivo examplepolicy en ese directorio y después seleccione el botón Abrir.

Esto rellenará la ventana de Herramienta de Políticas con la información del archivo de políticas examplepolicy, incluyendo el nombre de archivos de políticas y la parte CodeBase de la entrada de políticas creada por la lección Crear un Archivo de Políticas.
