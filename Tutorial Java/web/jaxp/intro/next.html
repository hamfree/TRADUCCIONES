¿A Dónde Ir Desde Aquí?

En este punto, tiene bastante información para empezar a elegir su propio camino a través de la bibliotecas JAXP. Su próximo paso depende de lo que quiera lograr. Es posible que desee ir a cualquiera de estas lecciones:

  • Si las estructuras de datos han sido ya determinadas, y está escribiendo una aplicación de servidor o un filtro de XML que necesita hacer procesado rápido, vea IPA Simple para XML.

  • Si necesita construir un árbol de objetos desde los datos XML de forma que pueda manipularlo en una aplicación, o convierte un árbol de objetos en memoria a XML, vea Modelo de Objetos del Documento.

  • Si necesita transformar etiquetas XML a otras formas, si quiere generar una salida XML, o (en combinación con la IPA de SAX) si quiere convertir estructuras de datos heredadas a XML, vea Transformaciones del Lenguaje Extensible de Hojas de Estilo (XLST).

  • Si desea una IPA de análisis de extracción, impulsada por eventos y basada en la tecnología Java de transmisión para leer y escribir documentos XML o desea crear analizadores XML bidireccionales que sean rápidos, relativamente fáciles de programar y que ocupen poco espacio en la memoria, consulte IPA de Transmisión para XML.
