Lección: Introducción a JAXP

La IPA de Java para Procesado de XML (JAXP) es para procesar datos XML usando aplicaciones escritas en el lenguaje de programación Java. JAXP aprovecha los estándares del analizador de IPA Simple para Análisis de XML (SAX) y el Modelo de Objetos de Documento (DOM) para que pueda elegir analizar sus datos como un flujo de eventos o crear una representación de objetos de ellos. JAXP también soporta el estándar de las Transformaciones de Lenguaje de Hoja de Estilos Extensible (XSLT), dándole el control sobre la presentación de los datos y permitiéndole convertir los datos en otros documentos XML o a otros formatos, tales como HTML. JAXP también proporciona soporte de espacios de nombres, permitiéndole trabajar con DTDs que de otra forma podrían tener conflictos de nombres. Finalmente, a partir de la versión 1.4, JAXP implementa el estándar IPA de Transmisión para XML (StAX).

Diseñado para ser flexible, JAXP le permite usar cualquier analizador compatible con XML desde su aplicación. Hace esto con lo que se llama una capa de capacidad de conexión, que le permite conectar una implementación de SAX o la IPA de DOM. La capa de capacidad de conexión también le permite conectar un procesador XSL, permitiéndole controlar cómo se visualizan los datos XML.
