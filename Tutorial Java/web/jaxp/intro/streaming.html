IPAs para la Transmisión de XML

StAX es la última IPA en la familia JAXP, y proporciona una alternativa a SAX, DOM, TrAX, y DOM para desarrolladores que busquen realizar un filtrado, procesamiento y modificación de secuencias de alto rendimiento, particularmente con poca memoria y requisitos de extensibilidad limitados.

En resumen, StAX proporciona una interfaz de analizador de extracción bidireccional estándar para el procesamiento de XML en tiempo real, que ofrece un modelo de programación más simple que SAX y una gestión de memoria más eficiente que DOM. StAX permite a los desarrolladores analizar y modificar flujos de XML como eventos, y extender los modelos de información XML para permitir adiciones específicas de la aplicación. Se proporcionan comparaciones más detalladas de StAX con varias IPAs alternativas en IPA de Transmisión para XML, y en Comparación de StAX con otras IPA de JAXP.

Paquetes StAX

Las IPAs de StAX se definen en los paquetes mostrados en la Tabla 1-4.

Tabla 1-4 Paquetes StAX

Paquete

Descripción

javax.xml.stream

Define la interfaz XMLStreamReader, que se usa para iterar sobre los elementos de un documento XML. La interfaz XMLStreamWriter especifica cómo debería ser escrito el XML.

javax.xml.transform.stax

Proporciona IPAs de transformación específicas de StAX.
