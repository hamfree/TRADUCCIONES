Ejemplos

Puede descargar el fichero samples.zip, que contiene dos ficheros de aplicación Java, LimitSamples.java y SampleBase.java, y un directorio, samples, que contiene los ficheros xml para el ejemplo.

Para ejecutar el ejemplo, descomprima el fichero samples.zip, compile, y ejecute lo siguiente en la línea de comandos: 

java LimitSamples
