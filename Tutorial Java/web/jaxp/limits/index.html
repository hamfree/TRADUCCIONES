Lección: Límites de Procesamiento

El procesado XML puede algunas veces ser una operación intensiva de memoria. Las aplicaciones, especialmente aquellas que aceptan XML, XSD y XSL de fuentes no confiables, deben tomar medidas para protegerse contra el consumo excesivo de memoria mediante el uso de límites de procesamiento JAXP proporcionados en el KDJ.

Los desarrolladores deben evaluar sus requisitos de aplicación y entorno operativo para determinar los límites aceptables pra sus configuraciones de su sistema y establecer estos límites en consecuencia. Se pueden usar los límites relacionados con el tamaño para prevenir que fuentes XML mal formadas se procesen sin consumir grandes cantidades de memoria, mientras un EntityExpansionLimit permitirá a una aplicación controlar el consumo de memoria por debajo de un nivel aceptable.

En este tutorial se le presentarán los límites, y aprenderá a cómo usarlos correctamente.
