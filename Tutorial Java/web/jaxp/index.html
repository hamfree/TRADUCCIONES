Recorrido: IPA de Java para Procesado de XML (JAXP)

El recorrido IPA de Java para Procesado de XML (JAXP) proporciona una introducción de la tecnología 1.4 a la IPA Java para Procesado de XML (JAXP), a través de ejemplos de aplicaciones JAXP.

Antes de Leer este Tutorial

Para hacer un uso completo de la información en el Tutorial de la IPA de Java para Procesado de XML (JAXP), debe tener el conocimiento de la tecnología siguiente:

  • El lenguaje de programación Java y su entorno de desarrollo.
  • El Lenguaje eXtesible de Marcado (XML)
  • El Modelo de Objeto de Documento (DOM), como se define por el Grupo de Trabajo DOM del Consorcio de World Wide Web (W3C).
  • IPA simple para XML (SAX), como se desarrolla cooperativemente por los miembros de la lista de correo XML-DEV.

Se asume un conocimiento previo de DOM y SAX. Sólo el código que se especifica en la IPA JAXP se examina en detalle en este tutorial.

  Introducción a JAXP proporciona un descripción breve de la tecnología JAXP, incluyendo su propósito y características principales.

  IPA Simple para XML introduce un concepto usado en la tecnología JAXP, la IPA Simple para XML (SAX): cuándo se usa SAX, cómo analizar un fichero XML, cómo implementar la validación de SAX, cómo ejecutar el analizador de SAX, y cómo gestionar eventos léxicos. Se proporcionan enlaces para obtener más información.

  Modelo de Objetos del Documento introduce la estructura de árbol usada por el Modelo de Objeto del Documento (DOM) y le muestra cómo usar las funciones DOM para crear nodos, eliminar nodos, cambiar los contenidos de los nodos, y recorrer la jerarquía de nodos.

  Transformaciones del Lenguaje Extensible de Hojas de Estilo (XLST) incluye información sobre cómo escribir un Modelo de Objeto de Documento como un fichero XML, y cómo generar un DOM de un fichero de datos arbitrario para convertirlo a XML.

  IPA de Transmisión para XML se centra en una IPA de análisis de extracción basada en la tecnología JAVA de transmisión por secuencias, impulsada por eventos, para leer y escribir documentos XML. StAX le permite crear analizadores XML bidireccionales que son rápidos, relativamente fáciles de programar y que ocupan poco espacio en memoria.

  JAXP 1.5 y Nuevas Propiedades presenta propiedades que se han agregado al KDJ 7u40 y 8.

  Límites de Procesamiento describe los límites de la implementación JAXP, incluyendo tres que fueron agregados en el KDJ 7u45.
