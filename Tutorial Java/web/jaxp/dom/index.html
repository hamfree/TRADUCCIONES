Lección: Modelo de Objetos del Documento

Esta lección presenta el Modelo de Objetos del Documento (DOM). Un DOM es una estructura de árbol estándar, donde cada nodo contiene uno de los componentes de una estructura XML. Los dos tipos más comunes de nodos son los nodos de elemento y los nodos de texto. Usar funciones DOM le permite crear nodos, eliminar nodos, cambiar sus contenidos, y recorrer la jerarquía de nodos.

Los ejemplos en esta lección demuestran cómo analizar un fichero XML existente para construir un DOM, visualizar e inspeccionar la jerarquía DOM, y explorar la sintaxis de los espacios de nombres. También muestra cómo crear un DOM desde cero, y ver cómo usar alguna de las características específicas de implementación en la implementación JAXP de Sun para convertir un conjunto de datos existentes a XML.
