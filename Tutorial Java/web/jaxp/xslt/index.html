Lección: Transformaciones del Lenguaje Extensible de Hojas de Estilo (XLST)

El estándar de Transformaciones del Lenguaje Extensible de Hojas de Estilo (XLST) (XSLT) define mecanismos para direccionar datos XML (XPath) y para especificar transformaciones sobre los datos para convertirlos en otras formas. JAXP incluye una implementación de interpretación de XSLT.

En esta lección, escribirá un Modelo de Objeto de Documento como un fichero XML, y verá cómo generar un DOM desde un fichero de datos arbitrarios para convertirlo a XML. Finalmente, convertirá datos XML en una forma diferente, aprendiendo sobre el mecanismo de direccionamiento XPath en el camino.
