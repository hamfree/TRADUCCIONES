Lección: IPA de Transmisión para XML

Esta lección se enfoca en la la IPA de Transmisión para XML (StAX), una IPA de análisis de extracción, impulsada por eventos y basada en la tecnología Java de transmisión para leer y escribir documentos XML. StAX le permite crear analizadores XML bidireccionales que son rápidos, relativamente fáciles de programar, y tienen una huella de memoria ligera.
