StAX

La especificación para StAX, JSR 173, aún no admite las nuevas propiedades. Sin embargo, StAX en el contexto de JAXP incluye la admisión de estas propiedades. Establecer las nuevas propiedades es similar a SAX o DOM, pero a través de la XMLInputFactory como sigue: 

XMLInputFactory xif = XMLInputFactory.newInstance();
xif.setProperty("http://javax.xml.XMLConstants/property/accessExternalDTD", "file");

Las propiedades existentes y características especificadas en StAX, especificacón JSR 173 tomará preferencia sobre las nuevas propiedades JAXP. La propiedad SupportDTD por ejemplo, cuando la establece a falso, causará que un programa lance una excepción cuando un fichero de entrada contiene una DTD que puede ser analizada. Para las aplicaciones que usan la propiedad SupportDTD para deshabilitar la DTD, la adición de las nuevas propiedades no tendrá ningún efecto.
