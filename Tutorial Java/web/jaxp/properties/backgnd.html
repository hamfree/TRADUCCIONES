Fondo

La función de procesamiento seguro de JAXP pone límites a los recursos en los procesadores XML para contrarrestar ciertos tipos de ataques de denegación de servicios. Sin embargo, no limita los medios por los cuales se pueden obtener recursos externos, lo que también es útil cuando se intenta procesar documentos XML de forma segura. La implementación actual de JAXP admite propiedades específicas de la implementación que se pueden usar para hacer cumplir tales restricciones, pero existe la necesidad de una forma estándar de hacerlo.

JAXP 1.5 agrega tres nuevas propiedades junto con ssus correspondientes Propiedades de Sistema que permitan a los usuarios especificar el tipo de conexiones externas que se pueden o se no se pueden permitir. Los valores de la propiedad son una lista de protocolos. Los procesadores JAXP comprueban si una conexión externa dada se permite haciendo coincidir el protocolo con los de la lista. Los procesadores intentarán establecer la conexión si está en la lista, y la rechazarán si no está.

JAXP 1.5 se ha integrado dentro del KDJ 7u40 y 8.
