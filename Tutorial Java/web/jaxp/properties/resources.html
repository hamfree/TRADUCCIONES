Recursos Externos

Los estándares XML, Esquema, y XSLT admiten las siguientes construcciones que requieren recursos externos. El comportamiento predetermiando de los procesadores XML del KDJ es hacer una conexión y obtener los recursos externos como se especifica.

  • DTD Externo: referencia a una Definición de Tipo de Documento externo (DTD), ejemplo: <!DOCTYPE root_element SYSTEM "url">
  • Referencia de Entidad Externa: se refiere a datos externos, sintaxis: <!ENTITY name SYSTEM "url">
    Referencia de entidad general como la siguiente: 
    <?xml version="1.0" standalone="no" ?>
<!DOCTYPE doc [<!ENTITY otherFile SYSTEM "otherFile.xml">]>
<doc>
    <foo>
    <bar>&otherFile;</bar>
    </foo>
</doc>
  • Entidades de Parámetros Externos, sintaxis: <!ENTITY % name SYSTEM uri>. Por ejemplo: 
    <?xml version="1.0" standalone="no"?>
    <!DOCTYPE doc [
      <!ENTITY % foo SYSTEM "http://www.example.com/student.dtd"<
      %foo;
    ]>
  • XInclude: incluye un conjunto de información externo en un documento XML
  • Referencia a componentes de Esquema XML usando el atributo schemaLocation, y elementos import e include. Ejemplo: schemaLocation="http://www.example.com/schema/bar.xsd"
  • Combinado hojas de estilo usando elementos import o include. Sintaxis: <xsl:include href="include.xsl"/>
  • Instrucción de procesado xml-stylesheet: usada para incluir una hoja de estilos en un documento XML, sintaxis: <?xml-stylesheet href="foo.xsl" type="text/xsl"?>
  • Función document() de XSLT: usada para acceder a nodos en un documento externo XML. Por ejemplo, <xsl:variable name="dummy" select="document('DocumentFunc2.xml')"/>.
