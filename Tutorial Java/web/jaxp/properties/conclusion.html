Conclusión

JAXP 1.5 proporciona nuevas propiedades para brindar controles a los usuarios para obtener recursos externos. El uso de las nuevas propiedades es el mismo como con otras propiedades existentes, excepto que las propiedades se proporcionan con las propiedades del sistema correspondientes y jaxp.properties de forma que puedan utilizarse para restricciones o permisos de todo el sistema.
