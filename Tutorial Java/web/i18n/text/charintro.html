Comprobando las Propiedades de los Caracteres

Puede categorizar caracteres de acuerdo a sus propiedades. Por ejemplo, X es una letra mayúscula y 4 es un dígito decimal. Comprobar propiedades de caracteres es una forma común de verificar los datos introducidos por los usuarios finales. Si está vendiendo libros en línea, por ejemplo, su pantalla de entrada de peticiones debe verificar que los caracteres en el campo de cantidad sean todos dígitos.

Los desarrolladores que no están acostumbrados a escribir software doble podrían determinar las propiedades de un carácter comparándolos con constantes de carácter. Por ejemplo, podrían escribir código como este: 

char ch;
//...

// This code is WRONG!

// check if ch is a letter
if ((ch >= 'a' && ch <= 'z') || (ch >= 'A' && ch <= 'Z'))
    // ...

// check if ch is a digit
if (ch >= '0' && ch <= '9')
    // ...

// check if ch is a whitespace
if ((ch == ' ') || (ch =='\n') || (ch == '\t'))
    // ...

El código anterior está equivocado porque funciona sólo con el inglés y otros pocos lenguajes. Para internacionalizar el ejemplo previo, reemplácelo con las sentencias siguientes: 

char ch;
// ...

// This code is OK!

if (Character.isLetter(ch))
    // ...

if (Character.isDigit(ch))
    // ...

if (Character.isSpaceChar(ch))
    // ...

Los métodos de Character confían en el Estándar Unicode para determinar las propiedades de un carácter. Unicode es una codificación de caracteres de 16 bits que soporta los principales idiomas del mundo. En el lenguaje de programación Java los valores char representan caracteres Unicode. Si comprueba las propiedades de un char con el método apropiado de Character, su código trabajará con todos los idiomas principales. Por ejemplo, el método Character.isLetter devuelve true si el carácter es una letra en chino, alemán, árabe, u otro idioma.

La lista siguiente da algunos de los métodos de comparación de Character más útiles. La documentación de la IPA de Character especifica completamente los métodos.

  • isDigit
  • isLetter
  • isLetterOrDigit
  • isLowerCase
  • isUpperCase
  • isSpaceChar
  • isDefined

El método Character.getType devuelve la categoría Unicode de un carácter. Cada categoría corresponde a una constante definida en la clase Character. Por ejemplo, getType devuelve la constante Character.UPPERCASE_LETTER para el carácter A. Para una lista completa de las constantes de categoría devueltas por getType, vea la documentación de la IPA de Character. El ejemplo siguiente muestra cómo usar getType y las constantes de categoría de Character. Todas las expresiones en estas sentencias if son true: 

if (Character.getType('a') == Character.LOWERCASE_LETTER)
    // ...

if (Character.getType('R') == Character.UPPERCASE_LETTER)
    // ...

if (Character.getType('>') == Character.MATH_SYMBOL)
    // ...

if (Character.getType('_') == Character.CONNECTOR_PUNCTUATION)
    // ...
