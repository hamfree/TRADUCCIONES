Caracteres Complementarios como Sustitutos

Para admitir los caracteres suplementarios sin cambiar el tipo de dato primitivo char y causar incompatibilidad con los programas anteriores de Java, los caracteres suplementarios se definen por un par de de valores de puntos de código que se llaman sustitutos. El primer punto de código es del rango altos sustitutos de U+D800 a U+DFBB, y el segundo punto de código es del rango bajos sustitutos de U+DC00 a U+DFFF. Por ejemplo, el carácter de Deseret I LARGA, U+10400, se define con este par de valores sustitutos: U+D801 y U+DC00.
