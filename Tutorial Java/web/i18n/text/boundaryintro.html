Detectando los Límites de Texto

Las aplicaciones que manipulan texto necesitan localizar los límites dentro del texto. Por ejemplo, considere algunas de las funciones comunes de un procesador de textos: subrayar un carácter, cortar una palabra, mover el cursor a la siguiente oración, y envolver una palabra al final de una línea. Para realizar cada una de estas funciones, el procesador de textos debe ser capaz de detectar los límites lógicos en el texto. Afortunadamente no tiene que escribir sus propias rutinas para realizar análisis de los límites. En vez de eso, puede aprovecharse de los métodos proporcionados por la clase BreakIterator .
