Límites de Palabra

Invoque el método getWordIterator para instanciar un BreakIterator que detecta los límites de palabra: 

BreakIterator wordIterator =
    BreakIterator.getWordInstance(currentLocale);

Querrá crear un BreakIterator cuando su aplicación necesite realizar operaciones sobre palabras individuales. Estas operaciones podrían ser funciones comunes de procesado de palabras, tales como seleccionar, cortar, pegar, y copiar. O, su aplicación podría buscar palabras, y debe ser capaz de distinguir palabras enteras de simples cadenas.

Cuando un BreakIterator analiza los límites de palabra, diferencia entre palabras y caracteres que no forman parte de palabras. Estos caracteres, que incluyen espacios, tabuladores, marcas de puntuación, y la mayoría de los símbolos, tienen límites de palabra en ambos lados.

El ejemplo que sigue el cual es del programa BreakIteratorDemo, marca los límites de palabra en algún texto. El programa crea el BreakIterator y después llama al método markBoundaries: 

Locale currentLocale = new Locale ("en","US");

BreakIterator wordIterator =
    BreakIterator.getWordInstance(currentLocale);

String someText = "She stopped. " +
    "She said, \"Hello there,\" and then went " +
    "on.";

markBoundaries(someText, wordIterator);

El método markBoundaries se define en BreakIteratorDemo.java. Este método marca los límites imprimiendo signos de intercalación (^) bajo la cadena objetivo. En el código que sigue, observe el bucle while donde markBoundaries explora la cadena llamando al método next: 

static void markBoundaries(String target, BreakIterator iterator) {

    StringBuffer markers = new StringBuffer();
    markers.setLength(target.length() + 1);
    for (int k = 0; k < markers.length(); k++) {
        markers.setCharAt(k,' ');
    }

    iterator.setText(target);
    int boundary = iterator.first();

    while (boundary != BreakIterator.DONE) {
        markers.setCharAt(boundary,'^');
        boundary = iterator.next();
    }

    System.out.println(target);
    System.out.println(markers);
}

A continuación, se muestra la salida del método markBoundaries. Tenga en cuenta dónde aparecen los signos de intercalación (^) en relación con los signos de puntuación y los espacios: 

She stopped.  She said, "Hello there," and then
^  ^^      ^^ ^  ^^   ^^^^    ^^    ^^^^  ^^   ^

went on.
^   ^^ ^^

La clase BreakIterator hace esto fácil al seleccionar las palabras en el texto. No tiene que escribir sus propias rutinas para manejar las reglas de puntuación de varios idiomas; la clase BreakIterator lo hace por usted.

El método extractWords en el ejemplo siguiente extrae e imprime palabras para una cadena dada. Tenga en cuenta que este método usa Character.isLetterOrDigit para evitar imprimir "palabras" que contengan caracteres de espacio. 

static void extractWords(String target, BreakIterator wordIterator) {

    wordIterator.setText(target);
    int start = wordIterator.first();
    int end = wordIterator.next();

    while (end != BreakIterator.DONE) {
        String word = target.substring(start,end);
        if (Character.isLetterOrDigit(word.charAt(0))) {
            System.out.println(word);
        }
        start = end;
        end = wordIterator.next();
    }
}

El programa BreakIteratorDemo invoca a extractWords, pasándole la misma cadena objetivo usada en el ejemplo previo. El método extractWords imprime la siguiente lista de palabras: 

She
stopped
She
said
Hello
there
and
then
went
on
