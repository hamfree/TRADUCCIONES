Ejemplo de Uso

Esta página contiene algunos trozos de código que le muestran varias escenarios comunes.

Crear un String desde un Punto de Código 

String newString(int codePoint) {
    return new String(Character.toChars(codePoint));
}

Crear un String desde un Punto de Código - Optimizado para Caracteres BMP

El método Character.toChars crea un vector temporal que se usa una vez y después es descartado. Si esto afecta negativamente al rendimiento, puede usar el enfoque siguiente que está optimizado para caracteres BMP (caracteres que se representan por un único valor char). En este método, toChars es invocado sólo para los caracteres suplementarios. 

String newString(int codePoint) {
    if (Character.charCount(codePoint) == 1) {
        return String.valueOf(codePoint);
    } else {
        return new String(Character.toChars(codePoint));
    }
}

Cración de Objetos String a Granel

Para crear un gran número de cadenas, la versión a granel del trozo anterior reusa el vector utilizado por el método toChars. Este método crea una instancia separada de String para cada punto de código y está optimizado para caracteres BMP. 

String[] newStrings(int[] codePoints) {
    String[] result = new String[codePoints.length];
    char[] codeUnits = new char[2];
    for (int i = 0; i < codePoints.length; i++) {
        int count = Character.toChars(codePoints[i], codeUnits, 0);
        result[i] = new String(codeUnits, 0, count);
    }
    return result;
}

Generar Mensajes

La IPA de formato admite caracteres suplementarios. El ejemplo siguiente es un forma simple de generar un mensaje. 

// recommended
System.out.printf("Character %c is invalid.%n", codePoint);

El enfoque siguiente es simple y evita la concatenación, lo cual hace el texto más difícil de traducir ya que no todos los idiomas insertan valores numéricos dentro de una cadena en el mismo orden que el inglés. 

// not recommended
System.out.println("Character " + String.valueOf(char) + " is invalid.");
