Límites de Carácter

Necesita ubicar los límites de carácter si su aplicación permite al usuario final subrayar caracteres individuales o mover el cursor a través del texto un carácter a la vez. Para crear un BreakIterator que localize los límites de carácter, invoque el método getCharacterInstance, como sigue: 

BreakIterator characterIterator =
    BreakIterator.getCharacterInstance(currentLocale);

Este tipo de BreakIterator detecta los límites entre los caracteres de usuario, no sólo de los caracteres Unicode.

Un carácter de usario puede estar compuesto por más de un carácter Unicode. Por ejemplo, el carácter de usuario ü puede estar compuesto por la combinación de los caracteres Unicode \u0075 (u) y \u00a8 (¨). Este no es el mejor ejemplo, sin embargo, porque el carácter ü puede ser representado también por el carácter único Unicode \u00fc. Dibujaremos con el idioma árabe para un ejemplo más realista.

Esta palabra contiene tres caracteres de usuario, pero está compuesta de los seis caracteres Unicode siguientes: 

String house = "\u0628" + "\u064e" + "\u064a" + "\u0652" + "\u067a" + "\u064f";

Los caracteres Unicode en las posiciones 1, 3 y 5 en la cadena house son diacríticos. El árable requiere diacríticos porque pueden alterar los significados de las palabras. Los diacríticos en el ejemplo son caracteres que no espacian, ya que aparecen sobre los caracteres base. En un procesador de texto árabe no puede mover el cursor en la pantalla una vez por cada carácter Unicode en la cadena. En su lugar, debe moverlo una vez por cada carácter de usuario, que puede estar compuesto por más de un carácter Unicode. Por lo tanto debe usar un BreakIterator para explorar los caracteres de usuario en la cadena.

El programa de ejemplo BreakIteratorDemo, crea un BreakIterator para explorar caracteres arábicos. El programa pasa este BreakIterator, junto con el objeto String creado anteriormente, a un método llamado listPositions: 

BreakIterator arCharIterator = BreakIterator.getCharacterInstance(
                                   new Locale ("ar","SA"));
listPositions (house, arCharIterator);

El método listPositions usa un BreakIterator para ubicar los límites de carácter en la cadena. Dese cuenta de que BreakIteratorDemo asigna una cadena particular a BreakIterator con el método setText. El programa recupera el límite de primer carácter con el método first y entonces invoca el método next hasta que la constante BreakIterator.DONE es devuelta. El código para esta rutina es como sigue: 

static void listPositions(String target, BreakIterator iterator) {
                
    iterator.setText(target);
    int boundary = iterator.first();

    while (boundary != BreakIterator.DONE) {
        System.out.println (boundary);
        boundary = iterator.next();
    }
}

El método listPositions imprime las posiciones límite siguientes para los caracteres de usuario en la cadena house. Note que las posiciones de los diacríticos (1, 3, 5) no están listados: 

0
2
4
6
