Lección: Trabajando con Texto

Casi todos los programas con interfaces de usuario manipulan texto. En un mercado internacional el texto que sus programas visualizan deben ajustarse a las reglas de los idiomas de todo el mundo. El lenguaje de programación de Java proporciona varias clases que le ayudan a manejar texto de una manera independiente a la configuración regional.

Comprobando las Propiedades de los Caracteres

Esta sección explica cómo usar los métodos de comparación de Character para comprobar las propiedades de carácter para todos los idiomas principales.

Comparando Cadenas

En esta sección aprenderá cómo realizar comparaciones de cadena independientes de la configuración regional con la clase Collator.

Detectando los Límites de Texto

Esta sección muestra como la clase BreakIterator puede detectar carácteres, palabras, oraciones y límites de línea.

Convirtiendo Texto No Unicode

Diferentes sistemas de computadoras alrededor del mundo almacenan texto en una variedad de esquemas de codificación. Esta sección describe las clases que le ayudan a convertir texto entre Unicode y otras codificaciones.

IPA de Normalizer

Esta sección explica cómo usar la IPA de Normalizer para transformar texto aplicando diferentes formas de normalización.

Trabajar con Texto Bidireccional con la CLase JTextComponent

Esta sección describe cómo trabajar con texto bidireccional, el cual es el texto que contiene texto que corre en dos direcciones, de izquierda a derecha y de derecha a izquierda.
