Recorrido: Internacionalización

Las leccones en este recorrido le enseñan como internacionalizar aplicaciones Java. Las aplicaciones internacionaliadas son fáciles de adaptar a las costumbres y los idiomas de los usuarios finales de todo el mundo.

Nota: Este recorrido del tutorial cubre la funcionalidad principal, la cual es la base requerida por las características adicionales proporcionadas para aplicaciones de escritorio, empresariales y móviles. Para información adicional, vea Java Internacionalización home page.

  Introducción define el término internacionalización, dando un ejemplo de programa rápido, y proporciona una lista de verificación que puede utilizar para internacionalizar un programa existente.

 Estableciendo la Localización explica cómo crear y como usar los objetos Locale.

  Aislando Datos de una Localización Específica muestra cómo acceder dinámicamente a objetos que varían con Locale.

  Formateando explica cómo dar formato a números, fechas, y mensajes de texto, de acuerdo a Locale, y cómo crear formatos personalizados con patrones.

  Trabajando con Texto proporciona técnicas para manipular texto de una manera independiente de la configuración regional.

  Internacionalización de Recursos en Red explica cómo proporcionar internacionalización para los IDNs.

  Proveedores de Servicio para Internacionalización explica cómo habilitar el plug-in de los datos y servicios que dependen de la configuración regional.
