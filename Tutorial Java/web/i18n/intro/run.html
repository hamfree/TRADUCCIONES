Ejecutando el Programa de Ejemplo

El programa internacionalizado es flexible; permite al usuario final especificar un idioma y un país en la línea de comandos. En el ejemplo siguiente el código del idioma es fr (francés) y el código de país es FR (Francia), por lo que el programa visualiza los mensajes en francés: 

% java I18NSample fr FR
Bonjour.
Comment allez-vous?
Au revoir.

En el ejemplo siguiente el código de idioma es en (inglés) y el código de país es US (Estados Unidos de América) así que el programa visualiza los mensajes en inglés: 

% java I18NSample en US
Hello.
How are you?
Goodbye.
