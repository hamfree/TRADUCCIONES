Antes de la Internacionalización

Suponga que ha escrito un programa que visualiza tres mensajes, como el que sigue: 

public class NotI18N {

    static public void main(String[] args) {

        System.out.println("Hello.");
        System.out.println("How are you?");
        System.out.println("Goodbye.");
    }
}

Ha decidido que este programa necesita visualizar estos mismos mensajes para la gente que vive en Francia y Alemania. Desafortunadamente su plantilla de programación no es multilingüe, así que necesitará ayuda traduciendo los mensajes al francés y alemán. Ya que los traductores no son programadores, tendrá que mover los mensajes aparte del código fuente e introducirlos en ficheros de texto que los traductores puedan editar. Además, el programa debe ser lo bastante flexible para que pueda visualizar mensajes en otros idiomsa, pero por ahora nadie sabe qué idiomas serán.

Parece que el programa necesita ser internacionalizado.
