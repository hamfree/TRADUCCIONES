Cambiando los Símbolos del Formato de Fecha

Nota de Versión: Esta sección de Fecha y Tiempo usa las IPAs de fecha y tiempo en el paquete java.util. Las IPAs de java.time, disponibles en la versión 8 del KDJ, proporcionan un modelo de fecha y tiempo completo que ofrece mejoras significativas sobre las clases de java.util. Las IPAs de java.time se describen en el recorrido Fecha y Tiempo. La página Código Date-Time Heredado puede ser de especial interés.

El método format de la clase SimpleDateFormat devuelve un objeto String compuesto de dígitos y símbolos. Por ejemplo, en la String "Friday, April 10, 2009," los símbolos son "Friday" y "April." Si los símbolos encapsulados en SimpleDateFormat no se ajustan a sus necesidades, puede cambiarlos con DateFormatSymbols. Puede cambiar los símbolos que representan nombres para los meses, días de la semana, y zonas horarias, entre otros. La tabla siguiente lista los métodos de DateFormatSymbols que le permiten modificar los símbolos:

Métodos de DateFormatSymbol
Método Establecedor Ejemplo de un Símbolo que el Método Modifica
setAmPmStrings PM
setEras AD
setMonths December
setShortMonths Dec
setShortWeekdays Tue
setWeekdays Tuesday
setZoneStrings PST

El ejemplo siguiente invoca setShortWeekdays para cambiar los nombres cortos de los días de la semana de caracteres minúsculas a mayúsculas. El código fuente completo para este ejemplo está en DateFormatSymbolsDemo. El primer elemento en el argumento de vector de setShortWeekdays es una String nula. Por lo tanto el vector está basado en uno en vez de en cero. El constructor SimpleDateFormat acepta el objeto DateFormatSymbols modificado como un argumento. Aquí está el código fuente: 

Date today;
String result;
SimpleDateFormat formatter;
DateFormatSymbols symbols;
String[] defaultDays;
String[] modifiedDays;

symbols = new DateFormatSymbols( new Locale("en", "US"));
defaultDays = symbols.getShortWeekdays();

for (int i = 0; i < defaultDays.length; i++) {
    System.out.print(defaultDays[i] + " ");
}
System.out.println();

String[] capitalDays = {
    "", "SUN", "MON",
    "TUE", "WED", "THU",
    "FRI", "SAT"
};
symbols.setShortWeekdays(capitalDays);

modifiedDays = symbols.getShortWeekdays();
for (int i = 0; i < modifiedDays.length; i++) {
    System.out.print(modifiedDays[i] + " ");
}
System.out.println();
System.out.println();

formatter = new SimpleDateFormat("E", symbols);
today = new Date();
result = formatter.format(today);
System.out.println("Today's day of the week: " + result);

El código anterior genera esta salida: 

 Sun Mon Tue Wed Thu Fri Sat 
 SUN MON TUE WED THU FRI SAT 

Today's day of the week: MON
