Usando Formatos Predefinidos

Nota de Versión: Esta sección de Fecha y Tiempo usa las IPAs de fecha y tiempo en el paquete java.util. Las IPAs de java.time, disponibles en la versión 8 del KDJ, proporcionan un modelo de fecha y tiempo completo que ofrece mejoras significativas sobre las clases de java.util. Las IPAs de java.time se describen en el recorrido Fecha y Tiempo. La página Código Date-Time Heredado puede ser de especial interés.

La clase DateFormat le permite dar formato a las fechas y horas con estilos predefinidos de una manera sensible a la configuración regional. Las secciones que siguen demuestran cómo usar la clase DateFormat con un programa llamado DateFormatDemo.java.

Fechas

Dar formato a las fechas con la clase DateFormat es un proceso de dos pasos. Primero, crea un formateador con el método getDateInstance. Segundo, invoca al método format, el cual devuelve un objeto String conteniendo la fecha formateada. El ejemplo siguiente da formato a la fecha de hoy llamando a estos dos métodos: 

Date today;
String dateOut;
DateFormat dateFormatter;

dateFormatter = DateFormat.getDateInstance(DateFormat.DEFAULT, currentLocale);
today = new Date();
dateOut = dateFormatter.format(today);

System.out.println(dateOut + " " + currentLocale.toString());

La salida generada por este código a continuación. Dese cuenta de que los formatos de las fechas varían con Locale. Ya que DateFormat es sensible a la configuración regional, se ocupa de los detalles de formato para cada Locale. 

30 juin 2009     fr_FR
30.06.2009       de_DE
Jun 30, 2009     en_US

El código de ejemplo anterior especificó el estilo de formato DEFAULT. El estilo DEFAULT es simplemente uno de los estilos de formato predefinidos que la clase DateFormat proporciona, como los siguientes:

  • DEFAULT
  • SHORT
  • MEDIUM
  • LONG
  • FULL

La tabla siguiente muestra cómo se da formato a las fechas para cada estilo con las configuraciones regionales de EE.UU y Francia:

Formatos de Fecha de Ejemplo
Estilo Configuración Regional EE.UU Configuración Regional Francia
DEFAULT Jun 30, 2009 30 juin 2009
SHORT 6/30/09 30/06/09
MEDIUM Jun 30, 2009 30 juin 2009
LONG June 30, 2009 30 juin 2009
FULL Tuesday, June 30, 2009 mardi 30 juin 2009

Horas

Los objetos Date representan fechas y horas. Dar formato a las horas con la clase DateFormat es similar a dar formato a las fechas, excepto que crea el formateador con el método getTimeInstance, como sigue: 

DateFormat timeFormatter =
    DateFormat.getTimeInstance(DateFormat.DEFAULT, currentLocale);

La tabla que sigue muestra los diversos estilos de formato predefinidos para las configuraciones locales de EE.UU y Alemania:

Formatos de Hora de Ejemplo
Estilo Configuración Regional EE.UU Configuración Regional Alemania
DEFAULT 7:03:47 AM 7:03:47
SHORT 7:03 AM 07:03
MEDIUM 7:03:47 AM 07:03:07
LONG 7:03:47 AM PDT 07:03:45 PDT
FULL 7:03:47 AM PDT 7.03 Uhr PDT

Tanto Fechas Como Horas

Para visualizar una fecha y hora en el mismo objeto String, cree el formateador con el método getDateTimeInstance. El primer parámetro es el estilo de la fecha, y el segundo es el estilo de la hora. El tercer parámetro es el Locale. Aquí tiene un ejemplo rápido: 

DateFormat formatter = DateFormat.getDateTimeInstance(
                           DateFormat.LONG, 
                           DateFormat.LONG, 
                           currentLocale);

La tabla siguiente muestra los estilos de formato de fecha y hora para las configuraciones regionales de EE.UU y Francia:
