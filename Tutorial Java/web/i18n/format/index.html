Lección: Formateando

Esta lección explica como dar formato a números, monedas, fechas, horas, y mensajes de texto. Ya que los usuarios finales pueden ver estos elementos de datos, su formato debe ajustarse a varias convenciones culturales. Siguiendo los ejemplos en esta lección le enseñarán a como:

  • Dar formato a los elementos de datos de una manera sensible a la configuración regional
  • Mantener su código independiente de la configuración regional
  • Evitar la necesidad de escribir rutinas de formato para configuraciones regionales específicas

Números y Monedas

Esta sección explica cómo usar las clases NumberFormat, DecimalFormat, y DecimalFormatSymbols.

Fechas y Horas

Nota de versión: Esta sección de Fecha y Hora usa las IPAs de fecha y tiempo del paquete java.util. Las IPAs de java.time, disponibles en la versión 8 del KDJ, proporcionan un modelo completo de fecha y hora que ofrece mejoras significativas sobre las clases de java.util. Las IPAs de java.time se describen en el recorrido Fecha y Tiempo. La página Código Date-Time Heredado podría ser de particular interés.

Esta sección se enfoca en las clases DateFormat, SimpleDateFormat, y DateFormatSymbols.

Mensajes

Esta sección muestra cómo las clases MessageFormat y ChoiceFormat pueden ayudarle a resolver algunos de los problemas que puede encontrar cuando da formato a los mensajes de texto.
