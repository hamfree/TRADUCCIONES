Mensajes

A todos nosotros nos gusta usar programas que nos permitan saber que está pasando. Los programas que nos mantienen informados a menudo lo hacen visualizando mensajes de estado y de error. Por supuesto, estos mensajes necesitan ser transladados para que puedan ser entendidos por los usuarios finales en todo el mundo. La sección Aislando Datos de una Localización Específica describe los mensajes de texto traducibles. Habitualmente, ha terminado después de mover un mensaje String dentro de un ResourceBundle. Sin embargo, si ha embebido datos variables en un mensaje, tendrá que realizar unos pasos adicionales para prepararlo para la traducción.

Un mensaje compuesto contiene datos variables. En la lista siguiente de mensajes compuestos, los datos variables están subrayados: 

The disk named MyDisk contains 300 files.
The current balance of account #34-09-222 is $2,745.72.
405,390 people have visited your website since January 1, 2009.
Delete all files older than 120 days.

Podría estar tentado a construir el último mensaje en la lista precedente concatenando frases y variables como sigue: 

double numDays;
ResourceBundle msgBundle;
// ...
String message = msgBundle.getString(
                     "deleteolder" +
                     numDays.toString() +
                     msgBundle.getString("days"));

Este enfoque funciona bien en inglés, pero no funcionará para idiomas en los cuales el verbo aparece al final de la oración. Ya que el orden de las palabras de este mensaje está codificado, sus traductores no serán capaces de crear traducciones gramaticalmente correctas para todos los idiomas.

¿Cómo puede hacer que su programa sea localizable si necesita usar mensajes compuestos? Puede hacerlo usando la clase MessageFormat, la cual es el tema de esta sección.

Precaución: 

Los mensajes compuestos son difíciles de traducir porque el mensaje de texto está fragmentado. Si usa mensajes compuestos, la traducción tomará más tiempo y costará más. Por lo tanto debe usar mensajes compuestos sólo cuando sea necesario.

Tratando con Mensajes Compuestos

Un mensaje compuesto puede contener varias clases de variables: fechas, horas, cadenas, números, monedas, y porcentajes. Para dar formato a un mensaje compuesto de un manera independiente de la configuración regional, construya un patrón que aplique a un objeto MessageFormat.

Manejando plurales

Las palabras en un mensaje habitualmente varían si son posibles las formas de palabra en plural y singular. Con la clase ChoiceFormat, puede asignar un número a una palabra o frase, permitiéndole construir mensajes que son gramaticalmente correctos.
