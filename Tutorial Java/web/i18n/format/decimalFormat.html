Personalizando Formatos

Puede usar la clase DecimalFormat para dar formato a los números decimales como cadenas específicas de la configuración regional. Esta clase le permite controlar la visualización de los ceros iniciales y finales, los prefijos y sufijos, los separador de agrupamiento (miles), y el separador decimal. Si quiere cambiar los símbolos de formato, tales como el separador de decimales, puede usar DecimalFormatSymbols en conjunción con la clase DecimalFormat. Estas clases ofrecen mucha flexibilidad en el formato de los números, pero pueden hacer su código más complejo.

El texto que sigue usa ejemplos que demuestran las clases DecimalFormat y DecimalFormatSymbols. El código de los ejemplos en este material son de un programa de ejemplo llamado DecimalFormatDemo.

Construir Patrones

Usted especifica las propiedades de formato de DecimalFormat con un patrón String. El patrón determina cómo se ve el número formateado. Para una descripción completa de la sintaxis del patrón, vea Sintaxis de Patrón de Formato de Número.

El ejemplo que sigue crea un formateados pasando un patrón de tipo String al constructor de DecimalFormat. El método format acepta un valor double como un argumento y devuelve el número formateado en un objeto String: 

DecimalFormat myFormatter = new DecimalFormat(pattern);
String output = myFormatter.format(value);
System.out.println(value + " " + pattern + " " + output);

La salida de las lineas de código anteriores se describen en la tabla siguiente. El valor es el número, un double. que va a ser formateado. El patrón es el objeto String que especifica las propiedades de formato. La salida , la cual es un objeto String, representa el número formateado.

123456.789 ###.## 123456.79 El valor tiene tres dígitos a la derecha del punto decimal, pero el patrón tiene sólo dos. El método format maneja esto redondeando hacia arriba.

Formato Sensible a la Configuración Regional

El ejemplo anterior creó un objeto DecimalFormat para el Locale predeterminado. Si quiere un objeto DecimalFormat para un Locale no predeterminado, instancie un NumberFormat y entonces conviértalo por moldeo en un DecimalFormat. Aquí hay un ejemplo: 

NumberFormat nf = NumberFormat.getNumberInstance(loc);
DecimalFormat df = (DecimalFormat)nf;
df.applyPattern(pattern);
String output = df.format(value);
System.out.println(pattern + " " + output + " " + loc.toString());

Ejecutar el código de ejemplo previo resulta en la salida que sigue. El número formateado, el cual está en la segunda columna, varía con el Locale: 

###,###.###      123,456.789     en_US
###,###.###      123.456,789     de_DE
###,###.###      123 456,789     fr_FR

Hasta ahora, los patrones de formato descritos aquí siguen las convenciones del inglés de EE.UU. Por ejemplo, en el patrón ###,###.## la coma es el separador de miles y el punto representa el punto decimal. Esta convención está bien, siempre que sus usuarios finales no esté expuestos a ella. Sin embargo, algunas aplicaciones, tales como las hojas de cálculo y los generadores de informes, permiten a los usuarios finales definir sus propios patrones de formato. Para estas aplicaciones los patrones de formato especificados por los usuarios finales deben usar notación localizada. En estos casos querrá invocar el método applyLocalizedPattern del objeto DecimalFormat.

Alterar los Símbolos de Formato

Puede usar la clase DecimalFormatSymbols  para cambiar los símbolos que aparecen en los números formateados producidos por el método format. Estos símbolos incluyen el separador decimal, el separador de agrupamiento, el signo menos, y el signo de porcentaje, entre otros.

El ejemplo siguiente demuestra la clase DecimalFormatSymbols aplicando un formato extraño a un número. El format inusual es el resultado de las llamadas a los métodos setDecimalSeparator, setGroupingSeparator, y setGroupingSize. 

DecimalFormatSymbols unusualSymbols = new DecimalFormatSymbols(currentLocale);
unusualSymbols.setDecimalSeparator('|');
unusualSymbols.setGroupingSeparator('^');

String strange = "#,##0.###";
DecimalFormat weirdFormatter = new DecimalFormat(strange, unusualSymbols);
weirdFormatter.setGroupingSize(4);

String bizarre = weirdFormatter.format(12345.678);
System.out.println(bizarre);

Cuando lo ejecuta, este ejemplo imprime el número en un formato extraño: 

1^2345|678

Sintaxis de Patrón de Formato de Número

Puede diseñar sus propios patrones de formato para números siguiendo las reglas especificadas por el siguietne diagrama BNF: 

patrón     := subpatrón{;subpatrón}
subpatrón  := {prefijo}entero{.fracción}{sufijo}
prefijo    := '\\u0000'..'\\uFFFD' - caracteresEspeciales
sufijo     := '\\u0000'..'\\uFFFD' - caracteresEspeciales
entero     := '#'* '0'* '0'
fracción   := '0'* '#'*

La notación usada en el diagrama anterior se explica en la tabla siguiente:

descripción de la notación
Notación Descripción
X* 0 o más instancias de X
(X | Y) ya sea X o Y
X..Y cualquier carácter desde X hasta llegar a Y, inclusive
S - T caracteres en S, excepto aquellos en T
{X} X es opcional

En el diagrama BNF precedente, el primer subpatrón especifica el formato para los números positivos. El segundo subpatrón, el cual es opcional, especifica el formato para los números negativos.

Aunque no se indica en el diagrama BNF, puede aparecer una coma dentro de la parte entera.

Dentro de los subpatrones, especifica el formato con símbolos especiales. Estos símbolos se describen en la tabla siguiente:

descripción del símbolo de formato
Símbolo Descripción
0 un dígito
# un dígito, el cero se muestra como ausente
. marca de posición para el separador decimal
, marca de posición para el separador de agrupaciones
E separa la mantisa y el exponenete en los formatos exponenciales
; separa formatos
- prefijo predeterminado de negativo
% multiplica por 100 y se muestra como porcentaje
? multiplica por 1000 y se muestra como por mil
¤ el signo de moneda, reemplazada por el símbolo de moneda; si se dobla, se reemplaza por el símbolo internacional de moneda; si está presente en un patrón, el separador decimal monetario se usa en vez del separador decimal
X cualesquiera otros caracteres pueden usarse en el prefijo o sufijo
' usado para citar caracteres especiales en un prefijo o sufijo
