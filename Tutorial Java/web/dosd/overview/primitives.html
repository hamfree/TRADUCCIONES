Primitivas Geométricas

La IPA de Java 2D proporciona un conjunto de formas estándar tales como puntos, líneas, rectangulos, arcos, elipses, y curvas. El paquete más importante para definir las primitivas geométricas comunes es el paquete java.awt.geom. Las formas arbitrarias se pueden representar mediante combinaciones de primitivas geométricas rectas.

La interfaz Shape representa un forma geométrica, que tiene un contorno y un interior. Esta interfaz proporciona un conjunto común de métodos para describir e inspeccionar los objetos geométricos de dos dimensiones y admite segmentos de línea curvadas y múltiples subformas. La clase Graphics admite solo segmentos de línea recta. La interfaz Shape puede admitir segmentos con curvas.

Para más detalles sobre como dibujar y rellenar formas, vea la lección Trabajando con la Geometría.

Puntos

La clase Point2D define un punto representando una ubicación (x,y) en el espacio de coordenadas. El término “point” en la IPA de Java 2D no es lo mismo que un pixel. Un punto no tiene área, no contiene un color, y no puede ser representado.

Los puntos se usan para crear otras formas. La clase Point2D también incluye un método para calcular la distancia entre dos puntos.

Líneas

La clase Line2D es una clase abstracta que representa una línea. las coordenadas de una línea pueden ser recuperadas como double. La clase Line2D incluye varios métodos para establecer los puntos finales de una línea.

Puede también crear un segmento de línea recta usando la clase GeneralPath descrita abajo.

Formas Rectangulares

Las primitivas Rectangle2D, RoundRectangle2D, Arc2D, y Ellipse2D se derivan todas de la clase RectangularShape. Esta clase define métodos para objetos Shape que pueden ser descritos por una caja delimitada rectangular. La geometría de un objeto RectangularShape puede ser extrapolada desde un rectángulo que encierra completamente el contorno de la Shape.

Curvas Cuadráticas y Cúbicas

La clase QuadCurve2D le permite crear segmentos de curvas paramétricas cuadráticas. Una curva cuadrática está definida por dos puntos finales y un punto de control.

La clase CubicCurve2D le permite crear segmentos de curva paramétricos cúbicos. Una curva cúbica se define por dos puntos finales y dos puntos de control. Formas Arbitrarias

La clase GeneralPath le permite construir una forma arbitraria especificando unas series de posiciones a lo largo del límite de la forma. Estas posiciones pueden ser conectadas por segmentos de línea, curvas cuadráticas, o curvas cúbicas (de Bézier). La siguiente forma se puede crear con tres segmentos de línea y una curva cúbica. Areas

Con la clase Area, puede realizar operaciones booleanas, tales como la unión, intersección, y substracción, sobre dos objetos Shape. Esta técnica, a menudo denominada como geometría de área constructiva, le permite crear rápidamente objetos Shape complejos sin tener que describir cada segmentos de línea o curva.
