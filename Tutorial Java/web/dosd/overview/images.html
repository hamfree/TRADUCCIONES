Imágenes

En la IPA 2D de Java una imagen es habitualmente una matriz rectangular bidimensional de píxeles, donde cada píxel representa el color en esa posición de la imagen y donde las dimensiones representan la extensión horizontal (ancho) y la extensión vertical (altura) de la imagen tal como se muestra.

La clase de imagen más importante para representar tales imágenes es la clase java.awt.image.BufferedImage. La IPA 2D de Java almacena los contenidos de tales imágenes en memoria de forma que puedan ser accedidas directamente.

Las aplicaciones pueden crear directamente un objeto BufferedImage u obtener una imagen desde un formato externo de imagen tal como PNG o GIF.

En cualquier caso, la aplicación puede luego dibujar en la imagen usando llamadas de gráficos IPA 2D de Java. Así, las imágenes no se limitan solo visualizar imágenes de tipo fotográfico. Diferentes objetos tales como arte de línea, texto, y otros gráficos e incluso otras imágenes se pueden dibujar dentro de una imagen (como se muestra en las siguientes imágenes).

La IPA 2D de Java le permite aplicar operaciones de filtrado de imagen a BufferedImage e incluye varios filtros integrados. Por ejemplo, el filtro ConvolveOp se puede utilizar para desenfocar o enfocar imágenes.

La imagen resultante puede después ser dibujada a una pantalla, enviada a una impresora, o salvada en un formato gráfico tal como PNG, GIF, etc. Para aprender más sobre imágenes vea la lección Trabajando con Imágenes.
