Lección: Vista general de los conceptos de la IPA Java 2D

La IPA de Java 2D propociona gráficos, texto y capacidades de imagen en dos dimensiones para los programas Java a través de extensiones al Kit de Herramientas de Ventanas Abstractas o Abstract Windowing Toolkit (AWT) en inglés. Este paquete de renderizado completo admite líneas de arte, texto e imágenes en un marco flexible y con todas las funciones para desarrollar interfaces de usuario más ricas, sofisticados programas de dibujo y editores de imágenes. Los objetos de Java 2D existen en un plano llamado espacio de coordenadas del usuario, o simplemente espacio de usuario. Cuando los objetos se presentan en una pantalla o una impresora, las coordenadas del espacio de usuario son transformadas en coordenadas del espacio del dispositivo. Los siguientes enlaces son útiles para empezar a aprender sobre la la IPA de Java 2D:

  • Clase Graphics
  • Clase Graphics2D

La IPA de Java 2D proporciona las siguientes capacidades:

  • Un modelo de representación uniforme para los dispostiivos de visualización e impresoras
  • Un amplio rango de primitivas geométricas, tales como curvas, rectángulos, y elipses, como tanbién un mecanismo para presentar virtualmente cualquier forma geométrica
  • Mecanismos para realizar la detección de colisiones en formas, textos e imagenes
  • Un modelo de composición que proporcina control sobre como se presentan los objetos que se se solapan.
  • Soporte de color mejorado que facilita la gestión del color
  • Soporte para imprimir documentos complejos
  • Control de la calidad del renderizado mediante el uso de sugerencias de renderizado
