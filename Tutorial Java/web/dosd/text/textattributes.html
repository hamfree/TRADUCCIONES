Usando Atributos de Texto para Estilizar el Texto

Las aplicaciones normalmente necesitan la capacidad de aplicar los siguientes atributos de texto:

  • Subrayado– Una línea que se dibuja debajo del texto.
  • Tachado– Una línea horizontal que se dibuja a través del texto.
  • Superíndice o Subíndice – Un texto o una letra que aparece ligeramente por encima de una línea o, en consecuencia, debajo de una línea.
  • Kerning– El ajuste del espacio entre caracteres

Estos y otros atributos de texto se pueden aplicar utilizando la clase Java 2D TextAttribute .

Para aplicar estos atributos de texto, agréguelos a un objeto Font. Por ejemplo: 

Map<TextAttribute, Object> map =
    new Hashtable<TextAttribute, Object>();
map.put(TextAttribute.KERNING,
    TextAttribute.KERNING_ON);
font = font.deriveFont(map);
graphics.setFont(font);

El ejemplo de código representado a continuación muestra la aplicación de atributos de texto en el siguiente orden:

  1. Cadena de muestra (no se aplican atributos de texto)
  2. Kerning
  3. Kerning y subrayado
  4. Kerning, subrayado y tachado
  5. Kerning, subrayado, tachado y color

Nota: Si no ve el applet ejecutándose, debe instalar al menos la versión 7 del Kit de Desarrollo de Java SE (KDJ).

El código completo para este applet está en AttributedText.java.
