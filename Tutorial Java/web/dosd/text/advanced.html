Visualización Avanzada de Texto

La IPA 2D de Java proporciona mecanismos para admitir un diseño de texto sofisticado. Esta sección describe las siguientes características de la visualización de texto avanzada.

Visualizando Texto Suavizado Usando Consejos de Renderizado

Esta sección explica cómo controlar la calidad de la representación mediante el uso de sugerencias de representación.

Usando Atributos de Texto para Estilizar el Texto

Esta sección explica cómo usar la clase TextAttribute para subrayar o tachar texto.

Dibujando Múltiples Líneas de Texto

Esta sección explica cómo colocar y renderizar un párrafo de texto con estilo usando las clases TextLayout y LineBreakMeasurer.

Trabajando con Texto Bidireccional

En esta sección se explica cómo trabajar con texto bidireccional con clases en los paquetes java.awt y java.awt.font.
