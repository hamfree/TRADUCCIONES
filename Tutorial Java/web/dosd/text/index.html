Lección: Trabajando con la IPA de Texto

Esta lección le presenta el concepto de trabajar con IPA de texto para aplicar las capacidades de representación de texto. También en este recorrido, ha usado la IPA de texto básica de Java 2D y conoce como establecer una fuente y una posición, y cómo dibujar texto.

Esta lección amplía ese material para ayudarlo a desarrollar una comprensión de cómo usar esos IPA y avanza en las capacidades de la visualización de texto Java 2D.

Estos temas se describen en las secciones siguientes.

Fuentes Físicas y Lógicas
Esta sección explica cómo usar los métodos de la clase Font para determinar qué fuentes están disponibles en su sistema, para crear un objeto Font, y obtener información sobre una familia de fuente.
Midiendo Texto
Esta sección explica cómo medir apropiadamente el texto usando una instancia de la clase FontMetrics.
Visualización Avanzada de Texto
Esta sección explica cómo posicionar y presentar un párrafo de texto estilizado, visualizar texto suavizado, usar atributos de texto para estilizar el texto, y trabajar con texto bidireccional.
