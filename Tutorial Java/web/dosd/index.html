Recorrido: Gráficos 2D

Este recorrido le presenta la IPA de JAva 2D™ y le muestra cómo mostrar e imprimir gráficos 2D en sus programas Java. El recorrido está destinado a desarrolladores que deseen enriquecer sus conocimientos sobre Java 2D IPA, así como a principiantes en gráficos por computadora. Casi cada sección contiene ejemplos relevantes para ilustrar capacidades específicas. La IPA de Java 2D le permite realizar fácilmente las tareas siguientes:

  • Dibuja líneas, rectángulos y cualquier otra forma geométrica.
  • Rellena esas formas con colores sólidos o degradados y texturas.
  • Dibujar texto con opciones para un control preciso sobre la fuente y el proceso de renderizado.
  • Dibujar imágenes, aplicando opcionalmente operaciones de filtrado.
  • Aplique operaciones como composición y transformación durante cualquiera de las operaciones de renderización anteriores.

Este capítulo también explica conceptos menos familiares tales como la composición.

Usar Gráficos de la IPA 2D para visualizar gráficos complejos

Usar operaciones de filtrado de imágenes

Este capítulo describe el concepto de dibujar imágenes en pantalla y fuera de pantalla, así como superficies y dispositivo de impresión. Este recorrido cubre los usos más comunes de la IPA de Java 2D y brevemente describe algunas de las características más avanzadas.

  Descripción general de la API de gráficos 2D de Java introduce los conceptos claves de Java 2D y describe el modelo de renderizado de Java 2D. Esta lección es más conceptural que otras lecciones de este recorrido, le permite profundizar en nociones básicas y descripciones de clases.

  Empezando con Graphics usa un ejemplo desarrollado para mostrar como obtener un objeto Graphics y usarlos paras las tareas comunes de renderizado de gráficos.

  Trabajar con Geometría le enseña cómo usar la IPA para dibujar primitivas de gráficos y formas arbitrarias, y como aplicar trazos elegantes y estilos de relleno.

  Trabajar con la IPA de Texto le muestra cómo usar efectivamente la IPA de texto, incluyendo cómo crear un objeto Font con los atributos deseados, medir el texto y determinar los nombres de las fuentes disponibles en su sistema.

  Trabajar con imágenes explica cómo crear un objeto BufferedImage, realizar operaciones de filtrado de imagen, y dibujar sobre una imagen.

  Imprimir le enseña cómo presentar gráficos 2D a un imppesora, imprimir documentos complejos, y usar los Servicios de Impresión.

  Temas Avanzados en Java 2D explica cómo realizar transformaciones, recortar la región de dibujo, componer gráficos superpuestos, especificar preferencias de representación y controlar la calidad de la representación.
