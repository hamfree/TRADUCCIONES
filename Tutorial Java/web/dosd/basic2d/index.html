Lección: Empezando con los Gráficos

La IPA 2D de Java es potente y compleja. Sin embargo, la vasta mayoría de los usos para la IPA 2D de Java utilizan un pequeño subconjunto de sus capacidades encapsuladas en la clase java.awt.Graphics. Esta lección cubre las necesidades más comunes de los desarrolladores de aplicaciones. Las necesidades menos comunes se describen más tarde en Temas Avanzados en Java 2D.

La mayoría de los métodos de la clase Graphics pueden ser divididos en dos grupos básicos:

  • Métodos de dibujo y rellenado, que le permiten presentar formas básicas, texto, e imágenes
  • Métodos de configuración de atributos, que afectan a como el dibujado y rellenado aparece

Métodos tales como setFont y setColor definen cómo renderizan los métodos de dibujado y rellenado.

Esta figura ilustra cómo se relacionan estos métodos con los objetos gráficos:

Los métodos de dibujando incluyen:

  • drawString – Para dibujar texto 
    g.drawString("Hello", 10, 10);
  • drawImage – Para dibujar imágenes 
    g.drawImage(img, 
            0, 0, width, height,
            0, 0, imageWidth, imageHeight,
            null);
  • drawLine, drawArc, drawRect, drawOval, drawPolygon – Para dibujar formas geométricas 
    g2.draw(new Line2D.Double(0, 0, 30, 40));

Dependiendo de su necesidad actual, puede eleguir uno de varios métodos en la clase Graphics basado en el criterio siguiente:

  • Si desea renderizar la imagen en la ubicación especificada en su tamaño original o escalarla para que quepa dentro del rectángulo dado
  • Si prefiere llenar las áreas transparentes de la imagen con color o mantenerlas transparentes

Los métodos de relleno se aplican a las formas geométricas e incluyen fillArc, fillRect, fillOval, fillPolygon.

Si usted dibuja una línea de texto o una imagen, recuerde que en los gráficos 2D cada punto se determina por sus coordenadas x,y. Todos los métodos de dibujo y relleno necesitan esta información que determina dónde se debe renderizar el texto o la imagen.

Por ejemplo, para dibujar una línea, una aplicación llama a lo siguiente:

java.awt.Graphics.drawLine(int x1, int y1, int x2, int y2)

En este código (x1, y1) es el punto de inicio de la línea, y (x2, y2) es el punto final de la línea.

Entonces, el código para dibujar una línea horizontal es el siguiente:

Graphics.drawLine(20, 100, 120, 100);

El método fillRect le permite dibujar un rectángulo relleno con el color especificado. 

...
Rectangle2D boundsTemp 
          = font.getStringBounds(tempString, frc);
Rectangle2D boundsCond 
          = font.getStringBounds(condStr, frc);
int wText = Math.max((int)boundsTemp.getWidth(),
            (int)boundsCond.getWidth());

            + (int)boundsCond.getHeight();

int rY = (size.height-hText)/2;
