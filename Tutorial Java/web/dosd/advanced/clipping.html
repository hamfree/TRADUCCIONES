Recortando la Región de Dibujo

Cualquier objeto Shape se puede utilizar como trazado de recorte que restringe la parte del área de dibujo que se renderizará. El trazado de recorte es parte del contexto Graphics2D para establecer el atributo de clip, llame a Graphics2D.setClip y pase la Shape que define la ruta de recorte que desea utilizar. Puede reducir la ruta de recorte llamando al método clip y pasando otra Shape; el clip se establece en la intersección del clip actual y la Shape especificada.

Ejemplo: ClipImage

Este ejemplo anima un trazado de recorte para revelar diferentes partes de una imagen.

Nota:   si no ve el applet ejecutándose, debe instalar al menos la versión 7 del kit de desarrollo de Java SE (KDJ).

ClipImage.java contiene el código completo de este applet. El applet requiere el archivo de imagen clouds.jpg .

El trazado de recorte se define por la intersección de una elipse y un rectángulo cuyas dimensiones se establecen de forma aleatoria. La elipse se pasa al método setClip, y luego se llama a clip para establecer la ruta de recorte en la intersección de la elipse y el rectángulo. 

private Ellipse2D ellipse = new Ellipse2D.Float();
private Rectangle2D rect = new Rectangle2D.Float();
...
ellipse.setFrame(x, y, ew, eh);
g2.setClip(ellipse);
rect.setRect(x+5, y+5, ew-10, eh-10);
g2.clip(rect);

Ejemplo: Starry

También se puede crear un área de recorte a partir de una cadena de texto. El siguiente ejemplo crea un TextLayout con la cadena La noche estrellada. Luego, obtiene el contorno del TextLayout. El método TextLayout.getOutline devuelve un objeto Shape y se crea un Rectangle a partir de los límites de este objeto Shape. Los límites contienen todos los píxeles que puede dibujar el diseño. El color en el contexto de gráficos se establece en azul y se dibuja la forma del contorno, como se ilustra en la siguiente imagen y fragmento de código. 

FontRenderContext frc = g2.getFontRenderContext();
Font f = new Font("Helvetica", 1, w/10);
String s = new String("The Starry Night");
TextLayout textTl = new TextLayout(s, f, frc);
AffineTransform transform = new AffineTransform();
Shape outline = textTl.getOutline(null);
Rectangle r = outline.getBounds();
transform = g2.getTransform();
transform.translate(w/2-(r.width/2), h/2+(r.height/2));
g2.transform(transform);
g2.setColor(Color.blue);
g2.draw(outline);

A continuación, se establece un área de recorte en el contexto de gráficos utilizando el objeto Shape creado a partir de getOutline. La imagen starry.gif, que es la famosa pintura de Van Gogh, La noche estrellada, se dibuja en este área de recorte comenzando en la esquina inferior izquierda del objeto Rectangle. 

g2.setClip(outline);
g2.drawImage(img, r.x, r.y, r.width, r.height, this);

Nota:   si no ve el applet ejecutándose, debe instalar al menos la versión 7 del kit de desarrollo de Java SE (KDJ).

Starry.java contiene el código completo de este programa. Este applet requiere el archivo de imagen Starry.gif.
