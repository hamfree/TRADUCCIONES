Transformando Formas, Texto e Imágenes

Puede modificar el atributo de transformación en el contexto Graphics2D para mover, rotar, escalar y distorsionar gráficos primitivos cuando se renderizan. El atributo de transformación está definido por una instancia de la clase AffineTransform. Una transformación afín es una transformación como trasladar, rotar, escalar o distorsionar en la que las líneas paralelas permanecen paralelas incluso después de transformarse.

La clase Graphics2D proporciona varios métodos para cambiar el atributo de transformación. Puede construir un nuevo AffineTransform y cambiar el atributo de transformación Graphics2D llamando a transform.

AffineTransform define los siguientes métodos de fábrica para facilitar la construcción de nuevas transformaciones:

  • getRotateInstance
  • getScaleInstance
  • getShearInstance
  • getTranslateInstance

Alternativamente, puede usar uno de los métodos de transformación Graphics2D para modificar la transformación actual. Cuando llama a uno de estos métodos de conveniencia, la transformación resultante se concatena con la transformación actual y se aplica durante el renderizado:

  • rotate — para especificar un ángulo de rotación en radianes
  • scale — para especificar un factor de escala en las direcciones x e y
  • shear — para especificar un factor de corte en las direcciones x e y
  • translate — para especificar un desplazamiento de traducción en las direcciones x e y

También puede construir un objeto AffineTransform directamente y concatenarlo con la transformación actual llamando al método transform.

El método drawImage también está sobrecargado para permitirle especificar un AffineTransform que se aplica a la imagen a medida que se renderiza. Especificar una transformación cuando llama a drawImage no afecta el atributo de transformación Graphics2D.

Ejemplo: Transformar

El siguiente programa es el mismo que StrokeandFill, pero también permite al usuario elegir una transformación para aplicar al objeto seleccionado cuando se renderiza.

Nota:   si no ve el applet ejecutándose, debe instalar al menos la versión 7 del kit de desarrollo de Java SE (KDJ).

Transform.java contiene el código completo para este applet.

Cuando se elige una transformación del menú Transformar, la transformación se concatena en AffineTransform at: 

public void setTrans(int transIndex) {
    // Establece AffineTransform.
    switch ( transIndex ) {
    case 0 :
        at.setToIdentity();
        at.translate(w/2, h/2);
        break;
    case 1 :
        at.rotate(Math.toRadians(45));
        break;
    case 2 :
        at.scale(0.5, 0.5);
        break;
    case 3 :
        at.shear(0.5, 0.0);
        break;
    }
}

Antes de mostrar la forma correspondiente a las opciones del menú, la aplicación primero recupera la transformación actual del objeto Graphics2D: 

AffineTransform saveXform = g2.getTransform();

Esta transformación se restaurará al Graphics2D después de la renderización.

Después de recuperar la transformación actual, se crea otro AffineTransform, toCenterAt, que hace que las formas se representen en el centro del panel. El at de tipo AffineTransform está concatenado en toCenterAt: 

AffineTransform toCenterAt = new AffineTransform();
toCenterAt.concatenate(at);
toCenterAt.translate(-(r.width/2), -(r.height/2));

La transformación toCenterAt se concatena en la transformación Graphics2D con el método transform: 

g2.transform(toCenterAt);

Una vez finalizada la renderización, la transformación original se restaura mediante el método setTransform: 

g2.setTransform(saveXform);
Nota: Nunca use el método setTransform para concatenar una transformación de coordenadas en una transformación existente. El método setTransform sobrescribe la transformación actual del objeto Graphics2D, que podría ser necesaria por otras razones, como colocar Swing y componentes ligeros en una ventana. Siga estos pasos para realizar transformaciones:
  1. Utilice el método getTransform para obtener la transformación actual.
  2. Utilice transform, translate, scale, shear o rotate para concatenar una transformación.
  3. Realice el renderizado.
  4. Restaure la transformación original usando el método setTransform.
