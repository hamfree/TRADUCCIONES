Lección: Temas Avanzados en Java 2D

Esta lección le muestra cómo usar Graphics2D para mostrar gráficos con elegantes estilos de contorno y relleno, transformar gráficos cuando se renderizan, restringir el renderizado a un área en particular y, en general, controlar el aspecto de los gráficos cuando se renderizan. También aprenderá a crear objetos complejos de Shape combinando objetos simples y cómo detectar cuándo el usuario hace clic en una primitiva gráfica mostrada. Estos temas se tratan en las siguientes secciones:

Transformando Formas, Texto e Imágenes

Esta sección le muestra cómo modificar la transformación predeterminada para que los objetos se trasladen, roten, escalen o corten cuando se rendericen.

Recortando la Región de Dibujo

Puede usar cualquier forma como un trazado de recorte, el área dentro de la cual tiene lugar la renderización.

Componiendo Gráficos

Esta sección ilustra los diversos estilos de composición admitidos por AlphaComposite y le muestra cómo configurar el estilo de composición en el contexto de representación de Graphics2D.

Controlando la Calidad del Renderizado

Esta sección describe las sugerencias de renderizado que admite Graphics2D y le muestra cómo especificar su preferencia en el equilibrio entre calidad de renderizado y velocidad.

Construyendo Formas Complejas a partir de Primitivas Geométricas

Esta sección le muestra cómo realizar operaciones booleanas en objetos Shape usando la clase Area.

Apoyo a la Interacción del Usuario

Esta sección le muestra cómo realizar la detección de aciertos en primitivas gráficas.
