Controlando la Calidad del Renderizado

Use el atributo de sugerencias de renderizado de la clase Graphics2D para especificar si desea que los objetos se rendericen lo más rápido posible o si prefiere que la calidad de renderizado sea lo más alta posible.

Para establecer o cambiar el atributo de sugerencias de representación en el contexto Graphics2D, construya un objeto RenderingHints y páselo a Graphics2D utilizando el método setRenderingHints. Si solo desea establecer una sugerencia, puede llamar a Graphics2D setRenderingHint y especificar el par clave-valor para la sugerencia que desea establecer. (Los pares clave-valor se definen en la clase RenderingHints).

Por ejemplo, para establecer una preferencia para que se use el antialiasing si es posible, puede usar setRenderingHint: 

public void paint (graphics g){
    Graphics2D g2 = (Graphics2D)g;
    RenderingHints rh = new RenderingHints(
             RenderingHints.KEY_TEXT_ANTIALIASING,
             RenderingHints.VALUE_TEXT_ANTIALIAS_ON);
    g2.setRenderingHints(rh);
...
}
Nota: no todas las plataformas admiten la modificación del modo de representación, por lo que especificar sugerencias de representación no garantiza que se utilizarán.

Contraste de Texto LCD KEY_TEXT_LCD_CONTRAST Los valores deben ser un número entero positivo en el rango de 100 a 250. Un valor más bajo (por ejemplo, 100) corresponde a un texto de mayor contraste cuando se muestra texto oscuro sobre un fondo claro. Un valor más alto (por ejemplo, 200) corresponde a un texto de menor contraste cuando se muestra texto oscuro sobre un fondo claro. Un valor útil típico está en el rango estrecho 140-180. Si no se especifica ningún valor, se aplicará un valor predeterminado del sistema o de la implementación.

Cuando una sugerencia se establece como predeterminada, se utiliza la representación predeterminada de la plataforma.
