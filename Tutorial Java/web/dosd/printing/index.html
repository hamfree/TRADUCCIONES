Lección: Imprimiendo

Dado que la IPA 2D de Java le permite dibujar en cualquier superficie, una extensión natural de eso es la capacidad de imprimir gráficos Java 2D. Una impresora puede considerarse un dispositivo gráfico como una pantalla.

La IPA de impresión Java 2D no se limita a la impresión de gráficos. También le permite imprimir el contenido de la interfaz de usuario de una aplicación. El contenido se puede imprimir enviando datos sin procesar a la impresora bajo el control de formato del IPA de impresión Java 2D, o utilizando el IPA de Gráficos 2D de Java.

En esta lección, explorará las funciones de control de trabajos e impresoras de la IPA de impresión Java 2D, que se complementan con los elementos de renderizado. Aprenderá cómo buscar impresoras configuradas en el sistema o la red y descubrir información sobre estas impresoras, como los tamaños de papel que admiten, y seleccionar estos atributos para imprimir y cuadros de diálogo de usuario.

Las principales clases e interfaces involucradas en la impresión están representadas en los paquetes java.awt.print y javax.print (el último paquete le permite acceder a los servicios de impresión).

Las operaciones básicas de impresión se representan en las siguientes secciones:

  • Un Programa para Impresión Básica – Esta sección describe la interfaz Printable y presenta un programa de impresión básico.
  • Usando los Cuadros de Diálogo de Configuración de Impresión – En esta sección se explica cómo mostrar el cuadro de diálogo Configuración de Impresión.
  • Imprimiendo un Documento de Múltiples Páginas – Esta sección explica cómo utilizar la paginación para imprimir un documento de varias páginas.
  • Trabajando con Servicios de Impresión y Atributos – Esta sección le enseña sobre los servicios de impresión, cómo especificar el formato de datos de impresión y cómo crear un trabajo de impresión utilizando el paquete javax.print.
  • Imprimiendo el Contenido del Interfaz de Usuario – Esta sección explica cómo imprimir el contenido de una ventana o un marco.
  • Soporte para Impresión en los Componentes Swing – Esta sección proporciona una breve descripción de la funcionalidad de impresión relacionada en Swing y se refiere a clases e interfaces específicas de Swing.
