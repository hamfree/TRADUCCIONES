Imprimiendo el Contenido del Interfaz de Usuario

Otra tarea de impresión común es imprimir el contenido de una ventana o un marco, ya sea en su totalidad o en parte. La ventana puede contener los siguientes componentes: barras de herramientas, botones deslizantes, etiquetas de texto, áreas de texto desplazables, imágenes y otro contenido gráfico. Todos estos componentes se imprimen utilizando los siguientes métodos de la IPA de impresión Java 2D: 

java.awt.Component.print(Graphics g);
java.awt.Component.printAll(Graphics g);

La siguiente figura representa una interfaz de usuario sencilla.

El código para crear esta interfaz de usuario se encuentra en el programa de ejemplo PrintUIWindow.java.

Para imprimir esta ventana, modifique el código en los ejemplos anteriores que imprimieron texto o imágenes. El código resultante debería aparecer de la siguiente manera: 

public int print(Graphics g, PageFormat pf, int page)
    throws PrinterException {
    if (page > 0) {
        return NO_SUCH_PAGE;
    }

    Graphics2D g2d = (Graphics2D)g;
    g2d.translate(pf.getImageableX(), pf.getImageableY());

    // Imprima todo el contenido visible de un
    // java.awt.Frame.
    frame.printAll(g);

    return PAGE_EXISTS;
}
Nota: la llamada al método printAll es la única diferencia entre este ejemplo y los ejemplos para imprimir texto o imagen. El método print(Graphics g) refleja el método java.awt.Component.paint (Graphics g) utilizado para la representación en pantalla. Utilice el método print() en lugar del método paint() ya que la clase Components puede haber anulado el método print() para manejar el caso de impresión de manera diferente.

El método printAll(Graphics g) imprime el componente y todos sus subcomponentes. Este método se usa generalmente para imprimir objetos como una ventana completa, en lugar de un solo componente.
