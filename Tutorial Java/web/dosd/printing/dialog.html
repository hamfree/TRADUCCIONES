Usando los Cuadros de Diálogo de Configuración de Impresión

Tradicionalmente, el usuario desea ver los cuadros de diálogo de configuración de página e impresión. Desde el cuadro de diálogo de impresión, puede seleccionar una impresora, especificar las páginas para imprimir y establecer el número de copias.

Una aplicación muestra un cuadro de diálogo de impresión cuando el usuario presiona un botón relacionado con el comando de impresión o elige un elemento del menú de impresión. Para mostrar este cuadro de diálogo, llame al método printDialog de la clase PrinterJob: 

PrinterJob pj = PrinterJob.getPrinterJob();
...
    if (pj.printDialog()) {
        try {pj.print();}
        catch (PrinterException exc) {
            System.out.println(exc);
         }
     }   
...

Este método devuelve true si el usuario hizo clic en Aceptar para salir del cuadro de diálogo, y false en caso contrario. Las opciones del usuario en el cuadro de diálogo están limitadas en función del número y formato de las páginas que se han configurado en PrinterJob.

El método printDialog en el fragmento de código anterior abre un diálogo de impresión nativo. El código de ejemplo PrintDialogExample.java muestra cómo mostrar un cuadro de diálogo de impresión multiplataforma.

Puede cambiar la información de configuración de página contenida en el objeto PageFormat utilizando el cuadro de diálogo de configuración de página.

Para mostrar el diálogo de configuración de página, llame al método pageDialog de la clase PrinterJob. 

PrinterJob pj = PrinterJob.getPrinterJob();
PageFormat pf = pj.pageDialog(pj.defaultPage());

El diálogo de configuración de página se inicializa usando el parámetro pasado a pageDialog. Si el usuario hace clic en el botón Aceptar en el cuadro de diálogo, la instancia de PageFormat se creará de acuerdo con las selecciones del usuario y luego se devolverá. Si el usuario cancela el diálogo, pageDialog devuelve el PageFormat original sin cambios.

Por lo general, la IPA de Impresión 2D de Java requiere una aplicación para mostrar un cuadro de diálogo de impresión, pero a veces es posible imprimir sin mostrar ningún cuadro de diálogo. Este tipo de impresión se denomina impresión silenciosa. Puede ser útil en casos específicos, como cuando necesita imprimir un informe semanal de una base de datos en particular. En los demás casos, siempre se recomienda informar al usuario cuando se inicia un proceso de impresión.
