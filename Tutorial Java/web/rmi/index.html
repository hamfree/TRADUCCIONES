Recorrido: RMI

El sistema de Invocación de Método Remoto de Java, RMI (Nota del Traductor: Java Remote Method Invocation o RMI en inglés) permite a un objeto ejecutándose en una máquina virtual de Java invocar métodos en un objeto que se está ejecutando en otra máquina virtual Java. RMI proporciona comunicación remota entre programas escritos en el lenguaje de programación Java.

Nota: Si se está conectando a un programa IDL existente, debe utiliza Java IDL en vez de RMI.

Este recorrido proporciona una breve introducción del sistema RMI y luego recorre un ejemplo completo de cliente/servidor que utiliza las capacidades únicas de RMI para cargar y ejecutar tareas definidas por el usuario en tiempo de ejecución. El servidor en el ejemplo implementa un motor de cálculo genérico, que usa el cliente para calcular el valor de .

  Una descripción de las Aplicaciones RMI (Remote Method Interface) describe el sistema RMI y enumera sus ventajas. Además, esta sección proporciona una descripción de una aplicación RMI típica, compuesta de un servidor y un cliente, e introduce términos importantes.

  Escribiendo un Servidor RMI recorre el código del servidor del motor de cálculo. Esta sección le enseñará cómo diseñar e implementar un servidor RMI.

  Creando un Programa Cliente echa un vistazo a un posible cliente de motor de cálculo y lo utiliza para ilustrar las características importantes de un cliente RMI.

  Compilando y Ejecutando el Ejemplo le muestra cómo compilar y ejecutar tanto el servidor del motor de cálculo como su cliente.
