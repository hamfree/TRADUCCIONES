Escribiendo un Servidor RMI

El servidor del motor de cálculo acepta tareas de los clientes, ejecuta las tareas, y devuelve los resultados. El código del servidor consiste de una interfaz y una clase. La interfaz define los métodos que se pueden invocar desde el cliente. Esencialmente, la interfaz define la vista del cliente del objeto remoto. La clase proporciona la implementación.

Diseñando un Interfaz Remoto

Esta sección explica la interfaz Compute, que proporciona la conexión entre el cliente y el servidor. También aprenderá aobre la IPA de RMI, que admite esta comunicación.

Implementando un Intefaz Remoto

Esta sección explora la clase que implementa la interfaz Compute, implementando así un objeto remoto. Esta clase también proporciona el resto del código que compone el programa del servidor, incluyendo un método main que crea una instancia del objeto remoto, lo registra con el registro RMI, y configura un gestor de seguridad.
