Recorrido: Fecha y Tiempo

El paquete de Fecha-Hora, java.time , presentado en la versión de Java SE 8, suministra un modelo exhaustivo para la fecha y tiempo y fue desarrollado bajo el JSR 310: Date and Time IPA . A pesar de que java.time está basado en el sistema de calendario de la Organización Internacional para la Estandarización (ISO), los calendarios globales utilizados comúnmente también están soportados.

Este recorrido cubre los fundamentos para usar la clases basadas en ISPO para representar la fecha y el tiempo y manipular los valores de fecha y tiempo.
