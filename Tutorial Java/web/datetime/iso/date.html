Clase Date

La IPA de Fecha-Tiempo suministra cuatro clases que tratan exclusivamente con información de fechas, sin respeto a la hora o zona horaria. El uso de estas clases es sugerido por los nombres de las clases: LocalDate, YearMonth, MonthDay, y Year.

LocalDate

Una LocalDate representa un año-mes-día en el calendario ISO y es útil para representar una fecha sin una hora. Usted podría usar una LocalDate para rastrear un evento significativo, tales como una fecha de nacimiento o la fecha de la boda. Los ejemplos siguientes usan los métodos of y with para crear instancias de LocalDate: 

LocalDate date = LocalDate.of(2000, Month.NOVEMBER, 20);
LocalDate nextWed = date.with(TemporalAdjusters.next(DayOfWeek.WEDNESDAY));

Para más información sobre el interfaz TemporalAdjuster, vea Temporal Adjuster.

Además de los métodos habituales, la clase LocalDate ofrece métodos getter para obtener información sobre una fecha dada. El método getDayOfWeek devuelve el día de la semana en el que una fecha particulara cae. Por ejemplo, la liínea siguiente de código devuelve "MONDAY": 

DayOfWeek dotw = LocalDate.of(2012, Month.JULY, 9).getDayOfWeek();

El ejemplo siguiente usa un TemporalAdjuster para recuperar el primer Miércles después de una fecha específica. 

LocalDate date = LocalDate.of(2000, Month.NOVEMBER, 20);
TemporalAdjuster adj = TemporalAdjusters.next(DayOfWeek.WEDNESDAY);
LocalDate nextWed = date.with(adj);
System.out.printf("Para la fecha %s, el próximo Miércoles es %s.%n",
                  date, nextWed);

Ejecutar el código produce lo siguiente: 

Para la fecha 2000-11-20, el próximo Miércoles es 2000-11-22.

La sección Period y Duration también tiene ejemplos usando la clase LocalDate.

YearMonth

La clase YearMonth representa el mes de un año específico. El ejemplo siguiente usa el método YearMonth.lengthOfMonth() para determinar el número de días para varias combinaciones de año y mes. 

YearMonth date = YearMonth.now();
System.out.printf("%s: %d%n", date, date.lengthOfMonth());

YearMonth date2 = YearMonth.of(2010, Month.FEBRUARY);
System.out.printf("%s: %d%n", date2, date2.lengthOfMonth());

YearMonth date3 = YearMonth.of(2012, Month.FEBRUARY);
System.out.printf("%s: %d%n", date3, date3.lengthOfMonth());

El resultado de este código es el siguiente: 

2013-06: 30
2010-02: 28
2012-02: 29

MonthDay

La clase MonthDay representa el día de un mes particular, tal como el Dia de Año Nuevo el 1 de Enero.

El ejemplo siguiente usa el método MonthDay.isValidYear para determinar si el 29 de Febrero es válido para el año 2010. La llamada devuelve false, confirmando que 2010 no es un año bisiesto. 

MonthDay date = MonthDay.of(Month.FEBRUARY, 29);
boolean validLeapYear = date.isValidYear(2010);

Year

La clase Year representa un año. El ejemplo siguiente usa el método Year.isLeap para determinar si el año dado es un año bisiesto. La llamada devuelve true, confirmando que 2012 es un año bisiesto. 

boolean validLeapYear = Year.of(2012).isLeap();
