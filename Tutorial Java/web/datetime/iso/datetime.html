Clases Date y Time

LocalTime

La clase LocalTime es similar a las otras clases cuyos nombres tienen el prefijo Local, pero sólo se ocupa del tiempo. Esta clase es útil para representar la hora del día que usan las personas, tales como horas de una película, o las horas de apertura y cierre de una biblioteca local. Podría también ser utilizada para crear un reloj digital, como se muestra en el ejemplo siguiente: 

LocalTime thisSec;

for (;;) {
    thisSec = LocalTime.now();

    // la implementación del código de del visor se deja al lector.
    display(thisSec.getHour(), thisSec.getMinute(), thisSec.getSecond());
}

La clase LocalTime no almacena la zona horaria o la información del horario de verano.

LocalDateTime

La clase que maneja tanto fecha como tiempo, sin una zona horaria, es LocalDateTime, una de las clases principales de la IPA de Fecha-Tiempo. Esta clase es usada para representar la fecha (mes-día-año) junto con la hora (hora-minuto-segundo-nanosegundo) y es, en efecto, una combinación de LocalDate con LocalTime. Esta clase puede ser usada para representar un evento específico, tal como la primera carrera de la Copa Final de Louis Vuitton en la Serie Challenger de la Copa de las Américas, que empezó a la 1:10 de la tarde el 17 de agosto de 2013. Note que esto significa 1:10 de la tarde en hora local. Para incluir una zona horaria, debe usar un ZonedDateTime o un OffsetDateTime, como se describe en Zonas horarias y Clases de Compensación.

Además del método now que toda clase basada-en-el-tiempo ofrece, la clase LocalDateTime tiene varios métodos of (o métodos con el prefijo of) que crean una instancia de LocalDateTime. Hay un método from que convierte una instancia de otro formato temporal a una instancia de LocalDateTime. Hay también métodos para agregar y substraer horas, minutos, días, semanas, y meses. El ejemplo siguiente muestra unos cuantos de estos métodos. Las expresiones de fecha-tiempo están en negrita: 

System.out.printf("ahora: %s%n", LocalDateTime.now());

System.out.printf("Apr 15, 1994 @ 11:30am: %s%n",
                  LocalDateTime.of(1994, Month.APRIL, 15, 11, 30));

System.out.printf("ahora (desde Instant): %s%n",
                  LocalDateTime.ofInstant(Instant.now(), ZoneId.systemDefault()));

System.out.printf("6 meses desde ahora: %s%n",
                  LocalDateTime.now().plusMonths(6));

System.out.printf("hace 6 meses: %s%n",
                  LocalDateTime.now().minusMonths(6));

Este código produce una salida que se verá similar a la siguiente: 

ahora: 2013-07-24T17:13:59.985
Apr 15, 1994 @ 11:30am: 1994-04-15T11:30
ahora (desde Instant): 2013-07-24T17:14:00.479
6 meses desde ahora: 2014-01-24T17:14:00.480
hace 6 meses: 2013-01-24T17:14:00.481
