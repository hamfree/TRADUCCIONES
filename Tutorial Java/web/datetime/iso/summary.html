Sumario

El paquete java.time contiene muchas clases que sus programas pueden usar para representar fecha y tiempo. Esta es una IPA muy rica. Los puntos de entrada clave para las fechas basadas en ISO son los siguientes:

  • La clase Instant proporciona una vista de máquina de la línea de tiempo.
  • Las clases LocalDate, LocalTime, y LocalDateTime suministran una vista humana de la fecha y el tiempo sin ninguna referencia a la zona horaria.
  • Las clases ZoneId, ZoneRules, y ZoneOffset describen zonas horarias, desplazamientos dede zona horaria, y reglas de zonas horarias.
  • La clase ZonedDateTime representa la fecha y el tiempo con una zona horaria. Las clases OffsetDateTime y OffsetTime representan fecha y tiempo, o tiempo, respectivamente. Estas clases tienen en cuentan el desplazamiento de zona horaria.
  • La clase Duration mide una cantidad de tiempo en segundos y nanosegundos.
  • La clase Period mide una cantidad de tiempo uando años, meses, y días.

Otros sistemas de calendarios no ISO pueden ser representandos usando el paquete java.time.chrono. Este paquete está más allá del alcance de este tutorial, aunque la página Conversión de Fechas No-ISO suministra información sobre convertir una fecha basada en ISO a otro sistema de calendario.

La IPA de Fecha y Hora fue desarrollada como parte del proceso de la comunidad Java bajo la designación del JSR 310. Para más información, vea JSR 310: Date and Time IPA.
