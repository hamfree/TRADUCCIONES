Conversión de Fechas No-ISO

Este tutorial no describe el paquete java.time.chrono en ningún detalle. Sin embargo, podría ser util saber que este paquete suministra varias cronologías predefinidas que no están basadas en ISO, tales como la Japonesa, la Malaya, la China y la Budista Tailandesa. Puede también usar este paquete para crear su propia cronología.

Esta sección le muestra cómo convertir entre una fecha basada en ISO y una fecha en una de las otras cronologías predefinidas.

Convertir a una Fecha No Basada en ISO

Puede convertir una fecha basada en ISO a una fecha en otra cronología usando el método from(TemporalAccessor), tal como en JapaneseDate.from(TemporalAccessor). Este método lanza una DateTimeException si es incapaz de convertir la fecha a una instancia válida. El código siguiente conviete una instancia de LocalDateTime a varias fechas de calendarios predefinidos no ISO: 

LocalDateTime date = LocalDateTime.of(2013, Month.JULY, 20, 19, 30);
JapaneseDate jdate     = JapaneseDate.from(date);
HijrahDate hdate       = HijrahDate.from(date);
MinguoDate mdate       = MinguoDate.from(date);
ThaiBuddhistDate tdate = ThaiBuddhistDate.from(date);

El ejemplo StringConverter convierte desde una LocalDate a una ChronoLocalDate a una String y de vuelta. El método toString toma una intancia de LocalDate y una Chronology y devuelve la cadena convertida usando la Chronology suministrada. La clase DateTimeFormatterBuilder es usada para construir una cadena que pueda ser usada para imprimir la fecha: 

/**
 * Convierte un valor LocalDate (ISO a una fecha ChronoLocalDate
 * usando la Chronology suministrada, y entonces formatea la
 * ChronoLocalDate a una String usando una DateTimeFormatter con un
 * patrón SHORT basado en la Chronology y en la Locale actual.
 *
 * @param localDate - la fecha ISO a convertir y dar formato.
 * @param chrono - una Chronology opcional. Si es null, entonces se usa IsoChronology.
 */
public static String toString(LocalDate localDate, Chronology chrono) {
    if (localDate != null) {
        Locale locale = Locale.getDefault(Locale.Category.FORMAT);
        ChronoLocalDate cDate;
        if (chrono == null) {
            chrono = IsoChronology.INSTANCE;
        }
        try {
            cDate = chrono.date(localDate);
        } catch (DateTimeException ex) {
            System.err.println(ex);
            chrono = IsoChronology.INSTANCE;
            cDate = localDate;
        }
        DateTimeFormatter dateFormatter =
            DateTimeFormatter.ofLocalizedDate(FormatStyle.SHORT)
                             .withLocale(locale)
                             .withChronology(chrono)
                             .withDecimalStyle(DecimalStyle.of(locale));
        String pattern = "M/d/yyyy GGGGG";
        return dateFormatter.format(cDate);
    } else {
        return "";
    }
}

Cuando el método es invocado con la fecha siguiente para las cronologías predefinidas: 

LocalDate date = LocalDate.of(1996, Month.OCTOBER, 29);
System.out.printf("%s%n",
     StringConverter.toString(date, JapaneseChronology.INSTANCE));
System.out.printf("%s%n",
     StringConverter.toString(date, MinguoChronology.INSTANCE));
System.out.printf("%s%n",
     StringConverter.toString(date, ThaiBuddhistChronology.INSTANCE));
System.out.printf("%s%n",
     StringConverter.toString(date, HijrahChronology.INSTANCE));

La salida se ve así: 

10/29/0008 H
10/29/0085 1
10/29/2539 B.E.
6/16/1417 1

Convertir a una Fecha basada en ISO

Puede convertir desde una fecha no basada en ISO a una instancia LocalDate usando el método estático LocalDate.from, como es mostrado en el ejemplo siguiente: 

LocalDate date = LocalDate.from(JapaneseDate.now());

Otras clases basadas en el tiempo también suministran este método, el cual lanza una DateTimeException si la fecha no puede ser convertida.

El método fromString, del ejemplo StringConverter, analiza una String que contiene una fecha no basada en ISO y devuelve una instancia LocalDate. 

/**
 * Analiza una String a una ChronoLocalDate usando una DateTimeFormatter
 * con un patrón corto basado en el Locale actual y la
 * Chronology suministrada, entonces convierte esto a un valor
 * LocalDate (ISO).
 *
 * @param text   - el texto de la fecha de entrada en el formato CORTO esperado
 *                 para la Chronology y la Locale actual.
 *
 * @param chrono - una Chronology opcional. Si es null, entonces
 *                 se usa IsoChronology.
 */
public static LocalDate fromString(String text, Chronology chrono) {
    if (text != null && !text.isEmpty()) {
        Locale locale = Locale.getDefault(Locale.Category.FORMAT);
        if (chrono == null) {
           chrono = IsoChronology.INSTANCE;
        }
        String pattern = "M/d/yyyy GGGGG";
        DateTimeFormatter df = new DateTimeFormatterBuilder().parseLenient()
                              .appendPattern(pattern)
                              .toFormatter()
                              .withChronology(chrono)
                              .withDecimalStyle(DecimalStyle.of(locale));
        TemporalAccessor temporal = df.parse(text);
        ChronoLocalDate cDate = chrono.date(temporal);
        return LocalDate.from(cDate);
    }
return null;
}

Cuando el método es invocado con las cadenas siguientes: 

System.out.printf("%s%n", StringConverter.fromString("10/29/0008 H",
    JapaneseChronology.INSTANCE));
System.out.printf("%s%n", StringConverter.fromString("10/29/0085 1",
    MinguoChronology.INSTANCE));
System.out.printf("%s%n", StringConverter.fromString("10/29/2539 B.E.",
    ThaiBuddhistChronology.INSTANCE));
System.out.printf("%s%n", StringConverter.fromString("6/16/1417 1",
    HijrahChronology.INSTANCE));

Las cadenas impresas deben volver a convertirse todas al 29 de octubre de 1996: 

1996-10-29
1996-10-29
1996-10-29
1996-10-29
