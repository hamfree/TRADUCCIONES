DayOfWeek y Enumeraciones de Month

La IPA de Date-Time suministra enumeraciones para especificar días de la semana y meses del año.

DayOfWeek

La enumeración DayOfWeek consiste en siete constantes que describen los días de la semana: de MONDAY a SUNDAY. Los valores enteros de las constantes de DayOfWeek se encuentran en el rango de 1 (Lunes) a 7 (Domingo). Usar las constantes definidas (DayOfWeek.FRIDAY) hacen su código más legible.

Esta enumeración también suministra una variedad de métodos, similares a los métodos proporcionada por las clases basadas en el tiempo. Por ejemplo, el código siguiente agrega tres días a "Monday" e imprime el resultado. La salida es "THURSDAY": 

System.out.printf("%s%n", DayOfWeek.MONDAY.plus(3));

Utilizando el método getDisplayName(TextStyle, Locale), puede recuperar una cadena para identificar el día de la semana en la localización del usuario. La enumeración TextStyle le habilita a especificar qué clase de cadena quiere visualizar: FULL, NARROW (típicamente una letra única), o SHORT (una abreviación). Las constantes STANDALONE de TextStyle son usadas en algunos lenguajes donde la salida es diferente cuando se usa como parte de una fecha que cuando se usa sola. El ejemplo siguiente imprime las tres formas primarias de TextStyle para "Monday": 

DayOfWeek dow = DayOfWeek.MONDAY;
Locale locale = Locale.getDefault();
System.out.println(dow.getDisplayName(TextStyle.FULL, locale));
System.out.println(dow.getDisplayName(TextStyle.NARROW, locale));
System.out.println(dow.getDisplayName(TextStyle.SHORT, locale));

Este código tiene la salida siguiente para la configuración regiional en: 

Monday
M
Mon

Month

La enumeración Month incluye constantes para los doce meses, de JANUARY a DECEMBER. Como con la enumeración DayOfWeek, la enumeración Month está fuertemente tipada, y el valor entero de cada constante corresponde al rango ISO de 1 (Enero) a 12 (Diciembre). Usar las constantes definidas (Month.SEPTEMBER) hace su código más legible.

La enumeración Month también incluye una variedad de métodos. La línea de código siguiente usa el método maxLength para imprimir el máximo número posible de días en el mes de Febrero. La salida es "29": 

System.out.printf("%d%n", Month.FEBRUARY.maxLength());

La enumeración Month también implementa el método getDisplayName(TextStyle, Locale) para recuperar una cadena que identifica el mes en la configuración regional del usuario usando el TextStyle especificado. Si un TextStyle en particular no está definida, entonces una cadena representando el valor numérico de la constante es devuelta. El código siguiente imprime el mes de Agosto usando los tres estilos de texto primarios: 

Month month = Month.AUGUST;
Locale locale = Locale.getDefault();
System.out.println(month.getDisplayName(TextStyle.FULL, locale));
System.out.println(month.getDisplayName(TextStyle.NARROW, locale));
System.out.println(month.getDisplayName(TextStyle.SHORT, locale));

Este código tiene la siguiente salida para la configuración regional en: 

August
A
Aug
