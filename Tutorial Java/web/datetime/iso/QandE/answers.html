Respuestas a Preguntas y Ejercicios: IPA de Fecha y Tiempo

Preguntas

Pregunta 1. ¿Qué clase usaría para almacenar tu día de nacimiento en años, meses, días, segundos, y nanosegundos?

Respuesta 1. Lo más probable es que utilices la clase LocalDateTime. Para tener en cuenta una zona horaria particular, debería usar la clase ZonedDateTime. Ambas clases registran la fecha y la hora en nanosegundos de precisión y ambas clases, cuando usadas en conjunción con Period, da un resultado usando una combinación de unidades basadas en humanos, tales como años, meses y días.

Pregunta 2. Dada una fecha aleatoria, ¿cómo encontraría la fecha del Jueves anterior?

Respuesta 2. Puede usar el método previous de un TemporalAdjuster: 

LocalDate date = ...;
System.out.printf("El jueves anterior es: %s%n",
          date.with(TemporalAdjuster.previous(DayOfWeek.THURSDAY)));

Pregunta 3. ¿Cual es la diferencia entre entre un ZoneId y un ZoneOffset?

Respuesta 3. Tanto ZoneId y ZoneOffset registran un desplazamiento desde la hora de Greenwich/UTC, pero la clase ZoneOffset registra sólo el desplazamiento absoluto desde Greenwich/UTC. La clase ZoneId también usa a ZoneRules para determinar cómo un desplazamiento varía para un tiempo particular del año y región.

Pregunta 4. 4. ¿Cómo convertiría un Instant en un ZonedDateTime? ¿Cómo convertiría un ZonedDateTime en un Instant?

Respuesta 4. Puede convertir un Instant a un ZonedDateTime usando el método ZonedDateTime.ofInstant. También es necesario suministrar una ZoneId: 

ZonedDateTime zdt = ZonedDateTime.ofInstant(Instant.now(),
                                            ZoneId.systemDefault());

Alternativamente, podría usar el método Instant.atZone: 

ZonedDateTime zdt = Instant.now().atZone(ZoneId.systemDefault());

Puede usar el método toInstant en la interfaz ChronoZonedDateTime, implementado por la clase ZonedDateTime, para convertir desde un ZonedDateTime a un Instant: 

Instant inst = ZonedDateTime.now().toInstant();
