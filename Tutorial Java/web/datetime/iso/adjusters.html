Temporal Adjuster

El interfaz TemporalAdjuster, en el paquete java.time.temporal, ofrece métodos que toman un valor Temporal y devuelven un valor ajustado. Los ajustadores pueden ser utilizados con cualquiera de los tipos basados-en-el-tiempo.

Si un ajustador es usado con una ZonedDateTime, entonces se calcula una nueva fecha que preserva el tiempo original y los valores de zona horaria.

Ajustadores Predefinidos

La clase TemporalAdjusters (note el plural) suministra un conjunto de ajustadores predefinidos para encontrar el primer o último día del mes, el primero o último día del año, el último miércoles del mes, o el primer martes después de una fecha específica, por nombrar unos pocos ejemplos. Los ajustadores predefinidos están definidos como métodos estáticos y están diseñados para ser usados con la sentencia static import.

El ejemplo siguiente usa varios métodos TemporalAdjusters, en conjunción con el método with definido en las clases basadas-en-el-tiempo, para calcular nuevas fechas basadas en la fecha original del 15 de octubre del 2000: 

LocalDate date = LocalDate.of(2000, Month.OCTOBER, 15);
DayOfWeek dotw = date.getDayOfWeek();
System.out.printf("%s es un %s%n", date, dotw);

System.out.printf("primer día del Mes: %s%n",
                  date.with(TemporalAdjusters.firstDayOfMonth()));
System.out.printf("primer lunes del Mes: %s%n",
                  date.with(TemporalAdjusters.firstInMonth(DayOfWeek.MONDAY)));
System.out.printf("último día del Mes: %s%n",
                  date.with(TemporalAdjusters.lastDayOfMonth()));
System.out.printf("primer día del siguiente Mes: %s%n",
                  date.with(TemporalAdjusters.firstDayOfNextMonth()));
System.out.printf("primer día del siguiente Año: %s%n",
                  date.with(TemporalAdjusters.firstDayOfNextYear()));
System.out.printf("primer día del Año: %s%n",
                  date.with(TemporalAdjusters.firstDayOfYear()));

Esto produce la salida siguiente: 

2000-10-15 es un SUNDAY
primer día del Mes: 2000-10-01
primer lunes del Mes: 2000-10-02
último día del Mes: 2000-10-31
primer día del siguiente Mes: 2000-11-01
primer día del siguiente Año: 2001-01-01
primer día del Año: 2000-01-01

Ajustadores Personalizados

Puede también crear sus propios ajustadores personalizados. Para hacer esto, cree una clase que implemente el interfaz TemporalAdjuster con un método adjustInto(Temporal). La clase PaydayAdjuster del ejemplo NextPayday es un ajustador personalizado. El PaydayAdjuster evalúa la fecha introducida y devuelve el siguiente día de paga, asumiendo que el día de paga ocurre dos veces al mes: en el día decimoquinto, y de nuevo el último día del mes. Si la fecha calculada ocurre en un fin de semana, entonces se usa el viernes anterior. Se asume el año del calendario actual. 

/**
 * El método adjustInto acepta una instancia Temporal
 * y devuelve un LocalDate ajustado. Si lo pasado en
 * el parámetro no es un LocalDate, entonces se lanzará una DateTimeException.
 */
public Temporal adjustInto(Temporal input) {
    LocalDate date = LocalDate.from(input);
    int day;
    if (date.getDayOfMonth() < 15) {
        day = 15;
    } else {
        day = date.with(TemporalAdjusters.lastDayOfMonth()).getDayOfMonth();
    }
    date = date.withDayOfMonth(day);
    if (date.getDayOfWeek() == DayOfWeek.SATURDAY ||
        date.getDayOfWeek() == DayOfWeek.SUNDAY) {
        date = date.with(TemporalAdjusters.previous(DayOfWeek.FRIDAY));
    }

    return input.with(date);
}

El ajustador es invocado de la misma forma que un ajustador predefinido, usando el método with. La siguiente línea de código es del ejemplo NextPayday: 

LocalDate nextPayday = date.with(new PaydayAdjuster());

En 2013, tanto el 15 de junio como el 30 de junio caen en fin de semana. Ejecutar el ejemplo NextPayday con las fechas respectivas del 3 de junio y 18 de junio (en 2013), da el siguiente resultado: 

Dada la fecha:  2013 Jun 3
el siguiente dia de paga: 2013 Jun 14

Dada la fecha:  2013 Jun 18
el siguiente dia de paga: 2013 Jun 28
