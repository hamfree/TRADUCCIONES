Lección: Calendario Estándar

El núcleo de la IPA de Fecha y Tiempo es el paquete java.time. Las clases definidas en java.time basan su sistema de calendario en el calendario ISO, el cual es el estándar mundial para representar fechas y tiempo. El calendario ISO sigue las reglas gregorianas prolépticas. El calendario Gregoriano se introdujo en 1582; en el calendario gregoriano proléptico, las fechas se extienden hacia atrás desde ese momento para crear una línea de tiempo uniforme y unificada y para simplificar los cálculos de fechas.

Esta lección cubre los siguientes temas:

Resumen

Esta sección compara los conceptos del tiempo humano y el tiempo de la máquina suministrando una tabla de las clases primarias basadas en el tiempo en el paquete java.time.

DayOfWeek y Enumeraciones de Month

Esta sección describe la enumeración que define los días de la semana (DayOfWeek) y la enumeración que define los meses (Month).

Clase Date

Esta sección muestra clases basadas en el tiempo que tratan sólo con fechas, sin horas o zonas horarias. Las cuatro clases son LocalDate, YearMonth, MonthDay y Year.

Clases Date y Time

Esta sección presenta las clases LocalTime y LocalDateTime, las cuales tratan con el tiempo, y fecha y tiempo, respectivamente, pero sin zonas horarias.

Zonas horarias y Clases de Compensación

Esta sección describe las clases basadas en el tiempo que almacenan la información de la zona horaria (o la compensación de la zona horaria), ZonedDateTime, OffsetDateTime, y OffsetTime. Las clases de soporte, ZoneId, ZoneRules, y ZoneOffset, son también descritas.

Clase Instant

Esta sección describe la clase Instant, la cual representa un momento instantáneo en la línea del tiempo.

Parseado y Formateado

Esta sección suministra un resumen de cómo usar los formateadores predefinidos para formatear y analizar los valores de fecha y hora.

El Paquete Temporal

Esta sección presenta un resument del paquete java.time.temporal, el cual soporta las clases temporales, campos (TemporalField y ChronoField) y unidades (TemporalUnit y ChronoUnit). Esta sección también explica cómo usar un ajustador temporal para recuperar un valor temporal ajustado, tal como "el primer Martes después del 11 de Abril", y cómo realizar una consulta temporal.

Period y Duration

Esta sección explica cómo calcular una cantidad de tiempo, usando tanto la clase Period como la clase Duration, como también el método ChronoUnit.between.

Clock

Esta sección suministra un breve resumen de la clase Clock. Puede usar esta clase para suministrar un reloj alternativo al reloj del sistema.

Conversión de Fechas No-ISO

Esta sección explica cómo convertir una fecha del sistema de calendario ISO a una fecha de un sistema de calendario No-ISO, tal como una JapaneseDate o una ThaiBuddhistDate.

Código Date-Time Heredado

Esta sección ofrece algunos consejos de cómo convertir código viejo con java.util.Date y java.util.Calendar a la IPA de Fecha-Tiempo.

Sumario

Esta sección suministra un sumario de la lección del Calendario Estándar.
