Lección: Conceptos de Programación Orientada-a-Objetos

Si usted no ha utilizado nunca antes un lenguaje de programación orientado a objetos, deberá aprender unos pocos conceptos básicos antes de que pueda comenzar a escribir código. Esta lección le presentará los objetos, clases, herencia, interfaces y paquetes. Cada discusión se centra en cómo estos conceptos se relacionan con el mundo real, mientras simultáneamente proporciona una introducción a la sintaxis del lenguaje de programación Java.

¿Qué Es un Objeto?

Un objeto es un agregado de software de comportamiento y estado relacionado. Los objetos de software son utilizados a menudo para modelar los objetos del mundo-real que usted encuentra cada día en la vida. Esta lección explica cómo el estado y el comportamiento son representados dentro de un objeto, introduce el concepto de la encapsulación de los datos, y explica los beneficios de diseñar su software de esta manera.

¿Qué Es una Clase?

Una clase es una plantilla o prototipo desde la cual los objetos son creados. Esta sección define una clase que modela el estado y el comportamiento de un objeto del mundo-real. Se enfoca intencionalmente en lo básico, mostrando cómo incluso una simple clase puede claramente modelar el estado y el comportamiento.

¿Qué Es la Herencia?

La herencia ofrece un mecanismo poderoso y natural para organizar y estructurar su software. Esta sección explica cómo las clases heredan el estado y el comportamiento de sus superclases, y explica cómo derivar una clase de otra usando la sintáxis simple ofrecida por el lenguaje de programación Java.

¿Qué Es una Interfaz?

Una interfaz es un contrato entre una clase y el mundo exterior. Cuando una clase implementa una interfaz, promete proporcionar el comportamiento publicado por esa interfaz. Esta sección define una interfaz simple y explica los cambios necesarios para cualquier clase que la implemente.

¿Qué Es un Paquete?

Un paquete es un espacio de nombres para organizar las clases y las interfaces de manera lógica. Colocando su código en paquetes hace los proyectos grandes de software más fáciles de gestionar. Esta sección explica porqué es esto útil, y le introduce en la Interfaz de Programación de Aplicaciones (IPA en inglés) ofrecida por la plataforma Java.

Preguntas y Ejercicios: Conceptos de Programación Orientada-a-Objetos

Use las preguntas y ejercicios presentados en esta sección para comprobar su entendimiento de los objetos, clases, herencia, interfaces, y paquetes.
