¿Qué Es una Interfaz?

Como usted ya ha aprendido, los objetos definen su interacción con el mundo exterorio a través de los métodos que exponen. Los métodos forman el interfaz del objeto con el mundo exterior; los botones enfrente de su televisión, por ejemplo, son la interfaz entre usted y el cableado eléctrico al otro lado del enjaulado plástico.. Usted presiona el botón de "encendido" para encender o apagar la televisión.

En su forma más común, un interfaz es un grupo de métodos relacionados con cuerpos vacíos. Un comportamiento de la bicicleta, si es especificado como una interfaz, podría aparecer como sigue: 

interface Bicicleta {

    // revoluciones de la rueda por minuto
    void cambiaCadencia(int nuevoValor);

    void cambiaMarcha(int nuevoValor);

    void acelerar(int incremento);

    void frenar(int decremento);
}

Para implementar esta interfaz, el nombre de su clase cambiará ( a una marca particular de bicicleta, por ejemplo, como BicicletaACME, y usted usa la palabra clave implements en la declaración de la clase: 

class BicicletaACME implements Bicicleta {

    int cadencia = 0;
    int velocidad = 0;
    int marcha = 1;

   // El compilador ahora requerirá que los métodos
   // cambiaCadencia, cambiaMarcha, acelerar, y frenar
   // sean todos implmentados. La compilación fallará si esos
   // métodos no se encuentran en la clase.

    void cambiaCadencia(int nuevoValor) {
         cadencia = nuevoValor;
    }

    void cambiaMarcha(int nuevoValor) {
         marcha = nuevoValor;
    }

    void acelerar(int incremento) {
         velocidad = velocidad + incremento;
    }

    void frenar(int decremento) {
         velocidad = velocidad - decremento;
    }

    void imprimirEstados() {
         System.out.println("cadencia:" +
             cadencia + " velocidad:" +
             velocidad + " marcha:" + marcha);
    }
}

La implementación de una interfaz permite a una clase ser más formal sobre la conducta que promete proporcionar. Las interfaces forman un contracto entre la clase y el mundo exterior, y este contrato es forzado en tiempo de construcción por el compilador. Si su clase pretende implementar una interfaz, todos los métodos definidos por esa interfaz deben aparecer en su código fuente antes de que la clase compilase exitosamente.

Nota: Para poder compilar la clase BicicletaACME , usted necesita añadir la palabra clave public al principio de los métodos de interfaz implementados. Usted aprenderá las razones para esto más tarde en las lecciones en Clases y Objetos y Interfaces y Herencia.
