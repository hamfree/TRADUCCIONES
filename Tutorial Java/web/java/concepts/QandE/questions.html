Preguntas y Ejercicios: Conceptos de Programación Orientado-a-Objeto

Preguntas

  1. Los objetos del mundo real contienen ___ y ___.
  2. El estado de un objeto de software es almacenado en ___.
  3. El comportamiento de un objeto de software es expuesto a través de ___.
  4. Ocultar los datos internos del mundo exterior, y accediendo a ellos sólo a través de métodos públicamente expuestos es conocido como ___ de datos.
  5. Una plantilla para un objeto de software es llamado una ___.
  6. El comportamiento común puede ser definido en una ___ y heredado en una ___ usando la palabra clave ___.
  7. Una colección de métodos sin implementación es llamado una ___.
  8. Un espacio de nombres que organiza clases e interfaces por funcionalidad es llamado un ___.
  9. El término IPA (IPA) significa ___?

Ejercicios

  1. Cree nuevas clases para cada objeto del mundo real que observó al principio de este recorrido. Refíerase a la clase Bicicleta si olvidó la sintaxis requerida.
  2. Para cada nueva clase que usted creó arriba, cree una interfaz que defina su comportamiento, entonces requiera que sus clases lo implementen. Omita uno o dos métodos e intente compilar. ¿Cómo se ve el error?

Compruebe sus respuestas.
