Formateando la Salida Impresa Numérica

Antes vió el uso de los métodos print y println para imprimir cadenas a la salida estándar (System.out). Ya que todos los números pueden ser convertidos a cadenas (como verá más tarde en esta lección), puede usar estos métodos para imprimir una mezcla arbitraria de cadenas y números. El lenguaje de programación Java tiene otros métodos, sin embargo, que le permiten ejercer mucho más control sobre su salida cuando los números son incluídos.

Los Métodos printf y format

El paquete java.ioincluye una clase PrintStream que tiene dos métodos de formateado que puede usar para reemplazar print y println. Estos métodos, format y printf, son equivalentes a los otros. El familiar System.out que ha estado usando pasa a ser un objeto PrintStream, así que puede invocar los métodos PrintStream en System.out. Así, puede usar format o printf en cualquier lugar en su código donde ha estado usando previamente print o println. Por ejemplo, 

System.out.format(.....);

La sintaxis para estos dos The syntax for these two java.io.PrintStream métodos es la misma: 

public PrintStream format(String format, Object... args)

donde format es una cadena que especifica el formato a ser usado y args es una lista de variables a ser impresas usando ese formato. Un ejemplo simple sería 

System.out.format("El valor de " + "la variable float es " +
     "%f, mientras que el valor de " + "la variable entera es %d, " +
     " y la cadena es %s", floatVar, intVar, stringVar);

El primer parámetro, format, es una cadena de formato especificando cómo los objetos en el segundo parámetro, args, van a ser formateados. La cadena de formato contiene tanto texto plano como especificadores de formato, los cuales son caracteres especiales que formatean los argumentos de Object... args. (La notación Object... args es llamada varargs, lo que significa que el número de argumentos puede variar).

Los especificadores de formato empiezan con un signo de porcentaje (%) y terminan con un convertidor. El convertidor es un carácter indicando el tipo del argumento a ser formateado. Entre el signo del porcentaje (%) y el convertidor puede tener banderas opcionales y especificadores. Hay muchos convertidores, banderas, y especificadores, los cuales están documentados en java.util.Formatter

Aquí tiene un ejemplo básico: 

int i = 461012;
System.out.format("El valor de i es: %d%n", i);

El %d especifica que la variable única es un entero decimal. El %n es un caractere de nueva línea independiente-de-la-plataforma. La salida es: 

The value of i is: 461012

Los métodos printf y format están sobrecargados. Cada uno tiene una versión con la siguiente sintaxis: 

public PrintStream format(Locale l, String format, Object... args)

Para imprimir números en el sistema Francés (donde una coma es utilizada en lugar del punto decimal en la representación Ingles de los números de coma flotante), por ejemplo, usaría: 

System.out.format(Locale.FRANCE,
    "El valor de la variable " + "decimal es %f, mientras que el " +
    "valor de la variable entera " + "es %d, y la cadena es %s%n",
    floatVar, intVar, stringVar);

El siguiente programa muestra algunos de los formatos que puede hacer con format. La salida es mostrada entre comillas dobles en el comentario embebido: 

import java.util.Calendar;
import java.util.Locale;

public class PruebaFormato {

    public static void main(String[] args) {
        long n = 461012;

        System.out.format("%s%n",  n);                    //  -->  "461012"
        System.out.format("%08d%n",n);                    //  -->  "00461012"
        System.out.format("%+8d%n", n);                   //  -->  "+461012"
        System.out.format("%,8d%n", n);                   //  -->  "461.012"
        System.out.format("%+,8d%n%n", n);                //  -->  "+461.012"

        double pi = Math.PI;
        System.out.format("%f%n", pi);                    //  -->  "3,141593"
        System.out.format("%.3f%n", pi);                  //  -->  "3,142"
        System.out.format("%10.3f%n", pi);                //  -->  "     3,142"
        System.out.format("%-10.3f%n", pi);               //  -->  "3,142     "
        System.out.format(Locale.forLanguageTag("ES"),
                "%-10.4f%n%n", pi);                       //  -->  "3.1416    "

        Calendar c = Calendar.getInstance();
        System.out.format("%tB %te, %tY%n", c, c, c);     //  -->  "octubre 21, 2016"
        System.out.format("%tl:%tM %tp%n", c, c, c);      //  -->  "10:07 pm"
        System.out.format("%tD%n", c);                    //  -->  "10/21/16"

    }
}
Nota: La descripción en esta sección cubre justo lo básicos de los métodos format y printf. Más detalles se pueden encontrar en la sección E/S Básica del Recorrido Clases Esenciales, en la página "Formateando".

La Clase DecimalFormat

Puede usar la clase java.text.DecimalFormat para controlar la visualización de los ceros iniciales y finales, prefijos y sufijos, separadores de agrupamiento (miles), y el separador decimal. DecimalFormat ofrece una gran flexibilidad en el formateo de números, pero puede hacer su código más complejo.

El ejemplo que siguie crea un objeto DecimalFormat , miFormateador, pasandole un cadena patrón al constructor DecimalFormat. El método format(), el cual DecimalFormat hereda de NumberFormat, es entonces invocado por miFormateador — acepta un valor double como un argumento y devuelve el número formateado en una cadena:

Aquí tiene un programa de ejemplo que ilustra el uso de DecimalFormat: 


import java.text.*;

public class DemoFormatoDecimal {

   static public void formatoPersonalizado(String patron, double valor ) {
      DecimalFormat miFormateador = new DecimalFormat(patron);
      String output = miFormateador.format(valor);
      System.out.println(valor + "  " + patron + "  " + output);
   }

   static public void main(String[] args) {

      formatoPersonalizado("###,###.###", 123456.789);
      formatoPersonalizado("###.##", 123456.789);
      formatoPersonalizado("000000.000", 123.78);
      formatoPersonalizado("$###,###.###", 12345.67);
   }
}

The output is: 

123456.789  ###,###.###  123,456.789
123456.789  ###.##  123456.79
123.78  000000.000  000123.780
12345.67  $###,###.###  $12,345.67

La siguiente tabla explica cada línea de salida The following table explains each line of output.

Salida de DemoFormatoDecimal.java Output
Valor Patrón Salida Explicación
123456.789 ###,###.### 123,456.789 El signo almohadilla (#) denota un dígito, la coma es un marcador de posición para el separador de agrupamiento, y el punto es un marcador de posición para el separador decimal.
123456.789 ###.## 123456.79 El valor tiene tres dígitos a la derecha del punto decimal, pero el patrón tiene solo dos. El método format maneja esto haciendo redondeo hacia arriba.
123.78 000000.000 000123.780 El patrón especifica ceros iniciales y finales, porque el caracter 0 es usado en vez del signo almohadilla (#).
12345.67 $###,###.### $12,345.67 El primer carácter en el patrón es el signo del dólar ($). Tenga en cuenta que inmediatamente precede al dígito de la izquierda en la salida formateada.
