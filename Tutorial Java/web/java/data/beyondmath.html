Más Allá de la Aritmética Básica

El lenguaje de programación Java soporta la aritmética básica con sus operadores aritméticos: +, -, *, /, y %. La clase Math en el paquete java.lang ofrece métodos y constantes para hacer cálculos matemáticos más avanzados.

Los métodos en la clase Math son todos estáticos, así que los llama directamentes desde la clase, de esta forma: 

Math.cos(angulo);
Nota: Usando la característica del lenguaje static import , no tiene que escribir Math enfrente de cada función matemática: 
import static java.lang.Math.*;

Esto le permite invocar los métodos de la clase Math por sus nombre simples. Por ejemplo: 

cos(angulo);

Constantes y Métodos Básicos

La clase Math incluye dos constantes:

  • Math.E, que es la base de los logaritmos naturales, y
  • Math.PI, que es el radio de la circunferencia de un circulo divido por su diámetro.

La clase Math también incluye más de 40 métodos estáticos. La siguiente tabla lista unos cuantos de los métodos básicos.

Métodos Básicos de Math
Metodo Descripción
double abs(double d)
float abs(float f)
int abs(int i)
long abs(long lng) 		Devuelve el valor absoluto del argumento.
double ceil(double d) Devuelve el entero más pequeño que es mayor o igual que el argumento. Lo devuelto es un decimal de doble precisión.
double floor(double d) Devuelve el entero más grande que es menor o igual que el argumento. Lo devuelto es un decimal de doble precisión.
double rint(double d) Devuelve un entero que es el más cercano en valor al argumento. Lo devuelto es un decimal de doble precisión.
long round(double d)
int round(float f) 		Devuelve el entero o entero largo más cercano, como es indicado por el tipo de retorno del método, al argumento.
double min(double arg1, double arg2)
float min(float arg1, float arg2)
int min(int arg1, int arg2)
long min(long arg1, long arg2) 		Devuelve el más pequeño de los dos argumentos.
double max(double arg1, double arg2)
float max(float arg1, float arg2)
int max(int arg1, int arg2)
long max(long arg1, long arg2) 		Devuelve el más grande de los dos argumentos.

El siguiente programa, DemoMatematicaBasica , ilustra cómo usar alguno de estos métodos: 


public class DemoMatematicaBasica {
    public static void main(String[] args) {
        double a = -191.635;
        double b = 43.74;
        int c = 16, d = 45;

        System.out.printf("El valor absoluto " + "de %.3f es %.3f%n",
                          a, Math.abs(a));

        System.out.printf("El redondeo por arriba de " + "%.2f es %.0f%n",
                          b, Math.ceil(b));

        System.out.printf("El redondeo por abajo de " + "%.2f es %.0f%n",
                          b, Math.floor(b));

        System.out.printf("El valor entero más cercano de %.2f " + "es %.0f%n",
                          b, Math.rint(b));

        System.out.printf("El máximo de %d y " + "%d es %d%n",
                          c, d, Math.max(c, d));

        System.out.printf("El mínimo de %d " + "y %d es %d%n",
                          c, d, Math.min(c, d));
    }
}

Aquí está la salida de este programa: 

El valor absoluto de -191,635 es 191,635
El redondeo por arriba de 43,74 es 44
El redondeo por abajo de 43,74 es 43
El valor entero más cercano de 43,74 es 44
El máximo de 16 y 45 es 45
El mínimo de 16 y 45 es 16

Métodos Exponenciales y Logarítmicos

La siguiente tabla lista los métodos exponenciales y logarítmicos de la clase Math.

Métodos Exponenciales y Logarítmicos
Método Descripción
double exp(double d) Devuelve la base de los logarimos naturales, e. a la potencia del argumento.
double log(double d) Devuelve el logaritmo natural del argumento.
double pow(double base, double exponent) Devuelve el valor del primer argumento elevado a la potencia del segundo argumento.
double sqrt(double d) Devuelve la raíz cuadrada del argumento.

El siguiente programa, DemoExponenciales , muestra los valores de e, y entonces llama a cada uno de los métodos listados en la tabla previa sobre números elegidos arbitrariamente: 


public class DemoExponenciales {
    public static void main(String[] args) {
        double x = 11.635;
        double y = 2.76;

        System.out.printf("El valor de " + "e es %.4f%n",
                          Math.E);

        System.out.printf("exp(%.3f) " + "es %,.3f%n",
                          x, Math.exp(x));

        System.out.printf("log(%.3f) es " + "%.3f%n",
                          x, Math.log(x));

        System.out.printf("pow(%.3f, %.3f) " + "es %,.3f%n",
                          x, y, Math.pow(x, y));

        System.out.printf("sqrt(%.3f) es " + "%.3f%n",
                          x, Math.sqrt(x));
    }
}

Aquí está la salida que verá cuando ejecute ExponentialDemo: 

El valor de e es 2,7183
exp(11,635) es 112.983,831
log(11,635) es 2,454
pow(11,635, 2,760) es 874,008
sqrt(11,635) es 3,411

Métodos Trigonométricos

La clase Math también ofrece una colección de funciones trigonométricas, las cuales están resumidas en la siguiente tabla. El valor pasado en cada uno de estos métodos en un ángulo expresado en radianes. Puede usar el método toRadians para convertir de grados a radianes.

Trigonometric Methods
Método Descripción
double sin(double d) Devuelve el seno del valor decimal de doble precisión especificado.
double cos(double d) Devuelve el coseno del valor decimal de doble precisión especificado.
double tan(double d) Devuelve la tangente del valor decimal de doble precisión especificado.
double asin(double d) Devuelve el arcoseno del valor decimal de doble precisión especificado.
double acos(double d) Devuelve el arcocoseno del valor decimal de doble precisión especificado.
double atan(double d) Devuelve la arcotangente del valor decimal de doble precisión especificado.
double atan2(double y, double x) Convierte las coordenadas rectangulares (x, y) en coordenadas polares (r, theta) y devuelve theta.
double toDegrees(double d)
double toRadians(double d) 		Convierte el argumento a grados o radianes.

Aquí tiene un programa DemoTrigonometria , que usa cada uno de estos métodos para calcular varios valores trigonométricos para un ángulo de 45 grados: 


public class DemoTrigonometria {
    public static void main(String[] args) {
        double grados = 45.0;
        double radianes = Math.toRadians(grados);

        System.out.format("El valor de pi " + "es %.4f%n",
                           Math.PI);

        System.out.format("El seno de %.1f " + "grados es %.4f%n",
                          grados, Math.sin(radianes));

        System.out.format("El coseno de %.1f " + "grados es %.4f%n",
                          grados, Math.cos(radianes));

        System.out.format("La tangente de %.1f " + "grados es %.4f%n",
                          grados, Math.tan(radianes));

        System.out.format("El arcoseno de %.4f " + "es %.4f grados %n",
                          Math.sin(radianes),
                          Math.toDegrees(Math.asin(Math.sin(radianes))));

        System.out.format("El arcocoseno de %.4f " + "es %.4f grados %n",
                          Math.cos(radianes),
                          Math.toDegrees(Math.acos(Math.cos(radianes))));

        System.out.format("La arcotangente de %.4f " + "es %.4f grados %n",
                          Math.tan(radianes),
                          Math.toDegrees(Math.atan(Math.tan(radianes))));
    }
}

La salida de este programa es como sigue: 

El valor de pi es 3,1416
El seno de 45,0 grados es 0,7071
El coseno de 45,0 grados es 0,7071
La tangente de 45,0 grados es 1,0000
El arcoseno de 0,7071 es 45,0000 grados
El arcocoseno de 0,7071 es 45,0000 grados
La arcotangente de 1,0000 es 45,0000 grados

Números Aleatorios

El método random() devuelve un número pseudo-aleatorio seleccionado entre 0,0 y 1,0. El rango incluye 0,0 pero no 1,0. En otras palabras: 0.0 <= Math.random() < 1.0. Para obtener un número en un rango diferente, puede realizar aritmética sobre el valor devuelto por el método rando. Por ejemplo, para generar un entero entre 0 y 9, puede escribir: 

int number = (int)(Math.random() * 10);

Multiplicando el valor por 10, el rango de valores posibles entre 0.0 <= number < 10.0.

Usar Math.random funciona bien cuando necesita generar un único número aleatorio. Si necesita generar una serie de números aleatorios, debería crear una instancia de java.util.Random e invocar los métodos del objeto para generar números.
