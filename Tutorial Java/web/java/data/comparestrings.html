Comparando Cadenas y Porciones de Cadenas

La clase String tiene una serie de métodos para comparar cadenas y porciones de cadena. La siguiente tabla lista esos métodos..

Métodos para Comparar Cadenas
Método Descripción
boolean endsWith(String suffix)
boolean startsWith(String prefix) 		Devuelve true si la cadena termina con o empieza con la subcadena especificada como un argumento al método.
boolean startsWith(String prefix, int offset) Considere la cadena empezando en el indice offset, y devuelve true si empieza cn la subcdeana especificad como un argumento.
int compareTo(String anotherString) Compara dos cadenas lexicograficamente. Devuelve un entero indicando si esta cadena es mayor que (resultado es > 0), igual a (resultado es = 0), o menor que (resultadp es < 0) el argumento.
int compareToIgnoreCase(String str) Compara dos cadenas lexicograficamente, ignorando las diferencias en el caso. Devuelve un entero indicando si la cadena es mayor que (resultado es > 0), igual a (resultado es = 0), o menos que (resultado es < 0) el argumento.
boolean equals(Object anObject) Devuelve true si y solo si el argumento es un objeto String que representa la misma secuencia de caracteres como este objeto.
boolean equalsIgnoreCase(String anotherString) Devuelve Returns true si y sólo si el argumento es un objeto String que representa la misma secuencia de caracteres como este objeto, ignorando diferencias en la calle.
boolean regionMatches(int toffset, String other, int ooffset, int len) Comprueba si la región espeocificada de esta cadena coincide con la región especificada del argumento String.

Region es de longitud len y empieza en el indice toffset para esta cadena y offset para la otra cadena.

boolean regionMatches(boolean ignoreCase, int toffset, String other, int ooffset, int len) Comprueba si la región especificada de esta cadena coinciden con la región especificada del argumento String.

Region es de longitud len y empieza en el índice toffset para esta cadena y ooffset para la otra cadena.

El argumento booleano indica que el caso debería ser ignorado.; si es verdad, el caso será ignorado cuando se comparen los caracteres.

boolean matches(String regex) Comprueba si esta cadena coincide con la expresión regular especificada. Las expresiones regulares son descritas en la lección titulada "Expresiones Regulares."

El siguiente programa, DemoCoincidenciaRegion, usando el método regionMatches para buscar una cadena dentro de otra cadena: 


ublic class DemoCoincidenciaRegion {
    public static void main(String[] args) {
        String buscaMe = "Huevos verdes y jamón";
        String encuentraMe = "verdes";
        int longitudBuscaMe = buscaMe.length();
        int longitudEncuentraMe = encuentraMe.length();
        boolean encontrada = false;
        for (int i = 0;
             i <= (longitudBuscaMe - longitudEncuentraMe);
             i++) {
           if (buscaMe.regionMatches(i, encuentraMe, 0, longitudEncuentraMe)) {
              encontrada = true;
              System.out.println(buscaMe.substring(i, i + longitudEncuentraMe));
              break;
           }
        }
        if (!encontrada)
            System.out.println("No se encontraron coincidencias.");
    }
}

La salida de este programa es verdes.

Los pasos del programa a través de la cadena referenciados por searchMe son de un caracter cada vez. Para cda carácter, el programa llama el método regionMatches para determinar si la subcadena que empieza con el carácter actual coincide con la cadena que el programa está buscando.
