Sumario de Caracteres y Cadenas

La mayoría del tiempo, si está usando un valor de carácter simple, usará el tipo primitivo char. Hay veces, sin embargo, cuando necesite usar un carácter como un objeto — por ejemplo, como un argumento de método donde un objeto se espera. El lenguaje de programación Java ofrece una clase envoltorio que "envuelve" el char en un objeto Character para este objetivo. Un objeto de tipo Character contiene un campo único cuyo tipo es char. Esta clase Character también ofrece una serie de métodos de clase (p.e., estáticos) para manipular caracteres.

Las cadenas son secuencias de caracteres y son ampliamente usadas en la programación Java. En el lenguaje de programación Java, las cadenas son objetos. La clase String tiene sobre 60 métodos y 13 constructores.

Más comunmente, usted crea una cadena con una sentencia como 

String s = "¡Hola mundo!";

en vez de usar uno de los constructores de String.

La clase String tiene muchos métodos para buscar y recuperar subscadenas; estos pueden entonces ser fácilmente reensamblados en nuevas cadenas usando el operador de concatenación +.

La clase String también incluye un número de métodos de utiidad, entre ellos split(), toLowerCase(), toUpperCase(), y valueOf(). El último método es indispensable en convertir las cadenas introducidas por el usuario en números. Las subclases de Number también tiene métodos para convertir cadenas en números y viceversa.

Además de la clase String, hay también una clase StringBuilder. Trabajar con objetos StringBuilder puede a veces ser más eficiente que trabajar con cadenas. La clase StringBuilder ofrece unos pocos métodos que puedan ser útiles para cadenas, entre ellos reverse(). En general, sin embargo, la clase String tiene una amplia variedad de métodos.

Una cadena puede ser convertida en un constructor de cadenas usando un constructor de StringBuilder. Un constructor de cadena puede ser convertido en una cadena con el método toString().
