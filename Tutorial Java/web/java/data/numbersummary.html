Sumario de Números

Use una de las clases envoltorio – Byte, Double, Float, Integer, Long, o Short – para envolver un número de tipo primitivo en un objeto. El compilador de Java automáticamente envuelve (boxes, en inglés) los datos primitivos por usted cuando es necesario y los "desenvuelve" (unboxes, en inglés) de nuevo cuando es necesario.

Las clases Number incluyen constantes y métodos de clase útiles. Las constantes MIN_VALUE y MAX_VALUE contienen los valores más pequeños y más grandes que pueden contenerse en un objeto de ese tipo. Los métodos byteValue, shortValue, y métodos similares convierten un tipo numérico en otro. El método valueOf convierte una cadena en un número, y el método toString convierte un número en una cadena.

Para formatear una cadena que contiene números para la salida, puede usar los métodos printf() o format() en la clase PrintStream. Alternativemente, puede usar la clase NumberFormat para personalizar los formatos numéricos usando patrones.

La clase Math contiene una variedad de métodos de clase para realizar funciones matemáticas, incluyendo métodos exponenciales, logarítmicos y trigonométricos. Math también incluye las funciones aritméticas básicas, tales como el valor absoluto y el redondeo, y un método, random(), para generar números aleatorios.
