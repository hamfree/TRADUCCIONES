Convirtiendo Entre Números y Cadenas

Convirtiendo Cadenas a Números

Frecuentemente, un programa termina con los datos numéricos en un objeto string — un valor entrado por el usuario, por ejemplo.

Las subclases de Number que envuelven los tipos numéricos primitivos ( Byte , Integer , Double , Float , Long , and Short ) establecen cada uno un método de clase llamado valueOf que convierte una cadena a un objeto de este tipo. Aquí tiene un ejemplo, DemoValorDe , que toma dos cadenas de la línea de comandos, las convierte a números, y realiza operaciones aritméticas con los valores: 


public class DemoValorDe {
    public static void main(String[] args) {

        // este programa requiere dos
        // argumentos en la lína de comandos
        if (args.length == 2) {
            // convierte cadenas a números
            float a = (Float.valueOf(args[0])).floatValue();
            float b = (Float.valueOf(args[1])).floatValue();

            // hace algunos cálculos
            System.out.println("a + b = " +
                               (a + b));
            System.out.println("a - b = " +
                               (a - b));
            System.out.println("a * b = " +
                               (a * b));
            System.out.println("a / b = " +
                               (a / b));
            System.out.println("a % b = " +
                               (a % b));
        } else {
            System.out.println("Este programa " +
                "requiere dos argumentos en la línea de comandos.");
        }
    }
}

Lo siguiente es la salida del programa cuando usa 4.5 y 87.2 como argumentos de la línea de comandos: 

a + b = 91.7
a - b = -82.7
a * b = 392.4
a / b = 0.0516055
a % b = 4.5
Nota: Cada una de las subclases de Number que envuelven los tipos numéricos primitivos tambien establecen un método parseXXXX() (por ejemplo, parseFloat()) que puede ser usado para convertir cadenas en números primitivos. Ya que se devuelve un tipo primitivo en vez de un objeto, el método parseFloat() es más directo que el método valueOf() . Por ejemplo, en el programa DemoValorDe, podríamos usar: 
float a = Float.parseFloat(args[0]);
float b = Float.parseFloat(args[1]);

Convirtiendo Números a Cadenas

Algunas veces necesita convertir un número en una cadena porque necesita operar sobre el valor en su forma de cadena. Hay varias formas fáciles de convertir un número en una cadena: 

int i;
// Concatenamos "i" con una cadena vacía: la conversión es manejada por usted.
String s1 = "" + i;

o 

// El método de clase valueOf.
String s2 = String.valueOf(i);

Cada una de las subclases de Number incluyen un método de clase, toString(), que convertirá su tipo primitivo en una cadena. Por ejemplo: 

int i;
double d;
String s3 = Integer.toString(i); 
String s4 = Double.toString(d);

El ejemplo DemoToString usa el método toString para convertir un número en una cadena. El programa entonces usa algunos métodos de string para computar el número de dígitos antes y después del punto decimal: 


public class DemoToString {

    public static void main(String[] args) {
        double d = 858.48;
        String s = Double.toString(d);

        int dot = s.indexOf('.');

        System.out.println(dot + " dígitos " +
            "antes de la coma decimal.");
        System.out.println( (s.length() - dot - 1) +
            " dígitos después de la coma decimal.");
    }
}

La salida de este programa es: 

3 dígitos antes de la coma decimal.
2 dígitos después de la coma decimal.
