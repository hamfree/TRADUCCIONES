Caracteres

La mayoría del tiempo, si está usando un valor de carácter simple, usará el tipo primitivo char. Por ejemplo:

char c = 'a';
// código Unicode para la mayúscula del caracter Griego omega
char uniChar = '\u03A9';
// una matriz de caracteres
char[] matrizCaracteres = { 'a', 'b', 'c', 'd', 'e' };

Hay veces, sin embargo, cuando necesita usar un carácter como un objeto — por ejemplo, como un argumento de método donde se espera un objeto. El lenguaje de programación Java ofrece una clase envoltorio que envuelve el char en un objeto Character para este propósito. Un objeto del tipo Character contiene un único campo, cuyo tipo es char. Esta clase Character también ofrece una serie de útiles métodos de clase (p.e., estáticos) para manipular caracteres.

Puede crear un objeto Character con el constructor Character: 

Character ch = new Character('a');

El compilador de Java también creará un objeto Character por usted bajo ciertas circunstancias. Por ejemplo, si pasa un char primitivo a un método que espera un objeto, el compilador automáticamente convierte el char en un Character por usted. Esta característica se llama autoboxing — o unboxing, si la conversión va en la otra dirección. Para más información sobre autoboxing y unboxing, vea Autoboxing y Unboxing (Conversiones Automáticas) .

Nota: La clase Character es immutable, así que una vez es creada, un objeto Character no puede ser cambiado.

La siguiente tabla lista algunos de los más útiles métodos en la clase Character, pero no es exhaustiva. Para un listado completo de todos los métodos en esta clase (hay más de 50), refíerase a la especificación del IPA  java.lang.Character .

Método Descripción
boolean isLetter(char ch)
boolean isDigit(char ch)		 Determina si el valor carácter especificado es una letra o un dígito, respectiavamente.
boolean isWhitespace(char ch) Determina si el valor de caracterer especificado es un espacion en blanco.
boolean isUpperCase(char ch)
boolean isLowerCase(char ch)		 Determina si el valor de caracterer especificado está en mayúsculas o minúsculas, respectivamente.

Secuencias de Escape

Un carácter precedido por una barra invertida (\) es una secuencia de escape y tiene un significado especial para el compilador. La siguiente tabla muestra las secuencias de escapa de Java:

Secuencias de Escape
Secuencia de Escape Descripción
\t Inserta un tabulador en el texto en es punto.
\b Inserta un retroceso en el texto en ese punto.
\n Inserta una nueva línea en el texto en ese punto.
\r Inserta un retorno de carro en el texto en ese punto.
\f Inserta un salto de página en el texto en ese punto.
\' Insertar un carácter de comillas simples en el texto en este punto.
\" Insertar un carácter de comillas dobles en el texto en este punto.
\\ Inserta un carácter de barra invertida en el texto en ese punto.

Cuando se encuentra una secuencia de escape en una sentencia de impresión, el compilador la interpreta en consecuencia. Por ejemplo, si quiere colocar comillas dentro de comillas debe usar la secuencia de escape, \", como comillas interiores. Para imprimir la oración: 

Ella me dijo "¡Hola!".

Usted escribirá 

System.out.println("Ella me dijo \"¡Hola!\".");
