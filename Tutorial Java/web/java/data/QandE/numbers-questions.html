Preguntas y Ejercicios: Números

Preguntas

  1. Use la documentación de la IPA para buscar las respuestas a las siguientes preguntas:

    1. ¿Qué método de Integer puede usar para convertir un int en una cadena que exprese el número en hexadecimal? Por ejemplo, ¿Qué método convierte el entero 65 en una cadena?

    2. ¿Qué método de Integer usaría para converitr una cadena expresada en base 5 en el int equivalente? Por ejemplo, ¿Cómo convertiría la cadena "230" en el valor entero 65? Muestre el ćodigo que ha usado para cumplir esta tarea.

    3. ¿Qué método Double puede usar para detectar si un número de coma flotante tiene el valor especial No a Number (NaN)?

  2. ¿Cual es el valor de la siguiente expresión, y porqué? 

    Integer.valueOf(1).equals(Long.valueOf(1))

Ejercicios

  1. Cambie DemoMaximoVariables para mostrar los valores mínimos en vez de los valores máximos. Puede borrar todo el código relacionado a estas variables: aChar y aBoolean. ¿Cual es la salida?

  2. Cree un programa que lea un número no especificado de argumentos enteros desde la línea de comandos y añádalos juntos. Por ejemplo, suponga que usted introduce lo siguiente: 

    java Adder 1 3 2 10

    El programa visualizaría 16 y entonces saldría. El programa visualizará un mensaje de error si el usuario entra sólo un argumento. Puede basar su programa en DemoValorDe .

  3. Cree un programa que sea similar al anterior pero que tenga las siguientes diferencias:

    • En vez de leer argumentos enteros, lee argumentos en coma-flotante.
    • Visualiza la suma de los argumentos, usando exactamente dos dígitos a la derecha del punto decimal.

    Por ejemplo, suponga que introduce lo siguiente: 

    java FPAdder 1 1e2 3.0 4.754

    El programa visualizaría 108.75. Dependiendo de su región, el punto decimal podría ser una coma (,) en vez de un punto (.).

Compruebe sus respuestas.
