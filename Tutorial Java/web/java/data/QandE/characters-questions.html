Preguntas y Ejercicios: Caracteres y Cadenas

Preguntas

  1. ¿Cual es la capacidad inicial del siguiente constructor de cadenas? 

    StringBuilder sb = new StringBuilder("Able was I ere I saw Elba.");

  2. Considere la siguiente cadena: 

    String hannah = "Did Hannah see bees? Hannah did.";

    1. ¿Cuál es el valor visualizado por la expresión hannah.length()?

    2. ¿Cuál es el valor retornado por la llamada al método hannah.charAt(12)?

    3. Escribe una expresión que se refiera a la letra b en la cadena referenciasda por hannah.

  3. ¿Cuánto mide la cadena devuelta por la siugiente expresión?? ¿Cuál es la cadena? 

    "Was it a car or a cat I saw?".substring(9, 12)

  4. En el siguiente programa, llamado ComputaResultado , ¿cuál es el valor de resultado después de cada línea numerada se ejecute? 

    public class ComputaResultado {
    public static void main(String[] args) {
        String original = "software";
        StringBuilder resultado = new StringBuilder("hi");
        int indice = original.indexOf('a');

/*1*/   resultado.setCharAt(0, original.charAt(0));
/*2*/   resultado.setCharAt(1, original.charAt(original.length()-1));
/*3*/   resultado.insert(1, original.charAt(4));
/*4*/   resultado.append(original.substring(1,4));
/*5*/   resultado.insert(3, (original.substring(indice, indice+2) + " "));

        System.out.println(resultado);
    }
}

Ejercicios

  1. Muestre dos formas de concatenar las siguientes dos cadenas juntas para obtener la cadena "Hi, mom.": 

    String hi = "Hi, ";
String mom = "mom.";

  2. Escriba un programa que compute sus iniciales de su nombre completo y visualízelas.

  3. Un anagrama es una palabra o frase hecha por la transposición de letras de otra palabra o frase; por ejemplo, "parliament" es un anagrama de "partial men," y "software" es un anagrama de "swear oft." Escriba un programa que se de cuenta de si una cadena es un anagrama de otra cadena. El programa ignorará los esacios en blancol y la puntuación.

Compruebe sus respuestas.
