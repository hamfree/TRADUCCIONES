Cadenas

Las cadenas, las cuales son usadas ampliamente en la programación Java, son una secuencia de caracteres. En el lenguaje de programación Java, las cadenas son objetos.

La plataforma Java ofrece la clase String para crear y manipular cadenas.

Creando Cadenas

La forma más directa de crear una candea es cmo sigue: 

String saludo = "¡Hola mundo!";

En este caso, "'Hola mundo!" es un literal de cadena—una serie de caracteres en su código que está encerrado entre dobles comillas. En cualquier lugar donde se encuentre un literal de cadena en tu código, el compilador crea un objeto String con su valor — en este caso, ¡Hola mundo!.

Como con cualquier otro objeto, puede crear objetos String usando la palabra clavenew y un constructor. La clase String tiene 13 constructores que le permiten establecer el valor inicial de la cadena usando diferentes fuentes, tales como una matriz de caracteres: 

char[] matrizHola = { 'h', 'o', 'l', 'a', '.' };
String holaCadena= new String(matrizHola);
System.out.println(helloString);

La útlima línea de este trozo de código muestra hola..

Nota: La clase String es inmutable, así que una vez creada un objeto String no puede ser cambiado. La clase String tiene una serie de métodos, algunos de ellos descritos más abajo que parecen modificar cadenas. Ya que las cadenas son inmutables, lo que estos métodos realmente hacen es crear y devolver una nueva cadena que contien el resultado de la operación.

Longitud de la Cadena

Los métodos usados para obtener información sobre un objeto son conocidos como métodos de acceso. Un método de acceso que puede usar con cadenas es el método length(), el cual devuelve el número de caracteres contenidos en el objeto cadena. Después las sigiuentes dos líneas de código han sido ejecutadas, longitud vale 13: 

String palindromo = "Aman a Panamá"
int len = palindromo.length();

Un palíndromo es una palabra u oración que es simétrica — se deletrea igual hacia adelante y hacia atrás, ignorando el caso y la puntuación. Aquí tiene un corto e ineficiente programa que invierte una cadena palíndromo. Invoca el método charAt(i) de String, la cual devuelve el iésimo caracter en la cadena, contando desde 0. 


public class DemoString {
    public static void main(String[] args) {
        String palindromo = "Aman a Panamá";
        int longitud = palindromo.length();
        char[] matrizTmpCars = new char[longitud];
        char[] matrizCars = new char[longitud];

        // coloco la cadena original en una
        // matriz de caracteres
        for (int i = 0; i < longitud; i++) {
            matrizTmpCars[i] =
                palindromo.charAt(i);
        }

        // invierto la matriz de caracteres
        for (int j = 0; j < longitud; j++) {
            matrizCars[j] =
                matrizTmpCars[longitud - 1 - j];
        }

        String palindromoInverso =
            new String(matrizCars);
        System.out.println(palindromoInverso);
    }
}

Ejecutar este programa produce esta salida: 

ámanaP a namA

Para llevar a cabo la inversión de la cadena, el programa tiene que convertir la cadena en una matriz de caracteres (primer bucle for), invertir la matriz en una segunda matriz (segundo bloque for), y entonces convertirla de vuelta en una cadena. La clase String incluye un método, getChars(), para convertir una cadena, o una porción de una cadena, en una matriz de caracteres de forma que podamos reemplazar el primer bucle for en el programa de arriba con 

palindromo.getChars(0, longitud, matrizTmpCars, 0);

Concatenando Cadenas

La clase String incluye un método para concatenar dos cadenas: 

string1.concat(string2);

Esto devuelve una nueva cadena que es string1 con string2 añadida a su final.

Puede también usar el método concat() con literales de cadenas, como en: 

"Mi nombre es ".concat("Rumplestiltskin");

Las cadenas son concatenadas más comunmente con el operador +, como en 

"¡" + "Hola," + " mundo" + "!"

que resulta ser 

"¡Hola, mundo!"

El operador + es ampliamente usando en sentencias print. Por ejemplo: 

String string1 = "vio que yo era ";
System.out.println("Dot " + string1 + "Tod");

que imprime 

Dot vio que yo era Tod

Una concatenación de este tipo puede ser una mezcla de varios objetos. Para cada objeto que no es una String, su método toString() es llamado para convertirlo en una String.

Nota: El lenguaje de programación Java no permite que cadenas literales ocupen varias líneas en los ficheros fuente, por lo que debe usar el operador + de concatenación al final de cada línea en una cadena multilinea. Por ejemplo: 
String cita =
    "Ahora es el tiempo para que todos los hombres " +
    "buenos vengan en ayuda de su país.";

Romper las cadenas en líneas usando el operador de concatenación + es, una vez más, muy común en las sentencias print.

Creando Cadenas de Formato

Usted ha visto el uso de los métodos printf() y format() para imprimir con números formateados. La clase String tiene un método de clase equivalente, format(), que devuelve un objeto String en vez de un objeto PrintStream.

Usar el método de estático format() de String le permite crear una cadena formateada que puede reutilizar, al contrario que la sentencia de impresión de una vez. Por ejemplo, en vez de 

System.out.printf("El valor de la variable " +
                  "float es %f, mientras " +
                  "el valor de la " +
                  "variable entera es %d, " +
                  "y la cadena es %s",
                  floatVar, intVar, stringVar);

puede escribir 

String fs;
fs = String.format(El valor de la variable " +
                  "float es %f, mientras " +
                  "el valor de la " +
                  "variable entera es %d, " +
                  "y la cadena es %s",
                   floatVar, intVar, stringVar);
System.out.println(fs);
