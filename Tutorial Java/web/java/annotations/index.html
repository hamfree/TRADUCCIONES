Lección: Anotaciones

Las Anotaciones, una forma de metadatos, establece datos sobre un programa que no son parte del programa en sí mismo. Las anotaciones no tienen efecto directo sobre la operación del código que ellas anotan.

Las anotaciones tienen un número de usos, entre ellos:

  • Información para el compilador — Las anotaciones pueden ser usadas por el compilador para detectar errores o suprimir avisos.
  • Procesamiento en tiempo de compilación y en tiempo de despliegue — Las herramientas de software pueden procesar información de la anotación para generar código, ficheros XML, Etcétera.
  • Procesamiento en tiempo de ejecución — Algunas anotaciones están disponibles para ser examinadas en tiempo de ejecución.

Esta lección explica dónde se pueden usar las anotaciones, cómo aplicar las anotaciones, que tipos de anotaciones predefinidas están disponibles en la Plataforma Java, Edición Estándar (Java SE IPA), cómo los tipos de anotaciones pueden ser usados en conjunción con los sistemas de tipos enchufables para escribir código con comprobación de tipos más fuerte, y como implementar anotaciones que se repiten.
