Declarando un tipo Anotación

Muchas anotaciones reemplazan comentarios en el código.

Suponga que un grupo de software tradicionalmente inicia el cuerpo de cada clase como comentarios estableciendo información importante: 

public class ListaGeneracion3 extends ListaGeneracion2 {

   // Autor: Juan Herrera
   // Fecha: 17/3/2002
   // Revision actual: 6
   // Última modificación: 12/4/2004
   // Por: Juana Expósito
   // Revisores: Alica, Guillermo, Cintia

   // el código de la clase va aquí

}

Para añadir estos mismos metadatos con una anotación, debe primero definir el tipo de anotación. La sintaxis para hacer esto es: 

@interface PreambuloClase {
   String autor();
   String fecha();
   int revisionActual() default 1;
   String ultimaModificacion() default "N/D";
   String por() default "N/D";
   // Note el uso de la matriz
   String[] revisores();
}

La definición del tipo de anotación parece similar a una definición de interfaz donde la palabra clave interface está precedida del signo (@) (@ = arroba, como un tipo de anotación). Los tipos de anotación son una forma de interfaz, el cual será cubierto en una lección más tarde. Por el momento, no necesita entender los interfaces.

El cuerpo de la definición de anotación previa contiene declaraciones de elementos del tipo anotación, los cuales se parecen mucho a los métodos. Observe que en ellos se pueden definir valores por defecto opcionales.

Después de que el tipo de anotación es definido, puede usar anotaciones de ese tipo, con los valores rellenos, como esto: 

@PreambuloClase (
   autor = "Juan Herrera",
   fecha = "17/3/2002",
   versionActual = 6,
   ultimaModificacion = "12/4/2004",
   por = "Juan Exposito",
   // Note como se anota la matriz
   revisores = {"Alicia", "Roberto", "Cintia"}
)
public class ListaGeneracion3 extends ListaGeneracion2 {

// el código de la clase va aquí

}
Nota: Para hacer que la información de @ClassPreamble aparezca en la documentación generada-por-Javadoc, debe anotar la definición @ClassPreamble con la anotación @Documented: 
// importe esto para usar @Documented
import java.lang.annotation.*;

@Documented
@interface PreambuloClase {

   // Definiciones de los elementos de la anotación
   
}
