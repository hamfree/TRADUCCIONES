Preguntas y Ejercicios: Anotaciones

Preguntas

  1. ¿Qué está mal en el siguiente interfaz? 

    public interface Casa {
    @Deprecated
    void abrir();
    void abrirPuertaPrincipal();
    void abrirPuertaTrasera();
}

  2. Considere esta implementación del interfaz Casa, mostrado en la Pregunta 1. 

    public class MiCasa implements Casa {
    public void abrir() {}
    public void abrirPuertaPrincipal() {}
    public void abrirPuertaTrasera() {}
}

    Si compila este programa, el compilador produce un aviso porque abrir estaba obsoleto (en el interfaz). ¿Qué puede hacer para deshacerse de esa advertencia?

  3. ¿Compilará el siguiente código sin error? ¿Porqué o porqué no? 

    public @interface Comida { ... }

@Comida("desayuno", platoPrincipal="cereal")
@Comida("almuerzo", platoPrincipal="pizza")
@Comida("cena", platoPrincipal="ensalada")
public void evaluaDieta() { ... }

Ejercicios

  1. Defina un tipo de anotación para una petición mejorada con elementos id, sinopsis, ingeniero, y fecha. Especifique el valor por defecto como sin asignar para ingeniero y desconocida para fecha.

Compruebe sus respuestas.
