La Sentencia switch

A diferencia de las sentencias if-then y if-then-else, la sentencia switch puede tener un número de posibles rutas de ejecución. Un switch funciona con los tipos de datos primitivos byte, short, char, y int. También funciona con los Tipos enumerados (descritos en Tipos Enum), la clase String , y unas pocas clases especiales que envuelven ciertos tipos primitivos: Character, Byte, Short, e Integer (descrita en Números y Cadenas).

El siguiente ejemplo de código, SwitchDemo , declara un int llamado mes cuyo valor representa un mes. El código muestra el nombre del mes, basado en el valor de mes, usando la sentencia switch. 


public class SwitchDemo {
    public static void main(String[] args) {

        int mes = 8;
        String cadenaMes;
        switch (mes) {
            case 1:  cadenaMes = "Enero";
                     break;
            case 2:  cadenaMes = "Febrero";
                     break;
            case 3:  cadenaMes = "Marzo";
                     break;
            case 4:  cadenaMes = "Abril";
                     break;
            case 5:  cadenaMes = "Mayo";
                     break;
            case 6:  cadenaMes = "Junio";
                     break;
            case 7:  cadenaMes = "Julio";
                     break;
            case 8:  cadenaMes = "Agosto";
                     break;
            case 9:  cadenaMes = "Septiembre";
                     break;
            case 10: cadenaMes = "Octubre";
                     break;
            case 11: cadenaMes = "Noviembre";
                     break;
            case 12: cadenaMes = "Diciembre";
                     break;
            default: cadenaMes = "Mes no v\u00E1lido";
                     break;
        }
        System.out.println(cadenaMes);
    }
}

En este caso, Agosto es impreso en la salida estándar.

El cuerpo de una sentencia switch es conocido como un bloque switch. Una sentencia en el bloque switch puede ser etiquetada con una o más etiquetas case o default. La sentencia switch evalúa su expresión, entonces ejecuta todas las sentencias que sigan a la etiqueta coincidente case.

Podría también mostrar el nombre del mes con sentencias if-then-else: 

int mes = 8;
if (mes == 1) {
    System.out.println("Enero");
} else if (mes == 2) {
    System.out.println("Febrero");
}
...  // y así

Decidir si usar sentencias if-then-else o una sentencia switch está basado en la legibilidad y la expresión que la sentencia está comprobando. Una sentencia if-then-else puede comprobar expresiones basadas en rangos de valores o condiciones, mientras que una sentencia switch comprueba expresiones basadas sólo en un único entero, valor enumerado, u objeto String.

Otro punto de interés es la sentencia break. Cada sentencia break termina la sentencia switch que la encierra. El flujo de control continúa con la primera sentecia que sigue al bloque switch. Las sentencias break son necesarias porque sin ellas, las sentencias en los bloques switch caerán a través: Todas las sentencias después de la etiqueta coincidente case serán ejecutadas en secuencia, independientemente de la expresión de las etiquetas case subsecuentes, hastaa que una sentencia break fuera encontrada. El programa SwitchDemoFallThrough muestra sentencias en un bloque switch que caen a través. El programa muestra el mes correspondiente al entero mes y los meses que le siguen en el año: 


public class SwitchDemoFallThrough {

    public static void main(String[] args) {
        java.util.ArrayList<String> mesesFuturos
                = new java.util.ArrayList<String>();

        int mes = 8;

        switch (mes) {
            case 1:
                mesesFuturos.add("Enero");
            case 2:
                mesesFuturos.add("Febrero");
            case 3:
                mesesFuturos.add("Marzo");
            case 4:
                mesesFuturos.add("Abril");
            case 5:
                mesesFuturos.add("Mayo");
            case 6:
                mesesFuturos.add("Junio");
            case 7:
                mesesFuturos.add("Julio");
            case 8:
                mesesFuturos.add("Agosto");
            case 9:
                mesesFuturos.add("Septiembre");
            case 10:
                mesesFuturos.add("Octubre");
            case 11:
                mesesFuturos.add("Noviembre");
            case 12:
                mesesFuturos.add("Diciembre");
                break;
            default:
                break;
        }

        if (mesesFuturos.isEmpty()) {
            System.out.println("N\u00FAmero de mes no v\u00E1lido");
        } else {
            for (String monthName : mesesFuturos) {
                System.out.println(monthName);
            }
        }
    }
}

This is the output from the code: 

Agosto
Septiembre
Octubre
Noviembre
Diciembre

Técnicamente, el break final no es requerido porque el flujo cae fuera de la sentencia switch. Usar un break está recomendado de forma que si modifica el código es más fácil y menos propenso a errores. La sección default maneja todos los valores que no son explícitamente manejados por una de las secciones case.

El siguiente código de ejemplo, SwitchDemo2 , muestra como una sentencia puede tener múltiples etiquetas case. El ejemplo de código calcula el número de días en un particular mes: 


class SwitchDemo2 {
    public static void main(String[] args) {

        int mes = 2;
        int año = 2000;
        int numDias = 0;

        switch (mes) {
            case 1:
            case 3:
            case 5:
            case 7:
            case 8:
            case 10:
            case 12:
                numDias = 31;
                break;
            case 4:
            case 6:
            case 9:
            case 11:
                numDias = 30;
                break;
            case 2:
                if (((año % 4 == 0)
                        && !(año % 100 == 0))
                        || (año % 400 == 0)) {
                    numDias = 29;
                } else {
                    numDias = 28;
                }
                break;
            default:
                System.out.println("Mes no v\u00E1lido.");
                break;
        }
        System.out.println("N\u00FAmero de D\u00EDas = "
                + numDias);
    }
}

Esta es la salida del código: 

Número de Días = 29

Usando Strings en Sentencias switch

En Java SE 7 y posteriores, pudes usar un objeto String en la expresión de la sentencia switch. El siguiente código de ejemplo, StringSwitchDemo , muestra el número del mes basado en el valor de la String llamada mes: 

public class StringSwitchDemo {

    public static int getMonthNumber(String mes) {

        int numeroMes = 0;

        if (mes == null) {
            return numeroMes;
        }

        switch (mes.toLowerCase()) {
            case "enero":
                numeroMes = 1;
                break;
            case "febrero":
                numeroMes = 2;
                break;
            case "marzo":
                numeroMes = 3;
                break;
            case "abril":
                numeroMes = 4;
                break;
            case "mayo":
                numeroMes = 5;
                break;
            case "junio":
                numeroMes = 6;
                break;
            case "julio":
                numeroMes = 7;
                break;
            case "agosto":
                numeroMes = 8;
                break;
            case "septiembre":
                numeroMes = 9;
                break;
            case "octubre":
                numeroMes = 10;
                break;
            case "noviembre":
                numeroMes = 11;
                break;
            case "diciembre":
                numeroMes = 12;
                break;
            default:
                numeroMes = 0;
                break;
        }

        return numeroMes;
    }

    public static void main(String[] args) {

        String mes = "Agosto";

        int numeroMesRetornado
                = StringSwitchDemo.getMonthNumber(mes);

        if (numeroMesRetornado == 0) {
            System.out.println("Mes no v\u00E1lido");
        } else {
            System.out.println(numeroMesRetornado);
        }
    }
}

La salida de este código es 8.

La String en la exxpresión switch es comparada con las expresiones asociadas con cda etiqueta case como si método String.equals estuviera siendo usado. Para que el ejemplo StringSwitchDemo acepte cualquier mes independientemente de la capitalización, mes es convertido a minúsculas (con el método toLowerCase ), y todas las cadenas asociadas con las etiquetas case estarán en minúsculas.

Nota: Este ejemplo comprueba si la expresión en la sentencia switch es null. Asegura que la expresión en cualquier sentencia switch no sea null para prevenir que una NullPointerException sea lanzada.
