Sumario de Variables

El lenguaje de programación Java utiliza ambos "campos" y "variables" como parte de su terminología. Las variables de instancia (campos no estáticos) son únicos a cada instancia de una clase. Las variables de clase (campos estáticos) son campos declarados con el modificador static; hay exactamente una copia de una variable de clase, independientemente de cuantas veces la clase haya sido instanciada. Las variables locales almacenan el estado temporalmente dentro de un método. Los parámetros son variables que proveen de información extra a un método; tanto las variables locales como los parámetros son siempre clasificados como "variables" (no "campos"). Cuando nombra sus campos o variables, hay reglas y convenciones que debería (o tiene) que seguir.

Los ocho tipos de datos primitivos son :byte, short, int, long, float, double, boolean, y char. La clase java.lang.String representa cadenas de caracteres. El compilador asignará un valor por defecto razonable para los campos de los tipos de arriba; para las variables locales, nunca se asigna un valor por defecto. Un literal es la representación en el código fuente de un valor fijo. Una matriz es un objeto contenedor que contiene un número fijo de valores de un único tipo. La longitud de una matriz es establecida cuando la matriz es creada. Después de la creación, su longitud está fijada.
