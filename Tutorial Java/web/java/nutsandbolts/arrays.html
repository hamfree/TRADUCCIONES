Matrices

Una matriz es un objeto contenedor que tiene un número fijo de valores de un tipo único. La longitud de una matriz es establecida cuando la matriz es creada. Después de la creación, su longitud está fijada. Usted ha visto un ejemplo de matrices ya, en el método main de la aplicación "¡Hola Mundo!". Esta sección habla de las matrices en gran detalle.


Una matriz de 10 elementos.


Cada ítem en una matriz es llamado un elemento, y cada elemento es accedido por su índice numérico. Como se mostró en la ilustración precedente, la numeración empieza en 0. El noveno elemento, por ejemplo, sería de esta manera accedido en el índice 8.

El siguiente programa, ArrayDemo , crea un matriz de enteros, introduce algunos valores en la matriz, e imprime cada valor a la salida estándar. 


class ArrayDemo {
    public static void main(String[] args) {
        // declara una matriz de enteros
        int[] anArray;

        // Asigna memoria para 10 enteros
        anArray = new int[10];

        // inicializa el primer elemento
        anArray[0] = 100;
        // inicializa el segundo elemento
        anArray[1] = 200;
        // Etcétera
        anArray[2] = 300;
        anArray[3] = 400;
        anArray[4] = 500;
        anArray[5] = 600;
        anArray[6] = 700;
        anArray[7] = 800;
        anArray[8] = 900;
        anArray[9] = 1000;

        System.out.println("Elemento en el índice 0: "
                           + anArray[0]);
        System.out.println("Elemento en el índice 1: "
                           + anArray[1]);
        System.out.println("Elemento en el índice 2: "
                           + anArray[2]);
        System.out.println("Elemento en el índice 3: "
                           + anArray[3]);
        System.out.println("Elemento en el índice 4: "
                           + anArray[4]);
        System.out.println("Elemento en el índice 5: "
                           + anArray[5]);
        System.out.println("Elemento en el índice 6: "
                           + anArray[6]);
        System.out.println("Elemento en el índice 7: "
                           + anArray[7]);
        System.out.println("Elemento en el índice 8: "
                           + anArray[8]);
        System.out.println("Elemento en el índice 9: "
                           + anArray[9]);
    }
}

La salida de este programa es: 

Elemento en el índice 0: 100
Elemento en el índice 1: 200
Elemento en el índice 2: 300
Elemento en el índice 3: 400
Elemento en el índice 4: 500
Elemento en el índice 5: 600
Elemento en el índice 6: 700
Elemento en el índice 7: 800
Elemento en el índice 8: 900
Elemento en el índice 9: 1000

En una situación de programación del mundo real, usted usaría probablemente una de las construcciones de bucle soportadas para iterar a través de cada elemento de la matriz, en vez de escribir cada línea individualmente como en el ejemplo precedente. Sin embargo, el ejemplo ilustra claramente la sintaxis de las matrices. Aprenderá sobre las distintas construcciones de bucle (for, while, y do-while) en la sección Sentencias de Control de Flujo .

Declarar una Variable para hacer referencia a una Matriz

El programa anterior declara una matriz (nombrada anArray) con la siguiente línea de código: 

// declara una matriz de enteros
int[] anArray;

Como las declaraciones para variables de otros tipos, una declaración de matriz tiene dos componentes: el tipo de la matriz y el nombre de la matriz. Un tipo de la matriz es escrito como tipo[], donde tipo es el tipo de dato de los elmentos contenidos; los corchetes son símbolos especiales indicando que ésta variable contiene una matriz. El tamaño de la matriz no forma parte de su tipo (por lo cual es por lo que los corchetes están vacíos). Un nombre de matriz puede ser cualquiera que quiera, a condición de que sigue las reglas y convenciones discutidas anteriormente en la sección de nombrado . Como con las variables de otros tipos, la declaración no crea realmente un matriz; simplemente le dice al compilador que ésta variable contendrá una matriz del tipo especificado.

Similarmente, usted puede declarar matrices de otros tipos: 

byte[] anArrayOfBytes;
short[] anArrayOfShorts;
long[] anArrayOfLongs;
float[] anArrayOfFloats;
double[] anArrayOfDoubles;
boolean[] anArrayOfBooleans;
char[] anArrayOfChars;
String[] anArrayOfStrings;

Usted puede también colocar los corchetes después del nombre de la matriz: 

// esta forma no se recomienda
float anArrayOfFloats[];

Sin embargo, la convención no recomienda esta forma; los corchetes identifican el tipo matriz y deberían aparecer con la designación de tipo.

Creando, Inicializando, y Accediendo una Matriz

Una forma de crear una matriz es con el operador new. La siguiente sentencia en el programa ArrayDemo asigna una matriz con suficiente memoria para 10 elementos enteros y asigna la matriz a la variable anArray. 

// Asigna memoria para 10 enteros
anArray = new int[10];

Si esta declaración no se encuentra, entonces el compilador imprime un error como el siguiente, y la compilación falla: 

ArrayDemo.java:4: La variable anArray puede no haber sido inicializada.

Las pocas líneas siguientes asignan valores a cada elemento de la matriz: 

anArray[0] = 100; // inicializa el primer elemento
anArray[1] = 200; // inicializa el segundo elemento
anArray[2] = 300; // Etcétera

Cada elemento de la matriz es accedido por su índice numérico: 

System.out.println("Elemento 1 en el índice 0: " + anArray[0]);
System.out.println("Elemento 2 en el índice 1: " + anArray[1]);
System.out.println("Elemento 3 en el índice 2: " + anArray[2]);

Alternativamente, puede usar una sintaxis corta para crear e inicializar una matriz: 

int[] anArray = {
    100, 200, 300,
    400, 500, 600,
    700, 800, 900, 1000
};

Aquí la longitud de una matriz está determinada por el número de valores previstos entre los corchetes y separados por comas.

Puede también declarar una matriz de matrices (también conocida como una matriz multidimensional) utilizando dos o más conjuntos de corchetes, tal como String[][] names. Cada elemento, por lo tanto, tiene que ser accedido por el correspondiente número de valores de índice.

En el lenguaje de programación Java, una matriz multidimensional es una matriz cuyos componentes son en sí mismos matrices. Esto es a diferencia de las matrices en C o Fortan. Una consecuencia de esto es que las filas pueden variar en longitud, como se muestra en el siguiente MultiDimArrayDemo programa: 

class MultiDimArrayDemo {
    public static void main(String[] args) {
        String[][] names = {
            {"Mr. ", "Mrs. ", "Ms. "},
            {"Smith", "Jones"}
        };
        // Mr. Smith
        System.out.println(names[0][0] + names[1][0]);
        // Ms. Jones
        System.out.println(names[0][2] + names[1][1]);
    }
}

La salida de este programa es: 

Mr. Smith
Ms. Jones

Finalmente, puede usar la propiedad incorporada length para determinar el tamaño de una matriz. El siguiente código imprime el tamaño de la matriz a la salida estandar: 

 System.out.println(anArray.length);

Copiando Matrices

La clase System tiene un método arraycopy que puede usar para copiar datos eficientemente de una matriz a otra: 

public static void arraycopy(Object src, int srcPos,
                             Object dest, int destPos, int length)

Los dos argumentos Object especifican la matriz a copiar y la matriz a la que copiar . Los tres argumentos int especifican la posición de inicio en la matriz origen, la posición de inicio en la matriz destino, y el número de elementos de la matriz origen a copiar.

El siguiente programa, ArrayCopyDemo , declara una matriz de elementos char, deletreando la palabra "descafeinado." Utiliza el método System.arraycopy para copiar una subsecuencia de componentes de la matriz en una segunda matriz: 


class ArrayCopyDemo {
    public static void main(String[] args) {
        char[] copiarDesde = {'d', 'e', 's', 'c', 'a', 'f', 'e',
            'i', 'n', 'a', 'd', 'o'};
        char[] copiarA = new char[7];

        System.arraycopy(copiarDesde, 3, copiarA, 0, 7);
        System.out.println(new String(copiarA));
    }
}

La salida de este programa es: 

cafeina

Manipulaciones de Matrices

Las matrices son un poderoso y útil concepto utilizado en programación. Java SE contiene métodos para ejecutar algunas de las más comunes manipulaciones relacionadas con las matrices. Por ejemplo, el ejemplo ArrayCopyDemo usa el método arraycopy de la clase System en vez de iterar manualmente a través de los elementos de la matriz origen y colocar cada uno en la matriz destino. Esto es ejecutado entre bastidores, habilitando al desarrollador a usar tan sólo una línea de código para llamar al método.

Para su convenencia, Java SE ofrece varios métodos para ejecutar manipulaciones de matrices ( tareas comunes, tales como copiar, ordenar y buscar matrices) en la clase java.util.Arrays . Por ejemplo, el ejemplo previo puede ser modificado para usar el método copyOfRange de la clase java.util.Arrays , como puede ver en el ejemplo ArrayCopyOfDemo . La diferencia es que usando el método copyOfRange no precisa que tenga que crear la matriz de destino antes de llamar al método, porque la matriz de destino es devuelta por el método. 


class ArrayCopyOfDemo {
    public static void main(String[] args) {
        
        char[] copiarDesde = {'d', 'e', 's', 'c', 'a', 'f', 'e',
            'i', 'n', 'a', 'd', 'o'};
            
        char[] copiarA = java.util.Arrays.copyOfRange(copiarDesde, 3, 10);
        
        System.out.println(new String(copiarA));
    }
}

Como puede ver, la salida de éste programa es la misma (caffein), aunque requiere menos líneas de código. Note que el segundo parámetro del método copyOfRange es el índice inicial del rango a copiar, inclusive, mientras que el tercer parámetro es el índice final del rango a ser copiado, exclusive. En este ejemplo, el rango a ser copiado no incluye el elemento de la matriz en el índice 9 (el cual contiene el carácter a).

Algunas otras útiles operaciones ofrecidas por los métodos en la clase java.util.Arrays, son:

  • Buscar en una matriz por un valor específico para obtener el índice en el cual se encuentra (el método binarySearch).
  • Comparar dos matrices para determinar sin son iguales o no (el método equals).
  • Rellenar una matriz para colocar un valor específico en cada índice (el método fill).
  • Ordenar una matriz en orden ascendente. Esto puede ser hecho o secuencialmente, usando el método sort, o concurrentemente, utilizando el método parallelSort introducido en Java SE 8. La ordenación paralela de grandes matrices en sistemas multiprocesador es más rápida que la ordenación de matrices secuencial.
