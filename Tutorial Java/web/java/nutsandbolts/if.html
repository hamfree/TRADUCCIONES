Las Sentencias if-then y if-then-else

La Sentencia if-then

La sentencia if-then es la más básica de todas las sentencias de control de flujo. Le dice a su programa que ejecute una cierta sección de código sólo si una prueba particular evalúa a verdadero. Por ejemplo, la clase Bicicleta podría permitir a los frenos decrementar la velocidad de la bicicleta sólo si la bicicleta está aún en movimiento. Una posible implementación del método aplicaFrenos podría ser como sigue: 

void aplicaFrenos() {
    // la cláusula "if" : la bicicleta debe estar moviéndose
    if (estaMoviendose){
        // la cláusula "then" : decrementa la veocidad actual
        velocidadActual--;
    }
}

Si este test evalúa a falso (significando que la bicicleta no está en movimiento), el control salta al final de la sentencia if-then.

Además, las llaves de apertura y cierre son opcionales, siempre que la cláusula "then" contenga una sola sentencia: 

void aplicaFrenos() {
    // lo mismo que arriba, pero sin llaves
    if (estaMoviendose)
        velocidadActual--;
}

Decidir cuándo omitir las llaves es una cuestión de gusto personal. Omitirlas pude hacer el código más frágil. Si una segunda sentencia es añadida más tarde a la cláusula "then", un errore común sería olvidar añadir las llaves nuevamente requeridas. El compilador no puede capturar esta clase de error; va a tener los resultados incorrectos.

La Sentencia if-then-else

La sentencia if-then-else ofrece un camino secundario de ejecución cuando la cláusula "if" evalúa a false. Podría usar una sentencia if-then-else en el método aplicaFrenos para tomar alguna acción si los frenos son aplicados cuando la bicicleta no está en movimiento. En este caso, la acción es simplemente imprimir un mensaje de error indicando que la bicicleta ya se ha detenido- 

void aplicaFrenos() {
    if (estaMoviendose) {
        velocidadActual--;
    } else {
        System.err.println("¡La bicicleta ya está parada!");
    } 
}

El siguiente programa, IfElseDemo , asigna un grado basado en el valor de un resultado de prueba: una A para un resultado del 90% o superior, una B para un resultado del 80% o superior, y así. 


class IfElseDemo {
    public static void main(String[] args) {

        int resultadoTest = 76;
        char grado;

        if (resultadoTest >= 90) {
            grado = 'A';
        } else if (resultadoTest >= 80) {
            grado = 'B';
        } else if (resultadoTest >= 70) {
            grado = 'C';
        } else if (resultadoTest >= 60) {
            grado = 'D';
        } else {
            grado = 'F';
        }
        System.out.println("Grado = " + grado);
    }
}

La salida de este programa es: 

    Grado = C

Puede haber notado que el valor de resultadoTest puede satisfacer más de una expresión en la sentencia compuesta: 76 >= 70 y 76 >= 60. Sin embargo, una vez una condición es satisfecha, las sentencias apropiadas son ejecutadas (grado = 'C';) y las condiciones restantes no son evaluadas.
