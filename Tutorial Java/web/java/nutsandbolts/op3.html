Operadores de Nivel de Bit y de Desplazamiento de Bit

El lenguaje de programación Java también ofrece operadores que ejecutan operaciones bit a bit y de desplazamiento de bits sobre tipos integrales. Los operadores descritos en ésta sección son comunmente los menos usados. Por lo tanto, su cobertura es breve; la intención es simplemente hacerle saber que existen estos operadores.

El operador unario de complemento de bits "~" invierte un patrón de bits; puede ser aplicado a cualquiera de los tipos integrales, haciendo cada "0" un "1" y cada "1" un "0". Por ejemplo, un byte contiene 8 bits; aplicando este operador a un valor cuyo patrón de bits es "00000000" cambiaría su patrón a "11111111".

El operador de desplazamiento a la izquierda con signo "<<" desplaza un patrón de bits a la izquierda, y el operador de desplazamiento a la derecha con signo ">>" desplaza un patrón de bits a la derecha. El patrón de bits es dado por el operando a la izquierda, y el numero de posiciones a desplazar por el operando a la derecha. El operador de desplazamiento a la derecha sin signo ">>>" desplaza un cero en la posición más a la izquierda, mientras la posición más a la izquierda después de ">>" depende de la extensión de signo.

El operador bit a bit & ejecuta una operación AND bit a bit.

El operador bit a bit ^ ejecuta una operación OR exclusiva bit a bit.

El operador bit a bit | ejecuta una operación OR inclusiva bit a bit.

El siguiente programa, BitDemo , usa el operador bit a bit AND para imprimer el número "2" a la salida estándar. 


class BitDemo {

    public static void main(String[] args) {
        int mascarabits = 0x000F;
        int valor = 0x2222;
        // imprime "2"
        System.out.println(valor & mascarabits);
    }
}
