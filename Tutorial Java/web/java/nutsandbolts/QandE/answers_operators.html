Respuestas a las Preguntas y Ejercicios: Operadores

Respuestas a las Preguntas

  1. Considere el siguiente trozo de código: 
    arrayOfInts[j] > arrayOfInts[j+1]
    Pregunta: ¿Qué operadores contiene el código?
    Respuesta: >, +

  2. Considere el siguiente trozo de código: 
    int i = 10;
int n = i++%5;
    1. Pregunta: ¿Cuáles son los valores de i y n después de que el code sea ejecutado?
      Respuesta: i es 11, y n es 0.

    2. Pregunta: ¿Cuáles son los valores finales de i y n si en vez de usar el operador de incremento postfijo (i++), usted usa la versión prefija (++i))?
      Respuesta: i es 11, y n es 1.

  3. Pregunta: Para invertir el valor de un booleano, ¿Qué operador usaría?
    Respuesta: El operador de complemento lógico "!".
  4. Pregunta: ¿Qué operador es usado para comparar dos valores, = o == ?
    Respuesta: El operador == es usado para la comparación, y = es usado para la asignación.
  5. Pregunta: Explique el siguiente código de ejemplo: result = someCondition ? value1 : value2;
    Respuesta: Este código debe leerse como: "Si someCondition es verdad, asigne el valor de value1 a result. Si no, asigne el valor de value2 a result."

Ejercicios

  1. Cambie el siguiente programa para usar asignaciones compuestas: 
    class ArithmeticDemo {

    public static void main (String[] args){
          
        int result = 1 + 2; // resultado vale ahora 3
        System.out.println(result);

        result = result - 1; // resultado vale ahora 2
        System.out.println(result);

        result = result * 2; // resultado vale ahora 4
        System.out.println(result);

        result = result / 2; // resultado vale ahora 2
        System.out.println(result);

        result = result + 8; // resultado vale ahora 10
        result = result % 7; // resultado vale ahora 3
        System.out.println(result);

    }
}

    Aquí tiene una solución: 

    class ArithmeticDemo {

    public static void main (String[] args){
        int result = 3;
        System.out.println(result);

        result -= 1; // resultado vale ahora 2
        System.out.println(result);

        result *= 2; // resultado vale ahora 4
        System.out.println(result);

        result /= 2; // resultado vale ahora 2
        System.out.println(result);

        result += 8; // resultado vale ahora 10
        result %= 7; // resultado vale ahora 3
        System.out.println(result);

    }
}
  2. En el siguiente programa, explique porqué el valor "6" es impreso dos veces en una fila: 
    class PrePostDemo {
    public static void main(String[] args){
        int i = 3;
        i++;
        System.out.println(i);    // "4"
        ++i;                     
        System.out.println(i);    // "5"
        System.out.println(++i);  // "6"
        System.out.println(i++);  // "6"
        System.out.println(i);    // "7"
    }
}
    El código System.out.println(++i); evalúa a 6, porque la versión prefija de ++ evalúa al valor incrementado. La siguiente línea, System.out.println(i++); evalúa al valor actual (6). entonces incrementa en uno. Así "7" no es impreso hasta la siguiente línea.
