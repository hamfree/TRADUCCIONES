Preguntas y Ejercicios: Expresiones, Sentencias, y Bloques

Preguntas

  1. Los operadores pueden ser usados en la construcción ___, los cuales calculan valores.
  2. Las expresiones son los componentes principales de ___.
  3. Las declaraciones y sentencias pueden ser agrupadas en ___.
  4. El siguiente trozo de código es un ejemplo de una expresión ___. 
     1 * 2 * 3
  5. Las declaraciones son más o menos equivalentes a las oraciones en lenguas naturales, pero en vez de terminar con un punto, una declaración termina con una ___.
  6. Un bloque es un grupo de cero o más declaraciones entre ___ pareadas y puede ser usada en cualquier parte donde una declaración simple es permitida.

Ejercicios:

Identifique las siguientes clases de declaraciones de expresión:

  • unValor = 8933.234;
  • unValor++;
  • System.out.println("¡Hola Mundo!");
  • Bicicleta miBicicleta = new Bicicleta();

Compruebe sus respuestas.
