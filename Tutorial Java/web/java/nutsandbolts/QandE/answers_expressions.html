Respuestas a las Preguntas y Ejercicios: Expresiones, Sentencias, y Bloques

Preguntas

  1. Los operadores pueden ser usados en la construcción de expresiones, las cuales calculan valores.
  2. Las expresiones son el componente principal de las sentencias.
  3. Las sentencias pueden ser agrupadas en bloques.
  4. El siguiente trozo de código es un ejemplo de una expresión compuesta. 
     1 * 2 * 3
  5. Las sentencias son aproximadamente equivalentes a las oraciones en los lenguajes naturales, pero en vez de acabar con un punto, una sentecia termina con un punto y coma.
  6. Un bloque es un grupo de cero o más sentencias entre llaves pareadas y pueden ser usados en cualquier lugar donde se permita una sentencia única.

Ejercicios

Identifica las siguientes clases de sentencias de expresión:

  • unValor = 8933.234; // sentencia de asignación
  • unValor++; // sentencia de incremento
  • System.out.println("¡Hola Mundo!"); // sentencia de invocación a método
  • Bicicleta miBicicleta = new Bicicleta(); // sentencia de creación de objeto
