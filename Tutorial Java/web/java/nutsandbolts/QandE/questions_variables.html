Preguntas y Ejercicios: Variables

Preguntas

  1. El término "variable de instancia" es otro nombre para ___.
  2. El término "variable de clase" es otro nombre para ___.
  3. Una variable local almacena un estado temporalmente; es declarada dentro de un ___.
  4. Una variable declarada entre paréntesis abiertos y cerrados de una firma de método es llamada un ____.
  5. ¿Cuáles son los ocho tipos de datos primitivos soportados por el lenguaje de programación Java?
  6. Las cadenas de caracteres están representadas por la clase ___.
  7. Una ___ es un objeto contenedor que contiene un número fijo de valores de un tipo único.

Ejercicios

  1. Cree un pequeño programa que defina algunos campos. Intente crear algunos nombres ilegales de campos y vea que clase de error produce el compilador. Utilize las reglas de nombrado y convenciones como guía.
  2. En el programa que usted creó en el Ejercicio 1, intente dejar los campos sin inicalizar e imprima sus valores. Intente lo mismo con una variable local y vea que clase de errores del compilador puede producir. Familiarizarse con errores de compilación comunes hará que sea más fácil reconocer los errores en el código.
