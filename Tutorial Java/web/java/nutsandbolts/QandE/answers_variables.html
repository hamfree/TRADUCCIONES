Respuestas a Preguntas y Ejercicios: Variables

Respuestas a las Preguntas

  1. El término "variable de instancia" es otro nombre para campo no estático.
  2. El término "variable de clase" es otro nombre para campo estático.
  3. Una variable local almacena un estado temporalmente; está declarada dentro de un método.
  4. Una variable declarada entre paréntesis abiertos y cerrados de una firma de método es llamada un parámetro.
  5. ¿Cuáles son los ocho tipos de datos primitivos soportados por el lenguaje de programación Java? byte, short, int, long, float, double, boolean, char
  6. Las cadenas de caracteres están representadas por la clase java.lang.String.
  7. Una matriz es un objeto contenedor que contiene un número fijo de valores de un tipo único.

Respuestas a los ejercicios

  1. Cree un pequeño programa que defina algunos campos. Intente crear algunos nombres ilegales de campos y vea que clase de error produce el compilador. Utilize las reglas de nombrado y convenciones como guía.

    No hay una respuesta única correcta aquí. Sus resultados variarán dependiendo de su código.

  2. En el programa que usted creó en el Ejercicio 1, intente dejar los campos sin inicalizar e imprima sus valores. Intente lo mismo con una variable local y vea que clase de errores del compilador puede producir. Familiarizarse con errores de compilación comunes hará que sea más fácil reconocer los errores en el código.

    De nuevo, no hay una respuesta única correcta para este ejercicio. Sus resultados variarán dependiendo de su código.
