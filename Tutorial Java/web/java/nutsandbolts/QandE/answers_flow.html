Respuestas a las Preguntas y Ejercicios: Sentencias de Control de Flujo

Respuestas a las Preguntas

  1. La sentencia de control más básica soportada por el lenguaje de programación Java es la sentencia if-then.
  2. La sentencia switch permite cualquier número de rutas posibles de ejecución.
  3. La sentencia do-while es similar a la sentencia while, pero evalúa su expresión al final del bucle.
  4. Pregunta: ¿Cómo escribe un bucle infinito usando la sentencia for?

    Respuesta: 

    for ( ; ; ) {

}
  5. Pregunta: ¿Cómo escribe un bucle infinito usando la sentencia while?

    Respuesta: 

    while (true) {

}

Ejercicios

  1. Considere el siguiente trozo de código. 
    if (unNumero >= 0)
    if (unNumero == 0)
        System.out.println("primera cadena");
else System.out.println("segunda cadena");
System.out.println("tercera cadena");
    1. Ejercicio: ¿Qué salida piensa usted que el código producirá si unNumero es 3?

      Solución: 

      segunda cadena
tercera cadena
    2. Ejercicio: Escriba un programa de prueba conteniendo el trozo de código anterior; haga que unNumero valga 3. ¿Cuál es la salida del programa? ¿Es lo que usted predijo? Explique porqué la salida es la que es; en otras palabras, ¿Cuál es el flujo de control para el trozo de código?

      Solución: NestedIf 

      segunda cadena
tercera cadena

      3 es mayor o igual que 0, así la ejecución progresa a la segunda sentencia if. El test de la segunda sentencia if falla porque 3 no es igual a 0. Así, la cláusula else ejecuta (desde que está enlazada a la segunda sentencia if). Así, segunda cadena es mostrada. El println final está completamente fuera de cualquier sentencia if, de froma que siempre será ejecutada, y así tercera cadena se muestra siempre.

    3. Ejercicio: Usando solo espacios y saltos de línea, reformatee el trozo de código para hacer el control de flujo más fácil de entender.

      Solución: 

      if (unNumero >= 0)
    if (unNumero == 0)
        System.out.println("primera cadena");
    else
        System.out.println("segunda cadena");

System.out.println("tercera cadena");
    4. Ejercicio: Use llaves { y } para aclarar aún más el código y reducir la posibilidad de errores por mantenedores futuros del código.

      Solución: 

      if (unNumero >= 0) {
    if (unNumero == 0) {
        System.out.println("primera cadena");
    } else {
        System.out.println("segunda cadena");
    }
}

System.out.println("tercera cadena");
