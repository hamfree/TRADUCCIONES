Preguntas y Ejercicios: Sentencias de Control de Flujo

Preguntas

  1. La sentencia de control de flujo más básica soportada por el lenguaje de programación Java es la sentencia ___.
  2. La sentencia ___ permite cualquier número de rutas posibles de ejecución.
  3. La sentencia ___ es similar a la sentencia while, pero evalúa su expresión al ___ del bucle.
  4. ¿Cómo escribe usted un bucle infinito usando la sentencia for?
  5. ¿Cómo escribe usted un bucle infinito usando la sentencia while?

Ejercicios

  1. Consideres el siguiente trozo de código. 

    if (unNumero >= 0)
    if (unNumero == 0)
        System.out.println("primera cadena");
else System.out.println("segunda cadena");
System.out.println("tercera cadena");
    1. ¿Qué salida piensa que el código producirá si unNumero es 3?
    2. Escriba un programa de prueba conteniendo el trozo de código anterior; haga que unNumero valga 3. ¿Cuál es la salida del programa? ¿Es lo que usted predijo? Explique porqué la salida es la que es; en otras palabras, ¿Cuál es el flujo de control para el trozo de código?
    3. Usando solo espacios y saltos de línea, reformatee el trozo de código para hacer el control de flujo más fácil de entender.
    4. Use llaves, { y }, para aclarar aún más el código.
