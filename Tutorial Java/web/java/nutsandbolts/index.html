Lección: Lo Básico del Lenguaje

Variables

Usted ya ha aprendido que los objetos almacenan su estado en campos. Sin embargo, el lenguaje de programación Java también usa el término "variable". Esta sección analiza esta relación, más las reglas de nombrado de variables y las convenciones, tipos de datos básicos (tipos primitivos, cadenas de caracteres, y vectores), valores por defecto, y literales.

Operadores

Esta sección describe los operadores del lenguaje de programación Java. Presenta los operadores más comunmente usados primero, y los menos comunmente usados al final. Cada discusión incluye ejemplos de código que usted puede compilar y ejecutar.

Expresiones, Sentencias, y Bloques

Los operadores pueden ser usados en la construcción de expresiones, las cuales calculan valores; las expresiones son los componentes principales de las sentencias; las sentencias pueden ser agrupadas en bloques. Esta sección analiza las expresiones, sentencias, y bloques usando ejemplos de código que usted ya ha visto.

Sentencias de Control de FLujo

Esta sección describe las sentencias de control de flujo soportadas por el lenguaje de programación Java. Cubre la toma de decisiones, los bucles, y las sentencias de bifurcación que habilita a sus programas a ejecutar condicionalmente bloques particulares de código.
