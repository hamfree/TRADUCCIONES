Sumario de Sentencias de Control de Flujos

La sentencia if-then es la más básica de todas las sentencias de control de flujo. Le indica a su programa que ejecute un cierta sección del codigo solo si un test particular evalua a verdadero. La sentencia if-then-else ofrece una ruta de ejecución secundaria cuando una cláusula "if" evalúa a falso. A diferencia de if-then y if-then-else, la sentencia switch permite cualquier número de posibles rutas de ejecución. Las sentencias while y do-while ejecutan continuamente un bloque de sentencias mientras una condición particular sea verdadera. La diferencia entre do-while y while es que do-while evalúa la expresión al final del bucle en vez de al principio. Por lo tanto, las sentencias dentro de un bloque do son ejecutadas siempre al menos una vez. La sentencia for ofrece un forma compacta de iterar sobre un rango de valores. Tiene dos formas, una de las cuales fue diseñada para iterar a través de colecciones y matrices.
