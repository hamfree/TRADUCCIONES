Sentencias de Control de Flujo

Las sentencias dentro de sus ficheros fuente son generalmente ejecutadas de arriba a abajo, en el orden que éstas aparecen. Las sentencias de control de flujo, sin embargo, desarticulan el flujo de ejecución empleando la toma de decisiones, los bucles, y la bifurcación, habilitando a su programa a condicionalmente ejecutar bloques particulares de código. Esta sección describe las sentencias de toma de decisión (if-then, if-then-else, switch), las sentencias de bucle o repetición (for, while, do-while), y las sentencias de bifurcación (break, continue, return) soportadas por el lenguaje de programación Java.
