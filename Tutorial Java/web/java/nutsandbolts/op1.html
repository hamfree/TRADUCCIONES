Operadores de Asignación, Aritméticos, y Unarios

El Operador de Asignación Simple

Uno de los más comunes operadores que usted encontrará es el operador de asignación simple "=". Usted vió éste operador en la clase Bicicleta; asigna el valor de su derecha al operando a su izquierda: 

    int cadencia = 0;
    int velocidad = 0;
    int marcha = 1;

Este operador puede también ser utilizado sobre objetos para asignarles referencias a objeto, como es descrito en Creando Objetos.

Los Operadores Aritméticos

El lenguaje de programación Java ofrece operadores que ejecutan sumas, restas, multiplicaciones, y divisiones. Hay una buena probabilidad de que usted los reconozca por sus homólogos en las matemáticas básicas. El único símbolo que podría parecerle nuevo es "%", el cual divide un operando por otro y devuelve el resto como su resultado.

Operadores Aritméticos
Operador Descripción
+ Operador de suma (también usado para concatenación de cadenas)
- Operador de resta
* Operador de multiplicación
/ Operador de división
% Operador de resto

El siguiente programa, AritmeticaDemo, prueba los operadores aritméticos. 


class AritmeticaDemo {

    public static void main (String[] args) {

        int resultado = 1 + 2;
        // resultado es ahora 3
        System.out.println("1 + 2 = " + resultado);
        int resultado_original = resultado;

        resultado = resultado - 1;
        // resultado es ahora 2
        System.out.println(resultado_original + " - 1 = " + resultado);
        resultado_original = resultado;

        resultado = resultado * 2;
        // resultado es ahora 4
        System.out.println(resultado_original + " * 2 = " + resultado);
        resultado_original = resultado;

        resultado = resultado / 2;
        // resultado es ahora 2
        System.out.println(resultado_original + " / 2 = " + resultado);
        resultado_original = resultado;

        resultado = resultado + 8;
        // resultado is now 10
        System.out.println(resultado_original + " + 8 = " + resultado);
        resultado_original = resultado;

        resultado = resultado % 7;
        // result is now 3
        System.out.println(resultado_original + " % 7 = " + resultado);
    }
}

Este programa imprime lo siguiente: 

1 + 2 = 3
3 - 1 = 2
2 * 2 = 4
4 / 2 = 2
2 + 8 = 10
10 % 7 = 3

Puede también combinar los operadores aritméticos con el operador de asignación simple para crear asignaciones compuestas. Por ejemplo, x+=1; y x=x+1; ambos incrementan el valor de x en 1.

El operador + puede también ser usado para concatenar (unir) dos cadenas juntas, como se muestra en el siguiente programa: ConcatDemo 


class ConcatDemo {

    public static void main(String[] args) {
        String primeraCadena = "Esta es";
        String segundaCadena = " una cadena concatenada.";
        String terceraCadena = primeraCadena + segundaCadena;
        System.out.println(terceraCadena);
    }
}

En el final de este programa, la variable terceraCadena contiene "Esta es una cadena concatenada.", la cual es impresa en la salida estándar.

Los Operadores Unarios

Los operadores unarios sólo requieren un operando; ejecutan varias operaciones tales como incrementar/decrementar un valor por uno, negar una expresión, o invertir el valor de un booleano.

Operadores unarios
Operador Descripción
+ Operador unario más; indica valores positivos (los números son positivos sin éste, sin embargo)
- Operador unario menos; niega una expresión
++ Operador incremento; incrementa un valor en 1
-- Operador decremento; decrementa un valor en 1
! Operador de complemento lógico; invierte el valor de un booleano

El siguiente programa, UnaryDemo , comprueba los operadores unarios: 


class UnaryDemo {

    public static void main(String[] args) {

        int resultado = +1;
        // resultado es ahora 1
        System.out.println(resultado);

        resultado--;
        // resultado es ahora 0
        System.out.println(resultado);

        resultado++;
        // resultado es ahora 1
        System.out.println(resultado);

        resultado = -resultado;
        // resultado es ahora -1
        System.out.println(resultado);

        boolean acierto = false;
        // falso
        System.out.println(acierto);
        // verdadero
        System.out.println(!acierto);
    }
}

Los operadores de incremento/decremento pueden ser aplicados antes (prefija) o después (postfija) del operando. El código resultado++; y ++resultado; terminarán incrementando resultado en uno. La única diferencia es que la versión prefija (++resultado) evalúa al valor incrementado, mientras que la versión postfija (resultado++) evalúa al valor original. Si se acaba realizando un simple incremento/decremento, en realidad no importa la versión que elija. Pero si usa éstos operadores como parte de una expresión más larga, el que usted elija uno u otro puede hacer una diferencia significativa.

El siguiente programa, PrePostDemo , ilustra el operador unario de incremento prefijo/postfijo: 


class PrePostDemo {
    public static void main(String[] args){
        int i = 3;
        i++;
        // imprime 4
        System.out.println(i);
        ++i;
        // imprime 5
        System.out.println(i);
        // imprime 6
        System.out.println(++i);
        // imprime 6
        System.out.println(i++);
        // imprime 7
        System.out.println(i);
    }
}
