Palabras Clave del Lenguaje Java

Aquí tiene una lista de palabras clave en el lenguaje de programación Java. No puede usar ninguna de las siguientes como identificadores en sus programas. Las palabras claves const y goto están reservadas, a pesar de que éstas no son empleadas actualmente. true, false, y null podrían parecer palabras clave, pero en realidad son literales; que no puede usar como identificadores en sus programas.

abstract continue for new switch
assert*** default goto* package synchronized
boolean do if private this
break double implements protected throw
byte else import public throws
case enum**** instanceof return transient
catch extends int short try
char final interface static void
class finally long strictfp** volatile
const* float native super while
*   no usado
**   añadido en 1.2
***   añadido en 1.4
****   añadido en 5.0
