Las Sentencias while y do-while

La sentencia while ejecuta continuamente un bloque de sentencias mientras una condición particular sea verdadera. Su sintaxis puede ser expresada como: 

while (expresion) {
     sentencia(s)
}

La sentencia while evalúa la expresión, la cual debe devolver un valor booleano. Si la expresión evalúa a verdadero, la sentencia while ejecuta las sentencia(s) en el bloque while. La sentencia while continúa evaluando la expresión y ejecutando su bloque hasta que la expresión evalúa a falso. Usando la sentencia while para imprimir los valores de 1 a 10 puede llevarse a cabo como en el siguiente programa: WhileDemo : 


class WhileDemo {
    public static void main(String[] args){
        int contador = 1;
        while (contador < 11) {
            System.out.println("Contador es: " + contador);
            contador++;
        }
    }
}

Puede implementar un bucle infinito usando la sentencia while como sigue: 

while (true){
    // su código va aquí
}

El lenguaje de programación Java también ofrece una sentencia do-while, la cual puede ser expresada como sigue: 

do {
     sentencia(s)
} while (expresión);

La diferencia entre do-while y while es que do-while evalúa su expresión al final del bucle en vez de al principio. Por lo tanto, las sentencias dentro del bloque do son siempre ejecutadas al menos una vez, como se muestra en el siguiente programa: DoWhileDemo : 


class DoWhileDemo {
    public static void main(String[] args){
        int contador = 1;
        do {
            System.out.println("Contador es : " + contador);
            contador++;
        } while (contador < 11);
    }
}
