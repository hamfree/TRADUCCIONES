Respuestas a Preguntas y Ejercicios: Tipos Enum

Preguntas

  1. Pregunta: Verdadero o falso: un tipo Enum puede ser una subclase de java.lang.String.

    Respuesta: Falso. Todas las enumeraciones implícitamente extienden java.lang.Enum. Debido a que una clase solo puede extender un padre, el lenguaje de programación Java no soporta herencia múltiple de estado, y así una enumeración no puede extender nada más.

Ejercicios

  1. Ejercicio: Reescriba la clase Carta del ejercicio en Preguntas y Ejercicios: Clases de forma que representa el rango y palo de la carta con tipos enumerados.

    Respuesta: Vea Carta3.java , Palo.java , y Rango.java.

  2. Ejercicio: Reescriba la clase Baraja.

    Respuesta: Vea Baraja3.java .
