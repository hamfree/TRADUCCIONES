Preguntas y Ejercicios: Clases Anidadas

Preguntas

  1. El programa Problema.java no compila. ¿Qué necesita hacer para hacerlo compilar? ¿Porqué?

  2. Use la documentación del IPA de Java para la clase Box (en el paquete javax.swing) para ayudarle a responder a las siguientes preguntas.

    1. ¿Qué clase anidada estática define Box?

    2. ¿Qué clase interior define Box?

    3. ¿Cuál es la superclase de la clase interior de Box?

    4. ¿Cuál de las clases anidadas de Box puede usted usar desde cualquier clase?

    5. ¿Cómo crea una instancia de la clase Filler de Box?

Ejercicios

  1. Obtenga el fichero Clase1.java . Compile y ejecute Clase1. ¿Cuál es la salida?

  2. Los siguientes ejercicios implican modificar la clase EstructuraDatos.java , la cual la sección Ejemplo de Clase Interior describe.

    1. Defina un método nombrado print(EstructuraDatosIterador iterator). Invoque este método con una instancia de la clase ParIterador de forma que ejecute la misma función que el método imprimePares.

    2. Invoque el método print(EstructuraDatosIterador iterator) de forma que imprima elementos que tengan un valor de índice par. Use una clase anónima como el argumento del método en vez de una instancia del interfaz EstructuraDatosIterador.

    3. Defina un método nombrado print(java.util.Function<Integer, Boolean> iterator) que ejecute la misma función que print(EstructuraDatosIterador iterador). Invoque este método con una expresión lambda para imprimir elementos que tengan un valor de índice impar. Invoque este método de nuevo con una expresión lambda para imprimir elementos que tengan un valor de índice par.

    4. Defina dos métodos de forma que las siguientes dos sentencias impriman elementos que tengan un valor de índice impar y elementos que tengan un valor de índice par: 

      EstructuraDatos ds = new EstructuraDatos()
// ...
ds.print(EstructuraDatos::esIndicePar);
ds.print(EstructuraDatos::esIndiceImpar);

Compruebe sus respuestas.
