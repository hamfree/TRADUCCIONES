Respuestas a Preguntas y Ejercicios: Objetos

Preguntas

  1. Pregunta: ¿Qué está mal en el siguiente programa? 

    public class AlgoEstaMal {
    public static void main(String[] args) {
        Rectangulo miRect;
        miRect.anchura = 40;
        miRect.altura = 50;
        System.out.println("El área de miRect's es " + miRect.area());
    }
}

    Respuesta: El código nunca crea un objeto Rectangulo . Con este programa simple, el compilador genera un error. Sin embargo, en una situación más realística, mRect podría ser inicializada a null en un lugar, digamos en un constructor, y usado más tarde. En ese caso, el programa compilará bien, pero generará una NullPointerException en ejecución.

  2. Pregunta: El siguiente código crea una matriz y un objeto string. ¿Cuantas referencias a esos objetos existen depués de que el código se ejecute? ¿Es cualquiera de los objetos apto para el reciclaje? 

    ...
String[] estudiantes = new String[10];
String nombreEstudiante = "Juan López";
estudiantes[0] = nombreEstudiante;
nombreEstudiante = null;
...

    Respuesta: Hay una referencia a la matriz estudiantes y esa matriz tiene una referencia a la cadena Juan López. Ninguno de los objetos es elegible para el reciclaje. La matriz estudiantes no es elegible para el reciclaje porque tiene una referencia al objeto nombreEstudiante a pesar de que ese objeto se le ha asignado el valor null. El objeto ombreEstudiante no es apto tampoco porque estudiantes[0] aún se refiere a él.

  3. Pregunta: ¿Cómo un programa destruye un objeto que crea?

    Respuesta: Un programa no destruye explícitamente objetos. Un pograma puede establecer todas las referencia a un objeto a null de forma que se conviertan en aptos para el reciclaje. Pero el programa en realidad no destruye los objetos.

Ejercicios

  1. Ejercicio: Arregla el programa llamado AlgoEstaMal mostrado en la Pregunta 1.

    Respuesta: Vea AlgoEstaBien : 

    
public class AlgoEstaBien {
    public static void main(String[] args) {
        Rectangulo miRect = new Rectangulo();
        miRect.anchura = 40;
        miRect.altura = 50;
        System.out.println("El área de miRect es " + miRect.area());
    }
}

  2. Ejercicio: Dada la siguiente clase, llamada PoseedorNumero , escriba algo de código que cree una instancia de la clase, inicialzie sus dos variables miembro, y entonces muestre el valor de cada variable miembro. 

    
public class PoseedorNumero {
    public int unEntero;
    public float unFlotante;
}

    Respuesta: Vea VisualizaPoseedorNumero : 

    
public class VisualizaPoseedorNumero {

    public static void main(String[] args) {
        PoseedorNumero unPoseedorDeNumeros = new PoseedorNumero();
        unPoseedorDeNumeros.unEntero = 1;
        unPoseedorDeNumeros.unFlotante = 2.3f;
        System.out.println(unPoseedorDeNumeros.unEntero);
        System.out.println(unPoseedorDeNumeros.unFlotante);
    }
}
