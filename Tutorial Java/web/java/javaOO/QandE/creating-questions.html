Preguntas y Ejercicios: Clases

Preguntas

  1. Considere la siguiente clase: 

    public class IdentificaMisPartes {
    public static int x = 7; 
    public int y = 3; 
}

    1. ¿Cuáles son las variables de clase?

    2. ¿Cuáles son las variables de instancia?

    3. Cuál es la salida del siguiente código: 

      IdentificaMisPartes a = new IdentificaMisPartes();
IdentificaMisPartes b = new IdentificaMisPartes();
a.y = 5;
b.y = 6;
a.x = 1;
b.x = 2;
System.out.println("a.y = " + a.y);
System.out.println("b.y = " + b.y);
System.out.println("a.x = " + a.x);
System.out.println("b.x = " + b.x);
System.out.println("IdentificaMisPartes.x = " + IdentificaMisPartes.x);

Ejercicios

  1. Escriba una clase cuyas instancias representen una carta única de juego de una baraja de cartas. Las cartas de juego tienen dos propiedades distintivas: color y palo. Asegúrese de mantener su solución ya que se le pedirá reescribirla en Tipos Enum.

    Consejo: 

    Puede usar la sentencia assert para comprobar sus asignaciones. Escriba: 

    assert (expresión booleana a comprobar);

    Si la expresión boolean es falsa, obtendrá un mensaje de error. Por ejemplo, 

    assert toString(ACE) == "Ace";

    devolverá verdadero, así que no habrá ningún mensaje de error.

    Si usa la sentencia assert , debe ejecutar su programa con la bandera ea flag: 

    java -ea SuPrograma.class

  2. Escriba una clase cuyas instancias representen una baraja completa de cartas. Deberá guardar también esta solución.

  3. Escriba un pequeño programa para comprobar sus clases de baraja y cartas. El programa puede ser tan simple como crear una baraja de cartas y visualizar sus cartas.

Compruebe sus respuestas.
