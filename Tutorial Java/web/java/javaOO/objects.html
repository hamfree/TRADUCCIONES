Objetos

Un programa típico de Java crea muchos objetos, los cuales como usted sabe, interactúan invocando métodos. A través de estas interacciones de los objetos, un programa puede llevar a cabo varias tareas, tales como implementar un IGU, ejecutar una animación, o enviar y recibir información en una red. Una vez un objeto ha completado el trabajo para el cual fue creado, sus recursos son reciclados para su uso por otros objetos.

Aquí tiene un pequeño programa, llamado CreateObjectDemo , que crea tres objetos: un objeto Point y dos objetos Rectangle . Necesitará los tres ficheros fuente para compilar este programa. 


public class CreateObjectDemo {

    public static void main(String[] args) {

        // Declra y crea un objeto point y dos objetos rectángulo.
        Point origenUno = new Point(23, 94);
        Rectangle rectUno = new Rectangle(origenUno, 100, 200);
        Rectangle rectDos = new Rectangle(50, 100);

        // Visualiza el ancho, alto. y el área de rectUno
        System.out.println("Ancho de rectUno: " + rectUno.ancho);
        System.out.println("Altura de rectUno: " + rectUno.altura);
        System.out.println("Area de rectUno: " + rectUno.area());

        // Establece la posición de rectDos
        rectDos.origen = origenUno;

        // Visualiza la posición de rectDos
        System.out.println("Posición X de rectDos: " + rectDos.origen.x);
        System.out.println("Posición Y de rectDos: " + rectDos.origen.y);

        // Mueve rectDos y visualiza su nueva posición
        rectDos.move(40, 72);
        System.out.println("Posición X de rectDos: " + rectDos.origen.x);
        System.out.println("Posición Y de rectDos: " + rectDos.origen.y);
    }
}

Este programa crea, manipula, y visualiza información sobre varios objetos. Aquí tiene la salida: 

Ancho de rectUno: 100
Altura de rectUno: 200
Area de rectUno: 20000
Posición X de rectDos: 23
Posición Y de rectDos: 94
Posición X de rectDos: 40
Posición Y de rectDos: 72

Las siguientes tres secciones usan el ejemplo de arriba para describir el ciclo de vida de un objeto dentro del programa. Desde ellos, aprenderá cómo escribir código que crea y usa objetos en sus propios programas. También aprenderá cómo el sistema limpia después un objeto cuando su vida ha terminado.
