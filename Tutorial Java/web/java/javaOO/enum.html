Tipos Enum

Un tipo enum es un tipo especial de dato que habilita que una variable sea un conjunto de constantes predefinidos. La variable debe ser igual a uno de los valores que han sido predefinidas para ello. Ejemplos comunes incluyen las direcciones de la brújula (valores de NORTE, SUR, ESTE, y OESTE) y los días de la semana.

Debido a que son constantes, los nombres de los campos de un tipo enum están en letras mayúsculas.

En el lenguaje de programación Java, define un tipo enum usando la palabra clave enum. Por ejemplo, especificaría un tipo enum de días-de-la-semana como: 


public enum Dia {
    DOMINGO, LUNES, MARTES, MIERCOLES,
    JUEVES, VIERNES, SABADO
}

Debe utilizar los tipos de enumeración cada vez que necesite representar un conjunto fijo de constantes. Eso incluye los tipos enum naturales como los planetas en nuestro sistema solar y conjuntos de datos de los que usted sabe todos los posibles valores en tiempo de compilación — por ejemplo, las elecciones de un menú, las opciones de línea de comando, y así.

Aquí tiene un código que muestra como usar la enumeración Dia definida arriba: 


public class PruebaEnum {
    Dia dia;
    
    public PruebaEnum(Dia dia) {
        this.dia = dia;
    }
    
    public void digaloComoEs() {
        switch (dia) {
            case LUNES:
                System.out.println("Los lunes son malos.");
                break;
                    
            case VIERNES:
                System.out.println("Los viernes son mejores.");
                break;
                         
            case SABADO: case DOMINGO:
                System.out.println("Los fines de semana son lo mejor.");
                break;
                        
            default:
                System.out.println("Los días de entre semana son así-así.");
                break;
        }
    }
    
    public static void main(String[] args) {
        PruebaEnum primerDia = new PruebaEnum(Dia.LUNES);
        primerDia.digaloComoEs();
        PruebaEnum tercerDia = new PruebaEnum(Dia.MIERCOLES);
        tercerDia.digaloComoEs();
        PruebaEnum quintoDia = new PruebaEnum(Dia.VIERNES);
        quintoDia.digaloComoEs();
        PruebaEnum sextoDia = new PruebaEnum(Dia.SABADO);
        sextoDia.digaloComoEs();
        PruebaEnum septimoDia = new PruebaEnum(Dia.DOMINGO);
        septimoDia.digaloComoEs();
    }
}

La salida es: 

Los lunes son malos.
Los días de entre semana son así-así.
Los viernes son mejores.
Los fines de semana son los mejores.
Los fines de semana son los mejores.

Los tipos enumerados del lenguaje de programación Java son mucho más poderosos que sus contrapartes en otros lenguajes. La declaración enum define una clase (llamada un tipo enum). El cuerpo de la clase enum puede incluir métodos y otros campos. El compilador automáticamente añade algunos métodos especiales cuando crea una enumeración. Por ejemplo, tienen un método estático values que devuelve una matriz conteniendo todos los valores de la enumeración en el orden que fueron declarados. Este método es comunmente usado en combinación con la construcción for-each para iterar sobre los valores de un tipo enumerado. Por ejemplo, este código de la clase Planeta de abajo itera sobre todos los planetas en el sistema solar. 

for (Planeta p : Planeta.values()) {
    System.out.printf("Su peso en %s es %f%n",
                      p, p.pesoEnSuperficie(masa));
}
Nota: Todas las enumeraciones implícitamente extienden java.lang.Enum. Debido a que una clase sólo puede extender de un padre (vea Declarando Clases ), el lenguaje Java no soporta herencia múltiple del estado (vea Herencia Múltiple de Estado, Implementación, y Tipo ), y así una enumeración no puede extender nada más.

En el siguiente ejemplo, Planeta es un tipo enum que representa los planetas en el sistema solar. Están definidos con propiedades como la constante de masa y el radio.

Cada constante enum está declarada con valores para los parámetros de masa y radio. Estos valores son pasados al constructor cuando la constante es creada. Java requiere que las constantes sean definidas primero, antes que cualquier campo o método. También, cuando hay campos y métodos, la lista de constantes enum debe termina con un punto y coma.

Nota: El constructor para un tipo enum debe ser de acceso privado-para-el-paqute o privado. Automáticamente crea las constantes que están definidas al principio del cuerpo de la enumeración. No puede invocar a un constructor de enumeración directamente.

Además de sus propiedades y constructor, Planeta tiene métodos que le permiten recuperar la gravedad en superficie y peso de un objeto en cada planeta. Aquí hay un programa de ejemplo que toma su peso en la Tierra (en cualquier unidad) y calcula e imprime su peso en todos los planetas (en la misma unidad): 


public enum Planeta {
    MERCURIO (3.303e+23, 2.4397e6),
    VENUS    (4.869e+24, 6.0518e6),
    TIERRA   (5.976e+24, 6.37814e6),
    MARTE    (6.421e+23, 3.3972e6),
    JUPITER  (1.9e+27,   7.1492e7),
    SATURNO  (5.688e+26, 6.0268e7),
    URANO    (8.686e+25, 2.5559e7),
    NEPTUNO  (1.024e+26, 2.4746e7);

    private final double masa;   // en kilogramos
    private final double radio; // en metros
    Planeta(double masa, double radio) {
        this.masa = masa;
        this.radio = radio;
    }
    private double masa() { return masa; }
    private double radio() { return radio; }

    // constante de gravitación universal (m3 kg-1 s-2)
    public static final double G = 6.67300E-11;

    double gravedadEnSuperficie() {
        return G * masa / (radio * radio);
    }
    double pesoEnSuperficie(double otraMasa) {
        return otraMasa * gravedadEnSuperficie();
    }
    public static void main(String[] args) {
        if (args.length != 1) {
            System.err.println("Uso: java Planeta <peso_en_la_tierra>");
            System.exit(-1);
        }
        double pesoEnLaTierra = Double.parseDouble(args[0]);
        double masa = pesoEnLaTierra/TIERRA.gravedadEnSuperficie();
        for (Planeta p : Planeta.values())
           System.out.printf("Su peso en %s es %f%n",
                             p, p.pesoEnSuperficie(masa));
    }
}

Si ejecuta Planeta.class desde la línea de comandos con un argumento de 175, obtendrá esta salida: 

$ java Planet 175
Su peso en MERCURIO is 66.107583
Su peso en VENUS is 158.374842
Su peso en TIERRA is 175.000000
Su peso en MARTE is 66.279007
Su peso en JUPITER is 442.847567
Su peso en SATURNO is 186.552719
Su peso en URANOS is 158.397260
Su peso en NEPTUNO is 199.207413
