Lección: Clases y Objetos

Con el conocimiento que ahora tiene de los fundamentos del lenguaje de programación Java, puede aprender a escribir sus propias clases. En esta lección, encontrará información sobre la definición de sus propias clases, incluyendo la declaración de variables miembro, métodos, y constructores.

Aprenderá a usar sus clases para crear objetos, y cómo usar objetos que crea.

Esta lección también cubre las clases anidadas dentro de otras clases, y las enumeraciones.

Clases

Esta sección le muestra la anatomía de una clase, y cómo declarar campos, métodos, y constructores..

Objetos

Esta sección cubre la creación y uso de objetos. Aprenderá cómo instanciar un objeto, y, una vez instanciado, cómo usar el operador dot para acceder a las variables de instancia del objeto y métodos.

Más sobre Clases

Esta sección cubre más aspectos de las clases que dependen de usar referencias a objeto y el operador dot sobre el que usted aprendió en la sección precedente: retornar valores de los métodos, la palabra clave this , miembros de instancia vs. miembros de clase, y el control de acceso.

Clases Anidadas

Las clases anidadas estáticas, clases internas, clases internaas anónimas, clases locales, y expresiones lambda son explicadas. También hay una discusión sobre cuándo utilizar qué enfoque.

Tipos Enum

Esta sección cubre las enumeraciones, clases especializadas que le permiten definir y usar un conjutno de constantes.
