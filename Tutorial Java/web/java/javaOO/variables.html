Declarando Variables Miembro

Hay varias clases de variables:

  • Variables miembro en una clase — estas son llamadas campos.
  • Variables en un método o bloque de código — estas son llamadas variables locales.
  • Variables en declaraciones de métodos — estas son llamadas parámetros.

La clase Bicicleta usa las siguientes líneas de código para definir sus campos: 

public int cadencia;
public int marcha;
public int velocidad;

Las declaraciones de campos están compuestas de tres componentes, en orden:

  1. Cero o más modificadores, tales como public o private.
  2. El tipo del campo.
  3. El nombre del campo.

Los campos de Bicicleta son nombrados cadencia, marcha, y velocidad y todos son datos de tipo entero (int). La palabra clave public identifica estos campos como miembros públicos, accesibles desde cualquier objeto que pueda acceder a la clase.

Modificadores de Acceso

El primer modificador (más a la izquierda) usado le permite controlar qué otras clases tienen acceso a un campo miembro. Por el momento, considere sólo public y private. Otros modificadores de acceso serán descritos más tarde.

  • modificador public — el campo es accesible desde todas las clases.
  • modificador private — el campo es accesible sólo dentro de su propia clase.

En el espíritu de la encapsulación, es común hacer los campos privados. Esto significa que sólo pueden ser directamente accedidos desde la clase Bicicleta. Necesitamos aún tener acceso a estos valores, sin embargo. Esto puede ser hecho indirectamente añadiendo métodos públicos para obtener los valores de los campos para nosotros: 

public class Bicicleta {
        
    private int cadencia;
    private int marcha;
    private int velocidad;
        
    public Bicicleta(int cadenciaInicial, int velocidadInicial, int marchaInicial) {
        marcha = marchaInicial;
        cadencia = cadenciaInicial;
        velocidad = velocidadInicial;
    }
        
    public int getCadencia() {
        return cadencia;
    }
        
    public void setCadencia(int nuevoValor) {
        cadencia = nuevoValor;
    }
        
    public int getMarcha() {
        return marcha;
    }
        
    public void setMarcha(int nuevoValor) {
        marcha = nuevoValor;
    }
        
    public int getVelocidad() {
        return velocidad;
    }
        
    public void frenar(int decremento) {
        velocidad -= decremento;
    }
        
    public void acelerar(int incremento) {
        velocidad += incremento;
    }
}

Tipos

Todas las variables deben tener un tipo. Puede usar tipos primitivos tales como int, float, boolean, etc. O puede usar tipos por referencia, como cadenas, matrices, u objetos.

Nombres de Variable

Todas las variables, ya sean campos, variables locales, o parámetros, siguen las mismas reglas y convenciones de nombrado que fueron especificadas en la lección Lo Básico del Lenguaje, Variables — Nombrado .

En esta lección, sea consciente de que se están usando las mismas reglas de nombrado y convenciones para los métodos y los nombres de clase, excepto que

  • la primera letra de un nombre de clase estará capitalizada, y
  • la primera (o sólo) palabra en un nombre de método será un verbo.
