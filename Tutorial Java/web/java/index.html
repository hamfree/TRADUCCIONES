Recorrido: Aprendiendo el Lenguaje Java

Este recorrido cubre los fundamentos de la programacion en el lenguaje de programación Java.

 Conceptos de Programación Orientada-a-Objetos le enseña los conceptos principales detrás de la programación orienta-a-objetos: objetos, mensajes, clases, y herencia. Esta lección finaliza enseñándole estos conceptos trasladados al código. Siéntase libre de saltarse esta lección si usted está ya familiarizado con la programación orientada-a-objetos.

 Lo Básico del Lenguaje describe las características tradicionales del lengauje, incluyendo variables, arrays, tipos de datos, operadores, y control del flujo.

 Clases y Objetos describe cómo escribir las clases de las cuales los objetos son creados, y cómo crear y usar los objetos.

 Anotaciones son una forma de metadata y ofrecen información para el compilador. Esta lección describe dónde y cómo usar anotaciones en un programa efectivamente.

 Interfaces y Herencia describe las interfaces—lo que son, porqué querríamos escribir una, y cómo escribir una. Esta sección también describe la forma en la cual puede derivar una clase de otra. Esto es, cómo una subclase puede heredar campos y métodos de una superclase. Aprenderá que todas las clases están derivadas de la clase Object, y cómo modificar los métodos que una subclase hereda de la superclase.

 Números y Cadenas (Strings) Esta lección describe como usar los objetos Number y String. La lección también le muestra como formatear los datos para la salida.

 Tipos Genéricos son una característica poderosa del lenguaje de programación de Java. Éstos mejoran la seguridad de tipos en su código, haciendo detectables la mayoría de los errores en tiempo de compilación.

 Paquetes son una característica del lenguaje de programación Java que le ayuda a organizar y estructurar sus clases y sus relaciones con otras clases.
