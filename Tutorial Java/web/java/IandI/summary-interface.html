Sumario de Interfaces

Una declaración de interfaz puede contener firmas de métodos, métodos por defecto, métodos estáticos y definiciones de constantes. Los únicos métodos que tienen implementación son los métodos estáticos y por defecto.

Una clase que implementa una interfaz debe implementar todos los métodos declarados en la interfaz.

Un nombre de interfaz puede ser usado en cualquier tipo donde un tipo puede ser usado.
