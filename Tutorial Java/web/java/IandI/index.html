Lección: Interfaces y Herencia

Interfaces

Usted vió un ejemplo de implementación de una interfaz en la lección anterior. Puede leer más sobre interfaces aquí—Para qué son, porqué podría querer escribir uno, y cómo escribir uno.

Herencia

Esta sección describe la forma en la cual usted puede derivar una clase de otra. Esto es, cómo una subclase puede heredar campos y métodos de una superclase. Aprenderá que todas las clases están derivadas de la clase Object, y cómo modificar los métodos que una subclase hereda de las superclases. Esta sección también cubre las parecidas-a-un-interfaz clases abstractas.
