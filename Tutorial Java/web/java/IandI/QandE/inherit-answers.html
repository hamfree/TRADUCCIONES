Respuestas a Preguntas y Ejercicios: Herencia

Preguntas

Pregunta 1: Considere las dos clases siguientes: 

public class ClaseA {
    public void metodoUno(int i) {
    }
    public void metodoDos(int i) {
    }
    public static void metodoTres(int i) {
    }
    public static void metodoCuatro(int i) {
    }
}

public class ClaseB extends ClaseA {
    public static void metodoUno(int i) {
    }
    public void metodoDos(int i) {
    }
    public void metodoTres(int i) {
    }
    public static void metodoCuatro(int i) {
    }
}

Pregunta 1a: ¿Qué método sobreescribe un método en la superclase?
Respuesta 1a: metodoDos

Pregunta 1b: ¿Qué método oculta un método en la superclase?
Respuesta 1b: metodoCuatro

Pregunta 1c: ¿Qué hacen los otros métodos?
Respuesta 1c: Causan errores en tiempo de compilación.

Pregunta 2: Considere las clases Carta , Baraja , and VisualizaBaraja que usted escribió en Preguntas y Ejercicios: Clases . ¿Qué métodos de Object deberían sobreescribir cada una de estas clases?
Respuesta 2: Carta y Baraja sobreescribirían equals, hashCode, y toString.
