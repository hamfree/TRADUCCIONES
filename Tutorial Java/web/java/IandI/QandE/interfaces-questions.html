Preguntas y Ejercicios: Interfaces

Preguntas

  1. ¿Qué métodos tendría una clase que implementa la interfaz java.lang.CharSequence que implementar?
  2. ¿Qué está mal en el siguiente interfaz? 
    public interface AlgoEstaMal {
    void unMetodo(int unValor){
        System.out.println("Hola mamá");
    }
}
  3. Arregle el interfaz de la pregunta 2.
  4. ¿Es el siguiente interfaz válido? 
    public interface Marcador {
}

Ejercicios

  1. Escriba una clase que implemente el interfaz CharSequence encontrado en el paquete java.lang. Su implementación devolverá la cadena hacia atrás. Seleccione una de las sentencias de este libro para usarla como los datos. Escriba un pequeño método main para comprobar su clase; asegúrese de llamar a los cuatro métodos.
  2. Suponga que ha escrito un servidor de tiempo que periódicamente notifica a sus clientes la fecha y hora actual. Escriba un interfaz que el servidor podría usar para hacer cumplir un protocolo particular a sus clientes.

Compruebe sus respuestas.
