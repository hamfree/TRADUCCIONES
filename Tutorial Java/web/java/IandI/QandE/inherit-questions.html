Preguntas y Ejercicios: Herencia

Preguntas

1. Considere las dos clases siguientes: 

public class ClaseA {
    public void metodoUno(int i) {
    }
    public void metodoDos(int i) {
    }
    public static void metodoTres(int i) {
    }
    public static void metodoCuatro(int i) {
    }
}

public class ClaseB extends ClaseA {
    public static void metodoUno(int i) {
    }
    public void metodoDos(int i) {
    }
    public void metodoTres(int i) {
    }
    public static void metodoCuatro(int i) {
    }
}

a. ¿Qué método sobreescribe un método en la superclase?
b. ¿Qué método oculta un método en la superclase?
c. ¿Qué hacen los otros métodos?

2. Considere las clases Carta , Baraja , y VisualizaBaraja que usted escribió en Preguntas y Ejercicios: Clases . ¿Qué métodos de Object deberían sobreescribir cada una de estas clases?
