Preguntas y Ejercicios: Creando y Usando Paquetes

Preguntas

Asuma que ha escrito algunas clases. Con retraso, decide que deberían ser divididas en tres paquetes, como se muestra en la siguiente tabla. Además, asuma que las clases están actualmente en el paquete por defecto (no tienen sentencias package).

 

Paquetes Destino
Nombre Paquete Nombre Clase
mijuego.servidor Servidor
mijuego.compartido Utilidades
mijuego.cliente Cliente
  1. ¿Qué linea de codigo necesitará añadir a cada fichero fuente para colocar cada clase en el paquete correcto?
  2. Para adherirse a la estructura de directorios, necesitará crear algunos subdirectorios en el directorio de desarrollo y colocar los ficheros fuente en los subdirectorios correctos. ¿Qué subdirectorios debe crear? ¿En que subdirectorio va cada fichero fuente?
  3. ¿Piensa que necesitará hacer cualquier otro cambio a los ficheros fuente para hacerlos compilar correctamente? Si es así, ¿el qué?

Ejercicios

Descargue los ficheros fuentes que se enumeran aquí.

  • Cliente
  • Servidor
  • Utilidades
  1. Implemente los cambios que propuso en las preguntas 1 al 3 usando los ficheros fuente que acaba de descargar.
  2. Compile los ficheros fuente revisados. (Consejo: Si invoca el compilador desde la línea de comandos (en lugar de utilizar un constructor), invoque el compilador desde el directorio que contiene el directorio mijuego que acaba de crear.)

 

Compruebe sus respuestas.
