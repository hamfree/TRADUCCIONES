Sumario de Creando y Usando Paquetes

Para crear un paquete para un tipo, coloque una sentencia package como la primera sentencia en el fichero fuente que contiene el tipo (clase, interfaz, enumeración, o tipo de anotación).

Para usar un tipo público que está en un paquete diferente, tiene tres elecciones: (1) usar el nombre completamente cualificado del tipo, (2) importar el tipo, o (3) importar el paquete entero del cual el tipo es un miembro.

Los nombres de ruta para los ficheros fuente y class de un paquete reflejan el nombre del paquete.

Es posible que tenga que configurar su CLASSPATH para que el compilador y la MVJ puedan encontrar los ficheros .class de sus tipos.
