Comodines Acotados Superiormente

Puede usar el comodín acotado superiormente para relajar las restricciones en una variable. Por ejemplo, digamos que quiere escribir un método que trabaje con List<Integer>, List<Double>, y List<Number>; puede conseguir esto usando un comodín acotado superiormente.

Para declarar un comodín acotado superiormente, use el carácter de comodín ('?'), seguido por la palabra clave extends, seguido por su límite superior. Note que, en este contexto, extends es usada en un sentido general de significar que o "extiende" (como en las clases) o "implementa" (como en las interfaces).

Para escribir el método que trabaja sobre listas de Number y los subitpos de Number, tales como Integer, Double, y Float, especificará List<? extends Number>. El término List<Number> es más restrictivo que List<? extends Number> porque el primero coincide con una lista de tipo Number sólo, mientras que el segundo coincide con una lista de tipo Number o cualquiera de sus subclases.

Considere el siguiente método procesa : 

public static void procesa(List<? extends Foo> list) { /* ... */ }

El comodín acotado superiormente <? extends Foo>, donde Foo es cualquier tipo, coincide con Foo y cualquier subtipo de Foo. El método procesa puede acceder la lista de elementos con el tipo Foo: 

public static void procesa(List<? extends Foo> list) {
    for (Foo elem : list) {
        // ...
    }
}

En la cláusula foreach , la variable elem itera sobre cada elemento en la lista. Cualquier método definido en la clase Foo puede ahora ser usado sobre elem.

El método sumaDeLista devuelve la suma de los números en una lista: 

public static double sumaDeLista(List<? extends Number> list) {
    double s = 0.0;
    for (Number n : list)
        s += n.doubleValue();
    return s;
}

El siguiente código, usando una lista de objetos Integer , imprime suma = 6.0: 

List<Integer> li = Arrays.asList(1, 2, 3);
System.out.println("suma = " + sumaDeLista(li));

Una lista de valores Double puede usar el mismo método sumaDeLista. El siguiente código imprime suma = 7.0: 

List<Double> ld = Arrays.asList(1.2, 2.3, 3.5);
System.out.println("suma = " + sumaDeLista(ld));
