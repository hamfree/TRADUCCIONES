Recorrido: Nombrado en Java e Interfaz de Directorio

Este recorrido describe JNDI™ (Interfaz de Directorio y Nombres de Java), una IPA para acceder a los servicios de nombres y directorio. Aquí aprenderá sobre los servicios básicos de nombres y directorios y cómo usar JNDI para escribir aplicaciones simples para usar estos servicios. Se usa el servicio de directorio LDAP más popular para demostrar el uso de JNDI para acceder a los servicios de directorio.

  Conceptos de Nombrado y Directorio Empiece aquí para obtener una descripción general de los conceptos de nombres y directorios.

  Visión General de JNDI Le da una descripción general de JNDI, su arquitectura y empaquetado.

  Configuración del Software Describe las instrucciones y pasos necesarios para configurar un entorno requerido para ejecutar los ejemplos descritos en este recorrido, así como cualquier otra aplicación JNDI.

  Operaciones de Nombrado y Directorio Describe varias operaciones de nombres y directorio y las demuestra a través de multitude de ejemplos que acceden a los servicios de nombres/directorio usando JNDI.

  Temas Avanzados para los Usuarios LDAP Una lección especializada para los usuarios de LDAP. Habla sobre el modelado de JNDI como una IPA de LDAP, como realizar la autenticación en LDAP, SSL y gestión de las conexiones en entornos de producción.

  Objetos de Java en el Directorio Le muestra cómo integrar su aplicación con el directorio, de forma que pueda almacenar y recuperar objetos Java a y desde el directorio.

  Nuevas funcionalidades en KDJ 5.0 y KDJ 6 Introduce las características en JNDI y el Proveedor de Servicio LDPA que están disponibles en el KDJ 5.0 y el KDJ 6.

Nota: Este recorrido del tutorial está basado en el tutorial de JNDI independiente ubicado en https://docs.oracle.com/javase/jndi/tutorial/. El tutorial independiente de JNDI, actualizado por última vez en el KDJ de Java 2, Edición Estándar, v. 1.4.2, proporciona una cobertura completa sobre JNDI, pero ya no es compatible. Este tutorial extrae los conceptos básicos del tutorial independiente e incluye características agregadas a JNDI en las versiones de la Plataforma Edición Estándar 5.0 y 6.

El tutorial de JNDI más viejo se presever como un archivo en docs.oracle.com.
