Recuento de Resultados

Algunas veces, una consulta podría producir demasiadas respuestas y usted quiere limitar el número de respuestas devueltas. Puede hacer esto usando el límite de recuento del control de búsqueda. Por defecto, una búsqueda no tiene un límite de recuento — devolverá todas las respuestas que encuentre. Para establecer el límite de recuento de una búsqueda, pase el número a SearchControls.setCountLimit().

El ejemplo siguiente establece el límite de recuento en 1. 

// Set the search controls to limit the count to 1
SearchControls ctls = new SearchControls();
ctls.setCountLimit(1);

Si el programa intenta obtener más que el límite de recuento de resultados, será lanzada una SizeLimitExceededException. Por tanto, si un programa ha establecido un límite de recuento, debería diferenciar esta excepción de otras NamingExceptions o realizar un seguimiento del límite de recuento y no pedir más que esa cantidad de resultados.

Especificar un límite de recuento para una búsqueda es una forma de controlar los recursos (tales como la memoria y el ancho de banda de red) que su aplicación consume. Otras formas de controlar los recursos consumidos son reducir su filtro de búsqueda (ser más específico sobre lo que está buscando), iniciar su búsqueda en el contexto apropiado, y usar el alcance apropiado.
