Alcance

El objeto predeterminado SearchControls especifica que la búsqueda va a ser realizada en el contexto llamado ( SearchControls.ONELEVEL_SCOPE). Este valor predetermiando se usa en los ejemplos en la sección de Filtros de Búsqueda.

Además de este valor predeterminado, puede especificar que la búsqueda sea realizada en el subárbol completo o sólo en el objeto nombrado.

Buscar el Subárbol

Una búsqueda del subárbol completo busca el objeto nombrado y todos sus descendientes. Para hacer que la búsqueda se comporte de esta forma, pase SearchControls.SUBTREE_SCOPE a SearchControls.setSearchScope() como sigue. 

// Specify the ids of the attributes to return
String[] attrIDs = {"sn", "telephonenumber", "golfhandicap", "mail"};
SearchControls ctls = new SearchControls();
ctls.setReturningAttributes(attrIDs);
ctls.setSearchScope(SearchControls.SUBTREE_SCOPE);

// Specify the search filter to match
// Ask for objects that have the attribute "sn" == "Geisel"
// and the "mail" attribute
String filter = "(&(sn=Geisel)(mail=*))";

// Búsqueda the subtree for objects by using the filter
NamingEnumeration answer = ctx.search("", filter, ctls);

Este ejemplo busca en el subárbol del contexto ctx por entradas que satisfagan el filtro especificado. Encuentra la entrada "cn= Ted Geisel, ou=People" en este subárbol que satisface el filtro. 

# java SearchSubtree
>>>cn=Ted Geisel, ou=People
attribute: sn
value: Geisel
attribute: mail
value: Ted.Geisel@JNDITutorial.example.com
attribute: telephonenumber
value: +1 408 555 5252

Buscar el Objeto Nombrado

Puede también buscar el objeto nombrado. Esto es útil, por ejemplo, para comprobar si el objeto nombrado satisface un filtro de búsqueda. Para buscar el objeto nombrado, pase SearchControls.OBJECT_SCOPE a setSearchScope(). 

// Specify the ids of the attributes to return
String[] attrIDs = {"sn", "telephonenumber", "golfhandicap", "mail"};
SearchControls ctls = new SearchControls();
ctls.setReturningAttributes(attrIDs);
ctls.setSearchScope(SearchControls.OBJECT_SCOPE);

// Specify the search filter to match
// Ask for objects that have the attribute "sn" == "Geisel"
// and the "mail" attribute
String filter = "(&(sn=Geisel)(mail=*))";

// Search the subtree for objects by using the filter
NamingEnumeration answer = 
    ctx.search("cn=Ted Geisel, ou=People", filter, ctls);

Este ejemplo comprueba si el objeto "cn=Ted Geisel, ou=People" satisface el filtro dado. 

# java SearchObject
>>>
attribute: sn
value: Geisel
attribute: mail
value: Ted.Geisel@JNDITutorial.example.com
attribute: telephonenumber
value: +1 408 555 5252

El ejemplo encontró una respuesta y la imprimió. Observe que el nombre del resultado es la cadena vacía. Esto se debe a que el nombre del objeto siempre se nombra en relación con el contexto de la búsqueda (en este caso, " cn=Ted Geisel, ou=People ").
