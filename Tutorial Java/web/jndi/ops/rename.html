Renombrar

Puede renombrar un objeto en un contexto usando Context.rename(). 

// Renombrar to Scott S
ctx.rename("cn=Scott Seligman", "cn=Scott S");

Este ejemplo renombra el objeto que fue vinculado a "cn=Scott Seligman" a "cn=Scott S". Después de verificar que se renombró el objeto, el programa lo renombra a su nombre original, ("cn=Scott Seligman"), como sigue. 

// Renombrar back to Scott Seligman
ctx.rename("cn=Scott S", "cn=Scott Seligman");

Para más ejemplos de renombrado de entradas LDAP compruebe la lección Temas Avanzados para los Usuarios LDAP.
