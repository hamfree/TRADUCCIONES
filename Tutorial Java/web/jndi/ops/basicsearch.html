Búsqueda Básica

La forma más simple de búsqueda requiere que especifique el conjunto de atributos que una entrada debe tener y el nombre del contexto objetivo en el cual realizar la búsqueda.

El código siguiente crea un conjunto de atributos matchAttrs, que tiene dos atributos "sn" y "mail". Especifica que las entradas cualificadas deben tener un atributo apellido ("sn") con un valor de "Geisel" y un atributo "mail" con cualquier valor. Después invoca a DirContext.search() para buscar en el contexto "ou=People" por entradas que tengan los atributos especificados en matchAttrs. 

// Specify the attributes to match
// Ask for objects that has a surname ("sn") attribute with 
// the value "Geisel" and the "mail" attribute

// ignore attribute name case
Attributes matchAttrs = new BasicAttributes(true); 
matchAttrs.put(new BasicAttribute("sn", "Geisel"));
matchAttrs.put(new BasicAttribute("mail"));

// Search for objects that have those matching attributes
NamingEnumeration answer = ctx.search("ou=People", matchAttrs);

A continuación, puede imprimir los resultados de la siguiente manera. 

while (answer.hasMore()) {
    SearchResult sr = (SearchResult)answer.next();
    System.out.println(">>>" + sr.getName());
    printAttrs(sr.getAttributes());
}

printAttrs() es similar al código en el ejemplo con getAttributes() que imprime un conjunto de atributos.

Devolver Atributos Seleccionados

El ejemplo anterior devolvió todos los atributos asociados con las entradas que satisfacían la consulta especificada. Puede seleccionar los atributos a devolver pasándole a search() una matriz de identificadores de atributos que quiera incluir en el resultado. Después de crear matchAttrs como se mostró previamente, también necesita crear la matriz de identificadores de atributo, como se muestra a continuación. 

// Specify the ids of the attributes to return
String[] attrIDs = {"sn", "telephonenumber", "golfhandicap", "mail"};

// Search for objects that have those matching attributes
NamingEnumeration answer = ctx.search("ou=People", matchAttrs, attrIDs);

Este ejemplo devuelve los atributos "sn", "telephonenumber", "golfhandicap", y "mail" de las entradas que tienen un atributo "mail" y tienen un atributo "sn" con el valor "Geisel". Este ejemplo produce el siguiente resultado. (La entrada no tiene un atributo "golfhandicap", así que esta no es devuelta). 

# java Search 
>>>cn=Ted Geisel
attribute: sn
value: Geisel
attribute: mail
value: Ted.Geisel@JNDITutorial.example.com
attribute: telephonenumber
value: +1 408 555 5252
