Límite de Tiempo

Un límite de tiempo en una búsqueda coloca un límite superior en la cantidad de tiempo que la operación de búsqueda bloqueará esperando las respuestas. Esto es útil cuando no quiere esperar demasiado por una respuesta. Si el límite de tiempo especificado se excede antes de que la operación de búsqueda pueda ser completada, entonces se lanzará una TimeLimitExceededException.

Para establecer el límite de tiempo de una búsqueda, pase el número de milisegundos a SearchControls.setTimeLimit(). El siguiente ejemplo establece el límite de tiempo en 1 segundo. 

// Set the search controls to limit the time to 1 second (1000 ms)
SearchControls ctls = new SearchControls();
ctls.setTimeLimit(1000);

Para hacer que este ejemplo en particular exceda su límite de tiempo, necesita reconfigurarlo para que use un servidor lento, o un servidor que haya perdido montones de entradas. Alternativamente, puede usar otras tácticas para hacer que la búsqueda tome más de 1 segundo.

Un límite de tiempo de cero significa que no hay límite de tiempo establedcido y que las llamadas al directorio esperarán indefinidamente por una respuesta.
