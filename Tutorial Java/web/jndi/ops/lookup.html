Buscando un Objeto

Para buscar un objeto en el servicio de nombrado, use Context.lookup() y pásele el nombre del objeto que quiere recuperar. Suponga que hay un objeto en el servicio de nombrado con el nombre cn=Rosanna Lee,ou=People. Para recuperar el objeto, escribiría 

Object obj = ctx.lookup("cn=Rosanna Lee,ou=People");

El tipo de objeto que se devuelve por lookup() depende tanto del sistema de nombrado subyacente y de los datos asociados con el mismo objeto. Un sistema de nombrado puede contener muchos tipos diferentes de objetos, y una búsqueda de un objeto en diferentes partes del sistema puede producir diferentes objetos de diferntes tipos. En este ejemplo, sucede que "cn=Rosanna Lee,ou=People" está vinculado a un objeto de contexto (javax.naming.ldap.LdapContext). Puede convertir el resultado de lookup() a su clase objetivo.

Por ejemplo, el código siguiente busca el objeto "cn=Rosanna Lee,ou=People" y lo convierte a un LdapContext. 

import javax.naming.ldap.LdapContext;
...
LdapContext ctx = (LdapContext) ctx.lookup("cn=Rosanna Lee,ou=People");

El ejemplo completo está en el fichero Lookup.java.

Hay dos nuevos métodos estáticos disponibles en Java SE 6 para buscar un nombre:

  • InitialContext.doLookup(Name name)
  • InitialContext.doLookup(String name)

Estos métodos proporcionan una forma abreviada de buscar un nombre sin instanciar un InitialContext.
