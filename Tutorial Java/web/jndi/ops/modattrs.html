Modificar Atributos

La interfaz DirContext contiene métodos para modificar los atributos y los valores de atributo de los objetos en el directorio.

Usar una Lista de Modificaciones

Una forma de modificar los atributos de un objeto es proporcionar una lista de peticiones de modificación ( ModificationItem). Cada ModificationItem consiste en un constante numérica indicando el tipo de modificación que hacer y un Attribute describiendo la modificación a hacer. A continuación están los tres tipos de modificaciones:

  • ADD_ATTRIBUTE
  • REPLACE_ATTRIBUTE
  • REMOVE_ATTRIBUTE

Las modificaciones se aplican en el orden en el que aparecen en la lista. Se ejecutan todas las modificaciones o ninguna.

El código siguiente crea una lista de modificaciones. Reemplaza el valor del atributo "mail" con un valor de "geisel@wizards.com", agrega un valor adicional al atributo "telephonenumber", y elimina el atributo "jpegphoto". 

// Specify the changes to make
ModificationItem[] mods = new ModificationItem[3];

// Replace the "mail" attribute with a new value
mods[0] = new ModificationItem(DirContext.REPLACE_ATTRIBUTE,
    new BasicAttribute("mail", "geisel@wizards.com"));

// Add an additional value to "telephonenumber"
mods[1] = new ModificationItem(DirContext.ADD_ATTRIBUTE,
    new BasicAttribute("telephonenumber", "+1 555 555 5555"));

// Remove the "jpegphoto" attribute
mods[2] = new ModificationItem(DirContext.REMOVE_ATTRIBUTE,
    new BasicAttribute("jpegphoto"));

Directorio Activo de Windows: Los Directorios Activos definen "telephonenumber" como un atributo de valor único, opuesto a RFC 2256. Para que este ejemplo funcione contra el Directorio Activo, debe usar otro atributo distinto de "telephonenumber", o cambiar DirContext.ADD_ATTRIBUTE por DirContext.REPLACE_ATTRIBUTE.

Después de crear esta lista de modificaciones, puede proporcionarla a modifyAttributes() como sigue. 

// Perform the requested modifications on the named object
ctx.modifyAttributes(name, mods);

Usar Atributos

Alternativamente, puede realizar modificaciones especificando el tipo de la modificación y los atributos a los cuales quiere aplicar la modificación.

Por ejemplo, la línea siguiente reemplaza los atributos (identificados en orig) asociados con name con esos en orig: 

ctx.modifyAttributes(name, DirContext.REPLACE_ATTRIBUTE, orig);

Cualquier otro atributo de name permanece sin cambios.

Ambos usos de modifyAttributes() se demuestran en el programa de ejemplo. Este programa modifica los atributos usando una lista de modificación y después usa la segunda forma de modifyAttributes() para restaurar los atributos originales.
