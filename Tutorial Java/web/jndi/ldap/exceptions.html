Cómo los Códigos de Error LDAP se Mapean a las Excepciones de JNDI

LDAP define un conjunto de códigos de estado que son devueltos con las respuestas LDAP enviadas por el servidor LDAP (vea RFC 2251). En JNDI, las condiciones de error se indican como excepciones comprobadas que son subclases de NamingException. Vea la sección Excepciones de Nombrado para una descripción de las clases de excepción de JNDI.

El proveedor de servicio LDAP traduce el código de estado de LDAP que recibe del servidor LDAP a la subclase apropiada de NamingException. La tabla siguiente muestra la correspondencia entre los códigos de estado LDAP y las excepciones JNDI.

4 Se excedió el límite de tamaño. SizeLimitExceededException
5 Comparación falsa. Usado por DirContext.search(). No genera una excepción.

8 Se requiere autenticación fuerte. AuthenticationNotSupportedException
