Notificaciones no Solicitadas de LDAP

LDAP v3 (RFC 2251) define una notificación no solicitada, un mensaje que se envía por un servidor LDAP al cliente sin ninguna provocación del cliente. Una notificación no solicitada se representa en JNDI por la interfaz UnsolicitedNotification.

Ya que las notificaciones no solicitadas se envían asincrónicamente por el servidor, puede usar el mismo modelo de eventos usado para recibir notificaciones sobre cambios del espacio de nombres y cambios del contenido del objeto. Usted registra interés en recibir notificaciones no solicitadas al registrar un UnsolicitedNotificationListener con un EventContext o un EventDirContext.

Aquí tiene un ejemplo de un UnsolicitedNotificationListener. 

public class UnsolListener implements UnsolicitedNotificationListener {
    public void notificationReceived(UnsolicitedNotificationEvent evt) {
        System.out.println("received: " + evt);
    }

    public void namingExceptionThrown(NamingExceptionEvent evt) {
        System.out.println(">>> UnsolListener got an exception");
            evt.getException().printStackTrace();
    }
}

Lo siguiente es un ejemplo que registra una implementación de UnsolicitedNotificationListener con una fuente de evento. Tenga en cuenta que sólo el argumento del listener a EventContext.addNamingListener() es relevante. El nombre y parámetros de alcance no son relevantes para las notificaciones no solicitadas. 

// Get the event context for registering the listener
EventContext ctx = (EventContext)
    (new InitialContext(env).lookup("ou=People"));

// Create the listener
NamingListener listener = new UnsolListener();

// Register the listener with the context (all targets equivalent)
ctx.addNamingListener("", EventContext.ONELEVEL_SCOPE, listener);

Cuando ejecuta este programa, necesita apuntarlo a un servidor LDAP que pueda generar notificaciones no solicitadas y presionar al servidor para que emita la notificación.

Un listener que implementa UnsolicitedNotificationListener puede también implementar otros interfaces NamingListener, tales como NamespaceChangeListener y ObjectChangeListener.
