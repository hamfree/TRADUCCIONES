Control de Ordenación

El control de ordenación se usa cuando un cliente quiere que el servidor envíe los resultados de la búsqueda ordenados. La clasificación del lado del servidor es útil en una situación donde el cliente necesita ordenar los resultados de acuerdo a algún criterio pero no está equipado para realizar el proceso de clasificación por sí mismo. El control de ordenación se especifica en RFC 2891. Las clases de abajo proporcionan la funcionalidad requerida para admitir el control de ordenación.

  • javax.naming.ldap.SortControl
  • javax.naming.ldap.SortKey
  • javax.naming.ldap.SortResponseControl

SortKey es una lista ordenada de claves según la cual el servidor clasifica el resultado.

¿Cómo usar el Control de Ordenación?

El ejemplo de abajo ilustra la interacción cliente-servidor entre un cliente que realiza una petición de búsqueda con una clasificación del lado del servidor basada en el atributo "cn".

  1. El cliente envia una petición de búsqueda pidiendo 
       
    // Activate sorting
     String sortKey = "cn";
     ctx.setRequestControls(new Control[] { 
         new SortControl(sortKey, Control.CRITICAL) });

     // Perform a search
     NamingEnumeration results = 
         ctx.search("", "(objectclass=*)", new SearchControls());
  2. El servidor responde con entradas que están clasificadas basándose en el atributo "cn" y su correspondiente regla de coincidencia. 
       
    // Iterate over sorted search results
     while (results != null && results.hasMore()) {
         // Display an entry
         SearchResult entry = (SearchResult)results.next();
         System.out.println(entry.getName());

         // Handle the entry's response controls (if any)
         if (entry instanceof HasControls) {
             // ((HasControls)entry).getControls();
         }
     }
     // Examine the sort control response 
     Control[] controls = ctx.getResponseControls();
     if (controls != null) {
         for (int i = 0; i < controls.length; i++) {
             if (controls[i] instanceof SortResponseControl) {
                 SortResponseControl src = (SortResponseControl)controls[i];
                 if (! src.isSorted()) {
                     throw src.getException();
                 }
             } else {
                 // Handle other response controls (if any)
             }
         }
     }

El ejemplo completo JNDI se puede encontrar aquí.

Nota: El control de ordenación se admite tanto en el Servidor de Directorio de Oracle como en el servidor Directorio Activo de Windows.
