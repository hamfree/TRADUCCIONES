Lección: Excepciones

El lenguaje de programación Java usa excepciones para manejar los errores y otros eventos excepcionales. Esta lección describe cuándo y cómo usar excepciones.

¿Qué es una Excepción?

Una excepción es un evento que ocurre durante la ejecución de un programa que interrumpe el flujo normal de las instrucciones.

La Captura o la Especificación del Requisito

Esta sección cubre como capturar y manejar excepciones. La discusión incluye los bloques try, catch, y finally, como también las excepciones encadenadas y las trazas.

Cómo Lanzar Excepciones

Esta sección cubre la sentencia throw y la clase Throwable y sus subclases.

La Sentencia try-with-resources

Esta sección describ la sentencia try-with-resources, la cual es una sentencia try que declra uno o más recursos. Un recurso es un objeto que debe ser cerrado después de que el programa haya terminado con el. La sentencia try-with-resources asegura que cada recurso es cerrado al final de la sentencia.

Excepciones No Comprobadas ─ La Controversia

Esta sección explica el uso correcto e incorrecto de las excepciones no comprobadas indicadas por las subclases de RuntimeException.

Ventajas de las Excepciones

El uso de las excepciones para gestionar errores tiene algunas ventajas sobre la técnicas tradicionales de gestión de errores. Aprenderá más en esta sección.
