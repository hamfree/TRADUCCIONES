    Ejercicios

    1. Añada un método leeLista a ListaDeNumeros.java. Este método leerá los valores int de un fichero, imprimirá cada valor, y los añadirá al final del vector. Capturará todos los errores apropiados. Necesitará también un fichero de texto conteniendo los números a leer.
    2. Modifique el siguiente método cat de forma que compile. 
      public static void cat(File file) {
    RandomAccessFile input = null;
    String line = null;

    try {
        input = new RandomAccessFile(file, "r");
        while ((line = input.readLine()) != null) {
            System.out.println(line);
        }
        return;
    } finally {
        if (input != null) {
            input.close();
        }
    }
}
