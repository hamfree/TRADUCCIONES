Preguntas y Ejercicios: Expresiones Regulares

Preguntas

  1. ¿Cuáles son las tres clases públicas en el paquete java.util.regex? Describa el objetivo de cada una.
  2. Considere la cadena literal "foo". ¿Cuál es el índice de inicio? ¿Cuál es el índice final? Explique lo que esos números significan.
  3. ¿Cuál es la diferencia entre un carácter ordinario y un metacarácter? De un ejemplo de cada uno.
  4. ¿Cómo fuerza a un metacarácter a actuar como un carácter ordinario?
  5. ¿Cómo se llama un conjunto de caracteres entre corchetes? ¿Para qué es esto?
  6. Aquí hay tres clases carácter predefinidas: \d, \s, y \w. Describa cada una, y reescríbalas usando corchetes.
  7. Para cada \d, \s, y \w, escriba dos expresiones simples que coincidan con el conjunto opuesto de caracteres.
  8. Considere la expresión regular (dog){3}. Identifique las dos subexpresiones. ¿Qué cadena coincide con la expresión?

Ejercicios

  1. Use una referencia inversa para escribir una expresión que coincidirá un nombre de persona sólo si el primer nombre y el último nombre de esa persona es el mismo.

Compruebe sus respuestas.
