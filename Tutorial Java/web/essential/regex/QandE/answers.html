Respuestas a Preguntas y Ejercicios: Expresiones Regulares

Preguntas

  1. Pregunta: ¿Cuáles son las tres clases públicas en el paquete java.util.regex? Describa el objetivo de cada una.

    Respuesta:

    • Las instancias de Pattern son representaciones compiladas de expresiones regulares.
    • Las instancias de Matcher son motores que interpretan los patrones y realizan operaciones de correspondencia contra cadenas de entrada.
    • La PatternSyntaxException define una excepción no comprobada que indica un error de sintaxis en una expresión regular.
  2. Pregunta: Considere la cadena literal "foo". ¿Cuál es el índice de inicio? ¿Cuál es el índice final? Explique lo que esos números significan.

    Respuesta: Cada carácter en la cadena reside en su propia celda. Las posiciones de índice apuntan entre las celdas. La cadena "foo" empieza en el índice 0 y termina en el índice 3, aunque los caracteres sólo ocupan las celdas 0, 1 y 2.

  3. Pregunta: ¿Cuál es la diferencia entre un carácter ordinario y un metacarácter? De un ejemplo de cada uno.

    Respuesta: Un carácter ordinario en una expresión regular coincide con sí mismo. Un metacarácter es un carácter especial que afecta a la forma en que un patrón es correspondido. La letra A es un carácter ordinario. La marca de puntuación . es un metacarácter que coincide con cualquier carácter único.

  4. Pregunta: ¿Cómo fuerza a un metacarácter a actuar como un carácter ordinario?

    Respuesta: Hay dos formas:

    • Preceder el metacarácter con una barra invertida (\);
    • Encerrar el metacarácter entre expresiones de entrecomillado, \Q (en el principio) y \E (al final).
  5. Pregunta: ¿Cómo se llama un conjunto de caracteres entre corchetes? ¿Para qué es esto?

    Respuesta: Esta es una clase de caracteres. Coincide con cualquier carácter singular que esté en la clase de caracteres especificada por la expresión entre los corchetes.

  6. Pregunta: Aquí hay tres clases carácter predefinidas: \d, \s, y \w. Describa cada una, y reescríbalas usando corchetes.

    Respuesta:

    \d Coincide con cualquier dígito. [0-9]
    \s Coincide con cualquier carácter de espacio en blanco. [ \t\n-x0B\f\r]
    \w Coincide con cualquier palabra. [a-zA-Z_0-9]
  7. Pregunta: Para cada \d, \s, y \w, escriba dos expresiones simples que coincidan con el conjunto opuesto de caracteres.

    Respuesta:

    \d \D [^\d]
    \s \S [^\s]
    \w \W [^\w]
  8. Pregunta: Considere la expresión regular (dog){3}. Identifique las dos subexpresiones. ¿Qué cadena coincide con la expresión?

    Respuesta: La expresión consiste de un grupo de captura, (dog), seguido por un cuantificador codicioso {3}. Coincide con la cadena "dogdogdog".

    Ejercicios

    1. Ejercicio: Use una referencia inversa para escribir una expresión que coincidirá un nombre de persona sólo si el primer nombre y el último nombre de esa persona es el mismo.

      Solución: ([A-Z][a-zA-Z]*)\s\1
