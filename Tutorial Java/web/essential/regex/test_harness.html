Clase Para Pruebas

Esta sección define una estructura de pruebas reutilizable, PruebasExpReg.java , para explorar las construcciones de expresión regular soportadas por esta IPA. El comando para ejecutar este código es java PruebasExpReg; sin que acepte ningún argumento de línea de comandos. La aplicación hace bucles repetidamente, pidiendo al usuario una expresión regular y una cadena de entrada. Usar esta prueba es opcional, pero puede encontrarla conveniente para explorar los casos de prueba descritos en las siguientes páginas. 


import java.io.Console;
import java.util.regex.Pattern;
import java.util.regex.Matcher;

public class PruebasExpReg {

    public static void main(String[] args){
        Console consola = System.console();
        if (consola == null) {
            System.err.println("Sin consola.");
            System.exit(1);
        }
        while (true) {

            Pattern patron =
            Pattern.compile(consola.readLine("%nIntroduzca su expresión regular: "));

            Matcher matcher =
            patron.matcher(consola.readLine("Introduza la cadena de entrada donde buscar: "));

            boolean seHaEncontrado = false;
            while (matcher.find()) {
                consola.format("He encontrado el texto" +
                    " \"%s\" empezando en el índice " +
                    "%d y terminando en el índice %d.%n",
                    matcher.group(),
                    matcher.start(),
                    matcher.end());
                seHaEncontrado = true;
            }
            if(!seHaEncontrado){
                consola.format("No se encontraron coincidencias.%n");
            }
        }
    }
}

Antes de continuar a la siguiente sección, salve y compile este código para asegurarse que su entorno de desarrollo soporta los paquetes requeridos.
