Lección: Expresiones Regulares

Esta lección explica cómo usar la IPA java.util.regex para la correspondencia de patrones con expresiones regulares. Aunque la sintaxis aceptada por este paquete es similar a la del lenguaje de programación Perl, el conocer Perl no es un prerequisito. Esta lección empieza con lo básico, y gradualmente evoluciona para cubrir técnicas más avanzadas.

Introducción
Ofrece una vista general de las expresiones regulares. También introduce las clases principales que comprenden estas IPA.
Clase Para Pruebas
Define una aplicación simple para probar la coincidencian de patrones con expresiones regulares.
Literales de Cadena
Introduce la correspondencia básica de patrones, los metacaracteres y el entrecomillado.
Clases Carácter
Describle las clases de caracteres simples, la negación, los rangos, las uniones, las intersecciones, y la substracción.
Clases Carácter Predefinidas
Describe las clases de caracteres básicos predefinidos para el espacio en blanco, palabras, y carácteres dígito.
Cuantificadores
Explica los cuantificadores codiciosos, reluctantes y posesivos para emparejar una expresión especificada x número de veces.
Grupos de Captura
Explica cómo tratar caracteres múltiples como una sola unidad.
Comparadores Límite
Describe los límites de líneas, palabras, y entradas.
Métodos de la Clase Pattern
Examina otros métodos útiles de la clase Pattern , y explorar las características avanzadas tales como compilación con banderas y el uso de expresiones de bandera embebidas.
Métodos de la Clase Matcher
Describe los métodos más comunmente usados de la clase Matcher.
Métodos de la Clase PatternSyntaxException
Describe cómo exainar una PatternSyntaxException.
Recursos Adicionales
Para leer más sobre expresiones regulares, consulte esta sección para recursos adicionales.
