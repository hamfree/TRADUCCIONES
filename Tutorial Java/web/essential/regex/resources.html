Recursos Adicionales

Ahora que ha completado esta lección sobre expresiones regulares, probablemente encontrará que sus principales referencias serán la documentación del IPA para las siguientes clases: Pattern, Matcher, and PatternSyntaxException.

Para una descripción más precisa del comportamiento de las construcciones de expresiones regulares, recomendamos leer el libro Mastering Regular Expressions por Jeffrey E. F. Friedl.
