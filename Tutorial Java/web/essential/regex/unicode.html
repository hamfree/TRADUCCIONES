Soporte de Unicode

A partir de la versión de KDJ 7, la asignación de patrones de expresiones regulares ha ampliado la funcionalidad para admitir Unicode 6.0.

  • Emparejar un punto de código específico
  • Propiedades del Carácter Unicode

Coincidir un Punto de Código Específico

Puede hacer coincidir un punto de código Unicode específico usando una secuencia de escape de la forma \uFFFF, donde FFFF es el valor hexadecimal del punto de código que quiere hacer coincidir. Por ejemplo, \u6771 coincide con el carácter Han para este.

Alternativamente, puede especificar un código de punto usando una notación hexadecimal de estilo Perl, \x{...}. Por ejemplo: 

String hexPattern = "\x{" + Integer.toHexString(codePoint) + "}";

Propiedades de caracteres Unicode

Cada carácter Unicode, además de su valor, tiene ciertos atributos, o propiedades. Puede hacer coincidir un carácter singular perteneciente a una categoría particular con la expresión \p{prop}. Puede hacer que un caracter singular coincida y no pertenezca a una categoría particular con la expresión \P{prop}.

Las tres tipos de propiedad soportados son los guiones, bloques, y un categoría "general".

Guiones

Para determinar si un punto de código pertenece a un guión específico, puede o usar la clave script , o la forma corta sc, por ejemplo, \p{script=Hiragana}. Alternativamente, puede prefijar el nombre del guión con la cadena Is, tal como \p{IsHiragana}.

Los nombres de guión válidos soportados por Pattern son aquellos aceptados por UnicodeScript.forName.

Bloques

Un bloque puede ser especificado usando la palabra clave block , o la forma corta blk, por ejemplo, \p{block=Mongolian}. Alternativamente, puede prefijar el nombre del bloque con la cadena In, tal como \p{InMongolian}.

Los nombre de bloque válidos soportados por Pattern son aquellos aceptados por UnicodeBlock.forName.

Categoría General

Las categorías pueden ser especificadas con prefijos opcionales Is. Por ejemplo, IsL coincide la categoría de las letras Unicode. Las categorías pueden ser también especificadas usando la palabra clave general_category, o la forma corta gc. Por ejemplo, una letra en mayúsculas puede ser coincidida usando general_category=Lu o gc=Lu.

Las categorías soportadas son aquellas del Estándar Unicode en la versión especificada por la clase Character.
