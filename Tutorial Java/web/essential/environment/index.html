Lección: El Entorno de la Plataforma

Una aplicación se ejecuta en un entorno de la plataforma, definido por el sistema operativo subyacente, la máquina virtual de Java, las librerías de clases, y varios datos de configuración suministrados cuando se inicia la aplicación. Esta lección describe algunas de las IPA que una aplicación usa para examinar y configurar su entorno de plataforma. La lección consiste de tres secciones:

  • Utilidades de Configuración describe las IPA usadas para acceder a los datos de configuración suministrados cuando la aplicación es desplegada, o por el usuario de la aplicación.
  • Utilidades del Sistema describe IPA misceláneas definidas en las clases System y Runtime .
  • PATH Y CLASSPATH describe las variables de entorno usadas para configurar las herramientas de desarrollo del KDJ y otras aplicaciones.
