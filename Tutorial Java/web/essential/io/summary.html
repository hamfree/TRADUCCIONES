Sumario

El paquete java.io contiene muchas clases que sus programa pueden usar para leer y escribir datos. La mayoŕia de las clases implementan acceso secuencial a flujos. El acceso secuencial a flujos puede ser dividido en dos grupos: aquellos que leen y escriben bytes y aquellos que leen y escriben caracteres Unicode. Cada flujo de acceso secuencial tiene una especialización, tal como leer desde o escribir en un fichero, fltrar los datos mientras son leídos o escritos, o la serialización de un objeto.

El paquete java.nio.file ofrece un amplio soporte para la E/S de ficheros y del sistema de ficheros. Esta es una IPA muy completa, pero los puntos clave de entrada son los siguientes:

  • La clase Path tiene métodos para manipular un path.
  • La clase Files tiene métodos para operaciones de ficheros, tales como mover, copiar, borrar, y también métodos para recuperar y establecer atributos de ficheros.
  • La clase FileSystem tiene una variedad de métodos para obtener información sobre el sistema de ficheros.

Más información sobre NIO.2 puede ser encontrada en la web del proyecto OpenJDK: NIO en java.net. Éste sitio incluye recursos para las características ofrecidas por NIO.2 que están fuera del escape de este tutorial, tales como la multidifusión, E/S asíncrona, y la creación de su propia implementación de sistema de ficheros.
