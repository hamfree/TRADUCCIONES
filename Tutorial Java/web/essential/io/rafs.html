Acceso Aleatorio a Ficheros

El acceso aleatorio a ficheros permite el acceso no secuencial, o aleatorio, al contenido de un fichero. Para acceder al fichero aleatoriamente, usted abre el fichero, busca una localización particular, y lee o escribe a ese fichero.

Esta funcionalidad es posible con el interfaz SeekableByteChannel. El interfaz SeekableByteChannel extiende el canal de E/S con la noción de la posición actual. Los métodos le habilitan a establecer o preguntar por la posición, y entonces puede leer los datos desde ahí, o escribir los dato ahí, a esa localización. La IPA consiste de unos pocos métodos, fáciles de usar:

  • position – Devuelve la posición actual del canal
  • position(long) – Esstablece la posición del canal
  • read(ByteBuffer) – Lee bytes y los introduce en el buffer desde el canal
  • write(ByteBuffer) – Escribe bytes desde el buffer al canal
  • truncate(long) – Trunca el fichero (u otra entidad) conectada con el canal

Leyendo, Escribiendo y Creando Ficheros muestra que los métodos Path.newByteChannel devuelven una instancia de un SeekableByteChannel. En el sistema de ficheros por defecto, puede usar ese canal como tal, o puede moldearlo a un FileChannel dándole acceso a características más avanzadas, tales como la asignación de una región del fichero directamente en memoria para un acceso más rápido, el bloqueo de una región del fichero, o leer y escribir bytes desde una localización absoluta sin afectar a la posición actual del canal.

El trozo de código siguiente abre un fichero para leer y escribir usando uno de los métodos newByteChannel. El SeekableByteChannel que es devuelto es moldeado a un FileChannel. Entonces, 12 bytes son leídos del principio del fichero, y la cadena "¡Yo estuve aquí!" es escrita en esa localización. La posición actual en el fichero es movida al final, y los 12 bytes del principio son añadidos. Finalmente, la cadena "¡Yo estuve aquí!" es añadida, y el canal sobre el fichero es cerrado. 

String s = "¡Yo estuve aquí!\n";
byte data[] = s.getBytes();
ByteBuffer out = ByteBuffer.wrap(data);

ByteBuffer copy = ByteBuffer.allocate(12);

try (FileChannel fc = (FileChannel.open(file, READ, WRITE))) {
    // Lee los primeros 12
    // bytes del fichero.
    int nread;
    do {
        nread = fc.read(copy);
    } while (nread != -1 && copy.hasRemaining());

    // Escribe "¡Yo estuve aquí!" al principio del fichero.
    fc.position(0);
    while (out.hasRemaining())
        fc.write(out);
    out.rewind();

    // Nos movemos al final del fichero.  Copiamos los primeros 12 bytes al
    // final del fichero.  Entonces escribimos "¡Yo estuve aquí!" de nuevo.
    long length = fc.size();
    fc.position(length-1);
    copy.flip();
    while (copy.hasRemaining())
        fc.write(copy);
    while (out.hasRemaining())
        fc.write(out);
} catch (IOException x) {
    System.out.println("Excepción E/S: " + x);
}
