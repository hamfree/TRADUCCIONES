Respuestas a Preguntas y Ejercicios: E/S Básica

Preguntas

Pregunta 1. ¿Qué clase y método usaría para leer unas pocas piezas de datos que están en posiciones conocidas cerca del final de un fichero grande?

Respuesta 1. Files.newByteChannel devuelve una instancia de SeekableByteChannel el cual le permite leer desde (o escribir a) cualquier posición en el fichero.

Pregunta 2. Cuando invoca format, ¿Cuál es la mejor forma de indicar una nueva línea?

Respuesta 2. Use la conversión %n — ¡el escape \n no es independiente de la plataforma!

Pregunta 3. ¿Cómo determinaría el tipo MIME de un fichero?

Respuesta 3. El método Files.probeContentType usa el detector de tipo de ficheros de la plataforma subyacente para evaluar y devolver el tipo MIME.

Pregunta 4. ¿Qué métodos usaría para determinar si un fichero es un enlace simbólico?

Respuesta 4. Usted usaría el método Files.isSymbolicLink.

Ejercicios

Ejercicio 1. Escriba un ejemplo que cuente el número de veces que un carácter particular, tal como e, aparece en un fichero. El carácter puede ser especificado en la línea de comandos. Puede usar xanadu.txt como el fichero de entrada.

Respuesta 1. Vea CuentaLetra.java para la solución.

Ejercicio 2. El fichero datafile empieza con un único long que le informa del desplazamiento de una sola pieza de datos int dentro del mismo fichero. Escriba un programa que obtenga la pieza de datos int. ¿Qué hay en el dato int?
