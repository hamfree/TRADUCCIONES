Escaneando y Formateando

La programación de E/S a menudo implica la traducción a y desde los datos perfectamente formateados que a los humanos les gusta trabajar. Para asistirle con estos quehaceres, la plataforma Java ofrece dos IPA. La IPA escaneador trocea la entrada en detalles individuales asociados con trocitos de datos. La IPA formateador reorganiza los datos en formato legible por los humanos.
