Recorrido: Clases Esenciales

Este recorrido describe las clases de la plataforma Java que son esenciales para la mayoría de los programadores.

Excepciones explica el mecanismo de excepción y cómo es usado para manejar los errores y otras condiciones excepcionales. Esta lección describe lo que es una excepción, cómo lanzar y capturar excepciones, qué hacer con una excepción una vez ha sido capturada, y cómo usar la jerarquía de clases de excepción.

E/S Básica cubre las clases de la plataforma Java usadas para la entrada y salida básica. Se centra principalmente en los Flujos E/S, un concepto poderoso que simplifica extremadamente las operaciones E/S. La lección también muestra la Serialización, que permite a un programa escribir objetos enteros a flujos de salida y leerlos de vuelta de nuevo. Entonces la lección muestra algunas operaciones del sistema de ficheros, incluyendo el acceso aleatorio a ficheros. Finalmente, toca brevemente las características avanzadsa del nuevo IPA de E/S.

Concurrencia explica como escribir aplicaciones que realizan múltiples tareas simultáneamente. La plataforma Java está diseñada desde sus bases para soportar la programación concurrente, con soporte para la concurrencia básica en el lenguaje de programación Java y las librerías de clases de Java. Desde la versión 5.0, la plataforma Java también ha incluído IPA de concurrencia de alto nivel. Esta lección introduce el soporte básico de concurrencia de la plataforma y resume algunas de las IPA de alto nivel en los paquetes java.util.concurrent.

El Entorno de la Plataforma es definido por el sistema operativo subyacente, la máquina virtual de Java, las librerías de clases, y varios datos de configuración suministrados cuando se lanza la aplicación. Esta lección describe algunas de las IPA que una aplicación usa para examinar y configurar su entorno de plataforma.

Expresiones Regulares son una forma de describir un conjunto de cadenas basadas en características comunes compartidas por cada cadena en el conjunto. Pueden ser usadas para buscar, editar, o manipular texto y datos. Las expresiones regulares varían en complejidad, pero una vez entiende lo fundamental de cómo son construídas, será capaz de descifrar (o crear) cualquier expresión regular. Esta lección le enseña la sintaxis de las expresiones regulares soportadas por la IPA java.util.regex, y presenta varios ejemplos funcionales para ilustrar como interactúan los distintos objetos.
