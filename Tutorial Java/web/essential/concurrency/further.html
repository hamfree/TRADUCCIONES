Lecturas Adicionales

  • Concurrent Programming in Java: Design Principles and Pattern (2nd Edition) por Doug Lea. Un trabajo comprensivo por un experto principal, el cual es también el arquitecto del marco de trabajo de concurrencia de la plataforma Java.
  • Java Concurrency in Practice por Brian Goetz, Tim Peierls, Joshua Bloch, Joseph Bowbeer, David Holmes, y Doug Lea. Una guía práctica diseñada para ser accesible al principiante.
  • Effective Java Programming Language Guide (2nd Edition) por Joshua Bloch. Aunque esta es una guía general de programación, su capítulo sobre hilos contiene las "mejores prácticas" eseneciales para la programación concurrente.
  • Concurrency: State Models & Java Programs (2nd Edition), por Jeff Magee y Jeff Kramer. Una introducción a la programación concurrente a través de la combinación del modelado y ejemplos prácticos.
  • Java Concurrente Animado: Animaciones que muestran el uso de las características de concurrencia.
