Lección: Genéricos

por Gilad Bracha

Introducido en J2SE 5.0, esta mejora tan esperada para el sistema de tipos permite que un tipo o método opere en objetos de varios tipos mientras proporciona seguridad de tipos en tiempo de compilación. Agrega seguridad de tipos en tiempo de compilación al Marco de Colecciones y elimina el trabajo pesado del moldeo (conversión de tipos o casting en inglés).
