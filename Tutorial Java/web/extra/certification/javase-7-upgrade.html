Examen de Actualización de Java SE 7

Esta página asocia secciones de los Tutoriales de Java a los temas cubiertos en la Actualización del examen de Programador de Java SE 7. Esta examen está asociado con el certificado "Oracle certified Professional, Java SE 7 Programmer". Los temas cubiertos en este examen son:

  1. Mejoras del Lenguaje
  2. Patrones de Diseño
  3. Aplicaciones de bases de datos con JDBC
  4. Concurrencia
  5. Localización
  6. E/S de Ficheros en Java (NIO.2)

Sección 1: Mejoras del Lenguaje

Tema 1: Usar String en una sentencia switch.

  • La Sentencia switch

Tema 2: Use literales binarios, literales numéricos con guiones bajos.

  • Tipos de Datos Primitivos

Tema 3: Usar try-with-resources.

  • La Sentencia try-with-resources

Tema 4: Utilice la captura múltiple en las declaraciones de excepción.

  • Los Bloques catch

Tema 5: Usar la sintaxis de diamante con las declaraciones genéricas.

  • El Diamante

Tema 6: Use repeticiones más precisas en excepciones.

  • Relanzar excepciones con una verificación de tipo más inclusiva (Guías Java)

Sección 2: Patrones de Diseño

Los Tutoriales de Java no cubren los temas de Patrones de Diseño. Las referencias siguientes cubren los patrones de diseño usando el lenguaje de programación Java:

  • Head First Design Patterns por Elizabeth Freeman, y otros
  • Java Design Pattern Essentials por Tony Bevis

Tema 1: Diseñar una clase usando el patrón de diseño Singleton.

Tema 2: Identificar cuando y como usar la composición para resolver problemas de negocio.

Tema 3: Escribir código para implementar el patrón DAO.

Tema 4: Diseñar una clase que use el patrón de diseño Factoría.

  • Lo Básico de JDBC: Empezando

  • Estableciendo una Conexión
  • Conectando con Objetos DataSource

Tema 3: Usar los nuevos interfaces de JDBC 4.1 RowSetProvider, RowSetFactory, y RowSet.

  • Usando Objetos RowSet
  • Utilizando Objetos JdbcRowSet

Tema 4: Usar transacciones JDBC.

  • Usando Transacciones

Tema 5: Use la IPA JDBC adecuada para enviar consultas y leer resultados de la base de datos.

  • Procesando Sentencias SQL con JDBC

Tema 6: Usar las sentencias de JDBC PreparedStatement y CallableStatement.

  • Usando Tipos de Datos Avanzados
  • Usando Procedimientos Almacenados

Tema 5: Usar el marco de trabajo en paralelo Fork/Join.
