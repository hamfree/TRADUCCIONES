Examen del Programador Nivel II

Esta página asigna secciones en los Tutoriales de Java a los temas cubiertos en el examen de Java SE 7 Programmer II. Este examen está asociado con el certificado "Oracle Certified Professional, Java SE 7 Programmer". Los temas cubiertos en este examen son:

  1. Diseño de Clases en Java
  2. Diseño Avanzado de Clases
  3. Principios del Diseño Orientado a Objetos
  4. Genéricos y Colecciones
  5. Procesado de Cadenas
  6. Excepciones y Aserciones
  7. Fundamentos de E/S en Java
  8. E/S de Ficheros en Java (NIO.2)
  9. Creación de Aplicaciones de Bases de Datos con JDBC
  10. Hilos
  11. Concurrencia
  12. Localización
