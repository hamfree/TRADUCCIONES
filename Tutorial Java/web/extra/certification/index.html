Lección: Preparación para la Certificación del Lenguaje del Programador Java

Oracle ofrece tres exámenes de certificación y dos certificados para los programadores de Java SE 7. Los Tutoriales de Java pueden ser un recurso valioso para ayudarle a preparar los exámenes de certificación. Los siguientes enlaces le ayudarán a aprender más sobre la capacitación y certificación de Java SE 7 que ofrece Oracle.

  • Certificación Java de Oracle
  • Entrenamiento Java de Oracle

Los enlaces siguientes ofrecen más información sobre los temas de certificación y su asignación al contenido del tutorial:

  • Examen del Programador Nivel I - Aprobar con éxito el examen Java SE 7 Programmer I da como resultado la certificación "Oracle Certified Associate, Java SE 7 Programmer".
  • Examen del Programador Nivel II - La aprobación exitosa del examen Java SE 7 Programmer II da como resultado la certificación "Oracle Certified Professional, Java SE 7 Programmer".
  • Examen de Actualización de Java SE 7 - La aprobación exitosa de la actualización a Java SE 7 Programmer resulta en la certificación "Oracle Certified Professional, Java SE 7 Programmer".

Vea el Programa de Certificación de Oracle para más información sobre clases, exámenes y prerequisitos.

Para obtener una guía de alto nivel sobre cómo abordar el aprendizaje del lenguaje Java, vea las Rutas de aprendizaje de los Tutoriales de Java.
