Ejemplos

CapabilitiesTest demuestra las distinas capacidades de búfer disponibles para la máquina en la cual se ejecuta.

DisplayModeTest muestra una aplicación Swing que usa representación pasiva. Si se encuentra disponible el modo exclusivo de pantalla completa, activará el modo exclusivo de pantalla completa. Si se permite el cambio del modo de pantalla, le permite cambiar entre modos de pantalla.

MultiBufferTest entra en modo de pantalla completa y usa múltiples búferes a través de un bucle de representación activo.
