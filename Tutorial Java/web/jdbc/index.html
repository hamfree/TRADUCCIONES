Recorrido: Acceso a Bases de Datos JDBC(TM)

La IPA de JDBC™ fue diseñada para simplificar las cosas simples. Esto significa que JDBC hace que las tareas de todos los días en la base de datos sean fáciles. Este recorrido lo guía a través de ejemplos de usar JDBC para ejecutar sentencias SQL comunes, y realizar otros objetivos comunes en las aplicaciones con bases de datos.

Este recorrido se divide en estas lecciones:

  Introducción a JDBC Lista las características de JDBC, describe la Arquitectura de JDBC y revisa los comandos de SQL y los conceptos de la Base de Datos Relacional.

  Lo Básico de JDBC cubre la IPA de JDBC, que está incluída en la versión 6 de Java™ SE.

Al final de la primera lección, sabrá cómo usar la IPA básica de JDBC para crear tablas, insertar valores en ellas, consultar las tablas, recuperar los resultados de las consultas, y actualizar las tablas. En este proceso, aprenderá como usar sentencias simples y sentencias preparadas, y verá un ejemplo de un procedimiento almacenado. También aprenderá cómo realizar transacciones y cómo capturar excepciones y avisos.
