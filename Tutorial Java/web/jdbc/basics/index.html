Lección: Lo Básico de JDBC

En esta lección aprenderá lo básico de la IPA de JDBC.

  • Empezando configura un entorno de desarrollo básico de base de datos y le muestra cómo compilar y ejecutar los ejemplos del tutorial de JDBC.

  • Procesando Sentencias SQL con JDBC describe los pasos necesarios para procesar cualquier declaración SQL. Las páginas que siguen describen estos pasos con más detalle:

    • Estableciendo una Conexión le conecta a su base de datos.

    • Conectando con Objetos DataSource le muestra como conectar a su base de datos con objetos DataSource, la forma preferida de obtener una conexión a una fuente de datos.

    • Manejando SQLExceptions le muestra como gestionar excepciones causadas por errores de la base de datos.

    • Configurando Tablas describe todas las tablas de la base de datos usadas en los ejemplos del tutorial JDBC y cómo crear y rellenar las tablas con la IPA de JDBC y scripts de SQL.

    • Recuperando y Modificando Valores de los Result Sets desarrolla el proceso de configurar su base de datos, enviar consultas, y recuperar datos desde su base de datos.

    • Usando Prepared Statements describe una forma más flexible de crear consultas de bases de datos.

    • Usando Transacciones le muestra cómo controlar cuando se ejecuta la consulta a un base de datos.

  • Usando Objetos RowSet le presenta los objetos RowSet; estos son objetos que contienen datos tabulares de una manera que los hace más flexibles y fáciles de usar que los conjuntos de resultados. Las páginas siguientes describen los diferentes tipos de objetos RowSet disponibles:

    • Utilizando Objetos JdbcRowSet

    • Usando CachedRowSetObjects

    • Utilizando Objetos JoinRowSet

    • Utilizando Objetos FilteredRowSet

    • Utilizando Objetos WebRowSet

  • Usando Tipos de Datos Avanzados le presenta otros tipos de datos; las páginas que siguen describen estos tipos de datos con más detalle:

    • Utilizando Objetos Grandes

    • Utilizando Objetos SQLXML

    • Utilizando Objetos Array

    • Usando el Tipo de Datos DISTINCT

    • Utilizando Objetos Estructurados

    • Usando Mapeados de Tipos Personalizados

    • Utilizando Objetos Datalink

    • Utilizando Objetos RowId

  • Usando Procedimientos Almacenados muestra cómo crear y usar un procedimiento almacenado, el cual es un grupo de sentencias SQL que pueden ser llamadas como un método Java con parámetros variables de entrada y salida.

  • Usando JDBC con una IPA de IGU demuestra cómo integrar JDBC con la IPA de Swing.
