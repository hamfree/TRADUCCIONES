Lección: Introducción a JAXB

La Arquitectura Java para Vinculación XML (JAXB) proporciona una forma rápida y conveniente para vincular esquemas XML y representaciones Java, facilitándole que los desarrolladores de Java incorporen datos XML y funciones de procesamiento en aplicaciones Java. Como parte de este proceso, JAXB proporciona métodos para separar (Nota del Traductor: el original en inglés es "unmarshalling" y la traducción es aproximada) instancias de documentos XML en árboles de contenido Java, y después agrupar (Nota del Traductor: el original en inglés es "marshalling" y la traducción es aproximada) árboles de contenido Java de vuelta a instancias de documentos XML. JAXB también proporciona una forma de generar esquemas de XML para objetos Java.

JAXB 2.0 incluye varias mejoras importantes a JAXB 1.0:

  • Admite todas las características de los Esquemas XML del W3C. (JAXB 1.0 no especificó vínculos para algunas de las características del Esquema XML del W3C).
  • Soporte para vinculación Java a XML, con la adicción del paquete javax.xml.bind.annotation para controlar esta vinculación. (JAXB 1.0 especificó la asignación del Esquema XML a Java, pero no el de Java a XML).
  • Un reducción significativa en el número de clases generadas derivadas del esquema.
  • Capacidades de validaci´no adicionales a través de las IPAs de validación de JAXP 1.3.
  • Librerías de tiempo de ejecución más pequeñas.

Esta lección describe la arquitectura JAXB, funciones, y conceptos principales, y proporciona ejemplos con los procedimientos paso a paso para usar JAXB.
