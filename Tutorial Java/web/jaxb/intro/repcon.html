Representando Contenido XML

Esta sección describe como JAXB representa contenido XML como objetos de Java.

Representación Java del Esquema XML

JAXB admite la agrupación de las clases generadas en paquetes Java. Un paquete consiste de lo siguiente:

  • Un nombre de clase Java que se deriva del elemento nombre de XML o se especifica por una personalización de la vinculación.
  • Una clase ObjectFactory, que es una factoría que se usa para devolver instancias de una clase Java vinculada.
