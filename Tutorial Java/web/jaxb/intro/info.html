Para Más Información

Para obtener más información sobre JAXB, XML, y Esquema XML, vea:

  • Página del Proceso de la Comunidad de Java (JCP o Java Community Process, en inglés):

    http://jcp.org/en/jsr/detail?id=222

  • Recomendación del W3C "Lenguaje de Marcado Extensible (XML) 1.0":

    http://www.w3.org/TR/REC-xml

  • Conjunto de Información XML:

    http://www.w3.org/TR/xml-infoset/

  • Esquema XML W3C Parte 0: Introducción, editado por David C. Fallside:

    http://www.w3.org/TR/xmlschema-0/

  • Lenguaje de Rutas XML (XPath), editado por James Clark y Steve DeRose:

    http://www.w3.org/TR/1999/REC-xpath-19991116
