Personalizando Clases Generadas y Elementos del Programa Java

Las secciones siguientes describen cómo personalizar las clases JAXB generadas y los elementos de programa Java.

Esquema a Java

Las declaraciones personalizadas de vinculación JAXB le habilitan a personalizar sus clases JAXB generadas más allá de las restricciones específicas de XML en un esquema XML para incluir refinamientos específicos de Java, tal com los mapeos de nombre de clase y paquete.

JAXB proporciona dos formas de personalizar un esquema XML:

  • Como anotaciones en línea en una fuente de esquema XML
  • Como declaraciones en un fichero de personalización externo de vinculación que se pasa al compilador de vinculación de JAXB

Los ejemplos de código que muestran cómo personalizar los vínculos de JAXB se proporcionan más tarde en este documento.

Java a Esquema

Las anotaciones de JAXB definidas en el paquete javax.xml.bind.annotation pueden usarse para personalizar la asignación de los elementos de programa Java al esquema XML. La tabla siguiente resume las anotaciones JAXB que pueden usarse con un paquete Java.

Tabla: Anotaciones JAXB Asociadas con un Paquete Java

Anotaciones JAXB
Anotacion Descripción y Configuración Predeterminada
@XmlSchema Mapea un paquete a un espacio de nombres XML destino.
Configuración predeterminada: 
@XmlSchema ( 
    xmlns = {}, 
    namespace = "", 
    elementFormDefault = XmlNsForm.UNSET, 
    attributeFormDefault = XmlNsForm.UNSET
)
@XmlAccessorType Controla la serialización predeterminada de los campos y propiedades.
Configuración predeterminada: 
@XmlAccessorType (
    value = AccessType.PUBLIC_MEMBER 
)
@XmlAccessorOrder Controla la ordenación predeterminada de las propiedades y campos mapeados a elementos XML.
Configuración predeterminada: 
@XmlAccessorOrder (
    value = AccessorOrder.UNDEFINED
)
@XmlSchemaType Permite un mapeo personalizado a a un tipo integrado en el Esquema XML.
Configuración predeterminada: 
@XmlSchemaType (
    namespace = 
    "http://www.w3.org/2001/XMLSchema", 
    type = DEFAULT.class
)
@XmlSchemaTypes Una anotación de contenedor para definir múltiples anotaciones @XmlSchemaType.
Configuración predeterminada: Ninguna

Anotación Descripción y Configuración Predeterminada
@XmlElement Mapea una propiedad o campo de un JavaBean a un elemento XML derivado de un nombre de campo o propiedad.

Configuración predeterminada: 
@XmlElementRef (
    name = "##default",

    type = DEFAULT.class
)
@XmlElementRefs Una anotación de contenedor para definir múltiples anotaciones @XmlElementRef.
Configuración predeterminada: Ninguna
@XmlElementWrapper Genera un elemento encapsulador alrededor de una representación XML. Se usa habitualmente como un elemento encapsulador XML alrededor de colecciones.
Configuración predeterminada: 
@XmlElementWrapper (
    name = "##default", 
    namespace = "##default", 
    nillable = false
)
@XmlAnyElement Mapea una propiedad JavaBean a una representación de un conjunto de informaicón XML o un elemento JAXB.
Configuración predeterminada: 
@XmlAnyElement (
    lax = false,

)
@XmlAttribute Mapea una propiedad de JavaBean a un atributo XML.
Configuración predeterminada: 
@XmlAttribute (
    name = ##default, 
    required = false, 
    namespace = "##default" 
)
@XmlAnyAttribute Mapea una propiedad de JavaBean a un mapa de atributos comodín.
Configuración predeterminada: Ninguna
@XmlTransient Impide la asignación de una propiedad JavaBeans a una representación XML.
Configuración predeterminada: Ninguna
@XmlValue Define la asignación de una clase a un tipo complejo del Esquema XML con un simpleContent o un tipo simple de Esquema XML.
Configuración predeterminada: Ninguna
@XmlID Mapea una propiedad de JavaBean a un ID de XML.
Configuración predeterminada: Ninguna
@XmlIDREF Mapea una propiedad de JavaBean a un IDREF de XML.
Configuración predeterminada: Ninguna
@XmlList Mapea una propiedad a un tipo de lista simple.
Configuración predeterminada: Ninguna
@XmlMixed Marca una propiedad de varios valores de JavaBeans para admitir contenido mixto.
Configuración predeterminada: Ninguna
@XmlMimeType Asocia el tipo MIME que controla la representación XML de la propiedad.
Configuración predeterminada: Ninguna
@XmlAttachmentRef Marca un campo/propiedad que su forma XML es una referencia URI al contenido mime.
Configuración predeterminada: Ninguna
@XmlInlineBinaryData Deshabilita la consideración de codificación XOP para tipos de datos que están vinculados a datos binarios codificados en base 64 en XML.
