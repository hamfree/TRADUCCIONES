Vinculando Esquemas XML

Esta sección describe las vinculaciones XML a Java predeterminadas usadas por JAXB. Todos estos vínculos pueden ser sobreescritos globalmente o caso por caso usando una declaración personalizada de vinculación. Vea la Especificación JAXB para obtener una información completa sobre las vinculaciones predeterminadas de JAXB.

Definiciones de Tipo Simple

Un componente de esquema que utiliza una definición de tipo simple normalmente se vincula a una propiedad de Java. Ya que hay diferentes clases de componentes de esquema, los siguientes atributos de propiedad de Java (comunes a los componentes del esquema) incluyen:

  • Tipo base
  • Tipo Colección, si lo hay
  • Predicado

El resto de los atributos de propiedad de Java se especifican en el componente de esquema usando la definición de tipo simple.

Objeto JAXBElement

Cuando la información de un elemento XML no puede ser inferida por la representación de Java derivada del contenido XML, se proporciona un objeto JAXBElement. Este objeto tiene métodos para obtener y establecer el nombre del objeto y el valor del objeto.
