Preguntas y Ejercicios: Implementaciones

Preguntas

  1. Usted planea escribir un programa que usa varias interfaces básicas de colección: Set, List, Queue, y Map. No está seguro de qué implementaciones trabajarán mejor, así que decide usar las implementaciones de uso general hasta que tenga una mejor idea de cómo su programa funcionará en el mundo real. ¿Qué implementaciones son estas?
  2. Si necesita una implementación Set que suministre una iteración ordenada-por-valor ¿qué clase usaría?
  3. ¿Qué clase usa para acceder a las implementaciones de envoltura?

Ejercicios

  1. Escriba un programa que lea un fichero de texto, especificado por el primer argumento de la línea de comandos, en una List. El programa imprimirá después líneas aleatorias del fichero, el número de líneas impresas estará especificado por el segundo argumento de la línea de comandos. Escriba el programa de forma que se asigne una colección de tamaño correcto, de una vez, en vez de ser gradualmente expandida mientras se lee el fichero. Consejo: Para determinar el número de líneas en el fichero, use java.io.File.length para obtener el tamaño del fichero, y después divida por el tamaño supuesto de una línea promedio.

Compruebe sus respuestas.
