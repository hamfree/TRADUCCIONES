La Interfaz Deque

Pronunciado usualmente como deck, una deque es una cola de doble final. Una cola de doble final es una colección lineal de elementos que admite la inserción y eliminación de elementos en ambos puntos finales. La interfaz Deque es un tipo de datos abstracto más rico que Stack y Queue porque implementa tanto las pilas como las colas al mismo tiempo. La interfaz Deque , define métodos para acceder a los elementos en ambos extremos de la instancia Deque. Los métodos son suministrados para insertar, eliminar, y examinar los elementos. Las clases predefinidas como ArrayDeque y LinkedList implementan el interfaz Deque.

Dese cuenta de que el interfaz Deque puede ser usado tanto como pilas últimas-en-entrar-primeras-en-salir como en colas primeras-en-entrar-primeras-en-salir. Los métodos dados en el interfaz Deque están divididos en tres partes:

Insertar

Los métodos addfirst y offerFirst insertan elementos al principio de la instancia Deque. Los métodos addLast y offerLast insertan elementos al final de la instancia Deque. Cuando la capacidad de la instancia Deque está limitada, los métodos preferidos son offerFirst y offerLast porque addFirst podría fallar para lanzar una excepción si ésta está llena.

Eliminar

Los métodos removeFirst y pollFirst eliminan elementos desde el principio de la instancia Deque. Los métodos removeLast y pollLast eliminan elementos del fina. Los métodos pollFirst y pollLast devuelven null si la Deque está vacía mientras que los métodos removeFirst y removeLast lanzan una excepción si la instancia Deque está vacía.

Recuperar

Los métodos getFirst y peekFirst recuperan el primer elemento de la instancia Deque. Estos métodos no eliminan el valor de la instancia Deque. Del mismo modo, los métodos getLast y peekLast recuperan el último elemento. Los métodos getFirst y getLast lanzan una excepción si la instancia deque está vacía mientras que los métodos peekFirst y peekLast devuelven NULL.

Además de estos métodos básicos para insertar, eliminar y examinar una instancia Deque, la interfaz Deque también tiene algunos métodos predefinidos más. Uno de estos es removeFirstOccurence, este método elimina la primera ocurrencia del elemento especificado si existiera en la instancia Deque. Si el elemento no existe entonces la instancia Deque permanece inalterada. Otro método similar es removeLastOccurence; este método elimina la última ocurrencia del elemento especificado en la instancia Deque. El tipo de retorno de estos métodos es un boolean, y devuelven true si el elemento existen en la instancia Deque.
