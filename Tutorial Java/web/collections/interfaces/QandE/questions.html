Preguntas y Ejercicios: Interfaces

Preguntas

  1. Al principio de esta lección, aprendió que el núcleo de interfaces de colección está organizado en dos árboles de herencia distintos. Un interfaz en particular no es considerado una Collection verdadera, y por lo tanto se sienta en la cima de su propio árbol. ¿Cuál es el nombre de esta interfaz?
  2. Cada interfaz en el marco de trabajo de colecciones se declara con la sintaxis <E>, la cual le indica que es un genérico. Cuando declara una instancia Collection, ¿cuál es la ventaja de especificar el tipo de objetos que lo contendrá?
  3. ¿Qué interfaz representa una colección que no permite elementos duplicados?
  4. ¿Qué interfaz forma la raíz de la jerarquía de colecciones?
  5. ¿Qué interfaz representa una colección ordenada que puede contener elementos duplicados?
  6. ¿Qué interfaz representa una colección que contiene elementos antes del procesamiento?
  7. ¿Qué interfaz representa un tipo que asoica claves a valores?
  8. ¿Qué interfaz representa una cola de doble extremo?
  9. Nombra tres formas diferentes para iterar sobre los elementos de una List.
  10. Verdadero o Falso: Las operaciónes de agregación son operaciones mutativas que modifican la colección subyacente.

Ejercicios

  1. Escriba un programa que imprima sus argumentos en orden aleatorio. No haga una copia de la matriz de argumentos. Demuestra cómo imprimir los elementos usando ambos flujos y la sentencia for mejorada tradicional.
  2. Toma el ejemplo FindDups y modifíquelo para usar un SortedSet en vez de un Set. Especifique un Comparator por lo que ese caso se ignora cuando se ordena e identifican elementos establecidos.
  3. Escriba un método que toma una List<String> y aplique String.trim a cada elemento.
  4. Considere las cuatro interfaces principales, Set, List, Queue, y Map. Para cada una de las siguientes cuatro asignaciones, especifique cuál de las cuatro interfaces principales se ajusta mejor, y explique cómo usarla para implementar la asignación.
    1. Whimsical Toys Inc (WTI) necesita grabar los nombres de todos sus empleados. Cada mes, un empleado será elegido aleatoriamente desde estos registros para recibir un juguete gratis.
    2. WTI ha decidido que cada producto nuevo sea nombrado después de que un empleado pero sólo los nombres se usarán, y cada nombre será usado sólo una vez. Prepare una lista de nombre únicos.
    3. WTI decide que sólo quiere usar los nombres más populares para sus juguetes. Cuente el número de empleados que tienen cada nombre.
    4. WTI adquiere boletos de temporada para el equipo local de lacrosse, para ser compartido por empleados. Cree una lista de espera para este deporte popular.

Compruebe sus respuestas.
