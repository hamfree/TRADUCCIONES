Lección: Interoperabilidad

En esta sección, aprenderá sobre los dos aspectos siguientes de interoperabilidad:

  • Compatibilidad: Esta subsección describe cómo las colecciones pueden ser hechas para trabajar con viejas IPAs que son anteriores a la adicción de las Collections a la plataforma Java.
  • Diseño IPA: Esta subsescción describe como diseñar nuevas IPAs de forma que que interactúen sin problemas entre sí.
