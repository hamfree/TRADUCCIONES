Tabla del IPA de Listeners

En la tabla que sigue, la primera columna da el nombre del interfaz del listener, con un enlace a la página del tutorial que describe el listener o, si el tutorial no describe el listener, un enlace a los documentos de la IPA. La columna segunda nombra la clase adaptadora correspondiente, si la hay. (Para una descripción del uso de adaptadores, vea see Usar Adaptadores y Clases Internas para Manejar Eventos.) La tercera columna lista los métodos que el interfaz del listener contiene y muestra el tipo de objeto de evento que se la pasa al método. Típicamente, el listener, el adaptador, y el tipo de evento tienen el mismo prefijo de nombre, pero no siempre es el caso.

Para ver qué componentes de Swing pueden disparar qué clases de eventos, vea Listeners Soportados por los Componentes Swing.
