Lección: Escribiendo Listener Events

Índices de Ejemplos

Esta lección le da detalles sobre la escritura de los event listeners (Nota del Traductor: En español sería algo así como "escuchadores de eventos", se usará la versión original).
Podría no necesitar leer esta sección. Su primera fuente de información para eventos debería ser la sección Cómo para el componente en cuestión. Cada sección de componentes muestra el código para manejar los eventos más necesitados normalmente cuando se implementa el componente. Por ejemplo, Cómo utilizar Casillas de Verificación muestra como manejar pulsaciones del ratón en las casillas de verificación usando un listener de elemento (Nota del Traductor: item listener en inglés).

Algunos Ejemplos Simples de Manejo-de-Eventos

Los programas en esta sección ilustran los eventos y la gestión de eventos.

Información General sobre la Escritura de Listener Events

Esta sección proporciona información que es útil para gestionar todos los tipos de eventos. Uno de los temas incluye información sobre el uso de clases adaptadoras e internas para implementar gestores de eventos.

Listeners Soportados por los Componentes Swing

Este es el lugar para averiguar qué componentes de Swing pueden activar qué tipos de eventos. Es posible que desee marcar esta sección para encontrar fácilmente su tabla de referencia rápida.

Implementando Listeners para los Eventos Comunmente Manejados

Esta sección tiene información detallada y ejemplos de cómo escribir cada tipo común de event listener.

Tabla del IPA de Listeners

Esta sección presenta una tabla de referencia rápida que muestra cada listener, su clase de adaptador (si corresponde), y sus métodos.
