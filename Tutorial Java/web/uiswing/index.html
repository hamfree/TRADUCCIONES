Recorrido: Creando una IGU (Interfaz Gráfica de Usuario) con JFC/Swing

También conocido como El Tutorial de Swing

Este recorrido le indica como crear interfaces gráficos de usuario (IGUs) para aplicaciones y applets, usando los componentes de Swing. Si desea incorporar JavaFX en su aplicación Swing, por favor vea Integración de JavaFX en Aplicaciones Swing.

Empezando con Swing es una lección de inicio rápido. Primero le pone en antecedentes sobre Swing. Después le indica cómo compilar y ejecutar programas que usen componentes Swing.

Aprendiendo Swing con el EID (Entorno Integrado de Desarrollo) NetBeans es la manera más rápida y fácil de empezar a trabajar con Swing. Esta lección explora el constructor de IGUs del EID Netbeans, una característica poderosa que le permite construir visualmente sus Interfaces Gráficas de Usuario.

Usando Componentes Swing le indica cómo usar cada uno de los componentes Swing — botones, tablas, componentes de texto, y todo el resto. También le indica cómo usar bordes e iconos.

Concurrencia en Swing describe la concurrencia tal como es aplicada a la programación en Swing. Se incluye información sobre el hilo de envío de eventos y la clase SwingWorker.

Usando Otras Características de Swing le indica cómo usar acciones, temporizadores, y la bandeja del sistema; cómo integrarse con el escritorio, cómo soportar tecnologías de asistencia, cómo imprimir tablas y texto, cómo crear una pantalla de bienvenida, y cómo usar la modalidad en los diálogos.

Disponer Componentes Dentro de un Contenedor le indica cómo elegir un gestor de disposición, cómo usar cada una de las clases de los gestores de disposición que la plataforma Java ofrece, cómo usar posicionamiento absoluto en vez de un gestor de disposición, y cómo crear su propio gestor de disposición.

Modificando la Apariencia le indica cómo especificar la apariencia de los componentes Swing.

Arrastrar y Soltar y Transferencia de Datos le indica lo que usted necesita saber para implementar la transferencia de datos en su aplicación.

Escribiendo Listener Events le indica cómo manejar eventos en sus programas.

Realización de Pintado Personalizado le da información sobre cómo pintar sus propios componentes Swing. Describe los problemas específicos del pintado de componentes Swing, ofrece una visión de conjunto de los conceptos de pintado, y tiene ejemplos de componentes personalizados que se pintan a sí mismos.

Otros Recorridos Relacionados con la Interfaz de Usuario (IU)

Aunque este es el recorrido principal para aprender sobre IGUs, no es el único recorrido con información relacionada con la IU.

  • Gráficos 2D, la cual describe las características de gráficos 2D disponibles en el KDJ (Kit de Desarrollo Java o JDK, en inglés).
  • Sonido, la cual describe las capacidades de sonido disponibles en el KDJ.
  • Applets de Java, la cual describe la IPA (Interfaz de Programación de Aplicaciones) disponible sólo en los applets.
  • Clases Esenciales, la cual cubre muchos tópicos, incluyendo las propiedades y los flujos de E/S estándar.
  • La Documentación de JavaFX, la cual describe cómo construir IUs con JavaFX.
  • El recorrido de Bonificación contiene la IPA del Modo Exclusivo de Pantalla-Completa, una lección que describe cómo usar la IPA introducida en la versión 1.4 para renderizar gráficos directamente en la pantalla.
