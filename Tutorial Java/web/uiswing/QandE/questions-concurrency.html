Preguntas y Ejercicios: Concurrencia en Swing

Preguntas

  1. Para cada una de las siguientes tareas, especifique en qué hilo debería ser ejecutado y porqué.
    • Iniciar el IGU.
    • Cargar un gran fichero.
    • Invocar javax.swing.JComponent.setFont para cambiar la fuente de un componente.
    • Invocar javax.swing.text.JTextComponent.setText para cambiar el texto de un componente.
  2. Un conjunto de hilos no es usado para ninguna de las tareas mencionadas en la pregunta previa. Nombre este hilo y explique porqué sus aplicaciones son muy limitadas.
  3. SwingWorker tiene dos parámetros de tipos. Explique cómo estos parámetros de tipos son usados, y porque a menudo no importa cuáles son.

Ejercicios

  1. Modifique el ejemplo Flipper de forma que haga una pausa de 5 segundos entre las "tiradas de monedas." Si el usuario pulsa "Cancelar", el bucle de lanzamiento de monedas termina inmediatamente.

Compruebe sus respuestas.
