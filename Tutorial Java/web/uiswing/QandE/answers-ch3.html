Respuestas: Usando Componentes Swing

Use la información en esta lección y el componente secciones cómo para ayudarle a completar estas preguntas y ejercicios.

Preguntas

Pregunta 1: 1. Encuentre el componente que encaje mejor en cada una de las siguientes necesidades. Escriba el nombre común del componente (como "marco") y busque la página de instrucciones del componente en línea.

Pregunta 1a: Un componente que permita al usuario elegir un color.
Respuesta 1a: selector de color

Pregunta 1b: Un componente que muestre un icono, pero que no reaccione a los clics del usuario.
Respuesta 1b: etiqueta

Pregunta 1c:Un componente que parezca como un botón y que, cuando sea presionado, haga aparecer un menú de elementos para que el usuario elija.
Respuesta 1c: cuadro combinado no editable

Pregunta 1d: Un contenedor que parezca como un marco, pero que aparezca (usualmente con otros, contenedores similares) con un marco real.
Respuesta 1d: marco interno

Pregunta 1e: Un contenedor que permita al usuario determinar cómo dos componentes comparten una cantidad limitada de espacio.
Respuesta 1e: panel de desplazamiento

Pregunta 2: ¿Qué método usa para añadir una barra de menú a un contenedor de nivel superior tal como un JFrame?
Respuesta 2: setJMenuBar

Pregunta 3: ¿Qué método usa para especificar el botón por defecto para un contenedor de nivel superior tal como un JFrame o un JDialog?
Respuesta 3: el método setDefaultButton de JRootPane. (Obtiene el panel raíz del contenedor de nivel superior usando el método getRootPane definido por la interfaz RootPaneContainer, la cual implementa todo contenedor de nivel superior).

Pregunta 4: ¿Qué método usa para habilitar o deshabilitar componentes tales como los JButtons? ¿En qué clase está definida?
Respuesta 4: setEnabled , que está definida en la clase Component.

Pregunta 5a: ¿Qué componentes de Swing usan ListSelectionModel? [Consejo: El enlace "Uso" en la parte superior de la especificación para cada interfaz y clase le lleva a una página que le muestra dónde se referencia a ese interfaz o clase en la IPA.]
Respuesta 5a: JList y JTable

Pregunta 5b: D ¿Los componentes usan otros modelos para manejar otros aspectos del estado del componente? Si es así,liste los otros tipos de modelos.
Respuesta 5b: JList también usa un ListModel, el cual mantiene los datos de la lista. JTable usa un TableModel para mantener sus datos y TableColumnModel para gestionar las columnas de la tabla.

Pregunta 6: ¿Qué tipo de modelo almacena el contenido de un campo de texto?
Respuesta 6: Document

Ejercicios

Ejercicio 1.Implemente un programa con una IGU que se vea como la que se muestra abajo. Coloque el método main en una clae llamada MyDemo1.

Respuesta 1: Vea MyDemo1.java . Aquí está el cídog que añade el texto en negrita e itálica: 

JLabel label = new JLabel("My Demo");
frame.getContentPane().add(BorderLayout.CENTER, label);
label.setFont(label.getFont().deriveFont(Font.ITALIC | Font.BOLD));
label.setHorizontalAlignment(JLabel.CENTER)

Ejercicio 2. Haga una copia de MyDemo1.java llamada MyDemo2.java. Agregue una barra de menús a MyDemo2.
Respuesta 2: Vea MyDemo2.java . La bara de menú puede ser implementada con este código: 

JMenu menu = new JMenu("Menu");
JMenuBar mb = new JMenuBar();
mb.add(menu);
frame.setJMenuBar(mb);

Ejercicio 3. Copie MyDemo1.java a MyDemo3.java. Agregue un botón (JButton) a MyDemo3.java. Hágalo el botón por defecto.
Respuesta 3: Vea MyDemo3.java . Aquí está el código que agrega un botón y lo hace su botón por defcto. 

JButton b = new JButton("A button");
frame.getContentPane().add(BorderLayout.PAGE_END, b);
frame.getRootPane().setDefaultButton(b);
