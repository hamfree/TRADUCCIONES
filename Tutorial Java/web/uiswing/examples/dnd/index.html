Arrastrar y Soltar y Transferencia de Datos: Ejemplos

La tabla que sigue lista cada ejemplo en la lección Arrastrar y Soltar y Transferencia de Datos, con enlaces a los ficheros requeridos y donde cada ejemplo es descrito. La primera columna de la tabla tiene enlaces a los ficheros JNLP que le permiten ejecutar los ejemplos usando Java™ Web Start.

NOTA: Se requiere la Versión 7.0 para ejecutar todos los applets y los ejemplos de Java Web Start. La mayoría de los ejemplos se ejecutarán en versiones anteriores pero deberá compilarlos y ejecutarlos localmente.

Para ejecutar un ejemplo usando Java Web Start, pulse el enlace [Lanza] en la primera columna de la tabla. La primera vez que ejecute un ejemplo, habrá un retardo mientras Java Web Start descarga el fichero JAR que contiene los ficheros class de estos ejemplos de la lección. Después, los ejemplos se ejecutarán más rápidamente.

Compilar y Ejecutar los Ejemplos Localmente

La segunda columna en la tabla de debajo tiene enlaces a los ficheros comprimidos para cada demostración que puede abrir y ejecutar en el EID NetBeans. Refíerase a Ejecutando los Ejemplos del Tutorial en el EID NetBeans para más información.

Si usted descarga un ejemplo individual, tenga cuidado en obtener todos los ficheros necesarios en la jeraquía apropiada cuando los compile y ejecute. Todos los ejemplos del tutorial de Swing están colocados en un paquete. Por ejemplo, los ejemplos de componentes están colocados en un paquete components. Vea la siguiente imagen para la estructura completa. Note que cualquier ejemplo que use imágnes espera que sus ficheros de imágenes estén en un directorio llamado images que está en el mismo directorio que el de los ficheros fuentes de los ejemplos.

Puede averiguar qué archivos necesita cada ejemplo consultando la tabla siguiente o mirando los comentarios al principio de cada archivo de origen.
