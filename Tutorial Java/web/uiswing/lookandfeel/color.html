Cambiando el Tema de Color

El aspecto Nimbus tiene un conjunto de colores por defecto, pero no se requiere usarlos. Puede cambiar los colores para que coincidan con los de su marca corporativa u otro esquema de color.

Todos los colores usados por Nimbus son almacenados como un conjunto de propiedades UIManager. Puede cambiar cualquiera o todas esas propiedades antes de que establezca el aspecto. Por ejemplo: 

UIManager.put("nimbusBase", new Color(...));
UIManager.put("nimbusBlueGrey", new Color(...));
UIManager.put("control", new Color(...));

for (LookAndFeelInfo info : UIManager.getInstalledLookAndFeels()) {
    if ("Nimbus".equals(info.getName())) {
        UIManager.setLookAndFeel(info.getClassName());
        break;
    }
}

Estos tres colores base, nimbusBase, nimbusBlueGrey, y control, abordarán la mayoría de sus necesidades. Vea una lista completa de las claves de color y sus valores por defecto en la página Valores por defecto de Nimbus.
