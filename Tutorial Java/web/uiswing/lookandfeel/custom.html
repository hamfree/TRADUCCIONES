Cambiando la Apariencia de Nimbus

Una de las grandes ventajas de Nimbus es que es altamente configurable. Puede cambiar la apariencia en casi todas las formas que pueda imaginar. Puede usar tres técnicas primarias para modificar el aspecto de Nimbus:

  • Cambiando el Tamaño de un Componente – Bajo Nimbus, los componentes están disponibles es tres tamaños adicionales: mini, pequeño, y grande.
  • Cambiando el Tema de Color – Puede cambiar cualquiera de los colores usados en el aspecto Nimbus.
  • Cambiar el aspecto de un componente – Tiene control completo sobre cómo un componente es representado. . Cambiar el aspecto, o personalizar la apariencia de, un componente está más allá del alcance de este tutorial. Vea Para Más Información para más enlaces útiles.
