Lección: Usando Otras Características de Swing

Índice de Ejemplos

Esta lección contiene una colección de páginas cómo para ayudarle a usar características misceláneas de Swing.

Cómo Integrar con la Clase Desktop

Con la clase Desktop puede habilitar a su aplicación Java a interactuar con las aplicaciones por defecto asociadas con tipos específicos de fichero en la plataforma anfitriona.

Cómo crear Ventanas Translúcidas y con Formas

A partir de la versión 10 de Actualización de la Versión 6 de Java Platform, Standard Edition, puede agregar ventanas translúcidas y con formas a sus aplicaciones Swing. Esta lección le muestra cómo.

Cómo Decorar Componentes con la Clase JLayer

JLayer es un poderoso y flexible decorador para componentes de Swing. Le habilita a dibujar componentes y responder a los eventos del componente sin modificar el componente subyacente directamente.

Cómo Usar Acciones

Con los objetos Action, puede coordinar el estado y el manejo de eventos de dos o más componentes que generan eventos de acción. Por ejemplo, puede usar una única Action para crear y coordinar un botón de barra de menús y un elemento de menú que realizan la misma función.

Cómo Usar los Temporizadores de Swing

Con la clase de Swing Timer, puede implementar un hilo que ejecute una acción después de un tiempo, y opcionalmente continuar para repetir la acción. La acción se ejecuta en el hilo despachador de eventos.

Cómo Soportar Tecnologías de Asistencia

Los componentes de Swing tienen soporte incorporado para las tecnologías de asistencia. Su programa puede suministrar incluso mejor soporte siguiendo unas pocas reglas.

Cómo Usar el Subsistema de Foco

Algunos programas necesitan manipular el foco — por ejemplo, para validar la entrada, o cambiar el orden de las tabulaciones de los componentes. Esta sección describe algunas técnicas que puede usar para personalizar el foco en su programa.

Cómo Usar Atajos de Teclado

Con los atajos de teclado, puede especificar cómo los componentes reaccionan a lo tecleado por el usuario.

Cómo Usar la Modalidad en los Diálogos

Esta sección describe un nuevo modelo de modalidad que surgió en Java™ SE version 6 y le habilita a aplicar diferentes tipos de modalidad a los cuadros de diálogo.

Cómo Imprimir Tablas

Esta sección describe las capacidades de impresión para tablas y explica cómo añadir soporte de impresión a su programa.

Cómo Imprimir Texto

Esta sección describe las capacidades de impresión de los componentes de texto y explica cómo añadir soporte de impresión a sus programas.

Cómo Crear una Pantalla de Bienvenida

Con la clase SplashScreen puede cerrar la pantalla de bienvenida, cambiar la imagen de la pantalla de bienvenida, obtener la posición de la imagen o su tamaño, y pintar en la pantalla de bienvenida.

Cómo Usar la Bandeja del Sistema

Esta sección describe como añadir un icono de bandeja a la bandeja del sistema y aplicar un tooltip de texto, un menú emergente, mensajes bocadillo, y un conjunto de oyentes asociados con ellos.

Resolviendo Problemas Comunes Usando Otras Características de Swing

Esta sección le cuenta como arreglar problemas que podría encontrar mientras intenta usar la información en esta lección.

Si está interesado en usar JavaFX para crear su IGU, vea Usar Propiedades y enlaces de JavaFX, Crear Efectos Visuales en JavaFX, Aplicar Transformaciones en JavaFX y Crear Transiciones y Animaciones en el tiempo en JavaFX.
