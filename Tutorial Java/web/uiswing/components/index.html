Lección: Usando Componentes Swing

Índice de Ejemplos

Esta lección le da la información de fondo que necesita para usar los componentes Swing, y entonces describe varios componentes Swing. Se asume que usted ha compilado y ejecutado un programa que usa componentes Swing exitosamente, y que usted está familiarizado con los conceptos básicos de Swing. Estos prerequisitos están cubiertos en Empezando con Swing y Aprendiendo Swing con el EID NetBeans.

Usando Contenedores de Nivel Superior

Describe cómo usar las características compartidas por las clases JFrame, JDialog, y JApplet ─ paneles de contenido, barra de menús, y paneles raíz. También describe la jerarquía de contención, la cual se refiere al árbol de los componentes contenidos por un contenedor de nivel superior.

La Clase JComponent

Le habla sobre las características que JComponent ofrece a sus subclases ─ el cual incluye casi todos los componentes de Swing ─ y da consejos de cómo aprovechar estas características. Ésta sección finaliza con tablas del IPA describiendo la IPA más comunmente usada por JComponent y sus superclases, Container y Component.

Usando Componentes de Texto

Describe las características e IPA compartidas por todos los componentes que descienden de JTextComponent. Usted probablemente no necesita leer esta sección si sólo está usando campos de texto (formateados o no) o áreas de texto.

Como...

Secciones sobre cómo usar cada componente Swing, en orden alfabético. No esperamos que lea estas secciones en orden. En vez de eso, recomendamos leer las secciones relevantes "Cómo" una vez que esté listo para empezar a usar componentes Swing en sus propios programas. Por ejemplo, si su programa necesita un marco, una etiqueta, un botón, y un selector de color, debería leer Cómo Hacer Marcos (Ventanas Principales), Cómo Usar Etiquetas, Cómo Usar Botones, Casillas de Verificación y Botones de Radio, y Cómo Usar Selectores de Color.

Cómo Usar HTML en Componentes Swing

Describe cómo variar la fuente, color, y los otros formateos del texto mostrado por componentes Swing usando etiquetas HTML.

Usando Modelos

Le habla sobre el modelo de arquitectura de Swing. Esta variación del Modelo-Vista-controlador (MVC) significa que uste puede, si lo desea, especificar cómo los datos y el estado de un componente Swing son almacenados y recuperados. Los beneficios son la capacidad de compartir datos y estado entre componentes, y para mejorar en gran medida el rendimiento de los componentes tales como las tablas que visualizan grandes cantidades de datos.

Usando Bordes

Los bordes son muy útiles para dibujar líneas, títulos, y espacio vacío alrededor de los bordes de los componentes. (Podría haber notado que los ejemplos en este recorrido usa un montón de bordes). Esta sección le habla de cómo añadir un borde a cualquier JComponent.

Usando Iconos

Muchos componentes Swing pueden mostrar iconos. Usualmente, los iconos están implementados como instancias de la clase ImageIcon.
