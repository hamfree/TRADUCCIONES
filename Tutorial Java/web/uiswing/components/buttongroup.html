Cómo Usar el Componente Grupo De Botones

El componente ButtonGroup gestiona el estado seleccionado/deseleccionado para un conjunto de botones. Para el grupo, la instancia ButtonGroup garantiza que sólo un botón puede ser seleccionado a la vez.

Inicialmente, todos los botones gestionados por una instancia de ButtonGroup están deseleccionados.

Cómo Usar las Características de ButtonGroup

Puede usar ButtonGroup con cualquier conjunto de objetos que hereden de AbstractButton. Típicamente un grupo de botones contiene instancias de JRadioButton, JRadioButtonMenuItem, o JToggleButton. No tendría sentido colocar una instancia de JButton o JMenuItem en un grupo de botones porque JButton y JMenuItem no implementan el estado seleccionar/deseleccionar del botón.

En general, típicamente seguirá estos pasos para escribir código que usa un componente ButtonGroup.

  1. Subclasifica JFrame
  2. Llama a ContextPane junto con un gestor de disposición
  3. Declara y configura un conjunto de botones de opción o botones conmutables
  4. Instancia un objeto ButtonGroup
  5. Llama al método add sobre ese objeto buttongroup con la intención de añadir cada botón al grupo.

Para los detalles y un ejemplo de código, vea Cómo Utilizar Botones de Opción. Muestra cómo usar un componente ButtonGroup para agrupar un conjunto de RadioButtons establecidos dentro de un JPanel.

Constructores/Métodos Comunmente Usados en Grupo de Botón
Constructor o Metodo Propósito
ButtonGroup() Crea una instancia de ButtonGroup.
void add(AbstractButton)
void remove(AbstractButton) 		Añade un botón al grupo, o elimina un botón del grupo.
public ButtonGroup getGroup()
(en DefaultButtonModel) 		Obtiene el ButtonGroup, si lo hay, que controla un botón. Por ejemplo:
ButtonGroup group = ((DefaultButtonModel)button.getModel()).getGroup();
public ButtonGroup clearSelection() Borra el estado de los botones seleccionados en el ButtonGroup. Ninguno de los botones en el ButtonGroup están seleccionados.
