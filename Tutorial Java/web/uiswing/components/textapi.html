La IPA del Componente de Texto

Esta sección lista las partes usadas comunmente de la IPA que son compartidas por los componentes de texto. La mayoría de esta IPA está definida por la clase JTextComponent. Características del Componente de Texto describe cómo usar alguna de esta IPA.

La Clase JComponent describe la IPA que los componentes de texto heredan de JComponent. Para información sobre la IPA relacionada con componentes de texto específicos, vea la página cómo para ese componente: campo de texto, campo de contraseña, campo de texto formateado, área de texto, o panel de edición y panel de texto.

Para detalles completos sobre la IPA de texto, vea la documentación de la IPA para JTextComponent y para las diversas clases e interfaces en el paquete de texto.

La IPA listada en esta sección incluye las siguientes categorías:

  • Atributos de Configuración
  • Manipular la Selección
  • Convertir Posiciones Entre el Modelo y la Vista
  • Comandos de Edición de Texto
  • Clases e Interfaces Que Representan Documentos
  • Trabajar Con Documentos
  • Manipular Cursores y Marcadores de Selección
  • Leer y Escribir Texto
Atributos de Configuración
Estos métodos están definidos en la clase JTextComponent.
MétodoDescripción
void setEditable(boolean)
boolean isEditable() 		Establece o indica si el usuario puede editar el txto en el componente de texto.
void setDragEnabled(boolean)
boolean getDragEnabled() 		Establece u obtiene la propiedad dragEnabled,la cual debe ser true para habilitar el manejo del arrastre en este componente. El valor por defecto es false. Vea Arrastrar y Soltar y Transferencia de Datos para más detalles.
void setDisabledTextColor(Color)
Color getDisabledTextColor() 		Establece u obtiene el color usado para visualizar el texto cuando el componente de texto está deshabilitado.
void setMargin(Insets)
Insets getMargin() 		Establece u obtiene el margen entre el texto y el borde del componente de texto.

Convertir Posiciones Entre el Modelo y la Vista
Estos métodos están definidos en la clase JTextComponent.
MétodoDescripción

Rectangle modelToView(int) Convierte la posición especificada dentro del texto a un rectángulo en el sistema de coordenadas de la vista.

Clases e Interfaces Que Representan Documentos
Interfaz o ClaseDescripción
Document Un interfaz que defina la IPA que debe ser implementada por todos los documentos.
AbstractDocument Una implementación de superclase abstracta del interfaz de Document. Esta es la superclase para todos los doucmentos ofrecidos por el paquete de texto de Swing.
PlainDocument Una clase que implementa el interfaz Document. Este es el documento por defecto para los componentes de texto plano (campo de texto, campo de contraseña, y área de texto). Adicionalmente, esta clase es usada por los paneles de edición y los paneles de texto cuando cargan texto plano o texto de formato desconocido.
StyledDocument Un subinterfaz de Document. Define una IPA que debe ser implementada por los documentos que soporten texto estilizado. JTextPane requiere que su documento sea de este tipo.
DefaultStyledDocument Una clase que implementa el interfaz StyledDocument. El documento por defecto para JTextPane.

Manipular Cursores y Marcadores de Selección
Estos métodos están definidos en la clase JTextComponent.
Interfaz, Clase, or MétodoDescripción
Caret Un interaz que define la IPA para objetos que representan un punto de inserción dentro de los documentos.
DefaultCaret El cursor por defecto usado por todos los componentes de texto.
void setCaret(Caret)
Caret getCaret() 		Establece u obtiene el objeto cursor usado por un componente de texto.
void setCaretColor(Color)
Color getCaretColor() 		Establece u obtiene el color del cursor.
void setCaretPosition(int)
void moveCaretPosition(int)
int getCaretPosition() 		Establece u obtiene la posición actual del cursor dentro del documento.
void addCaretListener(CaretListener)
void removeCaretListener(CaretListener) 		Añade o elimina un oyente de cursor de un componente de texto.
NavigationFilter La superclase para todos los filtros de navegación. Un filtro de navegación le permite modificar los cambios de cursor que van a ocurrir en un componente de texto.
void setNavigationFilter(NavigationFilter) Vincula un filtro de navegación a un componente de texto.
Highlighter Un interfaz que define la IPA para objetos usados para resaltar la selección actual.
DefaultHighlighter El resaltador por defecto usado por todos los componentes de texto.
void setHighlighter(Highlighter)
Highlighter getHighlighter() 		Establece u obtiene el resaltador usado por un componente de texto.
