Lección: Concurrencia en Swing

Índice de Ejemplos

Esta lección describe la concurrencia tal como se aplica a la programación de Swing. Se asume que ya está familiarizado con el contenido de la lección de Concurrencia en el recorrido Clases Esenciales.

El uso cuidadoso de la concurrencia es particularmente importante para el programador Swing. Un programa de Swing bien escrito usa la concurrencia para crear un interfaz de usuario que nunca se "congela" — el programa siempre responde a la interacción del usuario , sin importar lo que éste esté haciendo. Para crear un programa receptivo, el programador debe aprender cómo el marco de trabajo de Swing emplea los hilos.

Un programador de Swing trata con las clases siguientes de hilos:

  • Hilos iniciales, los hilos que ejecutan el código inicial de la aplicación.
  • El hilo de envío de eventos, donde todo el código de manejo-de-eventos se ejecuta. La mayoría del código que interactúa con el marco de trabajo de Swing debe también ejecutarse en este hilo.
  • Hilos de trabajo, también conocidos como hilos en segundo plano, donde las tareas de fondo que consumen mucho tiempo son ejecutadas.

El programador no necesita suministrar código que cree explícitamente estos hilos: estos son establecidos por el sistema en ejecución o el marco de trabajo de Swing. El trabajo del programador es utilizar estos hilos para crear un programa responsivo y mantenible de Swing.

Como cualquier otro programa ejecutándose en la plataforma Java, un programa de Swing puede crear hilos adicionales y grupos de hilos, usando las herramientas descritas en la lección de Concurrencia. Pero para los programa básicos de Swing los hilos descritos aquí serán suficiente.

Esta lección describe cada una de las tres clases de hilos a su vez. Los hilos de trabajo requieren el mayor análisis porque las tareas que se ejecutan sobre ellos son creadas usando javax.swing.SwingWorker. Esta clase tiene muchas características útiles, incluida la comunicación y coordinación entre las tareas del hilo de trabajo y las tareas en otros hilos.
