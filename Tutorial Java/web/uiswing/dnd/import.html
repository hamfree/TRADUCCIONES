Métodos de Importación

Ahora veremos los métodos usuados para importar datos en un componente. Estos métodos se invocan desde el gesto de soltado, o la acción de pegar, cuando el componente es el destino de la operación. Los métodos de TransferHandler para importar datos son:

  • canImport(TransferHandler.TransferSupport) — Este método se llama repetitivamente durante un gesto de arrastre y devuelve verdadero si el área debajo del cursor puede aceptar la transferencia, o falso si la transferencia será rechazada. Por ejemplo, si un usuario arrastra un color sobre un componente que acepta sólo texto, el método canImport para el TransferHandler del componente devolverá falso.

  • importData(TransferHandler.TransferSupport) — Este método se llama en un gesto de soltado exitoso (o en un pegado) e inicia la transferencia de datos al componente destino. Este método devuelve verdadero si la importación tuvo éxito y falso en caso contrario.

Nota de la versión: 

Estos métodos reemplazan versiones más antiguas que no usan la clase TransferSupport. A diferencia de su método de reemplazo, canImport(JComponent, DataFlavor[]) no se llama continuamente.

Notará que estos métodos de importación toman el argumento TransferHandler.TransferSupport. A continuación, observamos la clase TransferSupport y luego algunos métodos de importación de muestra.
