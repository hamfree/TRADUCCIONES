Demostración - BasicDnD

Ahora veremos una demostración simple, llamada BasicDnD, que le muestra lo que tiene gratis. Como ve en la captura de pantalla, BasicDnD contiene una tabla, una lista, un árbol, un selector de color, un área de texto, y un campo de texto.

Todos estos componentes son componentes estándar listos para usar excepto la lista. Esta lista ha sido personalizada para que aparezca un cuadro de diálogo que muestra dónde ocurriría el soltado, si se han aceptado los soltados.

La áreas siguientes aceptan soltados:

  • Campo de texto
  • Área de texto
  • El selector de color acepta soltados de tipo de color, pero para intentar esto, necesita ejecutar dos copias de la demostración (u otra demostración que contenga un selector de color).

De forma predeterminada, ninguno de los objetos tienen el arrastrar y soltar habilitado por defecto. En el inicio, puede marcar la casilla de verificación "Activar Arrastrar y Soltar" para ver qué comportamiento de arrastrar y soltar obtiene de forma gratuita.

Intente esto: 
  1. Pulse el botón Lanzar para ejecutar BasicDnd usando Java™ Web Start ( descargue KDJ 7 o posterior ). Alternativamente, para compilar y ejecutar el ejemplo por usted mismo, consulte el índice de ejemplos .
  2. Seleccione un ítemn en la lista y, mientras mantiene pulsado el botón del ratón, empieze a arrastrar. No ocurre nada porque el arrastre no ha sido activado aún en la lista.
  3. Seleccione la casilla de verificación "Activar Arrastrar y Soltar".
  4. Presione el ítem seleccionado en la lista y empieze a arrastrar. Suelte el texto de vuelta en la lista. Un diálogo muestra donde el texto aparecería si la lista en realidad acepta soltados. (El comportamiento por defecto para una lista sería mostrar un cursor "no acepto datos").
  5. Arrastre el texto seleccionado sobre el área de texto. El punto de inserción para el texto es indicado por un signo de intercalación intermitente. Además, el cursor cambia para indicar que el área de texto aceptará el texto como una copia.
  6. Suelte el ratón y observe como el texto aparece en el área de texto.
  7. Seleccione algún texto en una de las áreas de texto.
  8. Presione el botón del ratón mientras el cursor se encuentra sobre el texto seleccionado y empieze a arrastrar.
  9. Note que esta vez, el cursor para la acción de arrastrar aparece. Dejar caer este texto con éxito en otro componente hará que el texto se elimine del componente original.
  10. Mantenga la tecla de Control pulsada y presione de nuevo sobre el texto seleccionado. Empieze el arrastre y el cursor de copia aparece ahora. Mueva el cursor sobre el área de texto y suelte. El texto aparece en la nueva ubicación pero no es eliminado de su ubicación original. La tecla Control pude usarse para cambiar cualquier Movimiento a una Copia.
  11. Seleccione un color desde el selector de color. El color seleccionado aparece en el panel de Vista Previa. Presione y mantenga el botón del ratón sobre el color en el panel de Vista Previa y arrástrelo sobre los otros componentes. Note que ninguno de estos componentes aceptan el color.
  12. Intente soltar texto, colores, e incluso ficheros, sobre la lista. Un cuadro de diálogo informará sobre la acción a realizar. La suelta real se puede implementar con seis líneas de código adicionales que se han comentado en el fichero fuente BasicDnD.java .

A continuación veremos la clase TransferHandler, el caballo de batalla del mecanismo de arrastrar y soltar.
