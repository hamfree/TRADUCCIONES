Renderización de la Ubicación del Soltado

Este es un tema más avanzado y la mayoría de la gente no necesita preocuparse sobre esto. Sin embargo, si usted tiene un componente personalizado necesitará manejar la representación de la ubicación de soltado usted mismo.

Puede registrar el ser notificado siempre que la propiedad dropLocation cambie. Escucharía por este cambio y haría su propia representación de la ubicación de soltado en un dibujador personalizado (Nota del Traductor: Render en inglés) para el componente o en el método paintComponent, usando el método getDropLocation.

Aquí tiene un ejemplo de la escucha para la propiedad dropLocation: 

class Repainter extends PropertyChangeListener {
    public void propertyChange(PropertyChangeEvent pce) {
        repaintDropLocation(pce.getOldValue());
        repaintDropLocation(pce.getNewValue());
    }
}

comp.addPropertyChangeListener("dropLocation", newRepainter());

Aquí tiene un ejemplo de la variante paintComponent: 

public void paintComponent(Graphics g) {
    super.paintComponent(g);

    DropLocation loc= getDropLocation();
    if (loc == null) {
        return;
    }

    renderPrettyIndicatorAt(loc);
}
