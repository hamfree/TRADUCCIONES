Lección: Disponer Componentes Dentro de un Contenedor

Índice de ejemplos

Esta lección le indica cómo usar los gestores de diseño suministrados por la plataforma Java. También le indica cómo usar el posicionamiento absoluto (sin gestor de diseño) y le da un ejemplo de escribir un gestor de diseño personalizado. Para cada gestor de diseño (o la falta de él), esta lección apunta a un ejemplo que puede ejecutar usando Java™ Web Start. Al redimensionar la ventana de un ejemplo, puede ver como los cambios de tamaño afectan al diseño.

Nota: Esta lección cubre la escritura de código de diseño a mano, lo cual puede ser desafiante. Si no está interesado en aprender todos los detalles de la gestión del diseño, preferirá usar el gestor de diseño GroupLayout combinado con una herramienta de construcción para diseñar su IGU. Una de tales herramientas de construcción es el EID NetBeans. De otra manera, si quiere codificar a mano y no quiere usar GroupLayout, entonces se recomienda GridBagLayout como el gestor de diseño más flexible y potente.

Si está interesado en usar JavaFX para crear su IGU, vea Trabajar Con Diseños en JavaFX.

Creando un Gestor de Diseño Personalizado

En vez de usar uno de los gestores de diseño de la plataforma Java, puede escribir el suyo propio.

Los gestores de diseño deben implementar el interfaz LayoutManager, el cual especifica los cinco métodos que cada gestor de diseño debe definir. Opcionalmente, los gestores de diseño pueden implementar LayoutManager2, el cual es una subinterfaz de LayoutManager.

Haciendo Sin un Gestor de Diseño (Posicionamiento Absoluto)

Si es necesario, puede posicionar los componentes sin usar un gestor de diseño. Generalmente, esta solución es utilizada para especificar tamaños y posiciones absolutas de los componentes.

Resolviendo Problemas Comunes de Diseño

Algunos de los problemas más comunes con el diseño envuelven a componentes que son visualizados demasiado pequeños — o no se visualizan. Esta sección le indica cómo arreglar estos y otros problemas comunes con el diseño.
