Lección: Trabajando con Cookies

Aunque usted ya esté probablemente familiarizado con las cookies, podría no saber como tomar ventaja de ellas en su aplicación Java. Esta lección le guía a través del concepto de las cookies y explica cómo establecer un manejador de cookies de forma que sus conexiones HTTP URL las usarán.

Java SE proporciona una clase principal para esta funcionalidad java.net.CookieHandler, y las siguientes clases de soporte e interfaces: java.net.CookieManager, java.net.CookiePolicy, java.net.CookieStore, y java.net.HttpCookie.
