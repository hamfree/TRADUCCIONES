Leyendo Directamente de una URL

Después de que ha creado correctamente un objeto URL, puede llamar al método openStream() de URL para obtener un flujo desde el cual puede leer los contenidos de la URL. El método openStream() devuelve un objeto java.io.InputStream, así que leer desde un URL es tan fácil como leer desde un flujo de entrada.

El pequeño programa Java siguiente usa openStream() para obtener un flujo de entrada de la URL http://www.oracle.com/. Entonces abre un BufferedReader sobre el flujo de entrada y lee desde el BufferedReader y lee la URL. Todo lo que se lee se copia al flujo de la salida estándar: 

import java.net.*;
import java.io.*;

public class URLReader {
    public static void main(String[] args) throws Exception {

        URL oracle = new URL("http://www.oracle.com/");
        BufferedReader in = new BufferedReader(
        new InputStreamReader(oracle.openStream()));

        String inputLine;
        while ((inputLine = in.readLine()) != null)
            System.out.println(inputLine);
        in.close();
    }
}

Cuando ejecuta el programa, verá, desplazándose por su ventana de comandos, los comandos HTML y el contenido de texto del fichero HTML ubicado en http://www.oracle.com/. Alternativamente, el programa podría colgarse o podría ver un volcado de pila de una excepción. Si ocurre alguno de los dos últimos eventos, es posible que deba establecer la máquina proxy de forma que el programa puede encontrar el servidor de Oracle.
