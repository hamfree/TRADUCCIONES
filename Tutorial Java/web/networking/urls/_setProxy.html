Detalles Específicos de la Plataforma: Estableciendo la Máquina Proxy

Puede establecer la máquina proxy a través de la línea de comandos. Dependiendo de su configuración de red, puede que necesite establecer también el puerto del proxy. Si es necesario, pregunta a su administrador de sistemas el nombre de la máquina proxy en su red.

UNIX 
java -Dhttp.proxyHost=proxyhost
[-Dhttp.proxyPort=portNumber] URLReader
Consola DOS (Windows 95/NT) 
java -Dhttp.proxyHost=proxyhost
[-Dhttp.proxyPort=portNumber] URLReader
