Lección: Visión General de las Redes

Antes de analizar los ejemplos de las próximas lecciones, debe comprender algunos conceptos básicos de redes. Además, para aumentar su confianza, hemos incluido una sección que revisa lo que quizás ya sepa sobre las redes en Java sin siquiera darse cuenta.

Lo Que Puede Ya Saber sobre Redes en Java

Si ha estado trabajando en los otros recorridos en este tutorial, usted probablemente ha cargado un applet a través de Internet por ahora, y probablemente haya cargado imágenes de la red en los applets que se ejecutan en la red. Todo esto está usando la red y usted ya sabe cómo hacerlo. Esta página destaca las funciones de red que se tratan en otras rutas y lecciones de este tutorial, funciones con las que quizás ya esté familiarizado, y proporciona enlaces a las páginas que tratan sobre esas funciones.

Fundamentos de Redes

Aprenderá lo que necesita saber sobre TCP, UDP, sockets, datagramas, y puertos para aprovechar al máximo las lecciones restantes de este recorrido. Si ya está familiarizado con estos conceptos, no dude en omitir esta sección.
