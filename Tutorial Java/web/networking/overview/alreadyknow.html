Lo Que Puede Ya Saber sobre Redes en Java

La palabra networking infunde miedo en los corazones de muchos programadores. ¡No tema! Usar las capacidades de red proporcionadas en el entorno Java es bastante fácil. De hecho, ¡es posible que ya esté utilizando la red incluso sin siquiera darse cuenta!

Cargando Applets desde la Red

Si tiene acceso a un navegador con Java habilitado, indudablemente ya ha ejecutado muchos applets. Los applets que usted ha ejecutado son referenciados por una etiqueta especial en un fichero HTML — la etiqueta <APPLET>. Los applets pueden estar ubicados en cualquier parte, ya sea en su máquina local o en alguna parte fuera en Internet. La ubicación del applet es completamente invisible para usted, el usuario. Sin embargo, la localización del applet está codificada dentro de la etiqueta <APPLET>. El navegador decodifica esta información, localiza el applet, y lo ejecuta. Si el applet está en otra máquina que no es la suya, el navegador debe descargar el applet antes de que pueda ejecutarlo.

Este es el nivel más alto de acceso que usted tiene a Internet desde el entorno de desarrollo de Java. Alguien más se ha tomado el tiempo de escribir un navegador que hace todo el trabajo duro de conectarse a la red y obtener los datos de ella, lo que le permite ejecutar applets desde cualquier parte del mundo.

La Aplicación "¡Hola Mundo!" le muestra cómo escribir su primer applet y ejecutarlo.

El recorrido Applets de Java describe cómo escribir applets de Java de la A a la Z.

Cargando Imágenes desde URLs

Si se ha aventurado en escribir sus propios applets de Java y aplicaciones, es posible que se haya encontrado con una clase en el paquete java.net llamada URL. Esta clase representa un Localizador Uniforme de Recursos (Nota del Traductor: Uniform Resource Locator o URL en inglés, se usará la abreviatura inglesa) y es la dirección de algún recurso en la red. Sus applets y aplicaciones pueden usar un URL para referenciar e incluso conectar a recursos ahí fuera en la red. Por ejemplo, para cargar una imagen desde la red, su programa Java debe primero crear un URL que contenga la dirección de la imagen.

Este es el siguiente nivel más alto de interacción que puede tener con Internet — su programa Java obtiene una dirección de algo que quiere, crea un URL para ello, y entonces usa alguna función existente en el entorno de desarrollo de Java que hace el trabajo duro de conectarse a la red y recuperar el recurso.

Para más información:
Cómo Usar Iconos le muestra cómo cargar una imagen en su programa Java (ya sean applets o aplicaciones) cuando tiene su URL. Antes de que pueda cargar la imagen debe crear un objeto URL con la dirección del recurso en él.
