¿Qué es un Interfaz de Red?

Una interfaz de red es el punto de interconexión entre una computadora y una red privada o pública. Una interfaz de red es normalmente una tarjeta de interfaz de red (Nota del traductor: Network Interface Card o NIC, en inglés), pero no tiene que tener una forma física. En cambio, la interfaz de red se puede implementar por software. Por ejemplo, la interfaz de bucle local (127.0.0.1 para IPv4 y ::1 para IPv6) no es un dispositivo físico sino una pieza de software que simula una interfaz de red. La interfaz de bucle local es usada habitualmente en entornos de pruebas.

La clase java.net.NetworkInterface representa ambos tipos de interfaces.

NetworkInterface es útil para un sistema multi-hogar, el cual es un sistema con múltiples NICs. Usando NetworkInterface, puede especificar qué NIC usar para una actividad particular de red.

Por ejemplo, asuma que tiene una máquina con dos NICs configuradas, y quiere enviar datos a un servidor. Cree un socket como este: 

Socket soc = new java.net.Socket();
soc.connect(new InetSocketAddress(address, port));

Para enviar los datos, el sistema determina qué interfaz se usa. Sin embargo, si tiene una preferencia o necesita especificar qué NIC usar, puede consultar al sistema las interfaces adecuadas y encontrar una dirección en la interfaz que desea usar. Cuando crea el socket y lo enlaza a esa dirección, el sistema usa el interfaz asociado. Por ejemplo: 

NetworkInterface nif = NetworkInterface.getByName("bge0");
Enumeration<InetAddress> nifAddresses = nif.getInetAddresses();

Socket soc = new java.net.Socket();
soc.bind(new InetSocketAddress(nifAddresses.nextElement(), 0));
soc.connect(new InetSocketAddress(address, port));

Puede también usar NetworkInterface para identificar la interfaz local en la cual un grupo de multidifusión se ha unido. Por ejemplo: 

NetworkInterface nif = NetworkInterface.getByName("bge0");
MulticastSocket ms = new MulticastSocket();
ms.joinGroup(new InetSocketAddress(hostname, port), nif);

NetworkInterface puede usarse con las IPAs de Java de muchas otras formas más allá de los dos usos descritos aquí.
