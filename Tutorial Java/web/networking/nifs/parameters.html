Parámetros de Interfaz de Red

Puede acceder a los parámetros de red sobre una interfaz de red más allá del nombre y las direcciones IP asignadas.

Puede descubrir si la interfaz de red está “arriba” (es decir, ejecutándose) con el método isUP(). Los métodos siguientes indican el tipo de interfaz de red:

  • isLoopback() indica si la interfaz de red es una interfaz de bucle local.
  • isPointToPoint() indica si la interfaz es una interfaz punto-a-punto.
  • isVirtual() indica si la interfaz es una interfaz virtual.

El método supportsMulticast() indica si la interfaz de red soporta multidifusión. El método getHardwareAddress() devuelve la dirección física del hardware de la interfaz de red, usualmente llamada dirección MAC, cuando está disponible. El método getMTU() devuelve la Unidad Máxima de Transmisión (Nota del Traductor: Maximum Transmission Unit o MTU, en inglés), la cual es el tamaño más grande de paquete.

El siguiente ejemplo expande el ejemplo en Listado de Direcciones de Interfaces de Red agregando los parámetros adicionales de red descritos en esta página: 

import java.io.*;
import java.net.*;
import java.util.*;
import static java.lang.System.out;

public class ListNetsEx {

    public static void main(String args[]) throws SocketException {
        Enumeration<NetworkInterface> nets = NetworkInterface.getNetworkInterfaces();
        for (NetworkInterface netint : Collections.list(nets))
            displayInterfaceInformation(netint);
    }

    static void displayInterfaceInformation(NetworkInterface netint) throws SocketException {
        out.printf("Display name: %s\n", netint.getDisplayName());
        out.printf("Name: %s\n", netint.getName());
        Enumeration<InetAddress> inetAddresses = netint.getInetAddresses();
        
        for (InetAddress inetAddress : Collections.list(inetAddresses)) {
            out.printf("InetAddress: %s\n", inetAddress);
        }
       
        out.printf("Up? %s\n", netint.isUp());
        out.printf("Loopback? %s\n", netint.isLoopback());
        out.printf("PointToPoint? %s\n", netint.isPointToPoint());
        out.printf("Supports multicast? %s\n", netint.supportsMulticast());
        out.printf("Virtual? %s\n", netint.isVirtual());
        out.printf("Hardware address: %s\n",
                    Arrays.toString(netint.getHardwareAddress()));
        out.printf("MTU: %s\n", netint.getMTU());
        out.printf("\n");
     }
}
