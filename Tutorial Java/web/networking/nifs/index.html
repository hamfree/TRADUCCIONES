Lección: Acceso Prográmatico a los Parámetros de Red

Los sistemas a menudo se ejecuta con múltiples conexiones de red activas, tales como la Ethernet cableada, 802.11 b/g (inalámbrica), y bluetooth. Algunas aplicaciones podrían necesitar acceder a esta información para realizar la particular actividad de red en una conexión específica.

La clase java.net.NetworkInterface proporciona acceso a esta información.

Esta lección le guía a través de algunos de los usos más comunes de esta clase y proporciona ejemplos que listan todos los interfaces de red en una máquina así como tambien sus direcciones IP y estado.

¿Qué es un Interfaz de Red?

Esta página describe un interfaz de red y explica porqué podría querer usarlo.

Recuperando Interfaces de Red

Esta página contiene un ejemplo que ilustra cómo un programa cliente puede recuperar todos los interfaces de red en una máquina.

Listado de Direcciones de Interfaces de Red

Esta página le muestra cómo listar las direcciones IP asignadas a todos los interfaces de red en una máquina.

Parámetros de Interfaz de Red

Esta página le muestra cómo determinar si un interfaz de red está ejecutándose o si el interfaz de red es un interfaz de bucle local, un interfaz punto-a-punto, o un interfaz virtual. Puede también aprender cómo determinar si el interfaz soporta multidifusión.
