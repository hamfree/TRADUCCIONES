Habilitando el Protocolo SDP

La compatibilidad con SDP está deshabilitada de forma predeterminada. Para habilitar la compatibilidad con SDP, configure la propiedad del sistema com.sun.sdp.conf proporcionando la ubicación del archivo de configuración. El siguiente ejemplo inicia una aplicación usando un archivo de configuración llamado sdp.conf: 

% java -Dcom.sun.sdp.conf=sdp.conf -Djava.net.preferIPv4Stack=true  AplicaciónEjemplo

AplicaciónEjemplo se refiere a la aplicación cliente que intenta conectarse al adaptador IB.

Tenga en cuenta que este ejemplo especifica otra propiedad del sistema, java.net.preferIPv4Stack. Consulte la sección Problemas para obtener más información sobre por qué se utiliza esta propiedad.
