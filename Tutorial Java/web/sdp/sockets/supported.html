IPAs de Java Soportadas

Todas las IPA que usan TCP pueden usar SDP, incluidas específicamente las siguientes clases:

  • paquete java.net:
    • Socket
    • ServerSocket
  • paquete java.nio.channels:
    • SocketChannel
    • ServerSocketChannel
    • AsynchronousSocketChannel
    • AsynchronousServerSocketChannel

Cuando la compatibilidad con SDP está habilitada, simplemente funciona sin ningún cambio en su código. No es necesario compilar. Sin embargo, es importante saber que un socket se enlaza solo una vez. Una conexión es un vínculo implícito. Por lo tanto, si el socket no se ha enlazado previamente y se invoca connect, el enlace se produce en ese momento.

Por ejemplo, considere el siguiente fragmento de código: 

AsynchronousSocketChannel ch = AsynchronousSocketChannel.open();
ch.bind(local);
Future<Void> result = ch.connect(remote);

En este fragmento, el canal de socket asincrónico está vinculado a una dirección TCP local cuando se invoca bind en el socket. Luego, el código intenta conectarse a una dirección remota utilizando el mismo enchufe. Si la dirección remota usa InfiniBand, como se especifica en el archivo de configuración, la conexión no se convertirá a SDP porque el socket se vinculó previamente.
