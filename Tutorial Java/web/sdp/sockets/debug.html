Depurando SDP

Puede habilitar la depuración de mensajes utilizando el indicador -Dcom.sun.sdp.debug. Si especifica un archivo, los mensajes se enviarán a ese archivo. De lo contrario, los mensajes se imprimirán en una salida estándar.

Este primer ejemplo muestra mensajes de muestra impresos en la salida estándar: 

% java -Dcom.sun.sdp.conf=sdp.conf -Dcom.sun.sdp.debug AplicacionEjemplo
BIND to 192.0.2.1:5000 (socket convertido al protocolo SDP)
CONNECT to 129.156.232.160:80 (sin coincidencia)
CONNECT to 192.0.2.2:80 (socket convertido al protocolo SDP)

Este segundo ejemplo muestra la salida redirigida a un archivo llamado debug.log: 

% java -Dcom.sun.sdp.conf=sdp.conf -Dcom.sun.sdp.debug=debug.log AplicacionEjemplo 
[1] 27310
% tail -f debug.log
BIND to 192.0.2.1:5000 (socket convertido al protocolo SDP)
