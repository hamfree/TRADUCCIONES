Soporte de Linux y Solaris

Solaris 10

Para probar que SDP está habilitado, use el comando sdpadm (1M): 

% /usr/sbin/sdpadm status
SDP está habilitado

Puede usar el comando grep para buscar en el archivo /etc/path_to_inst la cadena "ibd" para ver una lista de adaptadores IB que son compatibles con su red.

Otros comandos que pueden resultarle útiles son ib (7D), ibd (7D) y sdp (7D).

Solaris 11

Puede utilizar el siguiente comando para obtener la información del enlace de la partición InfiniBand: 

% dladm show-port -o LINK

Otros comandos que pueden resultarle útiles son ib (7D), ibd (7D) y sdp (7D).

Linux

Puede utilizar el siguiente comando para obtener la lista de adaptadores InfiniBand: 

% egrep "^[ \t]+ib" /proc/net/dev
