Recorrido: Protocolo de Sockets Directos

El Protocolo de Sockets Directos (SDP) proporciona acceso a conexiones de red de alto rendimiento, como la que proporciona InfiniBand. Este recorrido muestra cómo habilitar SDP, qué IPA de Java está involucrado, y explica cómo funciona.
