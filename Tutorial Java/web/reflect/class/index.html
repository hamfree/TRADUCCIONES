Lección: Clases

Cada objeto es una referencia o un tipo primitivo. Todos los tipos de referencia heredan de java.lang.Object. Las clases, enumeraciones, matrices, e interfaces son todos tipos de referencia. Hay un conjunto fijo de tipos primitivos: boolean, byte, short, int, long, char, float, y double. Ejemplos de tipos referencia incluyen java.lang.String, todas las clases contenedoras para tipos primitivos como java.lang.Double, la interfaz java.io.Serializable, y la enumeración javax.swing.SortOrder.

Para cada tipo de objeto, la máquina virtual de Java instancia una instancia inmutable de java.lang.Class que proporciona métodos para examinar las propiedades del objeto en tiempo de ejecución incluyendo sus miembros e información de tipo. Class también proporciona la habilidad de crear nuevas clases y objetos. Lo más importante es que es el punto de entrada para todos la IPA de Reflexión. Esta lección cubre las operaciones de reflexión más comúnmente utilizadas que involucran clases:

  • Recuperando Objetos de la Clase describe la forma de obtener una Class
  • Examinando Modificadores de Clases y Tipos muestra cómo acceder a la información de declaración de clase
  • Descubriendo Miembros de Clases ilustra cómo listar los constructores, campos, métodos, y clases anidadas en una clase
  • Solución de Problemas describe los errores comunes encontrados cuando se usa Class
