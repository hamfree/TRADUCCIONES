Identificando los Tipos del Matriz

Los tipos matriz pueden ser identificados invocando Class.isArray(). para obtener una Class use uno de los métodos descritos en la sección Recuperando Objetos de la Clase de este recorrido.

El ejemplo ArrayFind identifica los campos en la clase nombrada que son del tipo matriz e informa del tipo de componente para cada uno de ellos. 


import java.lang.reflect.Field;
import java.lang.reflect.Type;
import static java.lang.System.out;

public class ArrayFind {
    public static void main(String... args) {
	boolean found = false;
 	try {
	    Class<?> cls = Class.forName(args[0]);
	    Field[] flds = cls.getDeclaredFields();
	    for (Field f : flds) {
 		Class<?> c = f.getType();
		if (c.isArray()) {
		    found = true;
		    out.format("%s%n"
                               + "           Field: %s%n"
			       + "            Type: %s%n"
			       + "  Component Type: %s%n",
			       f, f.getName(), c, c.getComponentType());
		}
	    }
	    if (!found) {
		out.format("No array fields%n");
	    }

        // production code should handle this exception more gracefully
 	} catch (ClassNotFoundException x) {
	    x.printStackTrace();
	}
    }
}

La sintaxis para el valor devuelto de Class.get*Type() se describe en Class.getName(). El número de caracteres '[' al principoo del nombre del tipo indica el número de dimensiones (p.e. profunidad del anidamiento) de la matriz.

A continuación se muestran ejemplos de la salida. La entrada del usuario está en cursiva. Una matriz de tipo primitivo byte: 

$java ArrayFind java.nio.ByteBuffer
final byte[] java.nio.ByteBuffer.hb
           Field: hb
            Type: class [B
  Component Type: byte

Una matriz de tipo referencia StackTraceElement: 

$ java ArrayFind java.lang.Throwable
private java.lang.StackTraceElement[] java.lang.Throwable.stackTrace
           Field: stackTrace
            Type: class [Ljava.lang.StackTraceElement;
  Component Type: class java.lang.StackTraceElement

predefined es una matriz de una dimensión de tipo referencia java.awt.Cursor y cursorProperties es una matriz de dos dimensiones de tipo referencia String: 

$ java ArrayFind java.awt.Cursor
protected static java.awt.Cursor[] java.awt.Cursor.predefined
           Field: predefined
            Type: class [Ljava.awt.Cursor;
  Component Type: class java.awt.Cursor
static final java.lang.String[][] java.awt.Cursor.cursorProperties
           Field: cursorProperties
            Type: class [[Ljava.lang.String;
  Component Type: class [Ljava.lang.String;
