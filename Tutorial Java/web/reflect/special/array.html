Matrices

Una matriz es un objeto de tipo referencia que contiene un número fijo de componentes del mismo tipo; la longitud de una matriz es inmutable. Crear una instancia de una matriz requiere conocer la longitud y el tipo del componente. Cada componente puede ser un tipo primitivo (p.e. byte, int, o double), un tipo referencia (p.e. String, Object, o java.nio.CharBuffer), o una matriz. Las matrices multidimensionales son en realidad simplemente matrices que contienen componetes del tipo matriz.

Las matrices se implementan en la máquina virtual de Java. Los únicos métodos sobre matrices son aquellos heredados de Object. La longitud de una matriz no es parte de su tipo; las matrices tienen un campo length que es accesible a través de java.lang.reflect.Array.getLength().

Reflexión proporciona métodos para acceder a los tipos matriz y a los tipos de componente matriz, crear nuevas matrices, ay recuperar y establecer valores en componentes de matriz. Las secciones siguientes incluyen ejemplos de operaciones comunes sobre matrices:

  • Identificando los Tipos del Matriz describe cómo determinar si un miembro de la clase es un campo de tipo matriz
  • Creando Nuevas Matrices ilustra cómo crear nuevas instancias de matrices con componentes de tipo simple y complejo
  • Obteniendo y estableciendo Matrices y sus Componentes muestra cómo acceder a campos de tipo matriz y acceder individualmente a elementos de la matriz
  • Solución de Problemas cubre los errores comunes y conceptos erróneos de programación

Todas estas operaciones se soportan a través de métodos static en java.lang.reflect.Array.
