Tipos Enumerados

Una enum es una construcción del lenguaje que se usa para definir enumeraciones de tipado seguro que se pueden usar cuando se desea un conjunto fijo de valores con nombre. Todas las enumeraciones implíctamente extienden de java.lang.Enum. Las enumeraciones pueden contener una o más constantes de enumeración, que definen instancias únicas del tipo enumerado. Una declaración de enumeración define un tipo de enumeración que es muy similar a una clase en que puede tener miembros tales como campos, métodos, y constructores (con algunas restricciones).

Ya que las enumeraciones son clases, la reflexión no necesita nada para definir una clase explícita java.lang.reflect.Enum. Las únicas IPA de Reflexión que son específicas parar las enumeraciones son Class.isEnum(), Class.getEnumConstants(), y java.lang.reflect.Field.isEnumConstant(). La mayoría de las operaciones de reflexión que involucran a enumeraciones son las mismas que en cualquier otra clase o miembro. Por ejemplo, las constantes de enumeraicón se implementan como campos public static final en la enumeración. Las secciones siguientes muestran cómo usar Class y java.lang.reflect.Field con enumeraciones.

  • Examinando las Enumeraciones ilustra cómo recuperar unas constantes de enumeración y cualesquiera otros campos, constructores, y métodos
  • Obteniendo y Estableciendo Campos en los Tipos Enumerados muestra cómo establecer y obtener campos con un valor constante de enumeración
  • Solución de Problemas describe los errores comunes asociados con las enumeraciones

Para una introducción a las enumeraciones, vea la lección Tipos Enum.
