Constructores

Un constructor se usa en la creación de un objeto que es una instancia de una clase. Habitualmente este realiza las operaciones requeridas para inicializar la clase antes de que se invoquen los métodos o se acceda a los campos. Los constructores nunca se heredan.

De forma similar a los métodos, la reflexión proporciona una IPA para descubrir y recuperar los constructores de una clase y obtener información de la declaración tal como los modificadores, parámetros, anotaciones, y excepciones lanzadas. También se pueden crear nuevas instancias de clases usando un constructor especificado. Las clases clave que se utilizan al trabajar con constructores son Class y java.lang.reflect.Constructor. Las operaciones comunes que involucran a los constructores están cubiertas en las secciones siguientes:

  • Encontrando Constructores ilustra cómo recuperar constructores con parámetros específicos
  • Recuperando y Analizando Modificadores del Constructor muestra cómo obtener los modificadores de una declaración de constructor y otra información sobre el constructor
  • Creando Nuevas Instancias de Clase muestra cómo instanciar una instancia de un objeto invocando a su constructor
  • Solución de Problemas describe los errores comunes que se pueden encontrar mientras buscamos o invocamos constructores
