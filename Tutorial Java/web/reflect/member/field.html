Campos

Un campo es una clase, interfaz o enumeración con un valor asociados. Los métodos en la clase java.lang.reflect.Field pueden recuperar información sobre el campo, tal como su nombre, tipo, modificadores, y anotaciones. (La sección Examinar Modificadores y Tipos de Clase en la lección Clases describe cómo recuperar anotaciones). Hay también métodos que habilitan el acceso dinámico y la modificación del valor del campo. Estas tareas se cubren en las siguientes secciones:

  • Obteniendo los Tipos de los Campos describe cómo obtener los tipos genéricos y declarados de un campo
  • Recuperando y Analizando Modificadores de los Campos muestra cómo obtener porciones de la declaración del campo tal como public o transient
  • Obteniendo y Estableciendo los Valores de los Campos ilustra cómo acceder al valor o valores del campo
  • Solución de Problemas describe algunos errores comunes de codificación que pueden causar confusión

Cuando escribe una aplicación tal como un navegador de clases, podría ser útil encontrar qué campos pertenecen a una clase particular. Los campos de una clase se identifican invocando Class.getFields(). El método getFields() devuelve una matriz de objetos Field conteniendo un objeto por cada campo público accesible.

Un campo público es accesible si este es un miembro de:

  • esta clase
  • una superclase de esta clase
  • una interfaz implementada por esta clase
  • una interfaz extendida desde una interfaz implementada por esta clase

Un campo puede ser un campo de clase (instancia), como java.io.Reader.lock , un campo estático, tal como java.lang.Integer.MAX_VALUE , o una constante de enumeración, como java.lang.Thread.State.WAITING.
