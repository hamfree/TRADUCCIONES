Lección: Creando y Usando Extensiones

Cualquier conjunto de paquetes o clases pueden ser hechos fácilmente para jugar el rol de una extensión. El primer paso para convertir un conjunto de clases en una extensión es empaquetarlas en un fichero JAR. Una vez hecho esto, puede convertir el software en una extensión de dos formas:

  • colocando el fichero JAR en una ubicación especial en la estructura de directorios del Entorno de Ejecución de Java, en cuyo caso es llamada una extensión instalada.
  • al referenciar el fichero JAR de una forma especificada en el manifiesto de otro fichero JAR, en cuyo caso es llamada una extensión descargada.

Esta lección le muestra cómo trabaja el mecanismo de extensión usando una extensión simple "toy" como ejemplo.
