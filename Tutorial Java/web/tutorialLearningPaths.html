Rutas de Aprendizaje de los Tutoriales de Java

Página de Inicio

¿Es usted un estudiante intentando aprender el lenguaje Java o un profesional buscando expandir su conjunto de habilidades? Si usted se siente un poco abrumado por la amplitud de la plataforma Java, aquí tiene unas pocas rutas de aprendizaje sugeridas para ayudarle a obtener lo máximo de su experiencia de aprendizaje de Java.

Nuevo en Java

Los siguientes recorridos son los más útiles para principiantes:

  • Comenzando – Una introducción a la tecnología Java y lecciones sobre la instalación del software de desarrollo Java y usándolo para crear un programa simple.
  • Aprendiendo el Lenguaje Java – Lecciones describiendo los conceptos esenciales como las clases, objetos, herencia, tipos de datos, tipos genéricos, y paquetes.
  • Clases Java Esenciales – Lecciones sobre excepciones, entrada/salida básica, concurrencia, expresiones regulares, y el entorno de la plataforma.

Basándose en la Fundación

¿Listo para bucear más profundo en la tecnología? Vea los siguientes temas:

  • Colecciones – Lecciones usando y extendiendo el Marco de trabajo de Colecciones de Java.
  • Expresiones Lambda: Aprenda cómo y porqué usar Expresiones Lambda en sus aplicaciones.
  • Operaciones de Agregación: Explore cómo las Operación de Agregación, Flujos y Expresiones Lambda trabajan juntas para proveer de unas capacidades de filtrado poderosas.
  • Empaquetando Programas en Ficheros JAR – Lecciones sobre la creación y firmado de ficheros JAR.
  • Internacionalización – Una introducción al diseño de software de forma que pueda ser facilmente adaptado (traducido) a varios lenguajes y regiones.
  • Reflexión – Una IPA que representa ("refleja") las clases, interfaces y objetos en la Máquina Virtual de Java actual.
  • Seguridad – Características de la platafora Java que ayudan a proteger las aplicaciones del software malicioso.
  • JavaBeans – La tecnología de componentes de la plataforma Java.
  • El Mecanismo de Extensión – Como hacer que las IPA personalizadas estén disponibles a todas las aplicaciones ejecutándose en la plataforma Java.
  • Tipos Genéricos – Una mejora al sistema de tipos que soporta operaciones sobre objetos de varios tipos mientras provee de seguridad de tipos en tiempo de compilación.

¿Aprecia el Cliente?

Si uste quiere enfocarse al desarrollo de soluciones de escritorio y aplicaciones richas de internet desarrrolando en Java , vea los siguientes temas:

  • Creando un IGU (Interfaz Gráfico de Usuario o GUI en inglés) con Swing – Una amplia introducción a la creación de IGU en la plataforma JAVA.
  • Implantación – Cómo empaquetar las aplicaciones y applets usando ficheros JAR, e implantándolos usando Java Web Start y el Plug-in de Java.
  • Gráficos 2D – Cómo visualizar e imprimir gráficos 2D en las aplicaciones.
  • IPA para el Modo Exclusivo de Pantalla-Completa – Cómo escribir aplicaciones que utilizan de forma más plena el hardware de gráficos del usuario.

¡Fervor por el Servidor!

Si usted está interesado en adquirir habilidades importantes para el middleware, el lado del servidor, o desarrollo de aplicaciones web, vea los siguientes recorridos:

  • Acceso a Bases de datos con JDBC – Introduce una IPA para la conectividad entre las aplicaciones Java y un amplio rango de bases de datos y fuentes de datos.
  • JMX – Las Extensiones de Gestión de Java proporcionan una forma estarndar de gestionar recursos tales como aplicaciones, dispositivos y servicios.
  • JNDI – El Interfaz de Nombrado y Directorio de Java (JNDI) activa el acceso a los Servicios de Nombrado y Directorio tales como el DNS y el LDAP.
  • JAXP – Introducción al IPA de Java para el Procesado de XML (JAXP) en tecnología 1.4.
  • RMI – La IPA de Invocación Remota de Métodos permite a un objeto invocar métodos de un objeto que está ejecutándose en otra Maquina Virtual de Java.
  • Concurrencia – La plataforma Java tiene IPA que le ayudan a desarrollar programas mult-hilo.
