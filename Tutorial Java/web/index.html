Los Tutoriales de Java son guías prácticas para programadores que quieren usar el lenguaje de programación Java para crear aplicaciones. Estos incluyen cientos de ejemplos completos y funcionales, y docenas de lecciones. Los grupos de lecciones relacionadas están organizadas en "Recorridos".

Los Tutoriales de Java describen ante todo las características en Java SE 8. Para mejores resultados, descargue KDJ 8.

Qué hay de nuevo

Los Tutoriales de Java son actualizados continuamente para mantenerse al día con los cambios en la plataforma Java e incorporar los comentarios de nuestros lectores.

Esta versión del tutorial corresponde a la versión KDJ 8u40.

Esta versión incluye una nueva lección en el recorrido de Despliegue que describe como usar las herramientas de empaquetado Java para generar aplicaciones auto-contenidas. Las aplicaciones auto-contenidas son aplicaciones Java que son empaquetadas con el EEJ que necesitan para ejecutarse. Estas aplicaciones son instaladas en el disco local del usuario y lanzadas de la misma forma que las aplicaciones nativas. Vea Desplegando Aplicaciones Auto-Contenidas para más información.

Recorridos Cubriendo lo Básico

Estos recorridos están disponibles en forma de libro como El Tutorial de Java, Sexta Edición. Para comprar este libro, consulte el cuadro de la derecha.
  • Empezando — Una introducción a la tecnología Java y lecciones sobre la instalación del software de desarrollo de Java y usarlo para crear un programa simple.
  • Aprendiendo el Lenguaje Java — Lecciones describiendo los conceptos esenciales y las características del Lenguaje de Programación Java.
  • Clases de Java Esenciales — Lecciones sobre las excepciones, entrada/salida básica, concurrencia, expresiones regulares, y el entorno de la plataforma.
  • Colecciones — Lecciones sobre el uso y ampliación del Framework de Colecciones de Java.
  • IPA de Fecha-Tiempo — Como usar las pagínas java.time para escribir código de fecha y tiempo.
  • Despliegue — Como los paquetes de aplicaciones y applets usan los ficheros JAR, y desplegarlas usando Java Web Star y el Plug-in de Java.
  • Preparación para la Certificación del Lenguaje de Programación Java  — Listado de todos los recursos disponibles de tutoriales y formación.

Creando Interfaces Gráficas de Usuario

  • Creando un IGU (Interfaz Gráfica de Usuario, GUI en inglés) con Swing  — Una introducción comprensiva a la creación del IGU en la plataforma de Java.
  • Creando un IGU en JavaFX — Una colección de tutoriales JavaFX.

Recorridos Especializados y Lecciones

Estos recorridos y lecciones sólo están disponibles como páginas web.

  • Redes personalizadas — Una introducción a las potentes capacidades de red de la plataforma de Java.
  • El Mecanismo de Extensión — Cómo hacer disponibles IPA personalizadas a todas las aplicaciones ejecutándose en la plataforma Java.
  • IPA del Modo Exclusivo de Pantalla-Completa— Cómo escribir aplicaciones que utilizen más completamente el hardare de gráficos del usuario.
  • Genéricos— Una mejora al sistema de tipos que soporta operaciones sobre objetos de varios tipos mientras proporciona seguridad en tiempos de compilación. Note que esta lección es para usuarios avanzados. Este recorrido de Lenguaje Java contiene una lección de Genéricos que es adecuada para principiantes.
  • Internacionalización— Una introducción al diseño de software que puede ser facilmente adaptado (traducido) a varios lenguajes y regiones.
  • JavaBeans — La tecnología de componentes de la plataforma Java.
  • Acceso a Bases de datos JDBC — Introduce una IPA para la conectividad entre las aplicaciones de Java y un amplio rango de bases de datos y fuentes de datos.
  • JMX — Extensiones de Gestión de Java proporciona una forma estándar de gestionar recursos como aplicaciones, dispositivos, y servicios.
  • JNDI — El Interfaz de Directorio y Nombrado de Java activa el acceso a los servicios de Nombrado y Directorio como un DNS y LDAP.
  • JAXP — Introduce la IPA de Java para la tecnología de Procesado de XML (JAXP).
  • JAXB — Introduce la tecnología de arquitectura de Enlace XML (JAXB) de Java.
  • RMI — La IPA de Invocación de Métodos Remotos permite a un objeto invocar métodos de un objeto que se ejecuta en otra Máquina Virtual de Java.
  • Reflexión — Una IPA que representa ("refleja") las clases, interfaces y objetos in la Máquina Virtual Java actual.
  • Seguridad — Capacidades de la plataforma Java que ayuda a proteger las aplicaciones del software malicioso.
  • Sonido — Una IPA para reproducir datos de sonido desde las aplicaciones.
  • Gráficos 2D — Cómo visualizar e imprimir gráficos 2D en las aplicaciones.
  • Protocolo de Sockets Directos — Cómo habilitar el Protocolo de Sockets Directo para tomar ventaja de InfiniBand.
