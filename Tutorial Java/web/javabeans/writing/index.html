Lección: Escribiendo Componentes JavaBeans

Escribr componentes JavaBeans es sorprendentemente fácil. No necesita una herramienta especial y no tiene que implementar ninguna interfaz. Escribir beans es simplemente una cuestión de seguir ciertas convenciones de codificación. Todo lo que tiene que hacer es hacer su clase parecer como un bean — herramientas que usan beans serán capaces de reconocer y usar su bean.

Sin embargo, NetBeans proporciona algunas características que hacen más fácil escribir beans. Además, la IPA de Java SE incluye algunas clases de soporte para ayudar a implementar las tareas comunes.

Los ejemplos de código en esta lección se basan en un componente gráfico simple llamado FaceBean.

Sólo código fuente de FaceBean:
FaceBean.java
Proyecto de NetBeans entero incluyendo el código fuente de FaceBean:
FaceBean.zip

Un bean es una clase de Java con nombres de métodos que siguen las directrices de los JavaBeans. Una herramienta constructora de bean usa la introspección para examinar la clase bean. Basado en esta inspección, la herramienta constructora de bean puede determinar las propiedades, los métodos y los eventos del bean.

Las secciones siguientes describen las directrices de los JavaBeans para las propiedades, métodos, y eventos. Finalmente, una sección sobre BeanInfo muestra cómo puede personalizar la experiencia del desarrollador con su bean.
