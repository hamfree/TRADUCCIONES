La Persistencia a Largo Plazo

La persistencia a largo plazo es un modelo que habilita que los beans se salven en formato XML.

Se puede encontrar información sobre el formato XML y sobre cómo implementar la persistencia a largo plazo para no beans en Esquema XML y Usando XMLEncoder.

Codificador y Decodificador

La clase XMLEncoder se asigna para escribir ficheros de salida para la reprentación textual de los objetos Serializable. El fragmento de código siguiente es un ejemplo de la escritura de un Java bean y sus propiedades en formato XML: 

XMLEncoder encoder = new XMLEncoder(
           new BufferedOutputStream(
           new FileOutputStream("Beanarchive.xml")));

encoder.writeObject(object);
encoder.close();

La clase XMLDecoder lee un documento XML que fue creado con XMLEncoder: 

XMLDecoder decoder = new XMLDecoder(
    new BufferedInputStream(
    new FileInputStream("Beanarchive.xml")));

Object object = decoder.readObject();
decoder.close();

¿Qué hay en XML?

Un archivo XML de bean tiene su propia sintaxis específica, el cual incluye las etiquetas siguientes para representar cada elemento del bean:

  • Un preámbulo XML para describir una versión de XML y el tipo de codificación
  • Una etiqueta <java> para incorporar todos los elementos del objeto del bean
  • Una etiqueta <object> para representar un conjunto de llamadas a métodos necesasrios para reconstruir un objeto desde su forma serializada 
    <object class="javax.swing.JButton" method="new">
    <string>Ok</string>
</object>
    o sentencias 
    <object class="javax.swing.JButton">
    <void method="setText">
        <string>Cancel</string>
            </void>
    </object>
  • etiquetas para definir tipos primitivos apropiados:
    • <boolean>
    • <byte>
    • <char>
    • <short>
    • <int>
    • <long>
    • <float>
    • <double>
    <int>5555</int>
  • Una etiqueta <class> representa una instancia de Class. 
    <class>java.swing.JFrame</class>
  • Una etiqueta <array> para definir un vector 
    <array class="java.lang.String" length="5">
</array>

El código siguiente representa un archivo XML que será generado por el componente SimpleBean: 

<?xml version="1.0" encoding="UTF-8" ?>
<java>
  <object class="javax.swing.JFrame">
    <void method="add">
      <object class="java.awt.BorderLayout" field="CENTER"/>
      <object class="SimpleBean"/>
    </void>
    <void property="defaultCloseOperation">
      <object class="javax.swing.WindowConstants" field="DISPOSE_ON_CLOSE"/>
    </void>
    <void method="pack"/>
    <void property="visible">
      <boolean>true</boolean>
    </void>
  </object>
</java>
