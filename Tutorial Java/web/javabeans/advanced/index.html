Lección: Temas Avanzados de JavaBeans

Empezar con el desarrollo de los JavaBeans es fácil, pero los beans tienen una profundidad sorprendente. Esta lección cubre algunos tópicos más avanzados, incluyendo cómo se pueden almacenar los beans (persistidos) y cómo puede suplir editores personalizados para los tipos de datos personalizados.

  • Persistencia del Bean describe los mecanismos para salvar y reconstituir un bean.
  • La Persistencia a Largo Plazo cubre la representación de un bean como un XML.
  • Personalización del Bean proporciona una descripción general de la creación de componentes de editor para tipos de datos personalizados.
