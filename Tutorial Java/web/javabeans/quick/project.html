Creando un Proyecto

La forma más fácil de aprender sobre JavaBeans es empezar a usarlos. Para empezar, descargue e instale la última versión de NetBeans. Este tutorial describe cómo usar la versión 7.0 de NetBeans.

NetBeans es una herramienta constructora de beans, lo cual significa que reconoce los componentes JavaBeans (beans) y le permite unir los componentes en una aplicación con facilidad.

Un Botón es un Bean

Inicie NetBeans. Elija Fichero > Nuevo Proyecto... desde el menú.


Pulse para imagen completa

Seleccione Java desde la lista Categorías y seleccione Aplicación Java desde la lista Projectos. Pulse Siguiente >.


Pulse para imagen completa

Introduzca SnapApp como nombre de la aplicación. Desmarque Crear Clase Principal y pulse Finalizar. NetBeans crea el nuevo proyecto y usted puede verlo en el panel Projectos de NetBeans:

Haga control-click sobre el proyecto SnapApp y elija Nuevo > Formulario JFrame... desde el menú emergente.


Pulse para imagen completa

Rellene con SnapFrame el nombre de clase y snapapp como paquete. Pulse Finish. NetBeans crea la nueva clase y muestra su diseñador visual:


Pulse para imagen completa

En el panel Projectos de la izquierda, puede ver la nueva clase SnapFrame creada. En el centro de la pantalla está el diseñador visual de NetBeans. En la parte derecha está la Paleta, la cual contiene todos los componentes que puede agregar al marco en el diseñador visual.
