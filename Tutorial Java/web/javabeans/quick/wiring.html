Cableando la Aplicación

Para conectar el botón y la etiqueta juntos, pulse en el botón Modo de Conexión en la barra de herramientas del diseñador visual.

Pulse en el botón en el formulario SnapFrame. NetBeans resalta el botón en rojo para mostrar que este es el componente que estará generando un evento.

Pulse en la etiqueta. El Asistente de Conexión de NetBeans aparece. Primero eligirá el evento al que usted desea responder. Para el botón, este es el evento action. Pulse en el + próximo a action y seleccione actionPerformed. Pulse Siguiente >.

Ahora tiene que elegir qué ocurre cuando el botón dispara su evento action. El Asistente de Conexión lista todas las propiedades del bean etiqueta. Seleccione text en la lista y pulse Siguiente.

En la pantalla final del Asistente de Conexión, rellene el valor que desee establecer para la propiedad text. Pulse sobre Valor, después teclee ¡Usted presionó el botón! o algo tan elocuente. Pulse Finalizar.

NetBeans conecta los componentes juntos y le muestra su trabajo en el editor de código fuente.

Pulse sobre el botón Diseño en la barra de herramientas del código fuente para volver al diseñador IU. Pulse Ejecutar Proyecto Principal o presione F6 para construir y ejecutar su proyecto.

NetBeans construye y ejecuta el proyecto. Le pide identificar la clase principal, la cual es SnapFrame. Cuando la ventana de la aplicación aparezca, pulse sobre el botón. Verá su prosa inmortal en la etiqueta.

Observe que no escribió ningún código. Este es el poder real de JavaBeans — con una buena herramienta constructora como NetBeans, puede conectar rápidamente componentes juntos para crear una aplicación ejecutable.
