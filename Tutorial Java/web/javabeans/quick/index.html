Lección: Inicio Rápido

Esta lección describe cómo usar NetBeans para construir una aplicación simple. Con una buena herramienta como NetBeans, puede ensamblar componentes JavaBeans dentro de una aplicación sin tener que escribir ningún código.

Las tres primeras páginas de esta lección muestran cómo crear una aplicación simple usando beans gráficos que son parte de la plataforma Java. La última página demuestra qué fácil es incorporar un bean de terceras partes dentro de su aplicación.

  • Creando un Proyecto describe los pasos para configurar un nuevo proyecto en NetBeans.
  • Un Botón es un Bean muestra cómo agregar un bean a la interfaz de usuario de la aplicación y describe las propiedades y los eventos.
  • Cableando la Aplicación cubre el uso de NetBeans para responder a los eventos del bean en su aplicación.
  • Usando Beans de Terceros muestra lo fácil que es agregar un nuevo bean a la paleta y usarlo en su aplicación.
