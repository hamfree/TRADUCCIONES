Recorrido: JavaBeans(TM)

JavaBeans™ hace fácil reutilizar componentes de software. Los desarrolladores pueden usar componentes de software escritos por otros sin tener que entender sus funcionamientos internos.

Para entender porqué los componentes de software son útiles, piense en una trabajadora ensamblando un coche. En vez de construir una radio desde cero, por ejemplo, ella simplemente obtiene una radio y la conecta con el resto del automóvil.

Este recorrido describe los JavaBeans usando las lecciones siguientes:

  Inicio Rápido proporciona una introducción rápida a JavaBeans mostrando cómo construir aplicaciones con NetBeans.

  Escribiendo Componentes JavaBeans describe los patrones de codificación usados para las propiedades, métodos, y eventos de los beans. También describe el uso de BeanInfo para personalizar la experiencia de desarrollo en una herramienta de construcción.

  Temas Avanzados de JavaBeans cubre la persistencia de los beans, la persistencia a largo plazo, y la personalización.
