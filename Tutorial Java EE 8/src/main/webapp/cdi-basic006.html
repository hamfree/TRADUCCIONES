Usando Calificadores

Puede usar calificadores para proporcionar varias implementaciones de un tipo de bean en particular. Un calificador es una anotación que se aplica a un bean. Un tipo de calificador es una anotación Java definida como @Target({METHOD, FIELD, PARAMETER, TYPE}) y @Retention(RUNTIME).

Por ejemplo, podría declarar un tipo de calificador @Informal y aplicarlo a otra clase que amplíe la clase Greeting. Para declarar este tipo de calificador, utilice el siguiente código: 

              
package greetings;
import static java.lang.annotation.ElementType.FIELD;
import static java.lang.annotation.ElementType.METHOD;
import static java.lang.annotation.ElementType.PARAMETER;
import static java.lang.annotation.ElementType.TYPE;
import static java.lang.annotation.RetentionPolicy.RUNTIME;
import java.lang.annotation.Retention;
import static java.lang.annotation.RetentionPolicy.RUNTIME;
import java.lang.annotation.Target;
import javax.inject.Qualifier;
@Qualifier
@Retention(RUNTIME)
@Target({TYPE, METHOD, FIELD, PARAMETER})
public @interface Informal {}

A continuación, puede definir una clase de bean que amplíe la clase Greeting y utilice este calificador: 

              
package greetings;
@Informal
public class InformalGreeting extends Greeting {
    public String greet(String name) {
        return "Hola, " + name + "!";
    }
}

Ambas implementaciones del bean ahora se pueden usar en la aplicación.

Si define un bean sin calificador, entonces el bean automáticamente tiene el calificador @Default. La clase Greeting sin anotaciones podría declararse de la siguiente manera: 

              
package greetings;
import javax.enterprise.inject.Default;
@Default
public class Greeting {
    public String greet(String name) {
        return "Hola, " + name + ".";
    }
}
