1 Visión de conjunto

Este capítulo le presenta el desarrollo de aplicaciones empresariales de Java EE. Aquí revisará los conceptos básicos del desarrollo, aprenderá sobre los Arquitectura e IPA de Java EE, familiarícese con términos importantes y conceptos, y descubra cómo abordar la programación, montaje y despliegue de aplicaciones Java EE.

Aquí se tratan los siguientes temas:

  • Introducción a Java EE

  • Aspectos destacados de la plataforma Java EE 8

  • Modelo de aplicación Java EE

  • Aplicaciones distribuidas de varios niveles

  • Contenedores de Java EE

  • Soporte de Servicios Web

  • Implementación y Ensamblaje de Aplicaciones Java EE

  • IPAs de Java EE 8

  • IPAs de Java EE 8 in the Plataforma Java, Edición Estándar 8

  • Herramientas del Servidor GlassFish
