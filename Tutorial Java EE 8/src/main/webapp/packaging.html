5 Empaquetado

Este capítulo describe el empaquetado. Una aplicación Java EE está empaquetada en una o más unidades estándar para su implementación en cualquier sistema compatible con la plataforma Java EE. Cada unidad contiene un componente o componentes funcionales, como un bean empresarial, una página web, un servlet o un subprograma, y un descriptor de implementación opcional que describe su contenido.
