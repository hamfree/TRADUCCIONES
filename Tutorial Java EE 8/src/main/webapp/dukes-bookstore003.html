Ejecución del Caso de Estudio de la Aplicación de Librería de Duke

Puede usar el EID de NetBeans o Maven para compilar, empaquetar, implementar y ejecutar la aplicación Librería de Duke.

Aquí se tratan los siguientes temas:

  • Para Compilar y Desplegar la Librería de Duke mediante el EID de NetBeans

  • Para Construir y Desplegar la Librería de Duke Usando Maven

  • Para Ejecutar la Librería de Duke

Para Compilar y Desplegar la Librería de Duke mediante el EID de NetBeans

  1. Asegúrese de que el servidor GlassFish se haya iniciado (consulte Arrancando y Parando el Servidor GlassFish).

  2. En el menú Archivo, elija Abrir proyecto.

  3. En el cuadro de diálogo Abrir proyecto, vaya a:

                          
tut-install/examples/case-studies

  4. Seleccione la carpeta dukes-bookstore.

  5. Haga clic en Abrir proyecto.

  6. En la pestaña Proyectos, haga clic con el botón derecho en el proyecto dukes-bookstore y seleccione Construir.

    Esto compilará, empaquetará e implementará Librería de Duke en el servidor GlassFish.

Para Construir y Desplegar la Librería de Duke Usando Maven

  1. Asegúrese de que el servidor GlassFish se haya iniciado (consulte Arrancando y Parando el Servidor GlassFish), así como el servidor de la base de datos (ver Arrancando y Parando Apache Derby).

  2. En una ventana de terminal, vaya a:

                          
tut-install/examples/case-studies/dukes-bookstore/

  3. Introduzca el siguiente comando:

                          
mvn install

    Este comando compila la aplicación y la empaqueta en un archivo WAR en el directorio _tut-install_/examples/case-studies/dukes-bookstore/target/. Luego implementa la aplicación en el servidor GlassFish.

Para Ejecutar la Librería de Duke

  1. En un navegador web, ingrese la siguiente URL:

                          
http://localhost:8080/dukes-bookstore/

  2. En la página principal de Librería de Duke, haga clic en un libro en el gráfico o haga clic en uno de los enlaces en la parte inferior de la página.

  3. Utilice las páginas de la aplicación para ver y comprar libros.
