La Aplicación de Ejemplo jsonpstreaming

Esta sección describe cómo compilar y ejecutar la aplicación de ejemplo jsonpstreaming. Este ejemplo es una aplicación web que demuestra cómo crear datos JSON a partir de datos de formulario, cómo analizar datos JSON y cómo escribir resultados JSON mediante la API de transmisión.

La aplicación de ejemplo jsonpstreaming está en el directorio tut-install`/examples/web/jsonp/jsonpstreaming`.

Aquí se tratan los siguientes temas:

  • Componentes de la Aplicación de Ejemplo jsonpstreaming

  • Ejecutando la Aplicación de Ejemplo jsonpstreaming

Componentes de la Aplicación de Ejemplo jsonpstreaming

La aplicación de ejemplo jsonpstreaming contiene los siguientes archivos.

  • Tres páginas de JavaServer Faces.

    • La página index.xhtml contiene un formulario para recopilar información.

    • La página filewrite.xhtml contiene un área de texto que muestra datos JSON.

    • La página parsed.xhtml contiene una tabla que enumera los eventos del analizador.

  • El bean administrado StreamingBean.java, un bean administrado con ámbito de sesión que almacena los datos del formulario y dirige la navegación entre las páginas de Facelets. Este archivo también contiene código que usa la IPA de transmisión JSON.

El código usado en StreamingBean.java para escribir datos JSON en un archivo es similar al ejemplo en Escribiendo Datos JSON usando un Generador. El código para analizar datos JSON de un archivo es similar al ejemplo en Leyendo Datos JSON usando un Analizador.

Ejecutando la Aplicación de Ejemplo jsonpstreaming

Esta sección describe cómo ejecutar la aplicación de ejemplo jsonpstreaming utilizando el EID de NetBeans y desde la línea de comandos.

Aquí se tratan los siguientes temas:

  • Para Ejecutar la Aplicación de Ejemplo jsonpstreaming Usando el EID NetBeans

  • Para Ejecutar la Aplicación de Ejemplo jsonpstreaming Usando Maven

Para Ejecutar la Aplicación de Ejemplo jsonpstreaming Usando el EID NetBeans

  1. Asegúrese de que el Servidor GlassFish se haya iniciado (consulte Arrancando y Parando el Servidor GlassFish).

  2. En el menú Archivo, elija Abrir proyecto.

  3. En el cuadro de diálogo Abrir proyecto, vaya a:

                            
tut-install/examples/web/jsonp

  4. Seleccione la carpeta jsonpstreaming.

  5. Haga clic en Abrir proyecto.

  6. En la pestaña Proyectos, haga clic con el botón derecho en el proyecto jsonpstreaming y seleccione Ejecutar.

    Este comando compila y empaqueta la aplicación en un archivo WAR (jsonpstreaming.war) ubicado en el directorio target, lo implementa en el servidor y abre una ventana del navegador web con el siguiente URL: 

                            
http://localhost:8080/jsonpstreaming/

  7. Edite los datos en la página y haga clic en Escribir un objeto JSON en un archivo para enviar el formulario y escribir un objeto JSON en un archivo de texto. La página siguiente muestra el contenido del archivo de texto.

  8. Haga clic en Analizar JSON desde archivo. La siguiente página contiene una tabla que enumera los eventos del analizador para los datos JSON en el archivo de texto.

Para Ejecutar la Aplicación de Ejemplo jsonpstreaming Usando Maven

  1. Asegúrese de que el Servidor GlassFish se haya iniciado (consulte Arrancando y Parando el Servidor GlassFish).

  2. En una ventana de terminal, vaya a:

                            
tut-install/examples/web/jsonp/jsonpstreaming/

  3. Ingrese el siguiente comando para implementar la aplicación:

                            
mvn install

  4. Abra una ventana del navegador web e ingrese la siguiente URL:

                            
http://localhost:8080/jsonpstreaming/

  5. Edite los datos en la página y haga clic en Escribir un objeto JSON en un archivo para enviar el formulario y escribir un objeto JSON en un archivo de texto. La página siguiente muestra el contenido del archivo de texto.

  6. Haga clic en Analizar JSON desde archivo. La siguiente página contiene una tabla que enumera los eventos del analizador para los datos JSON en el archivo de texto.
