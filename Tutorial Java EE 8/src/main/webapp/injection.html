4 Inyección

Este capítulo proporciona una descripción general de la inyección en Java EE y describe los dos mecanismos de inyección proporcionados por la plataforma: inyección de recursos e inyección de dependencias.

Java EE proporciona mecanismos de inyección que permiten que sus objetos obtengan referencias a recursos y otras dependencias sin tener que instanciarlos directamente. Declara los recursos requeridos y otras dependencias en sus clases decorando campos o métodos con una de las anotaciones que marcan el campo como un punto de inyección. A continuación, el contenedor proporciona las instancias necesarias en tiempo de ejecución. La inyección simplifica su código y lo desacopla de las implementaciones de sus dependencias.
