Pasos para Crear un Componente Personalizado

Puede aplicar los siguientes pasos mientras desarrolla su propio componente personalizado.

  1. Cree una clase de componente personalizado que haga lo siguiente:

  2. Anula el método getFamily para devolver la familia de componentes, que se utiliza para buscar representadores que pueden representar el componente.

  3. Incluye el código de renderizado o lo delega a un renderizador (explicado en el Paso 2)

  4. Permite que los atributos de los componentes acepten expresiones

  5. Pone en cola un evento en el componente si el componente genera eventos

  6. Guarda y restaura el estado del componente.

  7. Delegue el renderizado a un renderizador si su componente no maneja el renderizado. Para hacer esto:

  8. Cree una clase de renderizador personalizada extendiendo javax.faces.render.Renderer.

  9. Registre el renderizador en un kit de renderizado.

  10. Registre el componente.

  11. Cree un controlador de eventos si su componente genera eventos.

  12. Cree un descriptor de biblioteca de etiquetas (TLD) que defina la etiqueta personalizada.

Vea Registrando un Componente Personalizado y Registrando un Renderizador Personalizado con un Kit Render para obtener información sobre cómo registrar el componente personalizado y el renderizador. La sección Usando un Componente Personalizado analiza cómo utilizar el componente personalizado en una página de JavaServer Faces.
