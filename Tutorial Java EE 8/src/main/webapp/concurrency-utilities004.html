Concurrencia y Seguridad

Utilidades de Concurrencia para Java EE defiere la mayoría de las decisiones de seguridad a la implementación del servidor de aplicaciones. Sin embargo, si el contenedor admite un contexto de seguridad, ese contexto se puede propagar al subproceso de ejecución. El ContextService puede admitir varios comportamientos de tiempo de ejecución y el atributo security, si está habilitado, propagará la entidad de seguridad del contenedor.
