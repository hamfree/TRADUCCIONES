13 Usando Ajax con la Tecnología JavaServer Faces

Este capítulo describe el uso de la funcionalidad Ajax en las aplicaciones web JavaServer Faces. Ajax es un acrónimo de Asynchronous JavaScript and XML, un grupo de tecnologías web que permiten la creación de aplicaciones web dinámicas y altamente receptivas. Con Ajax, las aplicaciones web pueden recuperar contenido del servidor sin interferir con la visualización del cliente. En la plataforma Java EE, la tecnología JavaServer Faces proporciona soporte integrado para Ajax.
