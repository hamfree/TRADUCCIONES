Dando a los Beans Nombres con EL

Para hacer que un bean sea accesible a través de EL, use el calificador integrado @Named: 

              
import javax.enterprise.context.RequestScoped;
import javax.inject.Inject;
import javax.inject.Named;
@Named
@RequestScoped
public class Printer {
    @Inject @Informal Greeting greeting;
    ...
}

El calificador @Named le permite acceder al bean usando el nombre del bean, con la primera letra en minúsculas. Por ejemplo, una página de Facelets se referiría al bean como printer.

Puede especificar un argumento para el calificador @Named para usar un nombre no predeterminado: 

              
@Named("MyPrinter")

Con esta anotación, la página Facelets se referiría al bean como MyPrinter.
