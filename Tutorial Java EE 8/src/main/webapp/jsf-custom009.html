Definiendo la Etiqueta del Componente Personalizado en una Librería de Descripción de Etiquetas

Para utilizar una etiqueta personalizada, debe declararla en un descriptor de biblioteca de etiquetas (TLD). El archivo TLD define cómo se usa la etiqueta personalizada en una página de JavaServer Faces. El contenedor web utiliza el TLD para validar la etiqueta. El conjunto de etiquetas que forman parte del kit de representación HTML se define en el TLD HTML_BASIC, disponible en la biblioteca de etiquetas HTML estándar de JavaServer Faces.

El nombre del archivo TLD debe terminar con taglib.xml. En el caso de estudio de la librería de Duke, las etiquetas personalizadas area y map se definen en el archivo web/WEB-INF/bookstore.taglib.xml.

Todas las definiciones de etiquetas deben estar anidadas dentro del elemento facelet-taglib en el TLD. Cada etiqueta está definida por un elemento tag. Aquí están las definiciones de etiquetas para los componentes area y map: 


<facelet-taglib xmlns="http://xmlns.jcp.org/xml/ns/javaee"
...>
    <namespace>http://dukesbookstore</namespace>
    <tag>
        <tag-name>area</tag-name>
        <component>
            <component-type>DemoArea</component-type>
            <renderer-type>DemoArea</renderer-type>
        </component>
    </tag>
    <tag>
        <tag-name>map</tag-name>
        <component>
            <component-type>DemoMap</component-type>
            <renderer-type>DemoMap</renderer-type>
        </component>
    </tag>
</facelet-taglib>

El elemento component-type especifica el nombre definido en la anotación @FacesComponent, y el elemento renderer-type especifica el rendererType definido en la anotación @FacesRenderer.

El elemento facelet-taglib también debe incluir un elemento namespace, que define el espacio de nombres que se especificará en las páginas que utilizan el componente personalizado. Consulte Usando un componente personalizado para obtener información sobre cómo especificar el espacio de nombres en las páginas.

El archivo TLD se encuentra en el directorio WEB-INF. Además, se incluye una entrada en el descriptor de implementación web (web.xml) para identificar el archivo descriptor de la biblioteca de etiquetas personalizadas, de la siguiente manera: 


    <context-param>
        <param-name>javax.faces.FACELETS_LIBRARIES</param-name>
        <param-value>/WEB-INF/bookstore.taglib.xml</param-value>
    </context-param>
