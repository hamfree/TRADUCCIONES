20 Procesado JSON

Este capítulo describe la IPA de Java para procesamiento JSON (JSR 374). JSR 374 es una versión incremental sobre JSR 353. JSON es un formato de intercambio de datos ampliamente utilizado en servicios web y otras aplicaciones conectadas. JSR 374 proporciona una API para analizar, transformar y consultar datos JSON utilizando el modelo de objetos o el modelo de transmisión.
