Personalizando los Mensajes del Validador

Validación de Beans incluye un paquete de recursos de mensajes predeterminados para las restricciones integradas. Estos mensajes se pueden personalizar y localizar para lugares que no hablan inglés.

Aquí se tratan los siguientes temas:

  • El Paquete de Recursos ValidationMessages

El Paquete de Recursos ValidationMessages

El paquete de recursos ValidationMessages y las variantes de configuración regional de este paquete de recursos contienen cadenas que anulan los mensajes de validación predeterminados. El paquete de recursos ValidationMessages suele ser un archivo de propiedades, ValidationMessages.properties, en el paquete predeterminado de una aplicación.

Localizando Los Mensajes de Validación

Las variantes de configuración regional de ValidationMessages.properties se agregan agregando un guión bajo y el prefijo de configuración regional al nombre base del archivo. Por ejemplo, el paquete de recursos de la variante local en español sería ValidationMessages_es.properties.
