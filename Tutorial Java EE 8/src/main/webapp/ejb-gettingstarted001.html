Empezando Con los Enterprise Beans

Aquí hay una descripción general de los pasos que seguirá:

  1. Cree el enterprise bean: ConverterBean.

  2. Cree el cliente web.

  3. Implemente converter en el servidor.

  4. Usando un navegador, ejecute el cliente web.

Antes de continuar, asegúrese de haber hecho lo siguiente:

  • Lea el Capítulo 1, "Descripción general"

  • Familiarícese con los enterprise beans (consulte el Capítulo 35, "Enterprise Beans")

  • Iniciado el servidor (ver Arrancando y Parando el Servidor GlassFish)
