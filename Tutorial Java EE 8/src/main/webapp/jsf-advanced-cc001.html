Atributos de un Componente Compuesto

Un componente compuesto es un tipo especial de plantilla JavaServer Faces que actúa como un componente. Si es nuevo en los componentes compuestos, consulte Componentes Compuestos antes de continuar con este capítulo.

Defina un atributo de un componente compuesto utilizando la etiqueta composite:attribute. La Table 14-1 enumera los atributos de uso común de esta etiqueta.

Tabla 14-1 Atributos de uso común de la etiqueta composite:attribute

Atributo

Descripción

name

Especifica el nombre del atributo del componente compuesto que se utilizará en la página de uso. Alternativamente, el atributo name puede especificar manejadores de eventos estándar como action, actionListener y bean gestionado.

default

Especifica el valor predeterminado del atributo del componente compuesto.

required

Especifica si es obligatorio proporcionar un valor para el atributo.

method-signature

Especifica una subclase de java.lang.Object como el tipo de atributo del componente compuesto. El elemento method-signature declara que el atributo del componente compuesto es una expresión de método. El atributo type y el atributo method-signature son mutuamente excluyentes. Si especifica ambos, method-signature se ignora. El tipo predeterminado de un atributo es java.lang.Object.

Nota: Las expresiones de método son similares a las expresiones de valor, pero en lugar de admitir la recuperación dinámica y la configuración de propiedades, las expresiones de método admiten la invocación de un método de un objeto arbitrario, pasando un conjunto específico de parámetros y devolviendo el resultado del método llamado (Si existe alguno).

type

Especifica un nombre de clase completamente calificado como el tipo de atributo. El atributo type y el atributo method-signature son mutuamente excluyentes. Si especifica ambos, method-signature se ignora. El tipo predeterminado de un atributo es java.lang.Object.

El siguiente fragmento de código define un atributo de componente compuesto y le asigna un valor predeterminado: 


<composite:attribute name="username" default="admin"/>

El siguiente fragmento de código utiliza el elemento method-signature:


<composite:attribute name="myaction"
                     method-signature="java.lang.String action()"/>

El siguiente fragmento de código utiliza el elemento type:


<composite:attribute name="dateofjoining" type="java.util.Date"/>
