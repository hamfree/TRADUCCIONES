Formato de Fecha y Número

Los programas Java utilizan el método DateFormat.getDateInstance(int, locale) para analizar y dar formato a las fechas teniendo en cuenta la configuración regional. Los programas Java utilizan el método NumberFormat.get`XXX`Instance(locale), donde XXX puede ser Currency, Number o Percent , para analizar y formatear valores numéricos de manera sensible a la configuración regional.

Una aplicación puede usar convertidores de fecha/hora y número para dar formato a fechas y números en función de la configuración regional. Por ejemplo, una fecha de envío podría convertirse de la siguiente manera: 

              
<h:outputText value="#{cashier.shipDate}">
    <f:convertDateTime dateStyle="full"/>
</h:outputText>

Para obtener información sobre los convertidores JavaServer Faces, consulte Usando los Convertidores Estándar.
