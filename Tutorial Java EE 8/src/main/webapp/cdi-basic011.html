Usando un Bean Gestionado en una Página Facelets

Para usar el bean administrado en una página de Facelets, cree un formulario que use elementos de la interfaz de usuario para llamar a sus métodos y mostrar sus resultados. El siguiente ejemplo proporciona un botón que le pide al usuario que escriba un nombre, recupera el saludo y luego muestra el texto en un párrafo debajo del botón: 

              
<h:form id="greetme">
   <p><h:outputLabel value="Introduzca su nombre: " for="name"/>
      <h:inputText id="name" value="#{printer.name}"/></p>
   <p><h:commandButton value="Diga Hola"
                       action="#{printer.createSalutation}"/></p>
   <p><h:outputText value="#{printer.salutation}"/></p>
</h:form>
