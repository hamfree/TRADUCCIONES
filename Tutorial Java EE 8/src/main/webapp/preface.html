Prefacio

Este tutorial es una guía para el desarrollo de aplicaciones empresariales para Plataforma Java, Edición Empresarial 8 (Java EE 8), utilizando la Edición de Código Abierto del Servidor GlassFish.

Edición de Código Abierto del Servidor GlassFish es la plataforma líder de código abierto y comunidad abierta para crear e implementar aplicaciones y servicios de última generación. La Edición de Código Abierto del Servidor GlassFish, desarrollado por la comunidad de código abierto del proyecto GlassFish en https://javaee.github.io/glassfish/ , es la primera implementación compatible de la especificación de la plataforma Java EE 8. Este servidor de aplicaciones liviano, flexible y de código abierto permite a las organizaciones no solo aprovechar las nuevas capacidades introducidas en la especificación Java EE 8, sino también agregar a sus capacidades existentes a través de un ciclo de desarrollo e implementación más rápido y optimizado. En lo sucesivo, la Edición de Código Abierto del Servidor GlassFish se denomina Servidor GlassFish.

Audiencia

Este tutorial está dirigido a programadores interesados en desarrollar e implementar aplicaciones Java EE 8. Abarca las tecnologías que componen la plataforma Java EE y describe cómo desarrollar componentes Java EE e implementarlos en el kit de desarrollo de software (SDK) Java EE.

Accesibilidad de la documentación

Para obtener información sobre el compromiso de Oracle con la accesibilidad, visite el sitio web del Programa de Accesibilidad de Oracle en http://www.oracle.com/pls/topic/lookup?ctx=acc&id=docacc .

Acceso al soporte de Oracle

Los clientes de Oracle que han comprado soporte tienen acceso a soporte electrónico a través de My Oracle Support. Para obtener información, visite http://www.oracle.com/pls/topic/lookup?ctx=acc&id=info o visite http://www.oracle.com/pls/topic/lookup?ctx=acc&id=trs si tiene problemas de audición.

Antes de leer este libro

Antes de continuar con este tutorial, debe tener un buen conocimiento del lenguaje de programación Java. Una buena manera de llegar a ese punto es trabajar con los Tutoriales de Java ( http://docs.oracle.com/javase/tutorial/index.html ).

Documentación relacionada

El conjunto de documentación del Servidor GlassFish describe la planificación de la implementación y la instalación del sistema. Para obtener la documentación de la Edición de Código Abierto del Servidor GlassFish, vaya a https://javaee.github.io/glassfish/documentation .

La especificación de la IPA de Java EE 8 se puede ver en https://javaee.github.io/glassfish/documentation y también se proporciona en el SDK de Java EE 8.

Además, las especificaciones de Java EE en https://javaee.github.io/javaee-spec/ podrían ser útil.

Para obtener información sobre cómo crear aplicaciones empresariales en el Entorno de Desarrollo Integrado (EID) de NetBeans, consulte https://netbeans.org/kb/ .

Para obtener información sobre Apache Derby para usar con el Servidor GlassFish, consulte http://db.apache.org/derby/docs/10.13/adminguide/ .

El proyecto GlassFish Samples es una colección de aplicaciones de muestra que demuestran una amplia gama de tecnologías Java EE. Los ejemplos de GlassFish se incluyen con el kit de desarrollo de software (SDK) de Java EE y también están disponibles en la página del proyecto de ejemplos de GlassFish en https://javaee.github.io/glassfish-samples/ .

Rutas predeterminadas y nombres de archivo

La siguiente tabla describe las rutas predeterminadas y los nombres de archivo que se utilizan en este libro.

Marcador de Posición

Descripción

Valor por Defecto

as-install

Representa el directorio de instalación base para el Servidor GlassFish o el SDK del que el Servidor GlassFish forma parte.

Instalaciones en el sistema operativo Solaris, sistema operativo Linux y sistema operativo Mac:

directorio_de_inicio_del_usuario/glassfish5/glassfish

Windows, todas las instalaciones:

Unidad_del_sistema:\glassfish5\glassfish

as-install-parent

Representa el directorio principal del directorio de instalación base para el Servidor GlassFish.

Instalaciones en el sistema operativo Solaris, sistema operativo Linux y sistema operativo Mac:

directorio_de_inicio_del_usuario/glassfish5

Windows, todas las instalaciones:

Unidad_del_sistema:\glassfish5

tut-install

Representa el directorio de instalación base para el tutorial de Java EE después de instalar el Servidor GlassFish o el SDK y ejecutar la herramienta de actualización.

as-install-parent/docs/javaee-tutorial

domain-dir

Representa el directorio en el que se almacena la configuración de un dominio.

as-install/domains/domain1
