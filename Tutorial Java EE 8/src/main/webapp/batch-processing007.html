Empaquetamiento de Aplicaciones por Lotes

Los archivos de definición de trabajos y los artefactos por lotes no requieren paquetes separados y se pueden incluir en cualquier aplicación Java EE.

Empaquete las clases de artefactos por lotes con el resto de las clases de su aplicación e incluya los archivos de definición de trabajo en uno de los siguientes directorios:

  • META-INF/batch-jobs/ para paquetes jar

  • WEB-INF/classes/META-INF/batch-jobs/ para paquetes war

El nombre de cada archivo de definición de trabajo debe coincidir con su ID de trabajo. Por ejemplo, si define un trabajo de la siguiente manera y está empaquetando su aplicación como un archivo WAR, incluya el archivo de definición de trabajo en WEB-INF/classes/META-INF/batch-jobs/simplejob.xml: 

              
<job id="simplejob" xmlns="http://xmlns.jcp.org/xml/ns/javaee"
                    version="1.0">
  ...
</job>
