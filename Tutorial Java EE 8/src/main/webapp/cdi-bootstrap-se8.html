28 Lanzando un Contenedor CDI en Java SE

Este capítulo explica cómo usar la IPA para arrancar un contenedor CDI en Java SE. Esta capacidad le permite ejecutar aplicaciones CDI en Java SE y obtener beans, independientemente de un servidor de aplicaciones o cualquier IPA de Java EE.

Aquí se tratan los siguientes temas:

  • La IPA de Bootstrap

  • Configurando el Contenedor CDI

Para obtener más información sobre cómo arrancar un contenedor CDI en Java SE, consulte la Guía de Referencia de Weld en http://weld.cdi-spec.org/documentation/ .
