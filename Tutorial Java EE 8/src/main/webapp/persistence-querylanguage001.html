Descripción General Del Lenguaje de Consultas de la Persistencia de Java

El lenguaje de consulta utiliza esquemas de persistencia abstractos de entidades, incluidas sus relaciones, para su modelo de datos y define operadores y expresiones basados en este modelo de datos. El alcance de una consulta abarca los esquemas abstractos de entidades relacionadas que se empaquetan en la misma unidad de persistencia. El lenguaje de consulta utiliza una sintaxis similar a SQL para seleccionar objetos o valores en función de los tipos de esquemas abstractos de entidades y las relaciones entre ellos.

Este capítulo se basa en el material presentado en capítulos anteriores. Para información conceptual, véase Capítulo 40, "Introducción a la IPA de Persistencia de Java". Para ejemplos de código, consulte Capítulo 41, "Ejecutando los Ejemplos de Persistencia".
