34 JAX-RS: Temas Avanzados y un Ejemplo

El IPA de Java para Servicios Web RESTful (JAX-RS, definido en JSR 370) está diseñado para facilitar el desarrollo de aplicaciones que utilizan la arquitectura REST. Este capítulo describe las características avanzadas de JAX-RS. Si es nuevo en JAX-RS, consulte el Capítulo 32, "Construyendo Servicios Web RESTful con JAX-RS" antes de continuar con este capítulo.

JAX-RS está integrado con los Contextos e Inyección de Dependencias para Java EE (CDI), la tecnología Enterprise JavaBeans (EJB) y la tecnología Java Servlet.

Aquí se tratan los siguientes temas:

  • Anotaciones Para Campos y Propiedades de Bean de las Clases Recurso

  • Validar los Datos del Recurso con Validación de Bean

  • Subrecursos y Resolución de Recursos en Tiempo de Ejecución

  • Integrando JAX-RS con la Tecnología EJB y CDI

  • Peticiones HTTP Condicionales

  • Negociación de Contenido en Tiempo de Ejecución

  • Usando JAX-RS con JAXB

  • La Aplicación de Ejemplo customer
