Construyendo y Ejecutando los Ejemplos Avanzados de CDI

Los ejemplos están en el directorio tut-install/examples/cdi/. Para compilar y ejecutar los ejemplos, hará lo siguiente.

  1. Utilice el EID NetBeans o la herramienta Maven para compilar, empaquetar e implementar el ejemplo.

  2. Ejecute el ejemplo en un navegador web.

Vea Capítulo 2, "Uso de los Ejemplos del Tutorial", para obtener información básica sobre la instalación, compilación y ejecución de los ejemplos.

Aquí se tratan los siguientes temas:

  • El Ejemplo encoder: Usando Alternativas

  • El Ejemplo producermethodse: Usando un Método de Productor para Elegir una Implementación de Bean

  • El Ejemplo producerfields: Usando los Campos de Producer para Generar Recursos

  • El Ejemplo billpayment: Usando Eventos e Interceptores

  • El Ejemplo decorators: Decorando un Bean
