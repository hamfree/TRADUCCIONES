Usando La IPA de Streaming

Esta sección describe dos casos de uso de la IPA de Transmisión.

Aquí se tratan los siguientes temas:

  • Leyendo Datos JSON Usando un Analizador

  • Escribiendo Datos JSON Usando un Generador

Leyendo Datos JSON Usando un Analizador

La IPA de Transmisión es el enfoque más eficiente para analizar texto JSON. El siguiente código muestra cómo crear un objeto JsonParser y cómo analizar datos JSON mediante eventos: 


import javax.json.Json;
import javax.json.stream.JsonParser;
...
JsonParser parser = Json.createParser(new StringReader(jsonData));
while (parser.hasSiguiente()) {
   JsonParser.Event event = parser.next();
   switch(event) {
      case START_ARRAY:
      case END_ARRAY:
      case START_OBJECT:
      case END_OBJECT:
      case VALUE_FALSE:
      case VALUE_NULL:
      case VALUE_TRUE:
         System.out.println(event.toString());
         break;
      case KEY_NAME:
         System.out.print(event.toString() + " " +
                          parser.getString() + " - ");
         break;
      case VALUE_STRING:
      case VALUE_NUMBER:
         System.out.println(event.toString() + " " +
                            parser.getString());
         break;
   }
}

Este ejemplo consta de tres pasos.

  1. Obtiene una instancia de analizador llamando al método estático Json.createParser.

  2. Itera sobre los eventos del analizador con los métodos JsonParser.hasSiguiente y JsonParser.next.

  3. Realice el procesamiento local para cada elemento.

El ejemplo muestra los diez tipos de eventos posibles del analizador. El método next del analizador lo avanza al siguiente evento. Para los tipos de eventos KEY_NAME, VALUE_STRING y VALUE_NUMBER, puede obtener el contenido del elemento llamando al método JsonParser.getString . Para eventos VALUE_NUMBER, también puede utilizar los siguientes métodos:

  • JsonParser.isIntegralNumber

  • JsonParser.getInt

  • JsonParser.getLong

  • JsonParser.getBigDecimal

Consulte la referencia de la IPA de Java EE para la interfaz javax.json.stream.JsonParser para obtener más información.

La salida de este ejemplo es la siguiente:

KEY_NAME lastName - VALUE_STRING Java
KEY_NAME age - VALUE_NUMBER 18
KEY_NAME streetAddress - VALUE_STRING 100 Internet Dr
KEY_NAME city - VALUE_STRING JavaTown
KEY_NAME state - VALUE_STRING JA
KEY_NAME postalCode - VALUE_STRING 12345
KEY_NAME phoneNumbers - START_ARRAY
START_OBJECT
KEY_NAME type - VALUE_STRING mobile
KEY_NAME number - VALUE_STRING 111-111-1111
END_OBJECT
START_OBJECT
KEY_NAME type - VALUE_STRING home
KEY_NAME number - VALUE_STRING 222-222-2222

END_OBJECT

Escribiendo Datos JSON Usando un Generador

El siguiente código demuestra cómo escribir datos JSON en un archivo mediante la IPA de transmisión: 


FileWriter writer = new FileWriter("test.txt");
JsonGenerator gen = Json.createGenerator(writer);
gen.writeStartObject()
   .write("firstName", "Duke")
   .write("lastName", "Java")
   .write("age", 18)
   .write("streetAddress", "100 Internet Dr")
   .write("city", "JavaTown")
   .write("state", "JA")
   .write("postalCode", "12345")
   .writeStartArray("phoneNumbers")
      .writeStartObject()
         .write("type", "mobile")
         .write("number", "111-111-1111")
      .writeEnd()
      .writeStartObject()
         .write("type", "home")
         .write("number", "222-222-2222")
      .writeEnd()
   .writeEnd()
.writeEnd();
gen.close();

Este ejemplo obtiene un generador JSON llamando al método estático Json.createGenerator, que toma un escritor (writer en inglés) o un flujo de salida como parámetro. El ejemplo escribe datos JSON en el archivo test.txt anidando llamadas a write, writeStartArray, writeStartObject, y los métodos writeEnd. El método JsonGenerator.close cierra el escritor subyacente o el flujo de salida.
