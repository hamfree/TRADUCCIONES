Tareas Básicas de Seguridad para Aplicaciones Empresariales

Los administradores de sistemas, los desarrolladores de aplicaciones, los proveedores de beans y los implementadores son responsables de administrar la seguridad de las aplicaciones empresariales. Las tareas básicas de seguridad son las siguientes:

  • Configurar una base de datos de usuarios y asignarlos al grupo adecuado

  • Configuración de la propagación de identidad

  • Configuración de las propiedades del servidor GlassFish que permiten que las aplicaciones se ejecuten correctamente, como la configuración predeterminada de la asignación principal a la función

  • Anotar las clases y métodos de una aplicación empresarial para proporcionar información sobre qué métodos deben tener acceso restringido

Las secciones sobre los ejemplos de seguridad en este capítulo y el capítulo anterior explican cómo realizar estas tareas.
