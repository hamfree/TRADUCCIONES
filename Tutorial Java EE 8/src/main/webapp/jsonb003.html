Ejecutando la Aplicación de Ejemplo jsonbbasics

Esta sección describe cómo compilar y ejecutar la aplicación de ejemplo jsonbbasics. Este ejemplo es una aplicación web que muestra cómo serializar un objeto en JSON y cómo deserializar JSON en un objeto.

La aplicación de ejemplo jsonbbasics se encuentra en el directorio tut-install/examples/web/jsonb/jsonbbasics.

Aquí se tratan los siguientes temas:

  • Componentes de la Aplicación de Ejemplo jsonbbasics

  • Ejecutando la Aplicación de Ejemplo jsonbbasics

Componentes de la Aplicación de Ejemplo jsonbbasics

La aplicación de ejemplo jsonbbasics contiene los siguientes archivos.

  • Dos páginas JavaServer Faces.

    • La página index.xhtml contiene un formulario para recopilar datos que se utilizan para crear un objeto Person.

    • La página jsongenerated.xhtml contiene un área de texto que muestra los datos en formato JSON.

  • El bean administrado jsonbBean.java, que es un bean administrado con ámbito de sesión que almacena los datos del formulario y dirige la navegación entre las páginas de Facelets. Este archivo contiene código que usa la IPA de enlace JSON.

Ejecutando la Aplicación de Ejemplo jsonbbasics

Esta sección describe cómo ejecutar la aplicación de ejemplo jsonbbasics desde la línea de comandos usando Maven.

Para ejecutar la aplicación de ejemplo jsonbbasics usando Maven:

  1. Asegúrese de que el Servidor GlassFish se haya iniciado (consulte Arrancando y Parando el Servidor GlassFish).

  2. En una ventana de terminal, vaya a:

                          
tut-install/examples/web/jsonb/jsonbbasics

  3. Ingrese el siguiente comando para implementar la aplicación:

                          
mvn install

  4. Abra una ventana del navegador web e ingrese la siguiente dirección:

                          
http://localhost:8080/jsonbbasics/

  5. Ingrese los datos en el formulario y haga clic en Serializar en JSON para enviar el formulario. La siguiente página muestra el formato JSON de los datos del objeto.

  6. Haga clic en Deserializar JSON. La página de índice muestra y contiene los campos completados a partir de los datos del objeto.
