La Aplicación de Ejemplo el Interceptor

El ejemplo de interceptor demuestra cómo usar una clase de interceptor, que contiene un método de interceptor @AroundInvoke, con un bean de sesión sin estado.

El bean de sesión sin estado HelloBean es un enterprise bean simple con dos métodos de negocio, getName y setName, para recuperar y modificar una cadena. El método de negocio setName tiene una anotación @Interceptors que especifica una clase de interceptor, HelloInterceptor, para ese método: 

              
@Interceptors(HelloInterceptor.class)
public void setName(String name) {
    this.name = name;
}

La clase HelloInterceptor define un método interceptor @AroundInvoke, modifyGreeting, que convierte la cadena pasada a HelloBean.setName en minúsculas: 

              
@AroundInvoke
public Object modifyGreeting(InvocationContext ctx) throws Exception {
    Object[] parameters = ctx.getParameters();
    String param = (String) parameters[0];
    param = param.toLowerCase();
    parameters[0] = param;
    ctx.setParameters(parameters);
    try {
        return ctx.proceed();
    } catch (Exception e) {
        logger.warning("Error calling ctx.proceed in modifyGreeting()");
        return null;
    }
}

Los parámetros de HelloBean.setName se recuperan y almacenan en una matriz Object llamando al método InvocationContext.getParameters. Debido a que setName solo tiene un parámetro, es el primer y único elemento de la matriz. La cadena se establece en minúsculas y se almacena en la matriz parameters, luego se pasa a InvocationContext.setParameters. Para devolver el control al bean de sesión, se llama a InvocationContext.proceed.

La interfaz de usuario de interceptor es una aplicación web JavaServer Faces que consta de dos vistas Facelets: index.xhtml, que contiene un formulario para ingresar el nombre y response.xhtml, que muestra el nombre final.

Ejecutando el Ejemplo del Interceptor

Puede usar el EID NetBeans o Maven para compilar, empaquetar, implementar y ejecutar el ejemplo de interceptor.

Aquí se tratan los siguientes temas:

  • Para Ejecutar el Ejemplo del Interceptor usando el EID NetBeans

  • Para Ejecutar el Ejemplo del Interceptor Usando Maven

Para Ejecutar el Ejemplo del Interceptor usando el EID NetBeans

  1. Asegúrese de que el servidor GlassFish se haya iniciado (consulte Arrancando y Parando el Servidor GlassFish).

  2. En el menú Archivo, elija Abrir proyecto.

  3. En el cuadro de diálogo Abrir proyecto, vaya a:

                            
tut-install/examples/ejb

  4. Seleccione la carpeta interceptor y haga clic en Abrir proyecto.

  5. En la pestaña Proyectos, haga clic con el botón derecho en el proyecto interceptor y seleccione Ejecutar.

    Esto compilará, implementará y ejecutará el ejemplo de interceptor, abriendo un navegador web en la siguiente URL: 

                            
http://localhost:8080/interceptor/

  6. Introduzca un nombre en el formulario y haga clic en Enviar.

    El nombre será convertido a minúsculas por el método interceptor definido en la clase HelloInterceptor.

Para Ejecutar el Ejemplo del Interceptor Usando Maven

  1. Asegúrese de que el servidor GlassFish se haya iniciado (consulte Arrancando y Parando el Servidor GlassFish).

  2. Vaya al siguiente directorio:

                            
tut-install/examples/ejb/interceptor/

  3. Para compilar los archivos fuente y empaquetar la aplicación, use el siguiente comando: 

                            
mvn install

    Este comando compila y empaqueta la aplicación en un archivo WAR, interceptor.war, ubicado en el directorio target. Luego, el archivo WAR se implementa en el servidor GlassFish.

  4. Abra la siguiente URL en un navegador web:

                            
http://localhost:8080/interceptor/

  5. Introduzca un nombre en el formulario y haga clic en Enviar.

    El nombre será convertido a minúsculas por el método interceptor definido en la clase HelloInterceptor.
