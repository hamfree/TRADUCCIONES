19 IPA de Java para WebSocket

Este capítulo describe el IPA de Java para WebSocket (JSR 356), que brinda soporte para crear aplicaciones WebSocket. WebSocket es un protocolo de aplicación que proporciona comunicaciones full-duplex entre dos pares sobre el protocolo TCP.
