Reemplazo Estático

Reemplazar una String estática con otra puede realizarse de varias formas: 

public final class ReplaceSubstring {
  
  /**
  *  Lo más simple en Java 1.5, usando el método replace, que 
  *  toma objetos CharSequence.
  */
  public static String replace15(
    String aInput, String  aOldPattern, String aNewPattern
  ){
    return aInput.replace(aOldPattern, aNewPattern);
  }

  /**
  * No es tan simple en Java 1.4. El método replaceAll funciona,
  * pero requiere más cuidado, ya que utiliza expresiones regulares, que
  * puede contener caracteres especiales.
  */
  public static String replace14(
    String aInput, String aOldPattern, String aNewPattern
  ){

    /*
    * El método replaceAll es un poco peligroso de usar.
    * aOldPattern se convierte en una expresión regular.
    * Por tanto, si aOldPattern puede contener caracteres que tienen
    * un significado especial para las expresiones regulares, entonces deben
    * ser 'escapado' antes de pasar a replaceAll. Es fácil 
    * olvidarse de hacer esto. 
    *
    * Además, aNewPattern trata '$' como caracteres especiales
    * también: Ellos se refieren a 'referencias posteriores'.
    */
    return aInput.replaceAll(aOldPattern, aNewPattern);
    /*
    Aquí hay una implementación alternativa usando Pattern y Matcher, 
    que se prefiere cuando el mismo patrón se usa repetidamente
    final Pattern pattern = Pattern.compile( aOldPattern );
    final Matcher matcher = pattern.matcher( aInput );
    return matcher.replaceAll( aNewPattern );
    */
  }

  /**
  * Si Java 1.4 no está disponible, la técnica siguiente se puede usar.
  *
  * @param aInput es la String origina que puede contener la subcadena aOldPattern
  * @param aOldPattern es la subcadena no vacía que va a ser reemplazada
  * @param aNewPattern es el reemplazo para aOldPattern
  */
  public static String replaceOld(
    final String aInput,
    final String aOldPattern,
    final String aNewPattern
  ){
     if ( aOldPattern.equals("") ) {
        throw new IllegalArgumentException("El patrón antiguo debe tener contenido.");
     }

     final StringBuffer result = new StringBuffer();
     //startIdx y idxOld delimitan varios trozos de aInput; estos 
     //trozos simpre terminan donde aOldPattern empieza
     int startIdx = 0;
     int idxOld = 0;
     while ((idxOld = aInput.indexOf(aOldPattern, startIdx)) >= 0) {
       //toma una parte de aInput que no incluye aOldPattern
       result.append( aInput.substring(startIdx, idxOld) );
       //agregar aNewPattern para reemplazar a aOldPattern
       result.append( aNewPattern );

       //restablece startIdx justo después de la coincidencia actual, para ver
       //si hay más coincidencias
       startIdx = idxOld + aOldPattern.length();
     }
     //el trozo final ira al final de aInput
     result.append( aInput.substring(startIdx) );
     return result.toString();
  }

  /** Ejemplo: actualiza una dirección ip que aparece en un enlace.  */
  public static void main (String[] aArguments) {
    String OLD_IP = "45.23.102.12";
    //escapa el '.', un carácter especial en expresiones regulares
    String OLD_IP_REGEX = "45\\.23\\.102\\.12";
    String NEW_IP = "99.104.106.95";
    String LINK = "http://45.23.102.12:8080/index.html";
    
    log("Enlace viejo : " + LINK);
    
    String newLink = replace15(LINK, OLD_IP, NEW_IP);
    log("Nuevo enlace con replace de Java 1.5 : " + newLink);
    
    newLink = replace14(LINK, OLD_IP_REGEX, NEW_IP);
    log("Nuevo enlace con replaceAll de Java 1.4 : " + newLink);

    newLink = replaceOld(LINK, OLD_IP, NEW_IP);
    log("Nuevo enlace con el estilo más viejo: " + newLink);
  }
  
  private static void log(String aMessage){
    System.out.println(aMessage);
  }
}

Ejemplo de ejecución de esta clase: 

Enlace viejo : http://45.23.102.12:8080/index.html
Nuevo enlace con replace de Java 1.5 : http://99.104.106.95:8080/index.html
Nuevo enlace con replaceAll de Java 1.4 : http://99.104.106.95:8080/index.html
Nuevo enlace con el estilo más viejo: http://99.104.106.95:8080/index.html

Reemplazo Dinámico

Si la cadena de reemplazo no es fija, y se necesita crear dinámicamente, entonces se requiere otro enfoque. En el ejemplo siguiente, cadenas de la forma "href='TopicAction.do?Id=182'" son reemplazadas con una cadena correspondiente "href=#182".

El número 182 se toma solo como un ejemplo. De hecho, se extrae dinámicamente y se hace referencia en la cadena de reemplazo utilizando la referencia inversa "$1", donde 1 es el índice del grupo coincidente para estos dígitos.

import java.util.regex.*;

public final class ReplaceSubstringDynamically {

  public static void main (String... aArguments) {
    String htmlText = "<a href=\"Topic27.cjp\">xyz</a> blah <a href=Topic8.cjp>abc</a>";
    ReplaceSubstringDynamically replace = new ReplaceSubstringDynamically();
    System.out.println("Antiguo texto HTML :" + htmlText);
    System.out.println("Nuevo texto HTML :" + replace.replaceLinks(htmlText));
  }

  /**
  * Reemplaza los enlaces de documento en un trozo de HTML con fragmentos de enlace
  * correspondientes, que empiezan con el signo #, y se refieren a las ubicaciones etiquetadas
  * dentro de un único documento.
  */
  String replaceLinks(String aHtmlTextWithLinks){
    Pattern pattern = Pattern.compile(fLINK);
    Matcher matcher = pattern.matcher(aHtmlTextWithLinks);
    return matcher.replaceAll(fFRAGMENT);
  }

  /**
  * El único grupo coincidente de esta expresión regular son los dígitos ((?:\\d){1,3}),
  * que corresponden al grupo 1.
  */
  private static String fLINK = "href=(?:\"|\')?Topic((?:\\d){1,3})\\.cjp(?:\"|\')?";

  /**
  * El "$1" se refiere al grupo coincidiente 1 de fLINK (los dígitos).
  */
  private static String fFRAGMENT = "href=#$1";
}

Ejemplo de ejecución de esta clase:

Antiguo texto HTML : <a href='TopicAction.do?Id=27'>xyz</a> blah <a href='TopicAction.do?Id=8'>abc</a>
Nuevo texto HTML : <a href=#27>xyz</a> blah <a href=#8>abc</a>

Aquí tiene un segundo ejemplo, donde la cadena de reemplazo se calcula sin usar referencias hacia atrás. 

import java.util.regex.*;

public final class ReplaceSubstringAppendReplacement {

  public static void main (String... aArguments) {
    String text = "Manzanas y naranjas son mejores para todo.";
    ReplaceSubstringAppendReplacement repl = new ReplaceSubstringAppendReplacement();
    System.out.println("Antiguo texto : " + text);
    System.out.println("Nuevo texto : " + repl.getEditedText(text));
  }

  /**
  * Reemplaza todas las palabras que empiezan con la letra 'a' o 'A' con 
  * sus formas en mayúsculas.
  */
  String getEditedText(String aText){
    StringBuffer result = new StringBuffer();
    Matcher matcher = fINITIAL_A.matcher(aText);
    while (matcher.find()) {
      matcher.appendReplacement(result, getReplacement(matcher));
    }
    matcher.appendTail(result);
    return result.toString();
  }

  private static final Pattern fINITIAL_A = Pattern.compile(
    "(?:\\s|^)a(?:\\w)*",
    Pattern.CASE_INSENSITIVE
  );

  private String getReplacement(Matcher aMatcher){
    return aMatcher.group(0).toUpperCase();
  }

}

Ejemplo de ejecución de esta clase:

Antiguo texto : Manzanas y naranjas son mejores para todo.
Nuevo texto : MANZANAS Y naranjas SON mejores para TODO.

Advertencia
Los métodos:

  • String.replaceAll(String, String)
  • String.replaceFirst(String, String)
  • Matcher.appendReplacement(StringBuffer, String)
trata '$' y '\\' en el texto de reemplazao como caracteres especiales. Si el texto de reemplazo puede contener texto arbitrario, después estos caracteres serán habitualmente escapados usando Matcher.quoteReplacement(String) .
