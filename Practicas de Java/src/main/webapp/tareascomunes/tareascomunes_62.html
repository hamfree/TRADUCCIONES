Generar números aleatorios
Hay dos formas principales de generar números aleatorios (realmente pseudo-aleatorios):
  • la clase Random genera enteros aleatorios, dobles, enteros largos, etc, en varios rangos.
  • el método estático Math.random genera s doubles entre 0 (inclusive) y 1 (exclusivo).
Para generar enteros aleatorios:
  • no use Math.random (esto produce doubles, no enteros)
  • use la clase Random para generar enteros aleatorios entre 0 y N.

Para generar una serie de números aleatorios como una unidad, necesita usar un único objeto Random - no cree un nuevo objeto Random para cada nuevo número aleatorio.

Otras alternativas son:

  • SecureRandom , una subclase criptográficamente fuerte de Random
  • ThreadLocalRandom , destinado a casos de subprocesos múltiples
Aquí tiene algunos ejemplos que usan Random.

Ejemplo 1 

import java.util.Random;

/** Genera 10 enteros aleatorios en el rango 0..99. */
public final class RandomInteger {
  
  public static final void main(String... aArgs){
    log("Generando 10 números aleatorios en el rango 0..99.");
    
    //tenga en cuenta que aquí se reutiliza un solo objeto aleatorio
    Random randomGenerator = new Random();
    for (int idx = 1; idx <= 10; ++idx){
      int randomInt = randomGenerator.nextInt(100);
      log("Generado : " + randomInt);
    }
    
    log("Hecho.");
  }
  
  private static void log(String aMessage){
    System.out.println(aMessage);
  }
}

Ejemplo 2

Este ejemplo genera enteros aleatorios en un rango específico.

import java.util.Random;

/** Genera enteros aleatorios en un cierto rango. */
public final class RandomRange {
  
  public static final void main(String... aArgs){
    log("Generando enteros aleatorios en el rango 1..10.");
    
    int START = 1;
    int END = 10;
    Random random = new Random();
    for (int idx = 1; idx <= 10; ++idx){
      showRandomInteger(START, END, random);
    }
    
    log("Hecho.");
  }
  
  private static void showRandomInteger(int aStart, int aEnd, Random aRandom){
    if (aStart > aEnd) {
      throw new IllegalArgumentException("El inicio no puede exceder el final .");
    }
    //Obtenga el rango, convirtiendo a entero largo para evitar problemas de desbordamiento
    long range = (long)aEnd - (long)aStart + 1;
    // calcula una fracción del rango, 0 <= frac < range
    long fraction = (long)(range * aRandom.nextDouble());
    int randomNumber =  (int)(fraction + aStart);    
    log("Generado : " + randomNumber);
  }
  
  private static void log(String aMessage){
    System.out.println(aMessage);
  }
}

Ejemplo 3

Este ejemplo genera números aleatorios en coma flotante en una distribución Gaussiana (normal). 

import java.util.Random;

/** 
 Genera valores pseudoaleatorios en coma flotante, con una 
 distribución aproximadamente Gaussiana (normal).

 Muchas medidas físicas tiene una distribución aproximadamente 
 Gaussiana; esto ofrece una forma de simular dichos valores.
*/
public final class RandomGaussian {
  
  public static void main(String... aArgs){
    RandomGaussian gaussian = new RandomGaussian();
    double MEAN = 100.0f; 
    double VARIANCE = 5.0f;
    for (int idx = 1; idx <= 10; ++idx){
      log("Generado : " + gaussian.getGaussian(MEAN, VARIANCE));
    }
  }
    
  private Random fRandom = new Random();
  
  private double getGaussian(double aMean, double aVariance){
    return aMean + fRandom.nextGaussian() * aVariance;
  }

  private static void log(Object aMsg){
    System.out.println(String.valueOf(aMsg));
  }
}
