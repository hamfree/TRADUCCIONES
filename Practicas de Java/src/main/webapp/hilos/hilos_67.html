Sincronizado es detalle de implementación

Aunque la palabra clave synchronized puede aparecer en un encabezado de método, no forma parte de la IPA del método o contrato. La sincronización es parte de la implementación, no parte de la interfaz.

Nota: alguna documentación temprana de Sun (javadoc) incluía la palabra clave synchronized en la descripción de los métodos. Este error fue corregido en Javadoc 1.2.

Ejemplo

Aquí, una implementación concreta de una interfaz usa synchronized para algunos métodos, incluso aunque los métodos de la interfaz por sí mismos no hagan mención de ello.

public interface ScientificTheory {
  
  void publish();
  
  void predict();
  
  boolean falsifyThroughMeasurement();
  
} 
    

public final class LoopQuantumGravity implements ScientificTheory {

  @Override public synchronized void publish() {
    //..omitido
  }
  
  @Override public synchronized void predict() {
    //..omitido
  }
  
  @Override public boolean falsifyThroughMeasurement() {
    return true; //esbozo
  }
  
}
