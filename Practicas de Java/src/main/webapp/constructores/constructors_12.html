Constructores de Copia

Constructores de copia:
  • Suministra una alternativa atractiva al más bien patológico método clone
  • son facilmente implementados
  • simplemente extrae los datos del argumento, y los reenvía a un constructor regular
  • son innecesarios para los objetos inmutables
Ejemplo 
/** Con correcciones de Bernardo Sulzbach. */
public final class Galaxy {

  /**
  * Constructor regular.
  */
  public Galaxy(double aMass, String aName) {
    fMass = aMass;
    fName = aName;
  }

  /**
  * Constructor de copia.
  */
  public Galaxy(Galaxy aGalaxy) {
    this(aGalaxy.getMass(), aGalaxy.getName());
    //no se crean copias defensivas aquí, ya que  
    //no hay campos de objetos mutables (String es inmutable)
  }

  /**
  * Estilo alternativo para un constructor de copia, usando un método estático newInstance.
  */
  public static Galaxy newInstance(Galaxy aGalaxy) {
    return new Galaxy(aGalaxy.getMass(), aGalaxy.getName());
  }

  public double getMass() {
    return fMass;
  }

  /**
  * Este es el único método que cambia el estado de un objeto Galaxy.
  * Si este método fue eliminado, entonces un constructor de copia 
  * no se proporcionará tampoco, ya que los objetos inmutables no necesitan 
  * un constructor de copia.
  */
  public void setMass(double aMass){
    fMass = aMass;
  }

  public String getName() {
    return fName;
  }

  // PRIVADO
  private double fMass;
  private final String fName;

  /** Metodo de prueba. */
  public static void main (String... aArguments){
    Galaxy m101 = new Galaxy(15.0, "M101");

    Galaxy m101CopyOne = new Galaxy(m101);
    m101CopyOne.setMass(25.0);

    Galaxy m101CopyTwo = Galaxy.newInstance(m101);
    m101CopyTwo.setMass(35.0);

    System.out.println("Masa de M101: " + m101.getMass());
    System.out.println("Masa de M101CopyOne: " + m101CopyOne.getMass());
    System.out.println("Masa de M101CopyTwo: " + m101CopyTwo.getMass());
  }
}

Ejemplo de ejecución de esta clase: 
>java -cp . Galaxy
Masa de M101: 15.0
Masa de M101CopyOne: 25.0
Masa de M101CopyTwo: 35.0
