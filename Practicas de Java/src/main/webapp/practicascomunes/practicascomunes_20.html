Espacio extra en la lista de argumentos

Algunos programadores sienten que usar espacios extra entre los paréntesis - como en ( this ) en vez de (that) - incrementa ligeramente la legibilidad del código. Sin embargo, este estilo no parece ser muy común.

Ejemplo

import java.util.*;

/**
* Cuando usa paréntesis, los espacios extra pueden ser usados para incrementar 
* ligeramente la legibilidad. Aquí, se ilustran ambos estilos para comparar.
*/
public final class ExtraSpace {

  public void explode( String aThing ) {
    expand( aThing );  //con espacios extra
    burst(aThing);     //sin espacios extra
  }

  // PRIVADO
  private static final String fBALLOON = "balloon";
  private static final String fTIRE = "tire";

  private void expand( String aThing ){
    if ( aThing.equals( fBALLOON ) ) { //con espacios extra
      //..omitido
    }
    else if (aThing.equals(fTIRE)){    //sin espacios extra
      //..omitido
    }
    else {
      //..omitido
    }
  }

  private void burst( String aThing ){  //con espacios extra
    //..omitido
  }
}
