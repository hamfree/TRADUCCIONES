Conozca las librerías principales

Java difiere de la mayoría de otros lenguajes en que el número de clases e interfaces en sus librerías estándar es muy grande. Muchas tareas comunes han sido ya implementadas por estas librerías.

Consejos para los principiantes podrían incluir:

  • no se intimide demasiado por el gran número de clases. Algunas son usadas muy a menudo, pero la mayoría se usan con relativa poca frecuencia.
  • Los paquetes más ampliamente usados son java.lang y java.util.
  • para trabajar con datos, vea java.sql, javax.sql, java.io, y java.nio.file.
  • para aplicaciones gráficas, vea la clases Swing (javax.swing, y así).
  • para aplicaciones de servidor, vea las IPAS de servlet y Java Server Page.
  • para otros paquetes, debe hojear su documentación solo para tener una idea de lo que está disponible. Más adelante, cuando surja una necesidad específica, a menudo sabrá qué paquetes pueden ser útiles.

Además, las bibliotecas del KDJ estándar:

  • son generalmente de alta calidad
  • a menudo mejoran su rendimiento con el paso del tiempo
  • son ampliamente usadas, y usualmente forman estándares defacto
Implementar algo que ya existe en las librerías probablemente sea un esfuerzo inútil.

Cambios significativos y adicciones a las librerías estándar ocurren en cada versión mayor, y vale la pena mantenerse actualizado.

La versión de Java 8 incluye:

  • java.time
  • interfaces con métodos de ayuda estáticos, y métodos predeterminados
  • interfaces funcionales (interfaces con un único método)
  • expresiones lambda
  • referencias de método
  • para las colecciones: flujos, tuberías y operaciones agregadas

La versión 7 de Java incluye:

  • inferencia de tipos
  • try-with-resources
  • java.nio.file - nuevas formas de interactuar con el sistema de ficheros
  • multi-catch - captura múltiples excepciones de una vez
  • JavaFX
  • literales binarios
La versión 6 de Java incluye:
  • scripting
  • la herramienta de monitorización JConsole
  • varias mejoras en Swing
La versión 5 de Java incluye:
  • genéricos
  • un bucle for mejorado
  • enumeraciones
  • autoconversión
  • y anotaciones

La versión 1.4 de Java incluye:

  • expresiones regulares
  • aserciones
  • servicios de trazado
