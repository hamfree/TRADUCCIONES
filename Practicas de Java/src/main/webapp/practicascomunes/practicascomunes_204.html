Citar texto dinámico al iniciar sesión

Los espacios en blanco extraños al principio o al final alrededor del texto son fáciles de agregar y difíciles de detectar: una mala combinación. Si el texto tiene algún significado lógico para el programa (por ejemplo, el nombre de un parámetro de configuración), los espacios en blanco extraños alrededor de este casi siempre son indeseados.

(Este problema ocurre para datos de texto, pero no para datos numéricos tales como BigDecimal o Integer).

Para ayudar a detectar tales espacios en blancos no queridos, simplemente entrecomille los valores de texto dinámicos cuando realize trazas. Probablemente sea una buena idea definir un método de utilidad simple para este propósito.

Ejemplo

Este ejemplo usa Objects.toString(Object) . Ese método tiene la ventaja de ser amistoso con los nulos.

import java.util.Objects;

public final class Util {

  /**
  * Rodear el resultado de <tt>Objects.toString(aObject)</tt> con comillas simples.
  */
  public static String quote(Object aObject){
    return "'" + Objects.toString(aObject) + "'";
  }

}
