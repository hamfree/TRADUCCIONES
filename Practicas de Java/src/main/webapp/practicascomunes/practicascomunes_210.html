Tenga cuidado con DecimalFormat

Analizar la entrada del usuario en números (especialmente números que representan dinero) es una tarea muy común. Tal análisis es a menudo implementado con DecimalFormat. Sin embargo, analizar con DecimalFormat debe ser hecho con cuidado. Puede incluso considerar una alternativa. Aquí está el porqué:
  • sorprendentemente, un Format no necesita analizar toda su entrada. Es decir, si encuentra una coincidencia al inicio de una String de entrada, cesará el análisis, y devuelve un valor. Cuando se analiza una entrada de usuario, esta clase de comportamiento no se desea habitualmente. Para acomodar este comportamiento, se necesita un paso extra - un objeto ParsePosition necesita ser pasado como un parámetro de salida a DecimalFormat.
  • la clase parece tener bastantes errores.
  • la clase es más difícil de entender que la mayoría.

Aquí hay un ejemplo de análisis con DecimalFormat. Una ejecución de ejemplo aparece debajo. Como puede ver,

  • analizar la entrada de usuarios siempre requerirá que el llamador use ParsePosition para detectar si la entrada completa ha sido analizada.
  • hay casos en los que el análisis parece simplemente incorrecto.

import java.text.*;

public final class BewareDecimalFormat {
  
  /**
  * analiza la entrada de usuario con DecimalFormat.
  */ 
  public static void main (String... aArguments) {

    String patternOne = "#,##0.00";
    log("Format : " + patternOne);
    parseUserInput("1000", patternOne);
    parseUserInput("1,000.00", patternOne);
    parseUserInput("1000.33", patternOne);
    parseUserInput(".20", patternOne);
    parseUserInput(".2", patternOne); 
    parseUserInput(".222", patternOne); 
    parseUserInput(".222333444", patternOne); 
    parseUserInput("100.222333444", patternOne); 
    parseUserInput(".22A", patternOne); 
    parseUserInput(".22BlahA", patternOne); 
    parseUserInput("22Blah", patternOne); 
    parseUserInput("Blah22", patternOne); 
    parseUserInput("100000,000.00", patternOne);
    parseUserInput("10,0000,000.00", patternOne);
    parseUserInput("1,0000,0000.00", patternOne);

    String patternTwo= "#,###0.00"; //agrupa cada 4 dígitos, no 3
    log(" ");
    log("Formato : " + patternTwo);
    parseUserInput("1000.00", patternTwo);
    parseUserInput("1,000.00", patternTwo);
    parseUserInput("1,0000.00", patternTwo);
    parseUserInput("1,00000.00", patternTwo);
    log("Hecho.");
  }

  // PRIVADO

  private static void parseUserInput(String aValue, String aDecimalFormatPattern) {
    DecimalFormat format = new DecimalFormat(aDecimalFormatPattern);
    ParsePosition parsePosition = new ParsePosition(0);
    Object object = format.parse(aValue, parsePosition);

    if(object == null) {
      log("Falló al analizar: " + aValue);
    }
    else if(parsePosition.getIndex() < aValue.length()) {
      log(
        aValue + " analizado OK (no la entrada completa) ParsePos:" + 
        parsePosition.getIndex() + ", Resultado del Análisis: " + object 
      );
    }
    else {
      log(
        aValue + " analizado OK. ParsePos: " + parsePosition.getIndex() + 
        ", Resultado del Análisis: " + object
      );
    } 
  }

  private static void log(Object aMessage){
    System.out.println(String.valueOf(aMessage));
  }
}
