Use la conversión automática de tipos con cuidado

El mecanismo de boxing se introdujo en el KDJ 1.5 para eliminar las conversiones triviales entre los tipos primitivos (int, boolean, etc.) y sus correspondientes objetos de envoltorio (Integer, Boolean, etc.). El mecanismo de boxing permite que algún código sea un poco más conciso y legible. Es también útil cuando usamos colecciones (que solo toman objetos, no tipos primitivos).

Hay dos aspectos complementarios del mecanismo de boxing:

  • el auto-boxing, que se refiere a la conversión automática de un int a un Integer, por ejemplo
  • el auto-unboxing, que se refiere a la conversión automáticia de un Boolean a un boolean, por ejemplo

El mecanismo de boxing oculta parcialmente la distinción entre los tipos primitivos y sus objetos encapsuladores correspondientes, pero no los elimina. Hay dos distinciones que no son cambiados por el mecanismo de boxing:

  • Los ojbetos puede ser null, mientras que los primitivos no pueden serlo
  • Los objetos tienen tanto estado como identidad, mientras que los primitivos solo tienen estado (el valor)
Ocasionalmente, estas diferencias pueden causar problemas cuando se usa el mecanismo de boxing.

Algunos puntos a recordar:

  • sea cuidadoso con los nulls. Realizar el auto-unboxing de un objeto null causará una NullPointerException.
  • comparar elementos con == y equals debe ser hecho con cuidado.
Las reglas para la comparación de primitivos y objetos envoltorio son las siguientes.

Si x e y son ambos primitivos, o ambos objetos, entonces no sucede el mecanismo de boxing:

Operación Dos primitivos Dos objetos
x == y compara valor compara identidad
x.equals(y) no compila compara valor

Si un elemento es un primitivo, y el otro elemento es un objeto de envoltura correspondiente, entonces puede ocurrir el mecanismo de boxing:

Operación Comportamiento
x == y tratar como dos primitivos y comparar valor
x.equals(y) no compila si x es un primitivo; de otra forma se trata como dos objetos, y compara por valor

Ejemplo 

import java.util.*;

public final class BoxingExamples {

  public static final void main(String... aArgs){
    //se pasa Integer donde se espera int
    explode(new Integer(3));

    //se pasa un literal donde se espera Boolean
    tellTruth(true);

    //calcula la "int-ercambiabilidad" con int e Integer
    Integer integerYear = new Integer(1989);
    Integer otherIntegerYear = integerYear + 10;
    int intYear = integerYear + new Integer(15);
    log(integerYear.toString());
    log(otherIntegerYear.toString());
    System.out.println(intYear);

    /*
    * Comparación de primitivas y objetos de envoltura uasndo == y equals.
    *
    * Cuando ambos elementos son del mismo tipo:
    *             2 primitivos          2 objetos
    * ----------------------------------------------------------
    * ==       :  valor                 identidad
    * equals() :  no aplicable          valor
    *
    *
    * Cuando un elemento es un primitivo, y el otro es un objeto:
    * ==          :  se trata como dos primitivos
    * x.equals(y) :  se trata como dos objetos, no compila si x es primitivo
    */
    intYear = 1880;
    integerYear = new Integer(1880);
    if (intYear == integerYear){
      log("intYear == integerYear: TRUE"); // si
    }
    else {
      log("intYear == integerYear : FALSE");
    }
    if (integerYear.equals(intYear)){
      log("integerYear.equals(intYear): TRUE"); //si
    }
    else {
      log("integerYear.equals(intYear): FALSE");
    }
    //no compila "int no puede ser desreferenciado" :
    //intYear.equals(integerYear);

    intYear = 1881; //nuevo valor
    if (intYear == integerYear){
      log("intYear == integerYear: TRUE");
    }
    else {
      log("intYear == integerYear : FALSE"); //si
    }
    if (integerYear.equals(intYear)){
      log("integerYear.equals(intYear): TRUE");
    }
    else {
      log("integerYear.equals(intYear): FALSE"); //si
    }
  }

  // PRIVADO
  private static void explode(int aNumTimes){
    log("Explotando" + aNumTimes + " veces.");
  }

  private static void tellTruth(Boolean aChoice){
    //en vez de si  ( aChoice.booleanValue() ) {
    if (aChoice) {
      log("Decir la verdad.");
    }
    else {
      log("Miente como una alfombra.");
    }
  }

  private static void log(String aText){
    System.out.println(aText);
  }
}

Decir la verdad.

intYear == integerYear : FALSE
