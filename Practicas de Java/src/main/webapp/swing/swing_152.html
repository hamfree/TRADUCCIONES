Tareas de utilidad de Swing

Una clase de utilidad que recopila tareas comunes puede reducir en gran medida la duplicación de código. Esto es especialmente verdadero para Swing, donde muchos elementos de construcción IGU son repetitivos.

Ejemplo

La clase UiUtil tiene un número de métodos estáticos para:

  • centrar las ventanas en la pantalla, o con respecto a una ventana principal
  • igualar tamaños de componentes
  • imponer una alineación en todos los elementos de un Box
  • crear bordes de un tamaño estándar
  • crear títulos de diálogo de un formato estándar
  • recuperar imágenes
  • crear pares de componentes, tanto la entrada de usuarios (JLabel y JTextField) y visualización simple (dos objetos JLabel)
  • usar GridBagConstraints de una manera concisa
  • dar formato a números, horas, y porcentajes
