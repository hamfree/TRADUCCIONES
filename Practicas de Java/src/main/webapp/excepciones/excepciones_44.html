Documente todas las excepciones

Todas las excepciones lanzadas por un método se pueden documentar con el comentario de javadoc @throws (el mismo que @exception). Algunos argumentan que un @throws debería ser incluído para cada excepción lanzada, ya que esta es la única forma de indicar claramente todos los requisitos que se le imponen al código llamador (las condiciones previas). Sin embargo, otros no están de acuerdo.

Si todos los parámetros del objeto null pasados a los métodos de clase causan una NullPointerException, entonces es aceptable indicar esto una vez en el comentario javadoc de la clase , en vez de repetirlo para cada método. Otra opción es indicar en overview.html (descripción resumida de una aplicación de javadoc) que todos los parámetros van a ser considerados como no nulos a menos que se indique explícitamente lo contrario.

Ejemplo

Este ejemplo tiene un número inusualmente grande de condiciones previas. Los comentarios sobre NullPointerException se colocan en el comentario javadoc de nivel de clase.

Tenga en cuenta que se lanza la SecurityException no comprobada indirectamente, a través de las llamadas a los métodos de la clase File.

import java.io.*;

/**
* Si un parametro de objeto nulo se pasa a cualquier método, entonces 
* se lanzará una <tt>NullPointerException</tt>.
*/
public final class WriteTextFile {

  //..otros métodos obviados

  /**
  * Cambia los contenidos de un fichero de texto en su totalidad, sobreescribiendo cualquier
  * texto existente.
  *
  * @param aFile es un fichero existente (no un directorio) que puede ser escrito.
  * @param aContents es el texto a ser colocado dentro de aFile.
  *
  * @exception FileNotFoundException si aFile no existe.
  * @exception IOException si el flujo a aFile no puede ser escrito o cerrado.
  *
  * @exception IllegalArgumentException si aFile es un directorio.
  * @exception IllegalArgumentException si aFile no puede ser escrito.
  * @exception SecurityException si existe un SecurityManager y no permite tener acceso de lectura o escritura a aFile.
  */
  static public void setContents(File aFile, String aContents)
                                 throws FileNotFoundException, IOException {
    if (aFile == null) {
      throw new NullPointerException("El fichero no debe ser nulo.");
    }
    if (aContents == null) {
      throw new NullPointerException("Los contenidos no deben ser nulos.");
    }
    if (!aFile.exists()) {
      throw new FileNotFoundException ("El fichero no existe: " + aFile);
    }
    if (!aFile.isFile()) {
      throw new IllegalArgumentException("No debe ser un directorio: " + aFile);
    }
    if (!aFile.canWrite()) {
      throw new IllegalArgumentException("El archivo no se puede escribir: " + aFile);
    }

    try (Writer output = new BufferedWriter(new FileWriter(aFile))) {
      output.write(aContents);
    }
  }

  public static void main (String... aArguments) throws IOException {
    File testFile = new File("C:\\Temp\\blah.txt");
    setContents(testFile, "bla bla bla");
  }
}
