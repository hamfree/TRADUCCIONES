Algunas clases necesitan readResolve

Hay dos casos comunes en los cuales se requiere el método readResolve para una deserialización apropiada. Ambos casos usan instancias específicas de objetos para definir su comportamiento:
  • formas más antiguas de enumeraciones con seguridad de tipos, que no utilizan enum
  • singletons
Vea los enlaces de abajo para ejemplos de versiones Serializable de este tipo de clases.
