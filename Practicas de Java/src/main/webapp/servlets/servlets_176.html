Use plantillas para estandarizar el diseño

Las plantillas se usan habitualmente para imponer una apariencia uniforme en un conjunto de páginas relacionadas - a menudo incluso para un sitio web entero.

Si usa JSPs, se pueden implementar fácilmente las plantillas usando la directiva <@include> o la acción <jsp:include>.

Ejemplo

Aquí, una plantilla JSP define el diseño. Incluye una cabecera, un pie de página y enlaces de navegación. Una <jsp:include> inserta el contenido principal o "cuerpo" de la página. Hay dos parámetros en la plantilla, que se pasan como parámetros de la petición:
  • TTitle, usada en la etiqueta <title>
  • TBody, que es simplemente el nombre de la JSP que proporciona todo el contenido de la página no definido de otra manera en la plantilla.
Los elementos <tags:xxx/> se refieren a archivos .tag, que contienen pequeños fragmentos de JSP. 
<%@ include file="/JspHeader.jsp" %>
<!doctype html>
<html lang='en'>
<head>
 <meta http-equiv="Content-Type" content="text/html; charset=uTF-8">
 <%-- TTitle es un parámetro de la petición para el titulo de la página con plantilla. --%>
 <title>
  Club de Pescado y Patatas Fritas - ${param.TTitle} </title>
 <link rel="stylesheet" type="text/css" href="../../stylesheet.css" media="all">
</head>

<body>
 <div align="center">
  <img class="no-margin" src="../../images/logo.jpg" alt="Club de Pescado y Patatas Fritas">
 </div>

 <div class="header">
  "El Total de Peces & Experiencia Gastronómica con Patatas Fritas"
 </div>

 <div class="menu-bar">
  <w:highlightCurrentPage styleClass='highlight'>
   <c:url value="/main/home/HomePageAction.do" var="homeURL"/> 
   <a href='${homeURL}'>Home</a>
   <c:url value="/main/rsvp/RsvpShow.do" var="showRsvpURL"/> 
   <a href='${showRsvpURL}'>Rsvp</a>
   <c:url value="/all/logoff/LogoffAction.do" var="logoffURL"/> 
   <a href='${logoffURL}'>Log Off</a>
  </w:highlightCurrentPage>
 </div>

 <%-- Mostrar mensajes de error e información. --%>
 <tags:displayMessages/>

 <%-- TBody es un parámetro de petición para el cuerpo de la página con plantilla. --%>
 <div class="body">
  <c:if test="${not empty param.TBody}">
   <jsp:include page='${param.TBody}' flush="true"/>
  </c:if>
  <c:if test="${empty param.TBody}">
   <jsp:include page="Error.jsp" flush="true"/>
  </c:if>
 </div>

 <tags:footer/>

</body>
</html>
