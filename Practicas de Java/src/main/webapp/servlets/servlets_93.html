Minimizar el tiempo de respuesta de un sitio

Los tiempos de respuesta de los sitios web son de forma habitual lentos comparados con aquellos del software de escritorio. Es muy útil minimizar los tiempos de respuesta del sitio web para aumentar la satisfacción del usuario.

Algunas técnicas útiles:

  • minimizar el número de gráficos asociados con la página
  • si es necesario un gráfico, considere formas de reducir su tamaño en bytes
  • considere seriamente servir contenido comprimido (todos los navegadores modernos son capaces de descomprimir páginas al vuelo)
  • elimine espacios en blanco innecesarios
  • durante el desarrollo, utilize las herramientas de depuración de su navegador para mostrar el tamaño del archivo y el tiempo de descarga para cada elemento en la página
