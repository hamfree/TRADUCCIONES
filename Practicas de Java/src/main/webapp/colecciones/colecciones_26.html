Utilice referencias a interfaces en vez de Colecciones

En general, las referencias a objetos deben ser lo más genéricas posible. El usuario de dicha referencia estará protegido de posibles cambios en la clase de implementación subyacente. Los efectos de ondulación de dicho cambio se limitan a la única línea de código que crea el objeto, y no se propagarán más.

En el caso de las colecciones, esto significa referirse habitualmente a objetos de colección usando las referencias de interfaz de List, Map, Set, Queue, y Deque.

Tenga en cuenta también que se puede hacer referencia a todos estos, excepto Map, utilizando el aún más genérico Collection.

Ejemplo

import java.util.*;

public class Nucleus {

  public static void main (String... arguments) {
    Nucleus lithium = new Nucleus (3,4);
    //tenga en cuenta que aquí se usa la referencia de Map genérica, no LinkedHashMap
    Map<String, Integer> quarks = lithium.getQuarkSummary();
    log("Número de quarks arriba en el núcleo de litio: " + quarks.get("Up"));
    log("Número de quarks abajo en el núcleo de litio: " + quarks.get("Down"));
  }

  public Nucleus(int aNumProtons, int aNumNeutrons) {
    fNumProtons = aNumProtons;
    fNumNeutrons = aNumNeutrons;
  }

  /** Tenga en cuenta que el método get es final. */
  public final int getNumProtons() {
     return fNumProtons;
  }

  /** Tenga en cuenta que el método get es final. */
  public final int getNumNeutrons() {
     return fNumNeutrons;
  }

  /**
  * Este método devuelve un Map el cual sumariza cuantos quarks de cada 
  * sabor existe en el núcleo.
  *
  * @return una referencia genérica Map, en vez de un LinkedHashMap; el 
  * usuario será protegido del detalle de qué implementación de Map
  * ha sido seleccionada aquí.
  */
  public final Map<String, Integer> getQuarkSummary() {
    LinkedHashMap<String, Integer> result = new LinkedHashMap<>();
    int numUp =
      fNumProtons * fUP_QUARKS_PER_PROTON +
      fNumNeutrons * fUP_QUARKS_PER_NEUTRON
    ;
    int numDown =
      fNumProtons * fDOWN_QUARKS_PER_PROTON +
      fNumNeutrons * fDOWN_QUARKS_PER_NEUTRON
    ;
    //esto hace uso de la autoconversión de enteros en Integers:
    result.put("Up", numUp);
    result.put("Down", numDown);
    return result;
  }

  //PRIVADO
  private final int fNumProtons;
  private final int fNumNeutrons;

  private static final int fUP_QUARKS_PER_PROTON = 2;
  private static final int fDOWN_QUARKS_PER_PROTON = 1;

  private static final int fUP_QUARKS_PER_NEUTRON = 1;
  private static final int fDOWN_QUARKS_PER_NEUTRON = 2;

  private static void log(String aMessage){
    System.out.println(aMessage);
  }
}
